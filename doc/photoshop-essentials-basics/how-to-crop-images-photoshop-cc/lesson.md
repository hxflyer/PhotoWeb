Learn everything you need to know about cropping images with the Crop Tool in Photoshop, including how to crop images non-destructively! Now updated for Photoshop 2021.

In this first tutorial in my [series on cropping images](/basics/cropping-images-in-photoshop-complete-lesson-guide/), I show you how to crop images, how to straighten images, and how to crop an image non-destructively with Photoshop. We'll start with the basics and learn how to crop images using the Crop Tool. Then we'll learn how to straighten an image with the Crop Tool. And finally, we'll look at a great feature that lets you crop an image without losing any of the original pixels!
