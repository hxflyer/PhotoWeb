## Lesson Expectations
- Frame Tool (`K`) draws rectangular/elliptical frames; Shift constrains, Space repositions mid-drag.
- File > Place Embedded puts an image into a Frame layer as a Smart Object; content and frame transform independently; drag new image to swap.
- Stroke can be added around frame from Options Bar.
