# Match Fonts

- Entry Type: Font recognition workflow
- Category: Text and typography > Select and manage fonts
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/select-manage-fonts/match-fonts.html

**Function Summary**
`Match Font` analyzes text visible in an image and suggests similar installed or available fonts.

**What This Function Does**
- Uses a selected image region to identify likely matching typefaces.
- Helps recreate text from flattened images, screenshots, scans, or reference artwork.
- Provides suggestions rather than editing the sampled raster text directly.

**Where You Find It**
- `Type > Match Font`.
- Usually used after making a selection around the text sample.

**Primary Interface Elements**
- Rectangular selection around sample text.
- Match Font dialog with detected text preview.
- Suggested font list.
- Font activation or selection controls where available.

**How To Use It**
1. Use a selection tool to draw a tight selection around the text in the image.
2. Choose `Type > Match Font`.
3. Review the suggested similar fonts.
4. Pick a font and create a new type layer to rebuild the text.
5. Adjust size, tracking, color, and placement manually to match the source.

**Expected Result**
- Photoshop suggests visually similar fonts for the sampled raster text.
