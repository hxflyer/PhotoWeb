# Duplicate an Embedded Smart Object

- Entry Type: Smart Object duplication workflow
- Category: Create and manage layers > Smart objects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/smart-objects/duplicate-an-embedded-smart-object.html

**Function Summary**
This workflow creates either a connected duplicate or an independent duplicate of an embedded Smart Object. The two duplication paths determine whether later edits to one copy affect the other.

**What This Function Does**
- Creates a linked copy through normal layer duplication
- Creates an independent Smart Object copy through `New Smart Object Via Copy`
- Lets users decide whether duplicates should stay synchronized or diverge

**Where You Find It**
- `Window > Layers`
- `Layer > New > Layer Via Copy`
- `Layer > Smart Objects > New Smart Object Via Copy`

**Primary Interface Elements**
- selected Smart Object layer
- `Layer Via Copy`
- `New Smart Object Via Copy`

**How To Create a Linked Copy**
1. Open `Window > Layers`.
2. Select the Smart Object layer.
3. Choose `Layer > New > Layer Via Copy`.

**How To Create an Independent Copy**
1. Select the Smart Object layer.
2. Choose `Layer > Smart Objects > New Smart Object Via Copy`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the Smart Object in the Layers panel and choose the duplication path from the menus.
- `Menus`
  Use either `Layer > New > Layer Via Copy` or `Layer > Smart Objects > New Smart Object Via Copy`.
- `Keyboard`
  No dedicated default hotkey is described in this entry.

**User Interface Behavior**
- A normal layer copy keeps the Smart Object relationship tied to the same underlying source.
- `New Smart Object Via Copy` creates a separate Smart Object instance that can diverge from the original when edited.

**Expected Result**
- You can choose between synchronized duplicates and independently editable duplicates.
