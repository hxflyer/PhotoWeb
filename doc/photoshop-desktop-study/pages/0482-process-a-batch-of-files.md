# Process a Batch of Files

- Entry Type: Batch automation topic hub
- Category: Automate tasks
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/process-batch-files.html

**Function Summary**
Batch processing applies actions, conversion settings, or export workflows to multiple files without opening and editing each one manually.

**Where You Find It**
- `File > Automate > Batch`.
- `File > Scripts > Image Processor`.
- Droplets created from actions.

**Primary Interface Elements**
- Batch dialog.
- Source and destination folder selectors.
- Action set/action selector.
- File naming, save, and error-handling options.

**Expected Result**
- Many files can be processed consistently using the same recorded or scripted workflow.
