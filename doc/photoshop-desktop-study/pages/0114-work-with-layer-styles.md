# Work with Layer Styles

- Entry Type: Custom layer-style workflow
- Category: Create and manage layers > Apply layer effects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/apply-layer-effects/work-with-layer-styles.html

**Function Summary**
This workflow applies custom layer styles and saves custom default settings inside the `Layer Style` dialog. It is useful when you want repeatable effect behavior without creating a preset style library item first.

**What This Function Does**
- Opens and edits custom style settings on a layer
- Saves adjusted effect defaults for later reuse inside the dialog
- Lets users revert back to those saved defaults

**Where You Find It**
- Layers panel
- `Layer Style` dialog
- `Make Default`
- `Reset to Default`

**Primary Interface Elements**
- target layer
- `Layer Style`
- style effect options
- `Make Default`
- `Reset to Default`

**How To Apply or Edit a Custom Layer Style**
1. Select the layer in the Layers panel.
2. Double-click the layer outside the name and thumbnail.
3. Adjust the effect options in the `Layer Style` dialog.
4. Confirm with `OK`.
5. Expand the layer's effect listing in the Layers panel if you want to inspect the attached effects afterward.

**How To Save Custom Default Settings**
1. In the `Layer Style` dialog, select the effect category you want to customize.
2. Adjust the effect values.
3. Click `Make Default`.
4. Use `Reset to Default` later if you want to return to the saved custom default after experimenting.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Double-click the layer to open the style dialog and click the default-management buttons inside it.
- `Menus`
  The main interaction is through the Layer Style dialog rather than a single top-level menu path.
- `Keyboard`
  No dedicated default hotkey is described for saving style defaults.

**User Interface Behavior**
- Custom defaults affect the next time that specific effect dialog is opened.
- This is different from saving a reusable preset style thumbnail in the Styles panel.

**Expected Result**
- The layer receives a customized style, and frequently used settings can become the user's new effect defaults.
