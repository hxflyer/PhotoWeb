# Fill Adjacent Areas with Similar Colors

- Entry Type: Paint Bucket workflow
- Category: Apply painting techniques > Fill objects, selections, and layers
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/fill-objects-selections-layers/fill-paint-bucket-tool.html

**Function Summary**
The `Paint Bucket Tool` fills clicked areas based on color similarity, tolerance, and adjacency settings.

**Primary Interface Elements**
- `Foreground`
- `Pattern`
- `Mode`
- `Opacity`
- `Tolerance`
- `Anti-alias`
- `Contiguous`
- `All Layers`

**How To Use It**
1. Select the `Paint Bucket Tool`.
2. Choose foreground color or pattern fill.
3. Set blend mode, opacity, tolerance, and edge options.
4. Click the area to fill.

**Expected Result**
- Pixels within the tolerance range are filled with the selected color or pattern.
