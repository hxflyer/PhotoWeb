# Blend Objects and People into Any Background with Harmonize

- Entry Type: Generative compositing workflow
- Category: Repair and retouch > Remove objects and fill space
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/remove-objects-fill-space/blend-subjects-with-harmonize.html

**Function Summary**
`Harmonize` adjusts a placed subject so its color, lighting, tone, and shadows better match the background.

**What This Function Does**
- blends people or objects into a new environment
- reduces manual color and lighting correction work
- generates variations for different blending results

**Where You Find It**
- contextual task bar when a suitable pixel layer is selected

**Primary Interface Elements**
- `Harmonize`
- contextual task bar
- variation picker
- `Remove Background`
- `Place Embedded`

**How To Use It**
1. Open the background image.
2. Place the subject on a new layer above the background.
3. Remove the subject's original background if needed.
4. Position and scale the subject.
5. Select `Harmonize` from the contextual task bar.
6. Choose the variation that best matches the scene.

**User Interface Behavior**
- Harmonize works on pixel layers and consumes generative credits.
- Photoshop creates generated blending results rather than only applying standard adjustment sliders.

**Expected Result**
- The subject appears more naturally integrated into the target background.
