# Layer Masks

- Entry Type: Topic hub
- Category: Create masks
- Source: https://helpx.adobe.com/photoshop/desktop/create-masks/layer-masks.html

**Function Summary**
This hub collects the core Photoshop workflows for creating, editing, applying, disabling, unlinking, and duplicating layer masks.

**What This Entry Is For**
- organizes non-destructive layer-visibility controls
- groups mask creation, transfer, and cleanup actions
- serves as the entry point for most mask-based editing tasks

**Primary Function Areas**
- add masks
- object-based mask generation
- unlink and relink masks
- enable or disable mask effect
- apply or delete masks

**Expected Result**
- Users can move from this hub to the exact mask-management task they need without searching across unrelated selection tools.
