# Insert a Non-Recordable Menu Command

- Entry Type: Action menu command workflow
- Category: Automate tasks > Actions
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/create-record-actions/insert-non-recordable-menu-command.html

**Function Summary**
Insert Menu Item adds certain menu commands to an action even when they cannot be captured through normal recording.

**Where You Find It**
- Actions panel menu: `Insert Menu Item`.

**Primary Interface Elements**
- Insert Menu Item dialog.
- Menu command selection.
- Recorded action command list.

**How To Use It**
1. Select the position in the action where the command should be inserted.
2. Choose `Insert Menu Item` from the Actions panel menu.
3. Choose the desired menu command.
4. Confirm the insertion.
5. Test playback.

**Expected Result**
- The chosen menu command appears as a step in the action.
