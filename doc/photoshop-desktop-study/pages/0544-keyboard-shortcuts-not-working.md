# Keyboard Shortcuts Not Working

- Entry Type: Keyboard shortcut troubleshooting workflow
- Category: Troubleshoot
- Source: https://helpx.adobe.com/photoshop/desktop/troubleshoot/keyboard-shortcuts-not-working.html

**Function Summary**
Keyboard shortcut troubleshooting restores shortcut behavior when commands do not respond, conflict, or trigger the wrong action.

**Where You Find It**
- `Edit > Keyboard Shortcuts`.
- Tool options and active text-editing state.
- Operating system keyboard settings.

**Primary Interface Elements**
- Keyboard Shortcuts dialog.
- Shortcut set menu.
- Conflicts and command assignments.
- Active tool or text insertion cursor.

**How To Use It**
1. Check whether the cursor is inside an active text edit, because many shortcuts type characters instead of commands.
2. Open `Edit > Keyboard Shortcuts` and verify the command assignment.
3. Check for shortcut conflicts.
4. Reset or choose a different shortcut set if needed.
5. Test OS-level keyboard shortcuts or input methods that may intercept keys.

**Expected Result**
- Photoshop shortcuts respond correctly or the conflicting shortcut source is identified.
