# Enhance Animation Frames

- Entry Type: Animation optimization workflow
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/enhance-animation-frames.html

**Function Summary**
Enhancing animation frames improves frame appearance or export quality before saving an animated result.

**Where You Find It**
- Timeline panel, Layers panel, filters, adjustments, and export dialogs.

**Primary Interface Elements**
- Frame thumbnails.
- Adjustment layers.
- Filters and masks.
- Export preview controls.

**How To Use It**
1. Review frames in the Timeline panel.
2. Apply consistent adjustments or effects to shared layers where possible.
3. Fix individual frame visibility, timing, or artifacts.
4. Preview the animation before export.

**Expected Result**
- Animation frames export with cleaner timing, appearance, and consistency.
