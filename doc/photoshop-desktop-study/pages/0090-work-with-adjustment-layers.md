# Work with Adjustment Layers

- Entry Type: Adjustment-layer workflow
- Category: Create and manage layers > Color adjustment and fill layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/color-adjustment-fill-layers/work-with-adjustment-and-fill-layers.html

**Function Summary**
This workflow creates and edits adjustment and fill layers so color, tone, and graphic fills can be changed non-destructively. It is the main operational entry point for using these layer types in everyday editing.

**What This Function Does**
- Creates adjustment layers for tonal and color correction
- Creates fill layers for solid color, gradient, or pattern content
- Keeps edits separate from original pixel data
- Opens properties controls for later refinement

**Where You Find It**
- Layers panel
- `Create new fill or adjustment layer` control
- `Properties` panel

**Primary Interface Elements**
- `Create new fill or adjustment layer`
- `Properties`
- adjustment types such as `Brightness/Contrast`, `Levels`, `Curves`, `Hue/Saturation`, `Color Balance`, and `Black & White`
- fill types such as `Solid Color`, `Gradient`, and `Pattern`

**How To Use It**
1. Open the image you want to edit.
2. In the Layers panel, click `Create new fill or adjustment layer`.
3. Choose the adjustment or fill type you want.
4. Use the `Properties` panel to adjust sliders, colors, tones, or other options.
5. Name the new layer and confirm with `Enter` on Windows or `Return` on macOS if needed.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Create the layer from the Layers panel and edit its controls in the Properties panel.
- `Menus`
  The main workflow is centered on the Layers panel control rather than a single top-level menu command.
- `Keyboard`
  Use `Enter` or `Return` to confirm naming changes.

**User Interface Behavior**
- Adjustment layers affect underlying layers without permanently changing their pixels.
- Fill layers create their own editable layer content instead of directly altering lower layers.
- All controls remain editable after creation through the Properties panel.

**Expected Result**
- Non-destructive color, tone, or fill changes are added to the document as editable layers.
