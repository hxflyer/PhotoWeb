# Choose a CMYK Equivalent for a Non-Printable Color

- Entry Type: Print color warning workflow
- Category: Adjust color > Choose colors
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/choose-colors/choose-cmyk-equivalent-for-non-printable-color.html

**Function Summary**
This workflow replaces out-of-gamut colors with printable CMYK equivalents.

**Where You Find It**
- Adobe Color Picker gamut warning controls

**Primary Interface Elements**
- gamut warning triangle
- nearest printable color swatch
- CMYK values

**How To Use It**
1. Open the Color Picker.
2. Watch for the out-of-gamut warning.
3. Click the printable-color warning swatch to choose the closest CMYK-safe equivalent.

**Expected Result**
- The selected color is adjusted to a printable CMYK-compatible approximation.
