# Sharpen a Selection

- Entry Type: Targeted sharpening workflow
- Category: Effects and filters > Sharpening
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/sharpening/sharpen-a-selection.html

**Function Summary**
Sharpening a selection limits sharpening to a selected area instead of affecting the entire layer.

**Where You Find It**
- Selection tools and `Filter > Sharpen`.

**Primary Interface Elements**
- Active selection border.
- Sharpen filter dialog or Sharpen Tool brush.
- Layer or Smart Object target.

**How To Use It**
1. Select the area that needs sharpening.
2. Feather or refine the selection if the edge should blend softly.
3. Apply Smart Sharpen, Unsharp Mask, or another sharpen command.
4. Deselect and inspect the transition edge.

**Expected Result**
- Only the selected image area receives sharpening.
