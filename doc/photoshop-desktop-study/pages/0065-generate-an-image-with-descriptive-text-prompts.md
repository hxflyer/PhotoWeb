# Generate An Image With Descriptive Text Prompts

- Entry Type: AI image-generation feature
- Category: Create, open, and import images > Create images
- Source: https://helpx.adobe.com/photoshop/desktop/create-open-import-images/create-images/generate-image-with-descriptive-text-prompts.html

**Function Summary**
This feature creates new images from a text prompt directly inside Photoshop. It is designed for concept generation, visual exploration, and fast creation of new source imagery without leaving the current document workflow.

**What This Function Does**
- Generates original image variations from written prompts
- Places the selected result into the current Photoshop workflow
- Supports iterative ideation without switching to an external generator
- Helps create starting material, concept assets, or reference visuals

**Where You Find It**
- Start from a new or active document
- Use the `Contextual Task Bar`
- Launch `Generate Image`

**Main Controls**
- `Generate Image`
  Opens the prompt-driven generation flow
- prompt field
  Accepts the image description
- `Generate`
  Starts the image creation request
- variation selection
  Lets you choose which result to keep

**How To Use It**
1. Open a new blank document or an existing working file.
2. In the `Contextual Task Bar`, select `Generate Image`.
3. Type a descriptive prompt for the image you want.
4. Select `Generate`.
5. Review the returned image variations.
6. Choose the variation you want to place into the document.

**User Interface Behavior**
- The feature is driven through the task bar rather than a traditional permanent tool panel.
- Results are presented as multiple variations instead of a single fixed output.
- The workflow is selection-based: generate, compare, choose, then continue editing.

**Expected Result**
- Photoshop creates several candidate images based on the typed description.
- The selected result becomes part of the current document workflow.
- You can refine results by changing the prompt and generating again.

**Practical Usage Notes**
- Use short clear prompts first, then add style or mood detail if needed.
- Treat the first result set as exploration, not necessarily the final image.
- This feature is useful for mockups, early concepts, visual brainstorming, and quick compositional placeholders.
