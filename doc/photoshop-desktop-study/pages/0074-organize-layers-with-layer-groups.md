# Organize Layers with Layer Groups

- Entry Type: Layer organization workflow
- Category: Create and manage layers > Get started with layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/get-started-layers/organize-layers-with-layer-groups.html

**Function Summary**
Layer groups collect multiple layers into one folder-like container inside the Layers panel. They are used to keep complex documents organized and to manage several related layers as a unit.

**What This Function Does**
- Combines related layers into one group
- Reduces clutter in large layer stacks
- Allows grouped layers to be expanded or collapsed
- Supports group-level masks and multi-layer adjustments

**Where You Find It**
- Layers panel
- `Layer > New > Group from Layers`
- `Layer > Ungroup Layers`

**Primary Interface Elements**
- selected layers
- group folder
- expand/collapse arrow
- `Group from Layers`
- `Ungroup Layers`

**How To Use It**
1. Select the layers you want to combine in the Layers panel.
2. Open `Layer > New > Group from Layers`.
3. Enter a name for the group and confirm with `OK`.
4. Use the arrow beside the folder in the Layers panel to expand or collapse the grouped layers.
5. Drag additional layers into the folder later if you want to add more items to the group.
6. To separate the items again, select the group folder and choose `Layer > Ungroup Layers`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Multi-select layers, drag layers into a group folder, and click the folder arrow to expand or collapse it.
- `Menus`
  Use `Layer > New > Group from Layers` and `Layer > Ungroup Layers`.
- `Keyboard`
  No dedicated default hotkey is described for grouping in this entry.

**User Interface Behavior**
- The group appears as a folder in the Layers panel.
- Collapsing the group hides the internal layer list without removing the layers.
- Group-level edits such as masks or adjustments can affect several contained layers together.

**Expected Result**
- Related layers become easier to find, manage, and edit as a set.

**Practical Use Cases**
- Group all text layers for a poster.
- Keep character parts or UI elements organized in one folder.
- Apply one mask to multiple related layers at once.
