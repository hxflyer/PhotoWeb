# Create a Droplet from an Action

- Entry Type: Droplet creation workflow
- Category: Automate tasks > Batch processing
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/process-batch-files/create-droplet-action.html

**Function Summary**
A droplet is a small application-like file that runs a Photoshop action on files dragged onto it.

**Where You Find It**
- `File > Automate > Create Droplet`.

**Primary Interface Elements**
- Save Droplet location.
- Action set and action selector.
- Destination and file naming options.
- Error handling options.

**How To Use It**
1. Create and test an action.
2. Choose `File > Automate > Create Droplet`.
3. Choose where to save the droplet.
4. Select the action to run.
5. Set destination, naming, and error options.
6. Drag files or folders onto the droplet to process them.

**Expected Result**
- Dropped files are processed through the selected Photoshop action.
