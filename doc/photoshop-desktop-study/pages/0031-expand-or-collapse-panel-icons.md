# Expand or Collapse Panel Icons

- Entry Type: Workspace space-saving control
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/collapse-expand-icons.html

**Function Summary**
This panel-display feature compresses full panels into icon form so the workspace uses less horizontal space while keeping the panels quickly accessible. It can expand one panel temporarily, collapse an entire dock, or change whether icon labels are shown.

**What This Function Does**
- Collapses panel groups into icons
- Expands a single icon into a full panel when needed
- Lets icon docks show icons only or icons with labels
- Helps maintain access to tools on smaller screens

**Where You Find It**
- Panel icon dock
- double-arrow collapse controls in panel docks and panel title bars

**Primary Interface Elements**
- panel icon
- icon dock
- panel title bar
- double-arrow collapse button
- dock width resize edge

**How To Use It**
1. Click a panel icon to expand that panel.
2. Click the double-arrow control at the top of a dock to collapse the full dock into icon view.
3. Drag the dock edge narrower until labels disappear if you want icon-only mode.
4. Drag the dock edge wider again to show labels beside the icons.
5. Click the double-arrow in a panel title bar to collapse that panel back into icon form.
6. Drag a floating panel or panel group into the icon dock by its title bar to convert it into a docked icon panel.
7. Drag icons up or down to reorder them, or drag one away from the dock to make it float again.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Use icon clicks, double-arrow buttons, and dock-edge dragging to expand, collapse, resize, and reorder icon panels.
- `Keyboard`
  No dedicated default hotkey is described for icon collapse behavior.
- `Menus`
  This behavior is controlled mainly by direct manipulation inside the workspace.

**User Interface Behavior**
- Clicking one icon can temporarily expand only that panel instead of the full dock.
- Narrowing the dock removes labels and leaves icons visible.
- Panels dragged into the icon dock inherit the dock's collapsed icon style.
- Icons can be reorganized between docks without reopening the panels from the `Window` menu.

**Expected Result**
- The workspace takes up less room while key panels stay easy to reach.
- Users can balance quick access with a cleaner canvas area.

**Practical Use Cases**
- Keep `Layers`, `History`, or `Properties` available on a small display.
- Reduce visual clutter while still keeping one-click access to common panels.
- Build a compact side dock for touchpad or tablet workflows.
