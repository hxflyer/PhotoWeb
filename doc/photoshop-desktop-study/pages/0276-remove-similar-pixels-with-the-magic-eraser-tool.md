# Remove Similar Pixels with the Magic Eraser Tool

- Entry Type: Magic Eraser workflow
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/change-similar-pixels-with-the-magic-eraser-tool.html

**Function Summary**
The `Magic Eraser Tool` makes pixels transparent based on clicked color similarity.

**Primary Interface Elements**
- `Tolerance`
- `Anti-alias`
- `Contiguous`
- `Sample All Layers`
- `Opacity`

**How To Use It**
1. Select the `Magic Eraser Tool` from the eraser group.
2. Set `Tolerance`.
3. Choose anti-aliasing, contiguous behavior, layer sampling, and opacity.
4. Click or drag over the color area to erase.

**Expected Result**
- Similar pixels are erased to transparency according to tolerance and adjacency settings.
