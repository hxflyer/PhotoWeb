# International Text and Languages

- Entry Type: Topic hub
- Category: Text and typography
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/international-text-languages.html

**Function Summary**
International text features control language, script, direction, glyph shaping, and text engine behavior for multilingual Photoshop documents.

**What This Area Controls**
- Unified text engine behavior.
- Text entry for supported international scripts.
- Right-to-left, left-to-right, and complex script shaping where supported.
- Font selection for script coverage.
- Document and paragraph settings that affect language-specific text layout.

**Where You Find It**
- `Type Tool`.
- `Character` and `Paragraph` panels.
- Language and paragraph direction controls where available.
- Input methods from the operating system.

**Primary Interface Elements**
- Language menu.
- Paragraph direction and composer controls.
- Font coverage and missing glyph indicators.
- Text insertion cursor and paragraph text box.

**Expected Result**
- Users can set up type layers that render supported international scripts correctly.
