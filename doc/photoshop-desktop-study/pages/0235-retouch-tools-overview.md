- Category: Repair and retouch > Remove objects and fill space

**Function Summary**
This reference lists the main retouching tools used for repairing, erasing, softening, sharpening, and tonal editing.

**What This Entry Is For**
- compares the main retouch tool choices
- helps users choose the right tool before editing pixels
- serves as a tool lookup entry rather than a single workflow

**Expected Result**
- Users can identify the appropriate retouch tool based on whether they need removal, repair, edge cleanup, or local tonal change.
