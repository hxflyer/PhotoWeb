# Clone Source Panel

- Entry Type: Clone source control reference
- Category: Repair and retouch > Heal and clone
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/heal-clone/clone-source-panel.html

**Function Summary**
The `Clone Source` panel controls how sampled pixels behave when using the `Clone Stamp` or `Healing Brush` tools.

**What This Function Does**
- stores up to five sample sources
- scales, rotates, or flips the sampled source
- shows an overlay preview for alignment
- controls layer and document sampling behavior

**Where You Find It**
- `Window > Clone Source`

**Primary Interface Elements**
- source buttons
- offset controls
- scale controls
- rotation controls
- overlay preview
- sample source options

**How To Use It**
1. Open the `Clone Source` panel.
2. Choose a source slot.
3. Set the sample point with a cloning or healing tool.
4. Adjust scale, rotation, or overlay settings if needed.
5. Paint with the selected repair tool.

**Expected Result**
- Cloned or healed pixels align more accurately with the target area.
