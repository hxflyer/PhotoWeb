# Replace Missing Fonts

- Entry Type: Missing font repair workflow
- Category: Text and typography > Select and manage fonts
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/select-manage-fonts/replace-missing-fonts.html

**Function Summary**
Missing font replacement assigns available fonts to type layers whose original fonts are not installed or activated.

**What This Function Does**
- Detects type layers that reference unavailable fonts.
- Lets users activate missing Adobe Fonts when possible.
- Replaces unavailable fonts with a selected installed font.
- Preserves editable text while changing font appearance and possibly text reflow.

**Where You Find It**
- Missing font alert when opening a document.
- Font family menu in the options bar, `Properties`, or `Character` panel.
- `Type` menu and Creative Cloud font activation prompts where available.

**Primary Interface Elements**
- Missing font warning dialog.
- Highlighted missing font names in font menus.
- Replacement font picker.
- Type layers in the `Layers` panel.

**How To Use It**
1. Open a document and review any missing font warning.
2. Activate the original font if Photoshop offers an activation option.
3. If activation is not possible, select the affected type layer.
4. Choose a replacement font from the font family menu.
5. Inspect line breaks, spacing, and layout after the replacement.

**Expected Result**
- The document uses available fonts and the affected text layers remain editable.
