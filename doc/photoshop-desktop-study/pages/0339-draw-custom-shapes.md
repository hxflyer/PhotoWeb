# Draw Custom Shapes

- Entry Type: Custom Shape workflow
- Category: Draw shapes and paths > Create shapes
- Source: https://helpx.adobe.com/photoshop/desktop/draw-shapes-paths/create-shapes/draw-custom-shapes.html

**Function Summary**
The `Custom Shape Tool` creates shapes from installed or imported shape presets.

**How To Use It**
1. Select the `Custom Shape Tool`.
2. Choose a shape preset.
3. Set drawing mode, fill, and stroke.
4. Drag on the canvas to create the shape.

**Expected Result**
- Photoshop creates the selected custom shape as a shape, path, or pixel object.
