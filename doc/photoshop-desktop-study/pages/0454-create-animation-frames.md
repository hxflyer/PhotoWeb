# Create Animation Frames

- Entry Type: Frame animation workflow
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/create-animation-frames.html

**Function Summary**
Animation frames store different layer visibility and layer states for each frame in a frame-based animation.

**Where You Find It**
- `Window > Timeline`, then choose `Create Frame Animation`.

**Primary Interface Elements**
- Timeline panel frame thumbnails.
- New frame button.
- Frame delay menu.
- Looping menu and playback controls.

**How To Use It**
1. Open `Window > Timeline`.
2. Choose `Create Frame Animation`.
3. Create or reveal layers for each animation state.
4. Add frames and set layer visibility for each frame.
5. Preview playback.

**Expected Result**
- Photoshop creates frame thumbnails that can play as a simple animation.
