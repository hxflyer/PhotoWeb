# Draw Paths with the Pen Tool

- Entry Type: Pen Tool path workflow
- Category: Draw shapes and paths > Draw lines and curves
- Source: https://helpx.adobe.com/photoshop/desktop/draw-shapes-paths/draw-lines-curves/draw-paths-with-the-pen-tool.html

**Function Summary**
The `Pen Tool` creates precise vector paths with straight segments, curves, and controlled anchor handles.

**How To Use It**
1. Click to place corner points for straight segments.
2. Click-drag to create curve handles.
3. Hold `Shift` to constrain straight segments.
4. Hold `Alt` on Windows or `Option` on macOS to adjust handles while drawing.
5. Click the starting point to close or use `Ctrl + Enter` / `Command + Return` to leave the path open.

**Expected Result**
- A precise editable path is created from anchor points and handles.
