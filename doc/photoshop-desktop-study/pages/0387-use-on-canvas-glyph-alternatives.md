# Use On-Canvas Glyph Alternatives

- Entry Type: Alternate glyph workflow
- Category: Text and typography > Characters and glyphs
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/characters-glyphs/use-on-canvas-glyph-alternatives.html

**Function Summary**
On-canvas glyph alternatives show available alternate glyphs near selected text so users can swap characters visually.

**What This Function Does**
- Displays alternate forms for supported characters in supported fonts.
- Replaces the selected character with a chosen alternate.
- Keeps text editable and font-based.

**Where You Find It**
- On canvas after selecting a character with alternates.
- `Type Tool` editing mode.
- OpenType fonts with alternate glyphs.

**Primary Interface Elements**
- Highlighted character.
- Small alternate glyph popup near the selected character.
- Glyph preview choices.

**How To Use It**
1. Select a type layer with an OpenType font that includes alternates.
2. Highlight a character that has alternate forms.
3. Wait for the on-canvas alternate popup.
4. Click the alternate glyph to replace the selected character.

**Expected Result**
- The selected character changes to the chosen alternate glyph.
