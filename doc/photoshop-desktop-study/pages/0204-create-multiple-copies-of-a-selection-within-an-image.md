# Create Multiple Copies of a Selection Within an Image

- Entry Type: Repeat-duplicate workflow
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/create-multiple-copies-of-a-selection-within-an-image.html

**Function Summary**
This shortcut workflow duplicates a selected area repeatedly while offsetting each new copy by a controlled amount.

**What This Function Does**
- creates copy-after-copy quickly
- supports `1`-pixel or `10`-pixel offsets
- works well for repeated patterns or stepped layouts

**Where You Find It**
- `Move Tool` duplication behavior

**Primary Interface Elements**
- `Move Tool`
- active selection

**How To Use It**
1. Select the area you want to duplicate.
2. Activate the `Move Tool`.
3. Hold `Alt` on Windows or `Option` on macOS and drag to make the first copy.
4. Keep holding the modifier and press arrow keys for repeated offset copies.
5. Add `Shift` to offset in `10`-pixel jumps instead of `1`.

**Mouse, Keyboard, and Menu Input**
- Mouse: `Alt`-drag on Windows or `Option`-drag on macOS to duplicate.
- Keyboard: hold `Alt` or `Option` and press an arrow key for a `1`-pixel copy offset.
- Keyboard: hold `Alt + Shift` on Windows or `Option + Shift` on macOS and press an arrow key for a `10`-pixel copy offset.

**User Interface Behavior**
- Each repeated arrow-key press creates another duplicate based on the last copied position, not the original source only.

**Expected Result**
- A sequence of evenly offset copies is created on the same layer context.
