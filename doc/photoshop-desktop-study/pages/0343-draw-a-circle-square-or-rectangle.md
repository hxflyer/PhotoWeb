# Draw a Circle, Square, or Rectangle

- Entry Type: Basic shape workflow
- Category: Draw shapes and paths > Draw lines and curves
- Source: https://helpx.adobe.com/photoshop/desktop/draw-shapes-paths/draw-lines-curves/draw-circle-square-rectangle.html

**Function Summary**
This workflow creates standard geometric shapes with the rectangle and ellipse tool family.

**How To Use It**
1. Select the rectangle or ellipse tool.
2. Choose shape, path, or pixel mode.
3. Drag on the canvas.
4. Hold `Shift` when needed to constrain to a square or circle.

**Expected Result**
- Photoshop creates a rectangle, square, ellipse, or circle using the selected mode.
