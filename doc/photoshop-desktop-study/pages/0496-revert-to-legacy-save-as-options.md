# Revert to Legacy Save As Options

- Entry Type: Save preference workflow
- Category: Save and export > Save files
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/save-files/revert-legacy-save-as-options.html

**Function Summary**
Legacy Save As options restore older save dialog behavior for users who prefer the previous Save As format availability.

**Where You Find It**
- macOS: `Photoshop > Settings > File Handling`.
- Windows: `Edit > Preferences > File Handling`.

**Primary Interface Elements**
- File Handling preferences.
- Legacy Save As preference.
- Save As and Save a Copy commands.

**How To Use It**
1. Open File Handling preferences.
2. Enable the legacy Save As behavior option if available.
3. Restart Photoshop if prompted.
4. Use Save As and Save a Copy with the adjusted behavior.

**Expected Result**
- Photoshop uses the selected legacy-style save behavior.
