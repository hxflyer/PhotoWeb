# Use Undo and Redo Commands

- Entry Type: Edit-state control
- Category: Get started > Set up toolbars and panels
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/set-up-toolbars-panels/use-undo-redo-commands.html

**Function Summary**
Undo and Redo move backward or forward through recent editing actions. They are the primary controls for reversing mistakes, testing variations, and stepping through work without manually reconstructing the image.

**What This Function Does**
- Reverses the most recent action
- Reapplies an action that was just undone
- Shows the specific action name in the menu when available
- Supports legacy shortcut behavior for users from older Photoshop versions

**Where You Find It**
- `Edit > Undo`
- `Edit > Redo`
- `File > Revert`
- `Edit > Keyboard Shortcuts and Menus`

**Primary Interface Elements**
- `Undo`
- `Redo`
- `Revert`
- `Keyboard Shortcuts and Menus`
- `Use Legacy Undo Shortcuts`

**How To Use It**
1. Press `Ctrl + Z` on Windows or `Command + Z` on macOS to undo the most recent action.
2. Press `Shift + Ctrl + Z` on Windows or `Shift + Command + Z` on macOS to redo the action you just undid.
3. Open the `Edit` menu if you want to see the exact action name beside `Undo` or `Redo`, such as `Undo Typing`.
4. Use `File > Revert` to return to the last saved version of the document.

**Legacy Shortcut Option**
1. Open `Edit > Keyboard Shortcuts and Menus`.
2. Enable `Use Legacy Undo Shortcuts`.
3. Confirm with `OK`.
4. Restart Photoshop so the change takes effect.

**Mouse, Keyboard, and Menu Input**
- `Keyboard`
  `Ctrl + Z` / `Command + Z` for undo, and `Shift + Ctrl + Z` / `Shift + Command + Z` for redo.
- `Menus`
  Use `Edit > Undo`, `Edit > Redo`, or `File > Revert` when you want explicit menu control.
- `Mouse`
  Click the menu commands when you prefer visual confirmation of the named action being reversed.

**User Interface Behavior**
- Undo reverses the most recent change in the current edit sequence.
- Redo becomes available after an undo.
- `Revert` discards all unsaved changes and returns to the last saved file state.
- Legacy shortcut mode changes how repeated undo behavior feels for users accustomed to earlier Photoshop releases.

**Expected Result**
- Mistakes can be corrected immediately.
- Experiments become safer because changes can be stepped backward and forward.

**Practical Use Cases**
- Undo an accidental brush stroke or transformation.
- Redo a change after comparing before-and-after states.
- Revert a file completely when the current edit branch should be abandoned.
