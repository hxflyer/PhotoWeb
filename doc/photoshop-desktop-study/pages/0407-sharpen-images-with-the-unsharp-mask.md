# Sharpen Images with the Unsharp Mask

- Entry Type: Unsharp Mask workflow
- Category: Effects and filters > Sharpening
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/sharpening/sharpen-images-unsharp-mask.html

**Function Summary**
Unsharp Mask sharpens by increasing contrast around edges using Amount, Radius, and Threshold controls.

**Where You Find It**
- `Filter > Sharpen > Unsharp Mask`.

**Primary Interface Elements**
- Amount controls sharpening intensity.
- Radius controls edge width.
- Threshold controls how different neighboring pixels must be before sharpening applies.
- Preview window and Preview checkbox.

**How To Use It**
1. Select the image layer or Smart Object.
2. Choose `Filter > Sharpen > Unsharp Mask`.
3. Set Radius first for the image scale.
4. Increase Amount until edges look crisp.
5. Raise Threshold to protect smooth tones from noise.

**Expected Result**
- Edges become sharper using a classic three-control sharpening method.
