# Export Your Work Using the Quick Export As Option

- Entry Type: Quick Export workflow
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/export-quick-export-as-option.html

**Function Summary**
Quick Export creates a fast export copy using the default format and location preferences.

**Where You Find It**
- `File > Export > Quick Export As`.
- Layer or artboard context menu export options where available.

**Primary Interface Elements**
- Quick Export format preference.
- Destination prompt or default export location.
- Exported file copy.

**How To Use It**
1. Set Quick Export preferences if needed.
2. Choose `File > Export > Quick Export As`.
3. Pick a destination if Photoshop asks.
4. Save the exported copy.

**Expected Result**
- Photoshop exports the document or selected asset with minimal settings.
