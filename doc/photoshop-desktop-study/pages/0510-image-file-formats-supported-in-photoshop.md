# Image File Formats Supported in Photoshop

- Entry Type: Image format support reference
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/image-file-formats-supported-photoshop.html

**Function Summary**
Image format support lists raster, raw, document, exchange, and export formats Photoshop can work with.

**Where You Find It**
- Open, Place, Save, Save a Copy, and Export format menus.

**Primary Interface Elements**
- File format dropdowns.
- Format-specific options such as compression, quality, layers, profiles, and transparency.

**Expected Result**
- Users can pick a compatible format for source import or final output.
