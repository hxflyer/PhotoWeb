# Create a New Preset Brush

- Entry Type: Brush preset save workflow
- Category: Apply painting techniques > Brushes and presets
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/brushes-presets/create-new-preset-brush.html

**Function Summary**
This workflow saves the current brush settings as a named preset.

**Primary Interface Elements**
- `Create New Brush`
- brush name
- capture brush size
- include tool settings

**How To Use It**
1. Configure the brush.
2. Open the brush preset menu or `Brushes` panel.
3. Choose the new brush preset command.
4. Name and save the brush.

**Expected Result**
- The current brush setup becomes a reusable preset.
