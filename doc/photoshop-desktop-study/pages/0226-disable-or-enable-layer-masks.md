# Disable or Enable Layer Masks

- Entry Type: Mask-toggle workflow
- Category: Create masks > Layer masks
- Source: https://helpx.adobe.com/photoshop/desktop/create-masks/layer-masks/disable-or-enable-layer-masks.html

**Function Summary**
This workflow temporarily turns a layer mask off or back on without deleting it.

**What This Function Does**
- lets users compare masked and unmasked states
- preserves the mask while bypassing its effect
- supports quick troubleshooting of mask problems

**Where You Find It**
- `Properties` panel
- `Layers` panel
- `Layer > Layer Mask`

**Primary Interface Elements**
- `Disable/Enable Mask`
- mask thumbnail

**How To Use It**
1. Select the layer containing the mask.
2. Disable or enable the mask from the `Properties` panel, by `Shift`-clicking the mask thumbnail, or through the layer mask menu command.

**Mouse, Keyboard, and Menu Input**
- Mouse: `Shift`-click the mask thumbnail to toggle the mask effect.
- Menu path: `Layer > Layer Mask > Disable` or `Enable`

**Expected Result**
- The mask effect is temporarily bypassed or restored without changing the mask data itself.
