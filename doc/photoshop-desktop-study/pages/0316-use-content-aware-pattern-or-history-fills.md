# Use Content-Aware, Pattern, or History Fills

- Entry Type: Fill dialog workflow
- Category: Apply painting techniques > Fill objects, selections, and layers
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/fill-objects-selections-layers/content-aware-fills.html

**Function Summary**
The `Fill` dialog replaces a selected area with Content-Aware pixels, a pattern, or a previous history state.

**Where You Find It**
- `Edit > Fill`

**Primary Interface Elements**
- `Contents`
- `Content-Aware`
- `Pattern`
- `History`
- `Custom Pattern`
- blend `Mode`
- `Opacity`

**How To Use It**
1. Select the area to fill.
2. Choose `Edit > Fill`.
3. Pick `Content-Aware`, `Pattern`, or `History`.
4. Set blending mode and opacity.
5. Click `OK`.

**Expected Result**
- The selected area is replaced using the chosen fill source.
