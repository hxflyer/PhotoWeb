# Manage Measurement Scales

- Entry Type: Measurement scale workflow
- Category: Grids, guides, and measurement
- Source: https://helpx.adobe.com/photoshop/desktop/grids-guides-measurement/manage-measurement-scales.html

**Function Summary**
Measurement scales map image pixels to real-world units so Photoshop measurements can report physical dimensions.

**Where You Find It**
- `Image > Analysis > Set Measurement Scale > Custom`.
- `Image > Analysis > Set Measurement Scale`.

**Primary Interface Elements**
- Pixel Length field.
- Logical Length field.
- Logical Units field.
- Preset scale list.

**How To Use It**
1. Measure a known reference distance in the image.
2. Choose `Image > Analysis > Set Measurement Scale > Custom`.
3. Enter the pixel length and real-world length/unit.
4. Save the scale preset if needed.
5. Use measurement tools or logs with the scale active.

**Expected Result**
- Measurements report real-world units based on the configured scale.
