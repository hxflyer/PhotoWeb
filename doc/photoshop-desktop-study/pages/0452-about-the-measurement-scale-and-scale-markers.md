# About the Measurement Scale and Scale Markers

- Entry Type: Measurement scale reference
- Category: Grids, guides, and measurement
- Source: https://helpx.adobe.com/photoshop/desktop/grids-guides-measurement/about-measurement-scale-scale-markers.html

**Function Summary**
Measurement scale and scale markers connect pixel measurements to real-world dimensions and provide visual scale references in the document.

**What This Function Describes**
- Measurement scale converts pixels into units such as inches, centimeters, millimeters, or custom units.
- Scale markers display a visible labeled ruler bar in the image.
- Scale accuracy depends on using a known reference distance.

**Where You Find It**
- `Image > Analysis > Set Measurement Scale`.
- `Image > Analysis > Place Scale Marker`.

**Expected Result**
- Users understand how calibrated scale settings and visible scale markers support technical, scientific, or documentation workflows.
