# Filters Overview

- Entry Type: Filter reference
- Category: Effects and filters > Get started with filters
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/get-started-filters/filters-overview.html

**Function Summary**
Filters are image-processing commands that alter pixels or Smart Objects to sharpen, blur, distort, stylize, render, reduce noise, or create special effects.

**What This Function Describes**
- Destructive filters change the selected pixels directly.
- Smart Filters apply to Smart Objects and remain editable.
- Many filters open dialogs with previews and numeric settings.
- Some filters require RGB mode, 8/16-bit support, a selected layer, or an unlocked pixel layer.

**Where You Find It**
- `Filter` menu and its submenus.
- `Filter > Filter Gallery` for grouped gallery effects.
- `Layers` panel for Smart Filter management.

**Expected Result**
- Users understand filters as Photoshop commands for image effects and know when a filter can be edited later.
