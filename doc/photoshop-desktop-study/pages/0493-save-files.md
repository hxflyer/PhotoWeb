# Save Files

- Entry Type: Saving topic hub
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/save-files.html

**Function Summary**
Saving files preserves Photoshop documents, creates copies in other formats, and controls save-related compatibility, preview, and cloud behavior.

**Where You Find It**
- `File > Save`, `File > Save As`, `File > Save a Copy`.
- macOS: `Photoshop > Settings > File Handling`.
- Windows: `Edit > Preferences > File Handling`.

**Primary Interface Elements**
- File format menu, location picker, save options, cloud/local location choice, and File Handling preferences.

**Expected Result**
- Users can save editable originals and generate copies in formats appropriate for delivery.
