# Add Text Along Paths or Inside Shapes

- Entry Type: Path text workflow
- Category: Text and typography > Text on paths and shapes
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/text-on-paths-shapes/add-text-along-paths-inside-shapes.html

**Function Summary**
Path and shape text places editable type on a vector path or inside a closed shape boundary.

**What This Function Does**
- Creates text that follows a path contour.
- Creates area text that flows inside a closed shape.
- Links the text flow to the underlying path geometry.

**Where You Find It**
- `Type Tool`, shortcut `T`.
- `Pen Tool` or shape tools for creating the path or shape.
- `Path Selection Tool` and `Direct Selection Tool` for later path edits.

**Primary Interface Elements**
- Path or shape outline.
- Type cursor with path/area indicator.
- Start and end markers for path text.
- Text bounding behavior based on the path or shape.

**How To Use It**
1. Draw a path with the `Pen Tool` or create a shape.
2. Select the `Type Tool`.
3. Move the cursor over the path edge for path text, or inside a closed shape for area text.
4. Click when the cursor changes, then type.
5. Format the text with the options bar, `Properties`, or `Character` panel.

**Expected Result**
- Photoshop creates editable type attached to the selected path or shape.
