# Change Text Size in Panels and Tooltips

- Entry Type: Interface preference
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/change-text-size.html

**Function Summary**
This interface preference changes the size of Photoshop's UI text, including panel labels and tooltips. It is mainly used to improve readability on high-resolution displays or to reduce eye strain during long editing sessions.

**What This Function Does**
- Adjusts the UI font size used in panel text and tooltips
- Improves readability on dense or high-resolution displays
- Optionally scales more of the Photoshop interface to match the chosen text size

**Where You Find It**
- `Edit > Preferences > Interface` on Windows
- `Photoshop > Settings > Interface` on macOS

**Primary Interface Elements**
- `Interface` preferences panel
- `UI Font Size` dropdown
- `Presentation` section
- `Scale UI To Font` option
- `OK` button

**How To Use It**
1. Open Photoshop preferences and go to the `Interface` section.
2. Find the `UI Font Size` setting in the `Presentation` area.
3. Choose one of the available sizes: `Tiny`, `Small`, `Medium`, or `Large`.
4. Enable `Scale UI To Font` if you want more of the interface to scale with the font choice.
5. Confirm the change with `OK`.
6. Restart Photoshop if the change does not apply immediately.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Open the preferences window, choose the font size from the dropdown, and confirm the setting.
- `Menus`
  Use the interface preferences path for your platform.
- `Keyboard`
  No dedicated default hotkey is described for this setting.

**User Interface Behavior**
- Text in panels and tooltips becomes larger or smaller based on the selected font size.
- `Scale UI To Font` can enlarge or reduce more of the interface in addition to the text.
- Actual visual results may still depend partly on operating-system display scaling.
- Photoshop may require a restart before all interface text updates completely.

**Expected Result**
- Interface labels become easier to read.
- Navigation can feel more comfortable on small or very high-density screens.

**Practical Use Cases**
- Increase text size for a 4K or 5K display.
- Reduce eye strain during long retouching sessions.
- Match Photoshop text size more closely to the rest of the operating system interface.
