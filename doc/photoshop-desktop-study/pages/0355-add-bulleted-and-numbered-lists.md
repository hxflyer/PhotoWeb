# Add Bulleted and Numbered Lists

- Entry Type: Paragraph text formatting workflow
- Category: Text and typography > Get started with text
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/get-started-text/add-bulleted-numbered-lists.html

**Function Summary**
Bulleted and numbered lists format paragraph text into ordered or unordered list items inside a type layer.

**What This Function Does**
- Adds bullet markers or sequential numbers to selected paragraphs.
- Supports quick UI formatting for text blocks that need list structure.
- Keeps list text editable as part of the type layer.

**Where You Find It**
- `Type Tool` editing mode.
- `Properties` panel or paragraph/list controls when available.
- `Window > Paragraph` for related paragraph spacing, alignment, and indentation.

**Primary Interface Elements**
- Active paragraph text box.
- List style buttons for bullets or numbering.
- Indent, spacing, alignment, and paragraph controls.

**How To Use It**
1. Select a paragraph text layer or highlight the paragraphs to format.
2. Choose the bullet or numbering control from the available type formatting UI.
3. Adjust indentation and paragraph spacing so the list aligns cleanly.
4. Commit the text edit.

**Expected Result**
- Selected paragraphs display as a clean editable list inside the type layer.
