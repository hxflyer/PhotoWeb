# Preview Content Credentials

- Entry Type: Content Credentials preview workflow
- Category: Save and export > Metadata, CSS, and Content Credentials
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/metadata-css-content-credentials/preview-content-credentials.html

**Function Summary**
Previewing Content Credentials shows what provenance information will be included before export.

**Where You Find It**
- Content Credentials panel, preview command, or export-related credentials preview where available.

**Primary Interface Elements**
- Credentials preview.
- Included information list.
- Warnings for missing or unavailable data.
- Export readiness indicators.

**How To Use It**
1. Enable Content Credentials.
2. Open the preview.
3. Review identity, edit, asset, AI, and document information that will be included.
4. Adjust included fields if needed.
5. Continue to export.

**Expected Result**
- Users can verify Content Credentials before publishing or exporting the file.
