# Export Layers as Files

- Entry Type: Layer export workflow
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/export-layers-as-files.html

**Function Summary**
Export Layers as Files saves each layer or selected layer group as a separate file.

**Where You Find It**
- `File > Export > Layers to Files` or script/export command where available.

**Primary Interface Elements**
- Destination folder.
- File name prefix.
- File type menu.
- Visible layers only option.
- Format-specific options.

**How To Use It**
1. Prepare and name layers clearly.
2. Choose the Layers to Files command.
3. Pick destination and file type.
4. Choose whether to export visible layers only.
5. Run the export.

**Expected Result**
- Photoshop creates separate files for the document layers.
