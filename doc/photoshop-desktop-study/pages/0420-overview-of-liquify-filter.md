# Overview of Liquify Filter

- Entry Type: Liquify reference
- Category: Effects and filters > Distortion
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/distortion/overview-liquify-filter.html

**Function Summary**
Liquify is a dedicated distortion workspace for pushing, pulling, twirling, puckering, bloat, reconstructing, freezing, and face-aware reshaping.

**Where You Find It**
- `Filter > Liquify`.

**Primary Interface Elements**
- Liquify preview workspace.
- Distortion tools on the left.
- Brush size, density, pressure, and rate controls.
- Freeze/Thaw mask tools, mesh options, backdrop options, and Face-Aware controls.

**Expected Result**
- Users can reshape image areas interactively with brush-like distortion tools.
