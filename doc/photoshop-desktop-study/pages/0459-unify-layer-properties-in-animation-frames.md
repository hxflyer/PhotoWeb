# Unify Layer Properties in Animation Frames

- Entry Type: Frame consistency workflow
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/unify-layer-properties-animation-frames.html

**Function Summary**
Unifying layer properties applies selected layer changes consistently across animation frames.

**Where You Find It**
- Timeline panel and `Layers` panel in Frame Animation mode.
- Layer unify controls where available.

**Primary Interface Elements**
- Unify layer position, visibility, and style controls.
- Selected layer.
- Frame thumbnails.

**How To Use It**
1. Select the layer whose properties should stay consistent.
2. Use the unify controls for position, visibility, or style.
3. Confirm whether Photoshop should update matching properties across frames.
4. Preview the animation for unwanted jumps.

**Expected Result**
- Chosen layer properties become consistent across animation frames.
