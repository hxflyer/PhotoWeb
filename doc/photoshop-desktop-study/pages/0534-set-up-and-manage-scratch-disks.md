# Set Up and Manage Scratch Disks

- Entry Type: Scratch disk workflow
- Category: Troubleshoot
- Source: https://helpx.adobe.com/photoshop/desktop/troubleshoot/set-up-manage-scratch-disks.html

**Function Summary**
Scratch disks provide temporary storage Photoshop uses when RAM is not enough for editing operations.

**Where You Find It**
- macOS: `Photoshop > Settings > Scratch Disks`.
- Windows: `Edit > Preferences > Scratch Disks`.
- Startup scratch disk selector when launching with modifier keys where supported.

**Primary Interface Elements**
- Scratch disk drive list.
- Enable checkboxes.
- Drive order.
- Free space indicators.

**How To Use It**
1. Open Scratch Disks preferences.
2. Enable a fast drive with plenty of free space.
3. Move the best drive higher in priority.
4. Restart Photoshop if required.
5. Keep scratch drives clear of low-space conditions.

**Expected Result**
- Photoshop has enough temporary storage for large edits and avoids scratch disk errors.
