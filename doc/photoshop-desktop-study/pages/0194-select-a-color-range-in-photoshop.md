# Select a Color Range in Photoshop

- Entry Type: Color Range workflow
- Category: Make selections > Freehand selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/freehand-selections/select-a-color-range-in-photoshop.html

**Function Summary**
`Color Range` creates a selection from sampled colors or tonal categories instead of traced geometry.

**What This Function Does**
- selects matching colors across an image
- supports skin-tone targeting and face detection
- provides grayscale-style previews for refinement
- can save and load range presets

**Where You Find It**
- `Select > Color Range`

**Primary Interface Elements**
- `Select` menu in the dialog
- `Sampled Colors`
- `Skin Tones`
- `Detect Faces`
- `Localized Color Clusters`
- `Fuzziness`
- `Range`
- `Invert`
- `Selection Preview`
- `Save`
- `Load`

**How To Use It**
1. Open `Select > Color Range`.
2. Choose a selection basis such as `Sampled Colors` or `Skin Tones`.
3. Use the eyedropper to add or subtract sampled colors if needed.
4. Adjust `Fuzziness` and `Range` to widen or narrow the result.
5. Review the preview mode.
6. Confirm, or save the settings as a preset if you will reuse them.

**Mouse, Keyboard, and Menu Input**
- Menu path: `Select > Color Range`
- Mouse: click with the eyedropper inside the image preview to sample colors.
- Keyboard: hold `Shift` to add sample colors and `Alt` or `Option` to remove sampled colors.

**User Interface Behavior**
- Preview modes show selected pixels as white, gray, or matte overlays so you can judge the range before committing.

**Expected Result**
- Pixels matching the defined color or tonal criteria become selected throughout the image.
