# Create Smoother Brush Strokes Using Stroke Smoothing

- Entry Type: Brush stroke control reference
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/create-smoother-more-polished-brush-strokes-with-stroke-smoothing.html

**Function Summary**
Stroke smoothing reduces hand jitter and produces cleaner brush, pencil, mixer brush, and eraser strokes.

**Primary Interface Elements**
- `Smoothing`
- `Pulled String Mode`
- `Stroke Catch-up`
- `Catch-up on Stroke End`
- `Adjust for Zoom`

**How To Use It**
1. Select a supported painting tool.
2. Set the `Smoothing` percentage.
3. Open smoothing options from the gear menu.
4. Enable the modes that fit the stroke behavior you want.

**Expected Result**
- Freehand strokes look cleaner and less shaky, especially with a mouse or high smoothing value.
