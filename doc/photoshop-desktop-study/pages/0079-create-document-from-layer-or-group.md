# Create Document from Layer or Group

- Entry Type: Layer extraction workflow
- Category: Create and manage layers > Get started with layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/get-started-layers/create-document-from-layer-or-group.html

**Function Summary**
This workflow turns a selected layer or layer group into a separate Photoshop document. It is useful when one part of a composition needs to be edited independently, exported separately, or reused in another file.

**What This Function Does**
- Creates a new document from a chosen layer
- Creates a new document from a chosen group
- Lets selected content be split out without flattening the source document

**Where You Find It**
- Layers panel
- `Layer > Duplicate Layer`
- `Layer > Duplicate Group`
- duplicate dialog `Document` destination set to `New`

**Primary Interface Elements**
- selected layer or group
- `Duplicate Layer`
- `Duplicate Group`
- `Document` dropdown
- `New`
- `OK`
- `Layer Via Copy`

**How To Use It**
1. Select the layer or group you want to separate in the Layers panel.
2. Open `Layer > Duplicate Layer` for a single layer, or `Layer > Duplicate Group` for a group.
3. In the duplicate dialog, set the `Document` destination to `New`.
4. Confirm with `OK`.

**Related Preparation Option**
- `Layer > New > Layer Via Copy`
  Creates a copy of the current selection on its own layer before duplicating it into a new document

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the target layer or group, then use the duplicate dialog to choose `New` as the document destination.
- `Menus`
  Use the appropriate duplicate command from the `Layer` menu.
- `Keyboard`
  No dedicated default hotkey is described for this document-creation path.

**User Interface Behavior**
- Photoshop opens a separate document containing the duplicated layer or group content.
- The source document remains intact.
- This acts as a clean extraction workflow rather than a destructive move.

**Expected Result**
- The selected content becomes available in its own document for focused editing or reuse.

**Practical Use Cases**
- Pull one asset out of a larger composition for standalone refinement.
- Create a reusable document from a grouped UI component or graphic.
- Separate a selected element for export-oriented preparation.
