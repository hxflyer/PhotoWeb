# Record an Action Again

- Entry Type: Action re-recording workflow
- Category: Automate tasks > Actions
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/create-record-actions/record-action-again.html

**Function Summary**
Recording an action again rebuilds or updates an action when the existing command sequence is no longer correct.

**Where You Find It**
- Actions panel.

**Primary Interface Elements**
- Existing action.
- Delete, New Action, Record, Stop, and Play controls.

**How To Use It**
1. Save or duplicate the existing action if you may need it later.
2. Delete or rename the old action.
3. Create a new action with the desired name.
4. Record the corrected steps.
5. Test playback on sample files.

**Expected Result**
- The action reflects the new desired workflow.
