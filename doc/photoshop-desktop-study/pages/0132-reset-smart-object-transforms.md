# Reset Smart Object Transforms

- Entry Type: Smart Object transform reset
- Category: Create and manage layers > Smart objects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/smart-objects/reset-smart-object-transforms.html

**Function Summary**
This command removes the current transform state from a Smart Object and returns it to its original proportions and orientation without changing the underlying source content.

**What This Function Does**
- Removes scaling, rotation, and skew from a Smart Object
- Restores the object's original transform state
- Works only on Smart Objects

**Where You Find It**
- Layers panel
- `Layer > Smart Objects > Reset Transform`

**Primary Interface Elements**
- selected Smart Object layer
- `Reset Transform`

**How To Use It**
1. Select the Smart Object layer in the Layers panel.
2. Choose `Layer > Smart Objects > Reset Transform`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the Smart Object in the Layers panel before running the command.
- `Menus`
  Use `Layer > Smart Objects > Reset Transform`.
- `Keyboard`
  No dedicated default hotkey is described for this command.

**User Interface Behavior**
- The bounding box returns to the original scale and orientation.
- The command is available only for Smart Objects.

**Expected Result**
- The Smart Object looks untransformed again while preserving its source data.
