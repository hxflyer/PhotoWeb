# Set Guide and Grid Preferences

- Entry Type: Alignment preference workflow
- Category: Grids, guides, and measurement
- Source: https://helpx.adobe.com/photoshop/desktop/grids-guides-measurement/set-guide-grid-preferences.html

**Function Summary**
Guide and grid preferences define overlay color, line style, grid spacing, and subdivision behavior.

**Where You Find It**
- macOS: `Photoshop > Settings > Guides, Grid & Slices`.
- Windows: `Edit > Preferences > Guides, Grid & Slices`.

**Primary Interface Elements**
- Guide color and style.
- Grid color and style.
- Gridline Every value.
- Subdivisions value.

**How To Use It**
1. Open the Guides, Grid & Slices preferences.
2. Set guide color/style for visibility.
3. Set grid spacing and subdivisions for the document scale.
4. Confirm preferences and toggle grid/guides from the `View` menu.

**Expected Result**
- Alignment overlays display with the chosen style and spacing.
