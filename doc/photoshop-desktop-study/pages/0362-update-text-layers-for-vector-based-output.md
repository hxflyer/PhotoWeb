# Update Text Layers for Vector-Based Output

- Entry Type: Text compatibility workflow
- Category: Text and typography > Get started with text
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/get-started-text/update-text-layers-vector-based-output.html

**Function Summary**
Updating text layers refreshes older type layers so Photoshop can render them with current vector-based text output behavior.

**What This Function Does**
- Modernizes legacy type-layer data when a document was created in an older Photoshop version.
- Helps preserve crisp type output for printing, PDF export, and high-resolution workflows.
- May slightly change text metrics if old text engine behavior differs from the current engine.

**Where You Find It**
- Type-layer update prompts when opening older documents.
- Type-related update commands or warnings in Photoshop when legacy text is detected.
- `Layers` panel for selecting affected type layers.

**Primary Interface Elements**
- Legacy text warning dialogs.
- Type layer icons in the `Layers` panel.
- Before/after text rendering after update.

**How To Use It**
1. Open a document that contains legacy type layers.
2. Review any warning that Photoshop shows about updating text.
3. Save a copy first if the typography must match an old layout exactly.
4. Allow Photoshop to update the type layer when ready.
5. Inspect line breaks, spacing, and font rendering before final output.

**Expected Result**
- Legacy text layers use the current vector text engine and remain editable.
