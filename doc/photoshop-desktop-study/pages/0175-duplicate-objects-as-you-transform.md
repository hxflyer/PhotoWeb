# Duplicate Objects as You Transform

- Entry Type: Transform-duplicate shortcut workflow
- Category: Crop, resize, and transform > Transform, manipulate, and reshape
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/transform-manipulate-reshape/duplicate-objects-as-you-transform.html

**Function Summary**
This shortcut creates a duplicate and immediately places it into transform mode in one action.

**What This Function Does**
- duplicates the current layer or selection
- opens Free Transform on the duplicate
- speeds up layout repetition and variation work

**Where You Find It**
- transform shortcut command

**Primary Interface Elements**
- `Free Transform`
- transform bounding box
- duplicate layer or duplicate selected pixels

**How To Use It**
1. Select the target layer or selection.
2. Press `Ctrl + Alt + T` on Windows or `Command + Option + T` on macOS.
3. Move, rotate, or scale the duplicate on-canvas.
4. Commit the transform.

**Mouse, Keyboard, and Menu Input**
- Hotkey: `Ctrl + Alt + T` on Windows
- Hotkey: `Command + Option + T` on macOS
- Mouse: drag inside the box to move, drag handles to scale, drag outside a corner to rotate.

**User Interface Behavior**
- Photoshop creates a duplicate first, then shows transform handles around the duplicate instead of the original.

**Expected Result**
- A transformed copy is created while the original remains unchanged.
