# View Keyboard Shortcuts

- Entry Type: Shortcut reference access
- Category: Get started > Settings and preferences
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/settings-and-preferences/view-keyboard-shortcuts.html

**Function Summary**
This feature opens Photoshop's Keyboard Shortcuts and Menus dialog so you can inspect shortcut assignments for application menus, panel menus, and tools.

**What This Function Does**
- Displays the current shortcut map inside Photoshop
- Lets users inspect different categories of commands
- Helps users learn faster input paths without searching the full documentation

**Where You Find It**
- `Edit > Keyboard Shortcuts`
- Shortcut: `Alt + Shift + Ctrl + K` on Windows
- Shortcut: `Option + Shift + Command + K` on macOS

**Primary Interface Elements**
- `Keyboard Shortcuts and Menus` dialog
- `Shortcuts For` dropdown
- `Application Menus`
- `Panel Menus`
- `Tools`
- `OK`

**How To Use It**
1. Open `Edit > Keyboard Shortcuts`, or use the keyboard shortcut for your platform.
2. In the `Keyboard Shortcuts and Menus` dialog, open the `Shortcuts For` dropdown.
3. Choose the category you want to inspect: `Application Menus`, `Panel Menus`, or `Tools`.
4. Browse the listed commands and their assigned shortcuts.
5. Select `OK` when finished.

**Mouse, Keyboard, and Menu Input**
- `Keyboard`
  Press `Alt + Shift + Ctrl + K` on Windows or `Option + Shift + Command + K` on macOS to open the dialog directly.
- `Menus`
  Use `Edit > Keyboard Shortcuts` if you prefer the menu path.
- `Mouse`
  Use the dropdown and scroll through the categories to inspect the assignments.

**User Interface Behavior**
- The dialog groups shortcuts by command type rather than mixing every command into one flat list.
- What you see reflects the current Photoshop shortcut configuration.
- This screen is useful for discovery even if you are not changing any shortcut assignments.

**Expected Result**
- You can locate the shortcut for a tool or command without leaving Photoshop.
- Repetitive commands become easier to memorize and use.

**Practical Use Cases**
- Check the shortcut for a rarely used menu command.
- Learn the assigned key for a specific tool.
- Review available shortcuts while building faster workflows.
