# Add and Remove Panels

- Entry Type: Workspace configuration
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/add-remove-panels.html

**Function Summary**
This function controls which Photoshop panels are visible in the workspace. It lets you open panels you need, close panels you do not need, and shape the workspace around the current task.

**What This Function Does**
- Opens hidden panels from the menu
- Closes visible panels to reduce clutter
- Changes which controls are available on screen
- Helps build task-specific workspace layouts

**Where You Find It**
- `Window` menu
- panel tab context menu

**Primary Interface Elements**
- `Window` menu
- panel name list
- panel tab
- visibility checkmark
- `Close` command
- dock area

**How To Use It**
1. Open the `Window` menu.
2. Select a panel name to show that panel in the workspace.
3. Look for the checkmark beside the panel name to confirm that it is visible.
4. Open `Window` again and select the same panel name to hide it.
5. To remove a panel directly, right-click its tab on Windows or Control-click on macOS and choose `Close`.
6. If all panels are removed from a dock, the dock disappears until panels are added again.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Use the menu or panel-tab context menu to add or remove panels.
- `Menus`
  `Window` is the main control point for showing and hiding panels.
- `Keyboard`
  No dedicated default hotkey is described for panel add or remove actions.

**User Interface Behavior**
- A visible panel is marked with a checkmark in the `Window` menu.
- Closing a panel removes it from view but does not delete the feature or its settings.
- Empty docks disappear automatically.
- Adding panels back can recreate a dock when dropped into the dock area.

**Expected Result**
- Only the panels needed for the current workflow remain visible.
- The workspace becomes simpler and easier to manage.

**Practical Use Cases**
- Keep `Layers` and `Properties` visible while hiding less-used panels.
- Strip down the interface for focused editing or teaching.
- Rebuild a clean workspace after experimenting with many panels.
