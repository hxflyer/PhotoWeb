# Move Panels

- Entry Type: Workspace layout control
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/move-panels.html

**Function Summary**
This function repositions individual Photoshop panels or groups so the workspace matches your preferred editing flow. It supports docking, floating, docking prevention, canceling a drag, and locking the workspace to avoid accidental movement.

**What This Function Does**
- Moves a panel to another location
- Docks a panel into an available drop zone
- Leaves a panel floating when no valid dock target is used
- Prevents accidental panel placement during repositioning

**Where You Find It**
- Panel tabs
- blue drop zones while dragging
- `Window > Workspace > Lock Workspace`

**Primary Interface Elements**
- panel tab
- blue drop zone
- floating panel area
- workspace dock
- `Lock Workspace` command

**How To Use It**
1. Select the panel by its tab.
2. Drag the panel to a new location.
3. Move it over a blue drop zone if you want Photoshop to dock it there.
4. Release it away from any drop zone if you want it to stay floating.
5. Hold `Ctrl` on Windows or `Command` on macOS while dragging to prevent docking.
6. Press `Esc` during the drag to cancel the move.
7. If you are using a stylus or want to freeze the layout, enable `Window > Workspace > Lock Workspace`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Drag panels by their tabs to reposition them.
- `Keyboard`
  Hold `Ctrl` on Windows or `Command` on macOS to avoid docking, and press `Esc` to cancel the move.
- `Menus`
  Use `Window > Workspace > Lock Workspace` to stop accidental panel movement.

**User Interface Behavior**
- Blue drop zones indicate valid docking positions.
- No drop zone means the panel will remain floating when dropped.
- The modifier key temporarily changes the drag behavior so the panel does not snap into a dock.
- `Lock Workspace` prevents unintended layout changes during normal use.

**Expected Result**
- Panels can be placed exactly where they are most useful.
- The workspace becomes easier to tailor for mouse, tablet, or multi-monitor setups.

**Practical Use Cases**
- Move frequently used panels closer to the canvas.
- Keep floating reference panels on a second display.
- Lock a finished layout before long stylus sessions.
