# Record an Action

- Entry Type: Action recording workflow
- Category: Automate tasks > Actions
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/create-record-actions/record-action.html

**Function Summary**
Recording an action captures supported Photoshop commands into an action that can be replayed later.

**Where You Find It**
- `Window > Actions`, New Action and Record controls.

**Primary Interface Elements**
- New Action dialog.
- Action name, set, function key, and color options.
- Record and Stop buttons.
- Recorded command list.

**How To Use It**
1. Open the Actions panel.
2. Click `Create New Action`.
3. Name the action and choose a set.
4. Click `Record`.
5. Perform the commands to capture.
6. Click `Stop` when finished.

**Expected Result**
- The Actions panel stores the recorded command sequence.
