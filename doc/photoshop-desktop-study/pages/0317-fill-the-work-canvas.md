# Fill the Work Canvas

- Entry Type: Canvas background display workflow
- Category: Apply painting techniques > Fill objects, selections, and layers
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/fill-objects-selections-layers/fill-work-canvas.html

**Function Summary**
This setting changes the pasteboard or work canvas color around the document for better visual contrast.

**Primary Interface Elements**
- canvas context menu
- `Default`
- `Black`
- `Dark Gray`
- `Medium Gray`
- `Light Gray`
- `Custom`

**How To Use It**
1. Right-click or context-click the work canvas around the document.
2. Choose a preset color or `Custom`.

**Expected Result**
- The area outside the image displays the selected canvas color.
