# Draw Straight-Edged Segments of a Selection Border

- Entry Type: Polygonal Lasso workflow
- Category: Make selections > Freehand selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/freehand-selections/draw-straight-edged-segments-of-a-selection-border.html

**Function Summary**
The `Polygonal Lasso Tool` builds a selection from straight line segments connected point by point.

**What This Function Does**
- creates angular selections with precise corners
- supports constrained 45-degree segments
- can switch temporarily into freehand drawing

**Where You Find It**
- `Polygonal Lasso Tool` in the toolbar

**Primary Interface Elements**
- `Polygonal Lasso Tool`
- `Feather`
- `Anti-alias`
- selection mode buttons

**How To Use It**
1. Select the `Polygonal Lasso Tool`.
2. Set feathering or anti-aliasing if needed.
3. Click once to place the starting point.
4. Click additional points to build straight segments.
5. Close the selection by returning to the starting point or by double-clicking.

**Mouse, Keyboard, and Menu Input**
- Mouse: click to place segment endpoints.
- Keyboard: hold `Shift` to constrain a segment to `45°`.
- Keyboard: hold `Alt` on Windows or `Option` on macOS and drag to draw a temporary freehand segment.
- Keyboard: press `Delete` to remove the last segment.

**User Interface Behavior**
- Photoshop previews the next straight segment as you move the pointer between anchor clicks.

**Expected Result**
- A straight-edged polygon selection is created around the target area.
