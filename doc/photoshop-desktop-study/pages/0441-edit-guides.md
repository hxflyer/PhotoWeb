# Edit Guides

- Entry Type: Guide editing workflow
- Category: Grids, guides, and measurement
- Source: https://helpx.adobe.com/photoshop/desktop/grids-guides-measurement/edit-guides.html

**Function Summary**
Guide editing changes guide position, orientation, color, style, locking, or layout behavior depending on the guide command used.

**Where You Find It**
- `View > Guides`.
- `View > Lock Guides`.
- `Preferences > Guides, Grid & Slices`.

**Primary Interface Elements**
- Guide lines.
- Guide options dialogs.
- Guide color and style preferences.
- Lock/unlock guide toggle.

**How To Use It**
1. Show guides and unlock them if movement is needed.
2. Drag guides manually or use guide commands for exact placement.
3. Change guide color/style in preferences if visibility is poor.
4. Lock guides when the layout is finalized.

**Expected Result**
- Guides remain useful alignment references with the desired position and visibility style.
