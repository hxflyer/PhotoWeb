# Group and Ungroup Layers

- Entry Type: Layer grouping workflow
- Category: Create and manage layers > Transform and manipulate layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/transform-manipulate-layers/group-and-ungroup-layers.html

**Function Summary**
This workflow combines selected layers into a group and removes them from a group again when needed. It is used to keep the Layers panel organized and to manage related content together.

**What This Function Does**
- Builds layer groups from selected layers
- Removes layers from a group through ungrouping
- Supports adding layers and nested groups after the initial group is created

**Where You Find It**
- Layers panel
- `Layer > Group Layers`
- `Layer > Ungroup Layers`
- folder and new-group controls at the bottom of the Layers panel

**Primary Interface Elements**
- selected layers
- group folder
- `Group Layers`
- `Ungroup Layers`
- `Create a New Layer`
- `New Group`

**How To Group Layers**
1. Select multiple layers in the Layers panel.
2. Use `Layer > Group Layers`, or drag the selected layers to the folder control at the bottom of the panel while holding `Alt` on Windows or `Option` on macOS if needed for the described grouping shortcut.

**How To Ungroup Layers**
1. Select the group.
2. Choose `Layer > Ungroup Layers`.

**How To Add More Layers to a Group**
1. Drag a layer into the group folder.
2. Drag one group folder into another if you want nested groups.
3. Use the new layer or new group controls while the target group is active when building content inside it.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select layers, drag them to folders, and move layers or groups between groups directly in the panel.
- `Keyboard`
  Use `Alt` or `Option` for the drag-based grouping shortcut described in the source.
- `Menus`
  Use the `Layer` menu for explicit group and ungroup commands.

**User Interface Behavior**
- Grouping creates a folder container in the Layers panel.
- Nested groups stay intact when moved into larger groups.
- Ungrouping removes the folder structure while leaving the layers themselves in the document.

**Expected Result**
- Layer stacks become easier to organize and navigate, especially in large files.
