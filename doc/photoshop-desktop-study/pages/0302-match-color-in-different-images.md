# Match Color in Different Images

- Entry Type: Match Color overview
- Category: Adjust color > Selective color adjustments
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/selective-color-adjustments/match-color-in-different-images.html

**Function Summary**
`Match Color` shifts the color and luminance of a target image or layer toward a chosen source image or layer.

**Where You Find It**
- `Image > Adjustments > Match Color`

**Primary Interface Elements**
- `Source`
- `Layer`
- `Luminance`
- `Color Intensity`
- `Fade`
- `Neutralize`

**Expected Result**
- The target image or layer takes on color characteristics from the selected source.
