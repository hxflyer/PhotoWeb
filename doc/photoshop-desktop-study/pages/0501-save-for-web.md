# Save for Web

- Entry Type: Web export workflow
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/save-for-web.html

**Function Summary**
Save for Web exports optimized web images and GIF animations with previewed compression, color, transparency, and metadata controls.

**Where You Find It**
- `File > Export > Save for Web (Legacy)`.

**Primary Interface Elements**
- Optimized preview.
- Format menu for GIF, JPEG, PNG, and WBMP where available.
- Quality, colors, dithering, transparency, matte, metadata, and image size controls.

**How To Use It**
1. Choose `File > Export > Save for Web (Legacy)`.
2. Select the output format.
3. Adjust quality, colors, transparency, and size.
4. Preview file size and appearance.
5. Save the optimized file.

**Expected Result**
- Photoshop exports a web-ready copy without replacing the editable master document.
