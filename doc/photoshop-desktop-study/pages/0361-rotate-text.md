# Rotate Text

- Entry Type: Text transform workflow
- Category: Text and typography > Get started with text
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/get-started-text/rotate-text.html

**Function Summary**
Text rotation turns a type layer around a transform reference point while keeping the characters editable.

**What This Function Does**
- Rotates point text or paragraph text as a layer object.
- Supports freehand rotation, numeric angle entry, and preset transform operations.
- Maintains text editability after the transform is committed.

**Where You Find It**
- `Edit > Free Transform`, shortcut `Cmd+T` on macOS or `Ctrl+T` on Windows.
- Transform angle field in the options bar.
- `Edit > Transform` for rotate commands.

**Primary Interface Elements**
- Transform bounding box.
- Rotation cursor outside a transform corner.
- Angle field and reference point controls.
- Commit and cancel transform controls.

**How To Use It**
1. Select the type layer.
2. Start Free Transform.
3. Move the pointer outside a corner handle until the rotation cursor appears.
4. Drag with the mouse or enter a numeric angle.
5. Commit the transform.

**Expected Result**
- The type layer rotates to the new angle and remains editable text.
