# Convert Shapes or Text to Frames

- Entry Type: Frame conversion workflow
- Category: Create and manage layers > Layout and design tools
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/layout-design-tools/convert-to-frame.html

**Function Summary**
This workflow turns an existing shape or text layer into a frame so it can act as an image placeholder with automatic masking behavior.

**What This Function Does**
- Converts shape layers into frames
- Converts text layers into frames
- Creates more custom frame outlines than basic drawn frames alone

**Where You Find It**
- Layers panel context menu
- `Convert to Frame`
- `New Frame` dialog

**Primary Interface Elements**
- source shape or text layer
- `Convert to Frame`
- `New Frame`
- width and height fields

**How To Use It**
1. Create a shape or text layer.
2. In the Layers panel, right-click on that layer.
3. Choose `Convert to Frame`.
4. Name the frame and set size options in the dialog.
5. Confirm with `OK`.

**User Interface Behavior**
- The original shape or text becomes a frame container.
- You can then place or generate image content inside it.
