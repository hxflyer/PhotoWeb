# Create and Record Actions

- Entry Type: Actions topic hub
- Category: Automate tasks
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/create-record-actions.html

**Function Summary**
Actions record Photoshop commands so the same sequence can be replayed on other files or used in batch automation.

**Where You Find It**
- `Window > Actions`.

**Primary Interface Elements**
- Actions panel.
- Action sets and action names.
- Record, Stop, Play, New Action, and New Set buttons.
- Dialog toggle and command checkbox columns.

**Expected Result**
- Users can create reusable command sequences for repetitive editing and export work.
