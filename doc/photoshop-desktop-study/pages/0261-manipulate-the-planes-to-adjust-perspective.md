# Manipulate the Planes to Adjust Perspective

- Entry Type: Perspective Warp adjustment workflow
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/manipulate-the-planes-to-adjust-perspective.html

**Function Summary**
Warp mode reshapes the planes created in `Perspective Warp` to correct or stylize image perspective.

**Primary Interface Elements**
- `Warp`
- quad corners
- straightened edge highlight
- automatic horizontal and vertical correction icons
- commit icon

**How To Use It**
1. Define perspective planes.
2. Switch to `Warp` mode.
3. Drag quad corners to reshape the perspective.
4. `Shift`-click an edge to keep it straight.
5. Use automatic level or vertical correction icons if suitable.
6. Commit the warp.

**Expected Result**
- Perspective distortion is corrected or intentionally adjusted.
