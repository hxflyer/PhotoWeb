# Decrease the Fringe on a Selection

- Entry Type: Defringe workflow
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/decrease-fringe-on-selection.html

**Function Summary**
`Defringe` replaces edge contamination around a selected object using neighboring interior pixels.

**What This Function Does**
- removes narrow halos
- improves pasted cutout edges
- replaces contaminated edge colors using cleaner nearby pixels

**Where You Find It**
- `Layer > Matting > Defringe`

**Primary Interface Elements**
- `Defringe`
- `Width`

**How To Use It**
1. Select the affected layer.
2. Choose `Layer > Matting > Defringe`.
3. Enter the width value and confirm.

**Expected Result**
- The visible halo around the object edge becomes narrower or disappears.
