# Resize Images

- Entry Type: Image Size workflow
- Category: Crop, resize, and transform > Resize and adjust resolution
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/resize-adjust-resolution/resize-images.html

**Function Summary**
This workflow changes image dimensions for screen, web, or print output through the `Image Size` dialog.

**What This Function Does**
- changes width and height
- changes resolution
- optionally resamples pixels
- preserves aspect ratio when the link setting is enabled

**Where You Find It**
- `Image > Image Size`

**Primary Interface Elements**
- `Width`
- `Height`
- unit menus
- aspect-ratio link
- `Resolution`
- `Resample`
- preview area

**How To Use It**
1. Choose `Image > Image Size`.
2. Enter new `Width` and `Height` values.
3. Keep the aspect-ratio link enabled unless you intentionally want distortion.
4. Set `Resolution` for the target output.
5. Turn `Resample` on or off depending on whether pixel count should change.
6. Click `OK`.

**Mouse, Keyboard, and Menu Input**
- Mouse: edit numeric fields and choose units from the dropdowns.
- Menu path: `Image > Image Size`

**User Interface Behavior**
- Photoshop previews the change and updates dimension readouts immediately.
- With `Resample` disabled, changing print size updates resolution instead of inventing new pixels.

**Expected Result**
- The document is resized for its target medium with controlled pixel and print behavior.
