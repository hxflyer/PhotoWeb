# Spot Color Libraries

- Entry Type: Spot library reference
- Category: Adjust color > Choose colors
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/choose-colors/spot-color-libraries.html

**Function Summary**
Spot color libraries provide standardized named ink colors for commercial print production.

**What This Entry Is For**
- explains where named spot colors come from
- supports consistent print specification
- helps users choose the correct library for production requirements

**Expected Result**
- Users can select spot colors from a known library rather than approximate them manually.
