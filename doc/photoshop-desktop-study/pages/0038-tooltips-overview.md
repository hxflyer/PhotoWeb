# Tooltips Overview

- Entry Type: Interface help behavior
- Category: Get started > Set up toolbars and panels
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/set-up-toolbars-panels/show-or-hide-tool-tips.html

**Function Summary**
Tooltips provide on-hover help inside Photoshop. They can show simple labels and shortcuts or richer help content with descriptions and short videos, depending on your settings.

**What This Function Does**
- Shows tool names and shortcuts when the pointer hovers over interface controls
- Provides richer guidance for tools and features when enabled
- Lets users reduce or disable hover help if they prefer a cleaner interface

**Where You Find It**
- On hover over tools, panel options, and interface icons
- Preference path: `Edit > Preferences > Tools` on Windows
- Preference path: `Photoshop > Preferences > Tools` on macOS

**Primary Interface Elements**
- standard tooltip
- rich tooltip
- `Show Tooltips`
- `Show Rich Tooltips`
- `Tools` preferences

**How To Use It**
1. Move the pointer over a tool, icon, or panel control.
2. Read the standard tooltip for the tool name and shortcut.
3. If rich tooltips are enabled, wait briefly to see expanded help content such as a description or short demonstration clip.
4. Open the `Tools` preferences if you want to turn tooltip behavior on or off.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Hover over interface controls to trigger the tooltip.
- `Menus`
  Open `Edit > Preferences > Tools` or the macOS equivalent to configure tooltip behavior.
- `Keyboard`
  No dedicated default hotkey is described for enabling or disabling tooltips.

**User Interface Behavior**
- Standard tooltips show quick identification information.
- Rich tooltips can provide extra teaching context and short visual demonstrations.
- Disabling `Show Tooltips` removes basic hover labels.
- Disabling `Show Rich Tooltips` keeps basic labels available while turning off the richer help layer.

**Expected Result**
- Users can identify tools faster and understand unfamiliar controls without leaving the workspace.
- Advanced users can reduce on-screen interruptions by disabling tooltip features they do not need.

**Practical Use Cases**
- Learn tool shortcuts while working.
- Quickly identify rarely used icons in dense panels.
- Turn off rich hover media in a distraction-free or performance-sensitive setup.
