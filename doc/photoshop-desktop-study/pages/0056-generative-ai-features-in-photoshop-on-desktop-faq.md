# Generative AI Features in Photoshop on Desktop FAQ

- Entry Type: Feature-access reference
- Category: Generative AI in Photoshop
- Source: https://helpx.adobe.com/photoshop/desktop/generative-ai/frequently-asked-questions-about-generative-ai-features.html

**Function Summary**
This entry summarizes eligibility and general usage expectations for Photoshop's generative AI features. It is not a single tool command; it is a reference page that clarifies who can use the features and the basic workflow pattern behind them.

**What This Entry Covers**
- who can access Photoshop generative AI features
- general service availability expectations
- the common user flow for prompt-based generative editing

**Core Workflow Pattern**
1. Open or import an image.
2. Make a selection if the workflow edits part of an image.
3. Enter a text prompt when the feature requires prompt guidance.
4. Generate results and review the returned variations.
5. Save through the normal Photoshop save commands when keeping the result.

**Primary Interface Elements Mentioned**
- `Generate`
- generative layer
- `Save`
- `Save As`
- `Save a Copy`

**User Interface Behavior**
- Generative features typically create editable results rather than replacing the entire document destructively.
- Prompt-based workflows are designed for both new and experienced Photoshop users.
- Availability depends on Adobe service access and supported account or subscription status.

**Expected Result**
- Users understand the baseline requirements and the broad interaction model for Photoshop's AI-assisted generation features before entering feature-specific pages.
