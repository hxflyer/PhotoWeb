# Select a Preset Brush

- Entry Type: Brush preset selection workflow
- Category: Apply painting techniques > Brushes and presets
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/brushes-presets/select-preset-brush.html

**Function Summary**
This workflow selects an existing brush preset for painting or retouching.

**Where You Find It**
- brush preset picker
- `Brushes` panel

**How To Use It**
1. Select a brush-based tool.
2. Open the preset picker or `Brushes` panel.
3. Choose a brush preset.
4. Adjust size or other immediate options if needed.

**Expected Result**
- The active painting tool uses the selected preset's brush tip and behavior.
