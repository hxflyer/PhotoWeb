# Draw Shapes

- Entry Type: Shape Tool workflow
- Category: Draw shapes and paths > Create shapes
- Source: https://helpx.adobe.com/photoshop/desktop/draw-shapes-paths/create-shapes/draw-shapes.html

**Function Summary**
Shape tools create editable geometric shape layers, paths, or pixel shapes.

**Primary Interface Elements**
- shape tool group
- `Shape`
- `Path`
- `Pixels`
- fill
- stroke
- width and height controls

**How To Use It**
1. Select a shape tool.
2. Choose `Shape`, `Path`, or `Pixels`.
3. Set fill, stroke, and geometry options.
4. Drag on the canvas to create the shape.

**Expected Result**
- Photoshop creates the chosen shape object using the selected drawing mode.
