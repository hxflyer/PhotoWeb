# Select and Manage Fonts

- Entry Type: Topic hub
- Category: Text and typography
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/select-manage-fonts.html

**Function Summary**
Font management controls which typefaces are available, how Photoshop applies them, and how missing or similar fonts are resolved.

**What This Area Controls**
- Font family and style selection.
- Missing-font detection and replacement.
- Font matching from raster image samples.
- OpenType, variable font, and multi-layer font changes.

**Where You Find It**
- Options bar with the `Type Tool` active.
- `Properties` and `Character` panels.
- `Type` menu commands such as `Match Font`.
- Creative Cloud font activation prompts where available.

**Primary Interface Elements**
- Font family menu and font style menu.
- Font search field, filters, favorite markers, and recent fonts.
- Missing font warnings.
- Suggested font lists from Match Font.

**Expected Result**
- Users can choose, replace, activate, search, and apply fonts predictably across text layers.
