# Import Brushes and Brush Packs

- Entry Type: Brush import workflow
- Category: Apply painting techniques > Brushes and presets
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/brushes-presets/import-brushes-and-brush-packs.html

**Function Summary**
Brush import adds external brush packs to Photoshop's brush library.

**Primary Interface Elements**
- `Brushes` panel menu
- `Import Brushes`
- brush pack file

**How To Use It**
1. Open the `Brushes` panel.
2. Choose `Import Brushes`.
3. Select the brush pack file.
4. Confirm the import.

**Expected Result**
- Imported brushes appear in the `Brushes` panel for use with brush-based tools.
