# Image Processor Overview

- Entry Type: Image Processor reference
- Category: Automate tasks > Batch processing
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/process-batch-files/image-processor-overview.html

**Function Summary**
Image Processor converts groups of images to JPEG, PSD, or TIFF while optionally resizing and running an action.

**Where You Find It**
- `File > Scripts > Image Processor`.

**Primary Interface Elements**
- Source image selection.
- Destination folder.
- File type options for JPEG, PSD, and TIFF.
- Resize to Fit fields.
- Run Action option.

**Expected Result**
- Users can batch-convert and resize images without building a full Batch action workflow.
