# Unlink Layers and Masks

- Entry Type: Mask-link control
- Category: Create masks > Layer masks
- Source: https://helpx.adobe.com/photoshop/desktop/create-masks/layer-masks/unlink-layers-and-masks.html

**Function Summary**
This control separates a layer from its mask so each can move independently.

**What This Function Does**
- breaks the shared movement relationship between layer and mask
- allows moving the pixels without moving the mask, or vice versa
- can be restored later by relinking

**Where You Find It**
- `Layers` panel between the layer thumbnail and mask thumbnail

**Primary Interface Elements**
- link icon
- layer thumbnail
- mask thumbnail

**How To Use It**
1. In the `Layers` panel, click the link icon between the layer and mask thumbnails to unlink them.
2. Move either the layer or the mask as needed.
3. Click the empty space between the thumbnails to relink them.

**Expected Result**
- The layer and its mask can be repositioned independently until relinked.
