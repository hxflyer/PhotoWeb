# Warp and Unwarp Text

- Entry Type: Text distortion workflow
- Category: Text and typography > Type layers and creation
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/type-layers-creation/warp-unwarp-text.html

**Function Summary**
Text warp bends an editable type layer into preset distortions such as arc, flag, wave, fisheye, or bulge.

**What This Function Does**
- Applies live warp settings to a type layer.
- Controls bend, horizontal distortion, and vertical distortion.
- Can remove warp by setting the warp style back to `None`.

**Where You Find It**
- Warp text button in the options bar when the `Type Tool` is active.
- `Type > Warp Text` where available.

**Primary Interface Elements**
- Warp Text dialog.
- Style menu.
- Horizontal/Vertical orientation buttons.
- Bend, Horizontal Distortion, and Vertical Distortion sliders.

**How To Use It**
1. Select a type layer.
2. Activate the `Type Tool` or choose the warp text command.
3. Pick a warp style.
4. Adjust Bend and distortion sliders.
5. Choose `None` to unwarp the text later.

**Expected Result**
- The type layer displays the selected warp effect while remaining editable text.
