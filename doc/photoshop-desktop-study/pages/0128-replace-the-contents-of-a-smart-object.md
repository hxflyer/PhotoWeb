# Replace the Contents of a Smart Object

- Entry Type: Smart Object source replacement workflow
- Category: Create and manage layers > Smart objects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/smart-objects/replace-the-contents-of-a-smart-object.html

**Function Summary**
This workflow swaps the source content inside a Smart Object while preserving the layer's existing transformations and placement setup.

**What This Function Does**
- Replaces the Smart Object source file
- Keeps existing transforms and prior placement work
- Makes placeholder and template swaps faster

**Where You Find It**
- Layers panel context menu
- `Replace Contents`

**Primary Interface Elements**
- selected Smart Object
- `Replace Contents`
- file browser
- `Place`

**How To Use It**
1. Open `Window > Layers`.
2. Select the Smart Object layer.
3. Right-click the layer and choose `Replace Contents`.
4. Browse to the replacement file.
5. Confirm with `Place`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the Smart Object, open the context menu, and choose the replacement file.
- `Menus`
  The article centers this workflow on the layer context menu.
- `Keyboard`
  No dedicated default hotkey is described for replacing Smart Object contents.

**User Interface Behavior**
- Photoshop swaps the source while keeping prior transforms and placement behavior intact where possible.
- This is especially useful for mockups, templates, or repeated layout structures.

**Expected Result**
- The new source replaces the old one without requiring the entire Smart Object setup to be rebuilt.
