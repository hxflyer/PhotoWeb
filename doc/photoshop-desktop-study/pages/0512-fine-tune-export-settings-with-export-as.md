# Fine-Tune Export Settings with Export As

- Entry Type: Export As workflow
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/fine-tune-export-settings-export-as.html

**Function Summary**
Export As provides previewed controls for format, size, scale, quality, transparency, color space, and metadata before exporting.

**Where You Find It**
- `File > Export > Export As`.

**Primary Interface Elements**
- Format menu.
- Image size and scale controls.
- Quality, transparency, smaller file, and color conversion settings.
- Metadata and resampling options.
- Preview pane.

**How To Use It**
1. Choose `File > Export > Export As`.
2. Select the output format.
3. Adjust quality, scale, size, transparency, and color options.
4. Review the preview and estimated size.
5. Export to the chosen location.

**Expected Result**
- Photoshop creates an optimized exported copy using explicit export settings.
