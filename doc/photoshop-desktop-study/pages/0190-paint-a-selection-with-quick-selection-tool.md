# Paint a Selection with Quick Selection Tool

- Entry Type: Brush-based selection workflow
- Category: Make selections > Automatic and color-based selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/automatic-color-based-selections/paint-a-selection-with-quick-selection-tool.html

**Function Summary**
The `Quick Selection Tool` builds a selection by painting over an area while Photoshop expands the boundary to nearby edges and textures.

**What This Function Does**
- grows selections with brush strokes
- detects edges automatically during painting
- supports additive and subtractive refinement

**Where You Find It**
- `Quick Selection Tool` in the toolbar

**Primary Interface Elements**
- `Quick Selection Tool`
- `New Selection`
- `Add to Selection`
- `Subtract from Selection`
- brush size
- `Sample All Layers`
- `Enhance Edge`
- `Select and Mask`

**How To Use It**
1. Select the `Quick Selection Tool`.
2. Choose whether the next stroke should create, add to, or subtract from the selection.
3. Set brush size and edge behavior.
4. Paint over the target area.
5. Refine further with `Select and Mask` if needed.

**Mouse, Keyboard, and Menu Input**
- Mouse: paint directly over the subject area to expand the selection.

**User Interface Behavior**
- Selection edges snap outward toward contrast boundaries as you paint.
- Smaller brush sizes give more control around narrow details.

**Expected Result**
- The target region becomes selected quickly without manually tracing every edge.
