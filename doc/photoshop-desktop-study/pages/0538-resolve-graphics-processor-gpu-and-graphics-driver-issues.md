# Resolve Graphics Processor GPU and Graphics Driver Issues

- Entry Type: GPU troubleshooting workflow
- Category: Troubleshoot
- Source: https://helpx.adobe.com/photoshop/desktop/troubleshoot/resolve-graphics-processor-gpu-graphics-driver-issues.html

**Function Summary**
GPU troubleshooting resolves display glitches, disabled features, crashes, flicker, canvas errors, and acceleration problems related to graphics hardware or drivers.

**Where You Find It**
- `Preferences > Performance`.
- `Help > GPU Compatibility`.
- Operating system graphics driver update tools.

**Primary Interface Elements**
- Use Graphics Processor checkbox.
- Advanced Settings.
- GPU compatibility report.
- Driver version information.

**How To Use It**
1. Check GPU compatibility.
2. Update the graphics driver from the hardware vendor or operating system update channel.
3. Restart Photoshop.
4. Test with GPU disabled to confirm whether the issue is GPU-related.
5. Re-enable GPU features after updating if stable.

**Expected Result**
- GPU-related Photoshop problems are confirmed, reduced, or resolved.
