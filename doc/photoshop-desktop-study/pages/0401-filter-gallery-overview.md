# Filter Gallery Overview

- Entry Type: Filter Gallery reference
- Category: Effects and filters > Get started with filters
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/get-started-filters/filter-gallery-overview.html

**Function Summary**
The Filter Gallery is a visual workspace for previewing, adjusting, stacking, and reordering compatible creative filters.

**Where You Find It**
- `Filter > Filter Gallery`.

**Primary Interface Elements**
- Preview pane with zoom controls.
- Filter thumbnails and category folders.
- Settings sliders for the selected effect.
- Filter stack list for multiple gallery effects.

**Expected Result**
- Users can build layered creative filter looks while previewing the result before applying it.
