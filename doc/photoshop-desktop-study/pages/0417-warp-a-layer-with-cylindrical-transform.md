# Warp a Layer with Cylindrical Transform

- Entry Type: Cylindrical warp workflow
- Category: Effects and filters > Distortion
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/distortion/warp-layer-cylindrical-transform.html

**Function Summary**
Cylindrical Transform curves a layer around a cylinder-like surface for packaging, labels, mockups, and rounded objects.

**Where You Find It**
- Transform Warp preset or warp options for cylindrical deformation.

**Primary Interface Elements**
- Warp controls.
- Curvature or cylinder-related handles.
- Transform commit/cancel controls.

**How To Use It**
1. Select the layer or Smart Object.
2. Enter Transform Warp.
3. Choose the cylindrical warp option where available.
4. Drag handles to match the curve of the target object.
5. Commit the transform.

**Expected Result**
- The layer curves as if wrapped around a cylindrical form.
