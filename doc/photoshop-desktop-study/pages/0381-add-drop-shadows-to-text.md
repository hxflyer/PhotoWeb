# Add Drop Shadows to Text

- Entry Type: Text layer style workflow
- Category: Text and typography > Type layers and creation
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/type-layers-creation/add-drop-shadows-to-text.html

**Function Summary**
Drop Shadow adds a live layer style that casts a shadow behind an editable type layer.

**What This Function Does**
- Adds depth, separation, or emphasis to text.
- Controls shadow blend mode, color, opacity, angle, distance, spread, and size.
- Remains editable as a layer effect until rasterized or merged.

**Where You Find It**
- `Layer > Layer Style > Drop Shadow`.
- `fx` button at the bottom of the `Layers` panel.
- Double-click the effect name below a layer to reopen settings.

**Primary Interface Elements**
- Layer Style dialog.
- Drop Shadow checkbox.
- Blend Mode, Opacity, Angle, Distance, Spread, Size, and Contour controls.
- `Use Global Light` option.

**How To Use It**
1. Select the type layer.
2. Click `fx` in the `Layers` panel and choose `Drop Shadow`.
3. Set shadow color, opacity, angle, distance, spread, and size.
4. Preview the effect on canvas.
5. Confirm the Layer Style dialog.

**Expected Result**
- The type layer displays a live drop shadow effect that can be edited or hidden later.
