# Create Preset Brush Groups

- Entry Type: Brush organization workflow
- Category: Apply painting techniques > Brushes and presets
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/brushes-presets/create-preset-brush-groups.html

**Function Summary**
Brush groups organize related brush presets into folders inside the `Brushes` panel.

**How To Use It**
1. Open the `Brushes` panel.
2. Create a new group.
3. Name the group.
4. Move related brushes into it.

**Expected Result**
- Brush presets are grouped for faster selection and cleaner brush management.
