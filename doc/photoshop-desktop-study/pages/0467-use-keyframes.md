# Use Keyframes

- Entry Type: Timeline animation workflow
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/use-keyframes.html

**Function Summary**
Keyframes mark property values at specific times so Photoshop can animate changes between them.

**Where You Find It**
- `Window > Timeline`, Video Timeline mode.

**Primary Interface Elements**
- Timeline playhead.
- Stopwatch icons for animatable properties.
- Diamond keyframe markers.
- Layer property tracks.

**How To Use It**
1. Open the Timeline panel and use Video Timeline mode.
2. Expand the layer property to animate.
3. Click the stopwatch to enable keyframes.
4. Move the playhead and change the property value.
5. Preview playback.

**Expected Result**
- Photoshop interpolates motion, opacity, style, or transform changes between keyframes.
