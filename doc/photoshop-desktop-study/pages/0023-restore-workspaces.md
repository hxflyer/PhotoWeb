# Restore Workspaces

- Entry Type: Workspace reset
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/restore-workspaces.html

**Function Summary**
This function resets a workspace to its original saved layout. It can be used on the default `Essentials` workspace, on other preset workspaces, and on custom workspaces that have been changed since they were last saved.

**What This Function Does**
- Restores a workspace layout to its saved baseline
- Resets panel positions after the interface has been rearranged
- Lets you recover the default arrangement for built-in presets
- Lets you reset one workspace or all default preset workspaces

**Where You Find It**
- `Choose a workspace` dropdown in the application bar
- `Window > Workspace > Reset > Essentials (Default)` for the default workspace
- `Edit > Preferences > Workspace` on Windows
- `Photoshop > Settings > Workspace` on macOS

**Primary Interface Elements**
- `Choose a workspace` dropdown
- `Reset <Workspace Name>` command
- `Essentials` workspace
- `Window` menu
- `Workspace` submenu
- workspace preferences
- `Restore Default Workspaces` button

**How To Use It**
1. Open the `Choose a workspace` menu in the application bar.
2. To restore the default layout, select `Essentials`.
3. To reset a specific workspace, choose `Reset <Workspace Name>`.
4. To restore all Photoshop preset workspaces, open workspace preferences and select `Restore Default Workspaces`.
5. If needed, use `Window > Workspace > Reset > Essentials (Default)` for a direct reset path to the default layout.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Use the application-bar workspace selector or the preferences controls.
- `Menus`
  Switch or reset the workspace from `Window > Workspace`, or restore all default presets from workspace preferences.
- `Keyboard`
  No default dedicated hotkey is described for restoring workspaces.

**User Interface Behavior**
- Resetting a workspace changes panel layout and placement, not document content.
- Built-in presets such as `Essentials` and `Photography` can be restored to their original Adobe-defined arrangement.
- A saved custom workspace resets to its previously stored configuration.
- Restoring all default workspaces affects all Photoshop preset layouts at once.

**Expected Result**
- Misplaced, closed, or rearranged panels return to a known layout.
- The workspace becomes predictable again after experimentation or accidental changes.
- You can recover either one workspace or the complete set of built-in defaults.

**Practical Use Cases**
- Reset the interface after moving panels around during a temporary task.
- Recover a built-in workspace after it becomes cluttered.
- Standardize the layout across users or training sessions.
