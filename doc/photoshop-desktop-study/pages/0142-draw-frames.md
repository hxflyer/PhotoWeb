# Draw Frames

- Entry Type: Frame tool workflow
- Category: Create and manage layers > Layout and design tools
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/layout-design-tools/draw-frames.html

**Function Summary**
This workflow uses the Frame tool to create placeholder containers that automatically mask images placed inside them.

**What This Function Does**
- Draws empty image frames
- Supports multiple frame shapes
- Masks placed images automatically
- Works for both blank layouts and framing on top of existing images

**Where You Find It**
- `Frame` tool
- Options bar
- Contextual Task Bar for image generation or content insertion

**Primary Interface Elements**
- `Frame` tool
- frame shape options
- stroke settings
- `Generate Image`
- Contextual Task Bar

**How To Draw an Empty Frame**
1. Select the `Frame` tool.
2. Choose the frame shape in the Options bar.
3. Set any stroke or appearance options.
4. Drag on the canvas to draw the frame.

**How To Draw a Frame over Existing Content**
1. Select the `Frame` tool.
2. Choose the shape.
3. Draw it over the image region you want framed.

**Expected Result**
- A frame layer is created and is ready to accept placed or generated image content.
