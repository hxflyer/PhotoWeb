# Overview Of Adobe Photoshop (Beta) On Desktop

- Entry Type: Product overview page
- Category: What's new
- Source: https://helpx.adobe.com/photoshop/desktop/whats-new/photoshop-desktop-beta-overview.html

**Function Summary**
This page explains what Photoshop beta is, how it differs from the standard release, and when it should or should not be used. It describes the beta app as a separate installation intended for early-access testing rather than primary production work.

**What This Entry Explains**
- what Photoshop beta is
- how it runs alongside regular Photoshop
- when to use beta instead of stable Photoshop
- what credit behavior applies to generative beta features
- what commercial-use and indemnification limitations exist

**How To Use This Entry**
- Read this page before installing or depending on Photoshop beta.
- Use it to decide whether a task belongs in the stable app or the beta app.
- Treat this as environment guidance rather than feature-specific instruction.

**When To Use Regular Photoshop**
- when you need maximum stability
- when the project is time-sensitive
- when you want polished production behavior
- when you want the lowest risk of crashes or behavior changes

**When To Use Photoshop Beta**
- when you want early access to upcoming features
- when you are comfortable testing in-progress tools
- when the project is exploratory rather than mission-critical
- when you want to provide feedback before a feature reaches stable release

**Credit And Usage Notes**
- Most generative beta features still use Adobe generative credits.
- Credit balance is shared across the Photoshop and Photoshop beta applications.
- Some tools, such as certain Remove-tool generative behaviors, may not consume credits.
- Beta outputs may be commercially usable unless otherwise stated, but beta outputs are not covered by indemnification while still in beta.

**Expected Result**
- You should understand whether Photoshop beta is appropriate for your current work.
- You should know that beta is a parallel testing environment, not a replacement for stable Photoshop in all cases.
