# Match Color Between Two Images

- Entry Type: Cross-document Match Color workflow
- Category: Adjust color > Selective color adjustments
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/selective-color-adjustments/match-color-between-two-images.html

**Function Summary**
This workflow adjusts one document so its color appearance better matches another open document.

**Where You Find It**
- `Image > Adjustments > Match Color`

**How To Use It**
1. Open both source and target images.
2. Select the target image.
3. Choose `Image > Adjustments > Match Color`.
4. Select the source document.
5. Adjust luminance, color intensity, fade, and neutralize settings.
6. Confirm with `OK`.

**Expected Result**
- The target image shifts toward the source image's overall color and brightness character.
