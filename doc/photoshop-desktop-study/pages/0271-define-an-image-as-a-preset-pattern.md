# Define an Image as a Preset Pattern

- Entry Type: Pattern preset creation workflow
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/define-an-image-as-a-preset-pattern.html

**Function Summary**
`Define Pattern` stores a selected image area as a reusable pattern preset.

**Where You Find It**
- `Edit > Define Pattern`

**Primary Interface Elements**
- `Rectangular Marquee Tool`
- `Feather`
- `Define Pattern`
- `Name`

**How To Use It**
1. Select the source area with `Rectangular Marquee`.
2. Keep `Feather` at `0 px` for a clean tile boundary.
3. Choose `Edit > Define Pattern`.
4. Name the pattern and confirm.

**Expected Result**
- The selected image area becomes available as a pattern preset.
