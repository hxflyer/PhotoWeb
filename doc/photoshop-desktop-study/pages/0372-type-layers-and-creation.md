# Type Layers and Creation

- Entry Type: Type layer reference
- Category: Text and typography
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/type-layers-creation.html

**Function Summary**
Type layers store Photoshop text as editable vector-based text objects separate from pixel layers.

**What This Function Describes**
- Point text layers are created by clicking once with the `Type Tool`.
- Paragraph text layers are created by dragging a text box.
- Type layers can be transformed, styled, masked, clipped, and blended like other layers.
- Rasterizing, converting, or merging type can remove normal text editability.

**Where You Find It**
- Toolbar: `Type Tool`, shortcut `T`.
- `Layers` panel with the type layer icon.
- `Type` menu and layer context menu.

**Primary Interface Elements**
- Type layer thumbnail and `T` icon.
- Text bounding box.
- Options bar and text panels.
- Layer effects, masks, blend modes, and opacity controls.

**Expected Result**
- Users understand that Photoshop text is normally layer-based and editable until converted or rasterized.
