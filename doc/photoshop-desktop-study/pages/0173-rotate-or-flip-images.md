# Rotate or Flip Images

- Entry Type: Orientation command reference
- Category: Crop, resize, and transform > Transform, manipulate, and reshape
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/transform-manipulate-reshape/rotate-or-flip-images.html

**Function Summary**
This command group rotates or mirrors image content, layers, or selections to change orientation without manually dragging transform handles.

**What This Function Does**
- rotates content by fixed angles
- mirrors content horizontally or vertically
- works well for fast orientation corrections

**Where You Find It**
- `Edit > Transform`

**Primary Interface Elements**
- `Rotate 180°`
- `Rotate 90° Clockwise`
- `Rotate 90° Counter Clockwise`
- `Flip Horizontal`
- `Flip Vertical`

**How To Use It**
1. Select the layer or selection you want to affect.
2. Open `Edit > Transform`.
3. Choose the needed rotate or flip command.

**Mouse, Keyboard, and Menu Input**
- Menu path: `Edit > Transform`

**User Interface Behavior**
- Fixed rotate and flip commands apply immediately instead of entering a live transform box.

**Expected Result**
- The selected content changes orientation using the exact preset command chosen.
