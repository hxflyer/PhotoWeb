# Text on Paths and Shapes

- Entry Type: Topic hub
- Category: Text and typography
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/text-on-paths-shapes.html

**Function Summary**
Text on paths and shapes attaches editable type to vector paths, shape outlines, or shape interiors.

**What This Area Controls**
- Text that follows a drawn path or shape edge.
- Text that flows inside a closed vector shape.
- Path handles that define where path text begins, ends, flips, and repositions.
- Shape/path edits that change text flow without rasterizing the type.

**Where You Find It**
- `Type Tool`, `Pen Tool`, shape tools, `Path Selection Tool`, and `Direct Selection Tool`.
- `Paths` panel and `Layers` panel.

**Primary Interface Elements**
- Path outline.
- Text insertion cursor adapted to path or area type.
- Start and end brackets for path text.
- Direction and flip indicators.

**Expected Result**
- Users can create curved, shaped, or bounded text layouts while keeping the type editable.
