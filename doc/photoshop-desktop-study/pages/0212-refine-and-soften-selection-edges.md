# Refine and Soften Selection Edges

- Entry Type: Edge-quality reference
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/refine-and-soften-selection-edges.html

**Function Summary**
This reference explains the three main ways Photoshop softens or cleans selection edges: `Select and Mask`, anti-aliasing, and feathering.

**What This Entry Is For**
- helps users decide which edge-softening method to use
- explains the difference between edge smoothing and edge blurring
- improves composite realism

**Primary Function Areas**
- `Select and Mask`
- anti-aliasing
- feathering

**Where You Find It**
- `Select > Select and Mask`
- tool options bars for compatible selection tools
- `Select > Modify > Feather`

**User Interface Behavior**
- Anti-aliasing smooths jagged steps at the edge.
- Feathering creates a blurred transition zone.
- `Select and Mask` provides the highest level of edge-preview control.

**Expected Result**
- Users can choose the correct refinement method for either crisp but smooth edges or deliberately soft transitions.
