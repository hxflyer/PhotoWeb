# Painting Tools Overview

- Entry Type: Tool family reference
- Category: Apply painting techniques > Fill objects, selections, and layers
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/fill-objects-selections-layers/painting-tools-overview.html

**Function Summary**
This reference lists Photoshop tools used to paint, fill, sample, clone, and apply brush-based effects.

**Primary Tools**
- `Brush`
- `Pencil`
- `Color Replacement`
- `Mixer Brush`
- `History Brush`
- `Art History Brush`
- `Gradient`
- `Paint Bucket`
- `Eyedropper`
- `Color Sampler`
- `Clone Stamp`
- `Pattern Stamp`

**Expected Result**
- Users can identify the correct painting tool for fill, sampling, retouching, or expressive brush work.
