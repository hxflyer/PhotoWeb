# Use Spring-Loaded Shortcuts

- Entry Type: Temporary tool switching behavior
- Category: Get started > Set up toolbars and panels
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/set-up-toolbars-panels/spring-loaded-shortcuts.html

**Function Summary**
Spring-loaded shortcuts temporarily switch to another tool only while its shortcut key is held down. When the key is released, Photoshop returns to the previously active tool automatically.

**What This Function Does**
- Temporarily changes tools without permanently leaving the current tool
- Speeds up quick adjustments during painting, retouching, or layout work
- Lets users tune the delay before the temporary switch activates

**Where You Find It**
- Directly in the Tools panel workflow through tool shortcut keys
- Preference path: `Edit > Preferences > Tools` on Windows
- Preference path: `Photoshop > Preferences > Tools` on macOS

**Primary Interface Elements**
- active tool
- temporary tool shortcut
- `Spring-loaded Tool Shortcuts` preference
- delay slider
- `OK` button

**How To Use It**
1. Select your main working tool, such as `Brush`.
2. Press and hold the shortcut for another tool, such as `V` for `Move`.
3. Use the temporary tool while keeping the shortcut key held down.
4. Release the key to return to the original tool automatically.

**How To Adjust Timing**
1. Open the `Tools` preferences.
2. Make sure `Spring-loaded Tool Shortcuts` is enabled.
3. Adjust the delay slider.
4. Use a shorter delay if you want the temporary tool to activate more quickly.
5. Confirm with `OK`.

**Mouse, Keyboard, and Menu Input**
- `Keyboard`
  Hold a tool shortcut key for the temporary switch, then release it to return.
- `Menus`
  Use the platform-specific `Tools` preferences path to enable or adjust timing.
- `Mouse`
  Continue editing normally with the temporary tool while the key is held.

**User Interface Behavior**
- Photoshop does not permanently change the selected tool unless you release and reselect in the normal way.
- The feature is active only while the shortcut is held long enough to pass the configured delay.
- The default delay is described as `200 milliseconds`.

**Expected Result**
- Users can borrow another tool briefly without breaking editing flow.
- Repetitive switching between movement, brushing, selection, or navigation becomes faster.

**Practical Use Cases**
- Hold `V` to move content briefly while painting, then return to `Brush`.
- Borrow a selection tool for a quick adjustment and return immediately to the main tool.
- Reduce pointer travel by relying more on held shortcuts and less on the Tools panel.
