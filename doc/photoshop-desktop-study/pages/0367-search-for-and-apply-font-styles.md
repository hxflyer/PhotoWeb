# Search for and Apply Font Styles

- Entry Type: Font selection workflow
- Category: Text and typography > Select and manage fonts
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/select-manage-fonts/search-apply-font-styles.html

**Function Summary**
Font search and style controls locate a typeface quickly and apply its available style variants to selected text.

**What This Function Does**
- Filters font lists by typed search text, class, favorite status, or availability.
- Applies font family and style variants such as Regular, Italic, Bold, Light, Condensed, or Black.
- Updates selected characters or the full selected type layer.

**Where You Find It**
- Font family menu in the options bar.
- `Properties` panel.
- `Window > Character`.

**Primary Interface Elements**
- Search field inside the font menu.
- Font family list with previews.
- Style menu beside the font family control.
- Favorite, recent, and filter controls where available.

**How To Use It**
1. Select a type layer or highlight characters.
2. Open the font family menu.
3. Type a font name or filter the list.
4. Choose a font family.
5. Choose a style variant if the family includes one.

**Expected Result**
- The selected text updates to the chosen font family and style.
