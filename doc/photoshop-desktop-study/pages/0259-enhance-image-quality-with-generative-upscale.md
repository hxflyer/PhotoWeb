# Enhance Image Quality with Generative Upscale

- Entry Type: Generative upscaling workflow
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/enhance-image-quality-with-generative-upscale.html

**Function Summary**
`Generative Upscale` enlarges an image while improving perceived sharpness, clarity, or detail.

**What This Function Does**
- upscales low-resolution images
- offers multiple AI model choices
- creates the result in a new document
- uses generative credits based on model and output size

**Where You Find It**
- `Image > Generative Upscale`

**Primary Interface Elements**
- scale factor
- `Firefly Upscaler`
- `Topaz Gigapixel`
- `Topaz Bloom`
- `Upscale`

**How To Use It**
1. Open a low-resolution image.
2. Choose `Image > Generative Upscale`.
3. Select `2x` or `4x` and any model-specific options.
4. Choose the AI model.
5. Click `Upscale`.

**Expected Result**
- Photoshop opens an upscaled version in a new document.
