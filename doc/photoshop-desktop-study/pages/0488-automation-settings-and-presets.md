# Automation Settings and Presets

- Entry Type: Automation preset reference
- Category: Automate tasks
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/automation-settings-presets.html

**Function Summary**
Automation settings and presets store reusable choices for actions, batches, droplets, scripts, and processing workflows.

**Where You Find It**
- Actions panel sets.
- Batch, Droplet, Image Processor, and preset-related dialogs.
- Preset Manager or export/import options where available.

**Primary Interface Elements**
- Action sets.
- Saved presets and libraries.
- Import, export, load, save, and reset controls.

**Expected Result**
- Automation workflows can be reused, shared, backed up, or restored.
