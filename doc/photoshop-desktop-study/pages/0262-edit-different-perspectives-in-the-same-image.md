# Edit Different Perspectives in the Same Image

- Entry Type: Multi-plane Perspective Warp workflow
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/edit-different-perspectives-in-the-same-image.html

**Function Summary**
This workflow uses connected or independent quads to correct different perspective areas separately.

**What This Function Does**
- preserves one perspective while changing another
- edits separate planes independently
- supports complex architecture and object scenes

**Where You Find It**
- `Edit > Perspective Warp`

**How To Use It**
1. Open `Edit > Perspective Warp`.
2. Draw quads over the areas you need to preserve or adjust.
3. Connect quads when the planes should relate to each other.
4. Leave quads unconnected when perspectives should move independently.
5. Switch to `Warp` and adjust each plane.
6. Commit with `Enter`, `Return`, or the commit icon.

**Expected Result**
- Different perspective regions are corrected without forcing the whole image into one plane.
