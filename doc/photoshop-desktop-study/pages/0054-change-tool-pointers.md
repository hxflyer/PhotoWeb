# Change Tool Pointers

- Entry Type: Cursor display preference
- Category: Get started > Settings and preferences
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/settings-and-preferences/change-tool-pointers.html

**Function Summary**
This preference changes how Photoshop tool cursors are drawn on screen. It affects whether you see tool icons, crosshairs, or brush-size outlines, and it helps match cursor behavior to the level of precision you need.

**What This Function Does**
- Changes painting cursor appearance
- Changes non-painting cursor appearance
- Controls whether brush outlines and crosshairs are shown
- Helps balance accuracy, visibility, and performance

**Where You Find It**
- `Edit > Preferences > Cursors` on Windows
- `Photoshop > Settings > Cursors` on macOS

**Primary Interface Elements**
- `Painting Cursors`
- `Other Cursors`
- `Standard`
- `Precise`
- `Normal Brush Tip`
- `Full Size Brush Tip`
- `Show Crosshair in Brush Tip`
- `Show Only Crosshair While Painting`
- `Brush Preview`

**How To Use It**
1. Open the `Cursors` preferences page.
2. In `Painting Cursors`, choose the brush pointer style you want.
3. In `Other Cursors`, choose `Standard` or `Precise` for non-brush tools.
4. Set the `Brush Preview` color if needed.
5. Confirm with `OK`.

**Cursor Behavior Options**
- `Standard`
  Shows the pointer as the tool icon
- `Precise`
  Shows a crosshair-style cursor for exact placement
- `Normal Brush Tip`
  Shows an outline covering roughly half the affected brush area
- `Full Size Brush Tip`
  Shows an outline close to the full affected brush area
- `Show Crosshair in Brush Tip`
  Adds a center crosshair inside the brush outline
- `Show Only Crosshair While Painting`
  Can improve responsiveness with very large brushes

**Mouse, Keyboard, and Menu Input**
- `Menus`
  Open the `Cursors` preference screen from Photoshop settings.
- `Keyboard`
  On Windows, hold `Alt` and right-click-drag left or right to resize the brush, or drag up and down to change hardness. On macOS, use `Control + Option` with the equivalent drag gesture.
- `Mouse`
  Drag-based brush adjustments let you preview the cursor change directly on the canvas.

**User Interface Behavior**
- Cursor style changes affect how precisely tool hotspots and brush boundaries are shown.
- Large brush workflows may benefit from crosshair-only display to reduce visual overhead.
- Pointer appearance changes do not modify brush behavior itself, only how it is visualized.

**Expected Result**
- Cursor feedback better matches the user's editing style and precision needs.

**Practical Use Cases**
- Use `Precise` for detailed masking or pixel-level edits.
- Use `Full Size Brush Tip` for painting where edge coverage matters.
- Enable crosshair-only display when very large brushes feel visually heavy.
