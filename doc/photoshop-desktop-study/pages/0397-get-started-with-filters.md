# Get Started with Filters

- Entry Type: Topic hub
- Category: Effects and filters
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/get-started-filters.html

**Function Summary**
The filters starter area covers how filters are applied, previewed, blended, faded, stacked, and made non-destructive.

**Where You Find It**
- `Filter` menu.
- `Filter Gallery`.
- `Layers` panel when Smart Filters are used.

**Primary Interface Elements**
- Filter dialog previews.
- Filter Gallery category list and preview pane.
- Smart Filter mask and filter list under Smart Object layers.

**Expected Result**
- Users can apply filters intentionally and decide when to use destructive pixel edits or non-destructive Smart Filters.
