# Move Guides

- Entry Type: Guide positioning workflow
- Category: Grids, guides, and measurement
- Source: https://helpx.adobe.com/photoshop/desktop/grids-guides-measurement/move-guides.html

**Function Summary**
Moving guides repositions existing guide lines without changing document pixels.

**Where You Find It**
- `Move Tool` and visible guides.
- `View > Lock Guides` must be off to drag guides.

**Primary Interface Elements**
- Guide line.
- Move cursor when hovering over a guide.
- Ruler position feedback.

**How To Use It**
1. Make guides visible.
2. Disable `View > Lock Guides` if needed.
3. Select the `Move Tool`.
4. Drag the guide to a new location.
5. Use rulers or Info panel values for precision.

**Expected Result**
- The guide moves to the new alignment position.
