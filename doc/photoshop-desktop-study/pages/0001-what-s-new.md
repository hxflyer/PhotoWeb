# What's New

- Entry Type: Topic hub
- Category: Photoshop Desktop Help
- Source: https://helpx.adobe.com/photoshop/desktop/whats-new.html

**Function Summary**
This entry is a release-information hub, not a single Photoshop tool. Use it to check newly added desktop features, release notes, beta-specific updates, and technology preview items before you look up a specific function in detail.

**What This Hub Is For**
- Review major feature additions in current desktop releases
- Check release-note style changes and fixes
- Compare standard desktop updates with beta-only features
- Find preview features that are not yet part of the main stable workflow

**How To Use This Entry**
- Open this topic when you want to see what changed in Photoshop before learning an individual feature.
- Start with the current release overview if you want feature highlights.
- Open release notes if you need version-by-version change history.
- Open beta and technology preview entries if you want experimental or early-access tools.

**Included Subtopics**
- What’s new in Adobe Photoshop on desktop
- Adobe Photoshop on desktop release notes
- What’s new in Adobe Photoshop (beta) on desktop
- Overview of Adobe Photoshop (beta) on desktop
- Use technology previews
- List of technology preview features

**Interface Context**
- This topic behaves like a documentation index, not an in-app panel or dialog.
- It helps you decide which specific function page to open next.

**Expected Result**
After using this hub, you should know whether the feature you need is:
- already available in the current desktop release
- only available in beta
- still a technology preview
- documented in a separate release note entry
