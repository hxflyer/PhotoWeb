# Measurement Data Points Overview

- Entry Type: Measurement data reference
- Category: Grids, guides, and measurement
- Source: https://helpx.adobe.com/photoshop/desktop/grids-guides-measurement/measurement-data-points-overview.html

**Function Summary**
Measurement data points define which values Photoshop records in the Measurement Log.

**What This Function Describes**
- Common values include area, perimeter, width, height, angle, distance, count, color, and document metadata.
- Available data points depend on the active measurement tool, selection, or count.
- Presets can store useful groups of data points.

**Where You Find It**
- Measurement Log panel menu.
- `Image > Analysis > Select Data Points`.

**Expected Result**
- Users can choose the exact measurement fields needed for analysis or reporting.
