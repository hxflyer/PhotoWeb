# Dynamic Text Overview

- Entry Type: Dynamic text reference
- Category: Text and typography > Work with dynamic text
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/work-with-dynamic-text/dynamic-text-overview.html

**Function Summary**
Dynamic text describes editable text behavior that responds to formatting, resizing, and positional controls while remaining live text.

**What This Function Describes**
- Dynamic text is adjusted through text-specific controls rather than pixel editing.
- Layout can change when text size, box size, start/end points, or formatting changes.
- Users should inspect wrapping, clipping, and alignment after changing dynamic controls.

**Where You Find It**
- Supported text layers or dynamic text objects.
- On-canvas text handles.
- `Properties`, `Character`, and `Paragraph` panels.

**Primary Interface Elements**
- Text bounding or control handles.
- Start and end markers where applicable.
- Font, size, spacing, and paragraph controls.

**Expected Result**
- Users understand dynamic text as live editable text with layout controls that can change its position, flow, and formatting.
