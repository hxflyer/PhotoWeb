# Auto-Align Image Layers

- Entry Type: Automatic image-registration workflow
- Category: Create and manage layers > Create layer compositions
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/create-layer-compositions/align-image-layers.html

**Function Summary**
This workflow automatically aligns overlapping image layers based on shared visual content. It is commonly used for panoramas, composites, focus stacks, and multi-shot correction work.

**What This Function Does**
- Detects common content across selected image layers
- Repositions and transforms layers automatically
- Supports multiple projection and alignment strategies

**Where You Find It**
- Layers panel
- `Edit > Auto-Align Layers`

**Primary Interface Elements**
- selected image layers
- optional locked reference layer
- `Auto-Align Layers`
- projection options such as `Auto`, `Perspective`, `Collage`, `Cylindrical`, `Spherical`, and `Reposition`

**How To Use It**
1. Select the image layers you want to align.
2. If needed, keep one layer locked as the visual reference.
3. Rasterize incompatible layers if the source guidance calls for it.
4. Choose `Edit > Auto-Align Layers`.
5. Select the projection or alignment method that best fits the job.
6. Confirm the alignment.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the layer stack and choose the alignment mode from the dialog.
- `Menus`
  Use `Edit > Auto-Align Layers`.
- `Keyboard`
  No dedicated default hotkey is described for this dialog-driven workflow.

**User Interface Behavior**
- Photoshop analyzes corners, edges, and overlapping structures to calculate alignment.
- Different projection modes are better suited to different composite or panorama cases.

**Expected Result**
- Overlapping image layers line up automatically with less manual nudging and distortion work.
