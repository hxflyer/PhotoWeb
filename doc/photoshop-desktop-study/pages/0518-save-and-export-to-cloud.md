# Save and Export to Cloud

- Entry Type: Cloud save/export workflow
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/save-export-cloud.html

**Function Summary**
Cloud save and export workflows store editable work or output copies in Adobe cloud-connected locations for access and sharing.

**Where You Find It**
- Save dialog cloud document destination.
- Home screen cloud document area.
- Share/export cloud options where available.

**Primary Interface Elements**
- Cloud document save location.
- Sync status.
- Share controls.
- Cloud storage browser.

**How To Use It**
1. Choose a cloud document destination when saving editable work.
2. Name the file and save.
3. Watch sync status before closing or switching devices.
4. Use share/export options for cloud-based delivery when available.

**Expected Result**
- The Photoshop document or exported copy is stored in a cloud-connected location.
