# Conversion Options for Indexed Color Images

- Entry Type: Indexed Color settings reference
- Category: Adjust color > Color modes
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/color-modes/conversion-options-for-indexed-color-images.html

**Function Summary**
Indexed Color options define the palette, color count, transparency, and dithering behavior used during conversion.

**Primary Interface Elements**
- palette type
- color count
- forced colors
- transparency
- matte
- dither method
- amount

**Expected Result**
- Users can reduce colors while controlling banding, transparency edges, and palette quality.
