# Copy and Paste Text

- Entry Type: Clipboard text workflow
- Category: Text and typography > Get started with text
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/get-started-text/copy-paste-text.html

**Function Summary**
Copy and paste text moves text content between Photoshop type layers, other Photoshop documents, and external applications through the system clipboard.

**What This Function Does**
- Copies selected characters from an editable type layer.
- Pastes plain or formatted clipboard text into an active type layer.
- Creates new editable type when pasted with the `Type Tool` active on the canvas.

**Where You Find It**
- `Edit > Copy`, `Edit > Paste`, `Edit > Cut`.
- Keyboard: `Cmd+C`, `Cmd+V`, `Cmd+X` on macOS; `Ctrl+C`, `Ctrl+V`, `Ctrl+X` on Windows.
- `Type Tool` editing mode.

**Primary Interface Elements**
- Highlighted text range.
- Text insertion cursor.
- Active type layer or newly created type layer.

**How To Use It**
1. Activate the `Type Tool` and select the text range to copy, or copy text from another app.
2. Use the copy command or keyboard shortcut.
3. Click inside an existing type layer, or click/drag on the canvas to make a new text area.
4. Paste the clipboard content.
5. Adjust formatting if Photoshop receives the pasted text with different font attributes.

**Expected Result**
- Clipboard text appears in the selected insertion point or new type layer.
