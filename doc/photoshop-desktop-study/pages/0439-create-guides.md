# Create Guides

- Entry Type: Guide creation workflow
- Category: Grids, guides, and measurement
- Source: https://helpx.adobe.com/photoshop/desktop/grids-guides-measurement/create-guides.html

**Function Summary**
Creating guides adds non-printing horizontal or vertical alignment lines to the document.

**Where You Find It**
- Drag from rulers after enabling `View > Rulers`.
- `View > Guides > New Guide`.
- `View > Guides > New Guide Layout` where available.

**Primary Interface Elements**
- Rulers.
- Guide line.
- New Guide dialog with orientation and position.
- Guide Layout rows, columns, gutters, and margins.

**How To Use It**
1. Show rulers with `Cmd+R` or `Ctrl+R`.
2. Drag from the top ruler for a horizontal guide or left ruler for a vertical guide.
3. Or choose `View > Guides > New Guide` and enter an exact position.
4. Use guide layouts for repeated columns, rows, margins, or gutters.

**Expected Result**
- New guides appear on the canvas for alignment.
