# Heal and Clone

- Entry Type: Topic hub
- Category: Repair and retouch
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/heal-clone.html

**Function Summary**
This hub groups tools that repair image areas by copying, blending, or resampling pixels from other areas.

**What This Entry Is For**
- separates cloning and healing from generative removal workflows
- organizes Clone Source, Clone Stamp, Healing Brush, Patch, and related repair tools
- supports precise manual retouching where automatic removal is not enough

**Primary Function Areas**
- clone source management
- Clone Stamp retouching
- healing and patching
- sample-source overlay control

**Expected Result**
- Users can choose a manual repair workflow for controlled pixel replacement.
