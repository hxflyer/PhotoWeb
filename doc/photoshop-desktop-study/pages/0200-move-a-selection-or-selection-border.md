# Move a Selection or Selection Border

- Entry Type: Selection movement workflow
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/move-selection-or-selection-border.html

**Function Summary**
This reference covers moving selected pixels, moving the border only, showing or hiding edges, and inverting a selection.

**What This Function Does**
- moves selected content with the `Move Tool`
- repositions a selection outline without moving pixels
- toggles selection-edge visibility
- inverts selected and unselected areas

**Where You Find It**
- `Move Tool`
- selection tool options bar
- `View > Extras`
- `View > Show > Selection Edges`
- `Select > Inverse`

**Primary Interface Elements**
- `Move Tool`
- `New Selection`
- selection border
- `Selection Edges`
- `Inverse`

**How To Use It**
1. Use the `Move Tool` to drag selected pixels to a new location.
2. Use a selection tool in `New Selection` mode to drag only the selection border.
3. Use `View > Extras` or `View > Show > Selection Edges` to hide or reveal the border display.
4. Use `Select > Inverse` when you need the opposite region selected.

**Mouse, Keyboard, and Menu Input**
- Mouse: drag inside the selection with the `Move Tool` to move pixels.
- Mouse: drag inside the border with a selection tool to move only the border.
- Keyboard: hold `Shift` while dragging to constrain movement.
- Keyboard: arrow keys nudge by `1` pixel; `Shift` plus arrow keys nudge by `10`.

**User Interface Behavior**
- Moving the border alone changes which pixels are targeted without relocating the image content.
- Hiding selection edges does not cancel the selection; it only hides the visible outline.

**Expected Result**
- The selected content, or just its boundary, moves or updates according to the chosen command.
