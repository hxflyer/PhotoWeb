# Convert Layer Styles to Image Layers

- Entry Type: Style-to-layer conversion workflow
- Category: Create and manage layers > Apply layer effects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/apply-layer-effects/convert-layer-styles-to-image-layers.html

**Function Summary**
This workflow converts style effects into ordinary image layers so each effect can be edited independently with standard layer tools.

**What This Function Does**
- Breaks a layer style into separate pixel-based effect layers
- Lets users paint on, filter, or otherwise edit the former effects directly
- Removes the live link between the effect and the original layer content

**Where You Find It**
- Layers panel
- `Layer > Layer Style > Create Layers`

**Primary Interface Elements**
- styled layer
- `Create Layers`
- generated effect layers
- grouped result around the original layer

**How To Use It**
1. Select the layer that contains the style.
2. Choose `Layer > Layer Style > Create Layers`.
3. Edit the resulting effect layers like normal image layers.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the layer in the panel and then work with the generated layers after conversion.
- `Menus`
  Use `Layer > Layer Style > Create Layers`.
- `Keyboard`
  No dedicated default hotkey is described for this conversion.

**User Interface Behavior**
- Photoshop generates separate effect layers and groups them with the original layer.
- The new effect layers no longer update automatically if the source layer content changes.

**Expected Result**
- Former live layer styles become ordinary editable layers for deeper custom editing.
