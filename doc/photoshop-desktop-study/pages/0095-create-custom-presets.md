# Create Custom Presets

- Entry Type: Preset authoring and sharing workflow
- Category: Create and manage layers > Color adjustment and fill layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/color-adjustment-fill-layers/create-custom-presets.html

**Function Summary**
This workflow saves user-made adjustment configurations as reusable presets. It also covers preset organization, preview, renaming, export, and import.

**What This Function Does**
- Saves a selected adjustment layer as a custom preset
- Organizes presets into groups
- Supports preview, rename, delete, export, and import actions

**Where You Find It**
- Layers panel
- `Adjustments` panel
- `Presets` section and menu

**Primary Interface Elements**
- selected adjustment layer
- `Presets`
- `Save`
- preset group selector
- `More`
- `Import Presets`
- `.psap` export file

**How To Create a Custom Preset**
1. Add the adjustment you want to save.
2. Select that adjustment layer in the Layers panel.
3. In the `Adjustments` panel, use the add control in the `Presets` area and choose `Save`.
4. Choose the preset group where it should be stored.

**How To Manage Presets**
1. Hover over a preset to preview it on the canvas.
2. Rename or delete the preset in list or thumbnail view as needed.

**How To Export or Import**
1. Open the Presets menu.
2. Export a single preset, a group, or all custom presets.
3. Save the export as a `.psap` file.
4. Use `More > Import Presets` to load presets back in.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Save presets, choose groups, preview them on hover, and access export or import from the presets menu.
- `Menus`
  Preset management is centered on the Adjustments panel flyout and Presets interface.
- `Keyboard`
  No dedicated default hotkey is described for custom preset management.

**User Interface Behavior**
- Saved presets appear in the Presets area as thumbnails or list entries.
- Exported presets can be moved between systems or shared with others.
- Preset grouping helps keep repeated workflow looks organized.

**Expected Result**
- Reusable adjustment recipes become available for faster consistent editing.
