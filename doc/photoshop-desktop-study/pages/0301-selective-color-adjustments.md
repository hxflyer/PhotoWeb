# Selective Color Adjustments

- Entry Type: Topic hub
- Category: Adjust color > Color corrections
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/selective-color-adjustments.html

**Function Summary**
This hub groups color workflows that target only specific colors, objects, layers, or source-target relationships.

**Primary Function Areas**
- Match Color between images
- Match Color between layers
- saved Match Color statistics
- object color replacement with Hue/Saturation

**Expected Result**
- Users can choose a targeted color adjustment workflow instead of applying a global correction.
