# Trace Images with the Content-Aware Tracing Tool

- Entry Type: Technology Preview tracing workflow
- Category: Draw shapes and paths > Draw lines and curves
- Source: https://helpx.adobe.com/photoshop/desktop/draw-shapes-paths/draw-lines-curves/trace-images-easily-with-the-content-aware-tracing-tool.html

**Function Summary**
The `Content-Aware Tracing Tool` detects object edges and creates vector paths or selections from them.

**Where You Find It**
- Pen tool group after enabling the Technology Preview

**Primary Interface Elements**
- `Content-Aware Tracing`
- `Detail`
- create path option
- extend path option
- trim traced paths option

**How To Use It**
1. Enable the Technology Preview if required and restart Photoshop.
2. Select `Content-Aware Tracing`.
3. Hover near an object edge.
4. Adjust `Detail`.
5. Click to create or extend a traced path.

**Expected Result**
- Photoshop creates a vector path following the detected edge.
