# Use OpenType Variable Fonts

- Entry Type: Variable font workflow
- Category: Text and typography > Select and manage fonts
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/select-manage-fonts/use-opentype-variable-fonts.html

**Function Summary**
Variable fonts expose adjustable design axes, allowing one font file to produce many weights, widths, slants, or optical styles.

**What This Function Does**
- Adjusts font variation axes instead of switching only between fixed styles.
- Enables fine typography tuning for responsive layouts, logos, headings, and brand systems.
- Keeps the type layer editable with live variation settings.

**Where You Find It**
- Font family menu when a variable font is selected.
- `Properties` panel and `Character` panel variable font controls.

**Primary Interface Elements**
- Variable font icon or variable font label.
- Axis sliders such as weight, width, slant, optical size, or custom font-specific axes.
- Numeric axis values where available.

**How To Use It**
1. Select a type layer or highlighted text range.
2. Choose a variable font from the font menu.
3. Open the variable font controls in `Properties` or `Character`.
4. Drag axis sliders or enter values to tune the font.
5. Review spacing and line breaks after the variation changes.

**Expected Result**
- The selected text changes along the chosen font variation axes while remaining editable.
