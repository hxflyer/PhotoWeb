# Display or Hide Layer Styles

- Entry Type: Layer-style panel visibility control
- Category: Create and manage layers > Apply layer effects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/apply-layer-effects/display-or-hide-layer-styles.html

**Function Summary**
This workflow expands or collapses the style listing under a layer in the Layers panel. It changes panel visibility only and does not remove or disable the style effects themselves.

**What This Function Does**
- Shows or hides the list of style effects under a layer
- Makes the Layers panel easier to scan in complex documents
- Lets users adjust layer thumbnail display size for clearer panel reading

**Where You Find It**
- Layers panel
- layer expand/collapse triangle
- panel menu `Panel Options`

**Primary Interface Elements**
- layer style icon
- expand/collapse triangle
- `Panel Options`
- thumbnail size controls

**How To Use It**
1. Select a layer that has styles applied.
2. Click the small triangle near the layer name to expand or collapse the style list.
3. Open the panel menu and choose `Panel Options` if you want to change thumbnail size for easier reading.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Click the expand/collapse triangle and change panel display options from the menu.
- `Menus`
  Use the Layers panel menu for `Panel Options`.
- `Keyboard`
  No dedicated default hotkey is described for style-list visibility.

**User Interface Behavior**
- Expanding or collapsing affects only the display of the style rows in the Layers panel.
- The visual appearance of the layer on the canvas does not change.

**Expected Result**
- The Layers panel becomes easier to manage without altering the actual effects applied to the layer.
