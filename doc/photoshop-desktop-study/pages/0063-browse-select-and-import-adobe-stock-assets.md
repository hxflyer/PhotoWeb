# Browse, Select, and Import Adobe Stock Assets

- Entry Type: Asset import workflow
- Category: Create, open, and import images > Import files
- Source: https://helpx.adobe.com/photoshop/desktop/create-open-import-images/import-files/browse-select-and-import-adobe-stock-assets.html

**Function Summary**
This workflow opens the Adobe Stock browsing surface from within Photoshop and places selected free stock assets into the current document. It is designed to speed up mockups, compositing, and layout work without leaving the app.

**What This Function Does**
- Opens an in-app Adobe Stock browsing panel
- Lets users select free stock images during document creation or editing
- Places imported stock content into the Photoshop document
- Supports Smart Object placement for safer scaling and editing

**Where You Find It**
- `File > New`
- `File > Open`
- `Add free stock Images` in the Contextual Task Bar
- `Preferences > General` for Smart Object placement behavior

**Primary Interface Elements**
- `Add free stock Images`
- Contextual Task Bar
- `Browse Free Adobe Stock Images` panel
- stock category browser
- selected stock asset thumbnail
- `Always Create Smart Objects When Placing`

**How To Use It**
1. Create a new blank document with `File > New`, or open an existing image with `File > Open`.
2. Select `Add free stock Images` from the Contextual Task Bar.
3. Browse categories in the Adobe Stock panel.
4. Select the image you want to place into the document.
5. If needed, enable `Always Create Smart Objects When Placing` in `Preferences > General` so the imported asset stays non-destructive.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Browse categories, click asset thumbnails, and place the chosen stock image.
- `Menus`
  Start from `File > New` or `File > Open`, and use `Preferences > General` for Smart Object placement behavior.
- `Keyboard`
  No dedicated default hotkey is described for the Stock import panel.

**User Interface Behavior**
- The stock browser appears as an in-app panel rather than a separate standalone workflow.
- Imported assets can be placed as Smart Objects when the preference is enabled, preserving editability during scaling.
- Availability and free-download limits can vary depending on subscription status.

**Expected Result**
- A selected stock image is placed into the Photoshop document quickly without leaving the application.

**Related Assets**

- Add placeholder imagery for layouts and comps.
- Pull in quick background or concept assets during composition work.
- Test multiple stock options inside the same document before committing.
