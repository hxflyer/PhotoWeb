# Apply Content-Aware Fill While Cropping Images

- Entry Type: Smart crop-fill workflow
- Category: Crop, resize, and transform > Crop and straighten
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/crop-straighten/apply-content-aware-fill-while-cropping-images.html

**Function Summary**
This crop mode fills newly exposed empty areas automatically when you rotate, straighten, or expand an image during cropping.

**What This Function Does**
- combines crop changes with automatic fill generation
- reduces manual cleanup after canvas expansion
- works well for edge extensions caused by straightening or reframing

**Where You Find It**
- `Crop Tool` options bar

**Primary Interface Elements**
- `Crop Tool`
- `Fill`
- `Content-Aware Fill`
- `Use Classic Mode`
- crop handles

**How To Use It**
1. Select the `Crop Tool`.
2. Make sure `Use Classic Mode` is off.
3. In the `Fill` menu, choose `Content-Aware Fill`.
4. Expand, rotate, or straighten the crop area as needed.
5. Commit the crop with `Enter` or `Return`.

**Mouse, Keyboard, and Menu Input**
- Mouse: drag crop handles or rotate the crop frame.
- Keyboard: `Enter` on Windows or `Return` on macOS to commit.

**User Interface Behavior**
- Photoshop analyzes surrounding pixels and fills the blank areas created by the crop change.
- Results depend on surrounding texture and image complexity.

**Expected Result**
- Empty border areas created during crop adjustments are filled automatically instead of left transparent or blank.
