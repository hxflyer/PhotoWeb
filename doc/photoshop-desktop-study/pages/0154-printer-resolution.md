# Printer Resolution

- Entry Type: Print-output reference
- Category: Crop, resize, and transform > Resize and adjust resolution
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/resize-adjust-resolution/printer-resolution.html

**Function Summary**
This reference explains the relationship between printer `dpi`, image `ppi`, and halftone screen frequency.

**What This Entry Is For**
- Separates printer hardware resolution from image resolution
- Explains how print sharpness is influenced by output device behavior
- Gives a practical rule for matching image resolution to print conditions

**Primary Concepts**
- `dpi`: printer dot density
- `ppi`: image pixel density
- `lpi`: screen frequency used in halftone printing

**How To Use This Information**
1. Check your output device or print provider requirements first.
2. Set image resolution high enough for the target print method, but avoid extreme oversizing.
3. For halftone workflows, aim for image resolution around `1.5x` to `2x` the screen frequency.

**Mouse, Keyboard, and Menu Input**
- Related workflow: `Image > Image Size` for document resolution changes.

**User Interface Behavior**
- Photoshop lets you set image resolution, but final print behavior also depends on the printer driver and physical output method.

**Expected Result**
- You can choose resolution settings that support sharp output without unnecessarily large files.
