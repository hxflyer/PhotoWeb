# Common Questions on Photoshop Cloud Documents

- Entry Type: Cloud document FAQ reference
- Category: Save and export > Save files
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/save-files/common-questions-photoshop-cloud-documents.html

**Function Summary**
Photoshop cloud documents save editable Photoshop work to Adobe cloud storage for access across supported devices and collaboration workflows.

**Where You Find It**
- Home screen cloud documents area.
- Save dialog destination options.
- `File > Save As` cloud/local choices.

**Primary Interface Elements**
- Cloud document thumbnails.
- Sync status.
- Version and sharing controls where available.
- Cloud/local save destination choices.

**Expected Result**
- Users understand when a file is saved as a cloud document and how cloud storage differs from local PSD files.
