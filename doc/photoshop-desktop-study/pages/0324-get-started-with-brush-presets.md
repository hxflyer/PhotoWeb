# Get Started with Brush Presets

- Entry Type: Brush preset overview
- Category: Apply painting techniques > Brushes and presets
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/brushes-presets/get-started-brush-presets.html

**Function Summary**
Brush presets store brush tip, size, behavior, and related settings so a brush can be reused quickly.

**Where You Find It**
- brush preset picker
- `Brushes` panel
- `Brush Settings` panel

**Expected Result**
- Users can select and reuse predefined brushes instead of configuring brush behavior each time.
