# Select Pixels Using Anti-Aliasing

- Entry Type: Anti-alias workflow
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/select-pixels-using-anti-aliasing.html

**Function Summary**
`Anti-alias` smooths the visual edge of a new selection by softening the transition along boundary pixels.

**What This Function Does**
- reduces jagged edge appearance
- improves pasted or composited cutouts
- affects edge quality without creating a wide blur

**Where You Find It**
- options bar for supported selection tools such as `Lasso`

**Primary Interface Elements**
- `Anti-alias`

**How To Use It**
1. Select a supported tool such as `Lasso`.
2. Turn on `Anti-alias` in the options bar.
3. Make the selection normally.

**Mouse, Keyboard, and Menu Input**
- Mouse: click the `Anti-alias` checkbox before drawing the selection.

**Expected Result**
- The selection edge renders with smoother transitions and less stair-stepping.
