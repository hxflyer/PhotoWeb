# Create a Brush Tip from an Image

- Entry Type: Custom brush tip workflow
- Category: Apply painting techniques > Brushes and presets
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/brushes-presets/create-brush-tip-from-image.html

**Function Summary**
This workflow turns selected image pixels into a reusable brush tip.

**Where You Find It**
- `Edit > Define Brush Preset`

**How To Use It**
1. Prepare the brush source image.
2. Select the area to become the brush tip.
3. Choose `Edit > Define Brush Preset`.
4. Name the brush.

**Expected Result**
- The selected image shape becomes available as a brush tip preset.
