# Copy Frames with Layer Properties

- Entry Type: Frame copy workflow
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/copy-frames-layer-properties.html

**Function Summary**
Copying frames duplicates frame states, including visible layers and layer properties, to reuse animation steps.

**Where You Find It**
- Timeline panel menu in Frame Animation mode.

**Primary Interface Elements**
- Selected frame thumbnails.
- Copy Frames and Paste Frames commands.
- Paste method options where available.

**How To Use It**
1. Select one or more frames.
2. Choose `Copy Frames` from the Timeline panel menu.
3. Select the destination position or animation.
4. Choose `Paste Frames` and pick the paste method.

**Expected Result**
- Copied animation frames appear at the destination with their layer states.
