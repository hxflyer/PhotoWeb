# Flatten Frames into Layers

- Entry Type: Animation layer conversion workflow
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/flatten-frames-into-layers.html

**Function Summary**
Flattening frames into layers converts animation frames into separate flattened layers that represent each frame's visible result.

**Where You Find It**
- Timeline panel menu commands for frame-to-layer conversion where available.

**Primary Interface Elements**
- Frame Animation timeline.
- Layers panel.
- Flattened frame layers.

**How To Use It**
1. Open the frame animation.
2. Use the Timeline panel menu command for flattening frames into layers.
3. Review the created layers.
4. Export or edit the resulting frame layers as needed.

**Expected Result**
- Each animation frame becomes a layer representing that frame's flattened appearance.
