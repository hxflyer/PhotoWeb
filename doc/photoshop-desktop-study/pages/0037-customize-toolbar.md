# Customize Toolbar

- Entry Type: Toolbar configuration
- Category: Get started > Set up toolbars and panels
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/set-up-toolbars-panels/customize-the-toolbar.html

**Function Summary**
This feature changes which tools appear in the Photoshop toolbar and how they are grouped. It lets you reorganize the visible tool set, move less-used tools into the extra tools area, and save the layout as a reusable preset.

**What This Function Does**
- Reorders tools and tool groups
- Moves tools into or out of `Extra Tools`
- Saves customized toolbar layouts as presets
- Restores the default toolbar when needed

**Where You Find It**
- `Edit > Toolbar` on Windows
- `Photoshop > Settings > Toolbar` or equivalent toolbar configuration path on macOS, depending on version

**Primary Interface Elements**
- `Customize Toolbar` dialog
- main toolbar column
- `Extra Tools` area
- `Toggle` control
- `Save Preset`
- `Load Preset`
- `Restore Defaults`
- `Clear Tools`
- `Done`
- `Cancel`

**How To Use It**
1. Open the toolbar customization dialog from `Edit > Toolbar`.
2. Drag tools or tool groups to the positions you want in the main toolbar.
3. Move less-used tools into `Extra Tools` if you want the visible toolbar to stay shorter.
4. Use `Toggle` to control access to the extra tools slot in the toolbar.
5. Select `Save Preset` if you want to keep the customized arrangement for later reuse.
6. Select `Load Preset` to apply a saved toolbar setup.
7. Use `Restore Defaults` to return to Adobe's standard toolbar arrangement.
8. Use `Clear Tools` if you want to move all tools into `Extra Tools`.
9. Select `Done` to apply changes or `Cancel` to discard them.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Drag tools and groups inside the customization dialog, then click the preset or confirmation buttons you need.
- `Menus`
  Use `Edit > Toolbar` to open the feature.
- `Keyboard`
  No dedicated default hotkey is described for opening the toolbar customization dialog.

**User Interface Behavior**
- The dialog separates visible toolbar items from hidden or less-used tools.
- Changes do not affect image data; they only change the editing interface.
- Saved presets let multiple toolbar setups be reused across different workflows.
- `Restore Defaults` resets the customized arrangement back to the standard Photoshop tool layout.

**Expected Result**
- The toolbar better matches the tools you use most often.
- The visible interface can become simpler, faster to scan, and more task-specific.

**Practical Use Cases**
- Keep only retouching tools visible in the main toolbar.
- Move rare tools into `Extra Tools` to reduce clutter.
- Save one toolbar preset for teaching and another for advanced production work.
