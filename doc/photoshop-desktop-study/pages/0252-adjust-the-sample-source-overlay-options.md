# Adjust the Sample Source Overlay Options

- Entry Type: Clone Source overlay reference
- Category: Repair and retouch > Heal and clone
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/heal-clone/adjust-the-sample-source-overlay-options.html

**Function Summary**
The overlay options preview the sampled source over the target area so cloning or healing can be aligned before painting.

**What This Function Does**
- shows or hides a source overlay
- changes overlay opacity and blending
- clips the overlay to brush size or shows it more broadly
- supports easier visual alignment

**Where You Find It**
- `Window > Clone Source`

**Primary Interface Elements**
- `Show Overlay`
- `Auto Hide`
- `Clipped`
- `Opacity`
- `Normal`
- `Darken`
- `Lighten`
- `Difference`
- `Invert`

**How To Use It**
1. Set a clone or healing sample source.
2. Open `Window > Clone Source`.
3. Enable `Show Overlay`.
4. Adjust opacity, clipping, blending, and invert settings until alignment is easy to judge.

**Expected Result**
- The sampled source becomes visible as a guide before and during retouching.
