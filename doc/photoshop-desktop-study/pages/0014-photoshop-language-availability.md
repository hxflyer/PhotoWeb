# Photoshop Language Availability

- Entry Type: Localization reference
- Category: Get started > Technical requirements and installation
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/technical-requirements-installation/photoshop-language-availability.html

**Function Summary**
This page lists supported Photoshop desktop language availability and notes any regional language packaging differences. It is a deployment and localization reference rather than a Photoshop editing feature.

**What This Entry Is For**
- checking whether Photoshop is available in a target language
- understanding regional language packaging differences
- confirming support for right-to-left language environments

**Special Regional Notes**
- Arabic and Hebrew support is provided through specific regional editions rather than through one universal package.
- These regional versions are intended to handle right-to-left text and associated language behavior correctly.

**How To Use This Entry**
- Check this page before deploying Photoshop in a multilingual or region-specific environment.
- Use it when deciding which regional build or language variant is appropriate for a user or team.

**Expected Result**
- You can confirm whether the language you need is supported and whether it requires a special regional Photoshop variant.
