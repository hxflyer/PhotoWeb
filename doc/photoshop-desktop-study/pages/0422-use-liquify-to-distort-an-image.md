# Use Liquify to Distort an Image

- Entry Type: Liquify workflow
- Category: Effects and filters > Distortion
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/distortion/use-liquify-distort-image.html

**Function Summary**
Using Liquify distorts image areas interactively in a preview workspace with brush tools and optional masks.

**Where You Find It**
- `Filter > Liquify`.

**Primary Interface Elements**
- Liquify preview.
- Distortion tools and brush controls.
- Freeze and Thaw mask tools.
- OK, Cancel, and Load/Save Mesh controls.

**How To Use It**
1. Convert the layer to a Smart Object if you want editable Liquify settings.
2. Choose `Filter > Liquify`.
3. Pick a distortion tool and set brush size and pressure.
4. Drag on the preview to reshape the image.
5. Freeze areas that should not move.
6. Click `OK`.

**Expected Result**
- The image is reshaped according to Liquify brush strokes.
