# Selection Tools Overview

- Entry Type: Tool family reference
- Category: Make selections > Get started with selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/get-started-selections/selection-tools-overview.html

**Function Summary**
This reference lists the main Photoshop tools used to create pixel selections and the type of selection each one is best suited for.

**What This Entry Is For**
- compares the core selection tools at a glance
- helps users choose between shape-based, edge-based, and automatic tools
- acts as a lookup entry rather than a single action

**Primary Tool Groups**
- `Rectangular Marquee`
- `Elliptical Marquee`
- `Single Row Marquee`
- `Single Column Marquee`
- `Lasso`
- `Polygonal Lasso`
- `Magnetic Lasso`
- `Quick Selection`
- `Magic Wand`
- `Object Selection`

**How To Use This Information**
1. Choose a marquee tool for geometric regions.
2. Choose lasso tools for hand-drawn or edge-traced regions.
3. Choose `Quick Selection`, `Magic Wand`, or `Object Selection` when Photoshop can infer boundaries from image content.

**Expected Result**
- Users can identify the fastest selection method before starting detailed editing.
