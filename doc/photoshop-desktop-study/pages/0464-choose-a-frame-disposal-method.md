# Choose a Frame Disposal Method

- Entry Type: GIF frame behavior workflow
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/choose-frame-disposal-method.html

**Function Summary**
Frame disposal controls how a frame is cleared or preserved before the next frame appears, which affects GIF-style animation playback.

**Where You Find It**
- Timeline panel frame options where disposal controls are available.

**Primary Interface Elements**
- Disposal method icon or menu.
- Frame thumbnails.
- Preview playback.

**How To Use It**
1. Select the frame whose disposal behavior matters.
2. Open its disposal method option.
3. Choose whether the frame should preserve, clear, or use automatic behavior.
4. Preview the animation to check trails, flicker, or transparency artifacts.

**Expected Result**
- Frame-to-frame cleanup behaves correctly during exported animation playback.
