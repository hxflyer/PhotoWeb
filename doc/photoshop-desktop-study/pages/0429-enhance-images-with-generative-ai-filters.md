# Enhance Images with Generative AI Filters

- Entry Type: Generative filter workflow
- Category: Effects and filters
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/enhance-images-generative-ai-filters.html

**Function Summary**
Generative AI filter workflows use Adobe AI services to create or enhance image results from context, prompts, or filter-specific controls.

**Where You Find It**
- AI-enabled filter or contextual workflows in supported Photoshop versions.
- `Filter` menu or contextual task controls where available.

**Primary Interface Elements**
- Prompt field or filter settings.
- Generate button.
- Variation thumbnails.
- Generated layer or non-destructive output options.

**How To Use It**
1. Select the image layer or target area.
2. Open the supported AI filter workflow.
3. Enter a prompt or choose filter settings if the feature provides them.
4. Generate variations.
5. Choose the best result and refine with masks, layers, or additional edits.

**Expected Result**
- Photoshop creates AI-assisted image enhancements or generated variations based on the selected workflow.
