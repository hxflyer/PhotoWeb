# Change Pixels to Transparent with the Background Eraser Tool

- Entry Type: Background Eraser workflow
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/change-pixels-to-transparent-with-the-background-eraser-tool.html

**Function Summary**
The `Background Eraser Tool` erases sampled colors to transparency while trying to preserve foreground edges.

**Primary Interface Elements**
- `Continuous`
- `Once`
- `Background Swatch`
- `Limits`
- `Discontiguous`
- `Contiguous`
- `Find Edges`
- `Tolerance`
- `Protect Foreground Color`

**How To Use It**
1. Select the `Background Eraser Tool`.
2. Set brush size and sampling mode.
3. Choose limits and tolerance.
4. Enable `Protect Foreground Color` if needed.
5. Drag with the brush center over the color to erase.

**Expected Result**
- Matching background pixels become transparent while protected or nonmatching areas remain.
