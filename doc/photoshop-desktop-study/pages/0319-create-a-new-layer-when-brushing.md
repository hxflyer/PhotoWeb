# Create a New Layer When Brushing

- Entry Type: Non-destructive painting workflow
- Category: Apply painting techniques > Fill objects, selections, and layers
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/fill-objects-selections-layers/create-a-new-layer-when-brushing.html

**Function Summary**
Painting on a new layer keeps brush strokes separate from the original image pixels.

**Primary Interface Elements**
- `Layers` panel
- new layer button
- blend mode
- opacity
- brush tool

**How To Use It**
1. Create a new layer above the image.
2. Select a brush-based tool.
3. Paint on the new layer.
4. Adjust layer opacity or blend mode if needed.

**Expected Result**
- Brush strokes remain editable independently from the underlying image.
