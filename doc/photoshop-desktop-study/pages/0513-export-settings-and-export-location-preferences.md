# Export Settings and Export Location Preferences

- Entry Type: Export preference workflow
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/export-settings-export-location-preferences.html

**Function Summary**
Export preferences define the default Quick Export format, metadata behavior, color conversion, and destination handling.

**Where You Find It**
- macOS: `Photoshop > Settings > Export`.
- Windows: `Edit > Preferences > Export`.

**Primary Interface Elements**
- Quick Export format.
- Export location behavior.
- Metadata option.
- Convert to sRGB option.

**How To Use It**
1. Open Export preferences.
2. Choose the default Quick Export format.
3. Set export location behavior.
4. Configure metadata and sRGB conversion choices.
5. Use Quick Export or Export As with the updated defaults.

**Expected Result**
- Export commands use the configured default behavior.
