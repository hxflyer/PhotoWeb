# Edit the Contents of a Smart Object

- Entry Type: Smart Object source-edit workflow
- Category: Create and manage layers > Smart objects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/smart-objects/edit-the-contents-of-a-smart-object.html

**Function Summary**
This workflow opens the source content behind a Smart Object so it can be edited directly, then saved back into the document and any linked instances.

**What This Function Does**
- Opens Smart Object source content in the correct editor
- Lets users modify the underlying source without destroying Smart Object flexibility
- Updates the Smart Object in Photoshop after the source is saved

**Where You Find It**
- `Window > Layers`
- `Layer > Smart Objects > Edit Contents`

**Primary Interface Elements**
- selected Smart Object layer
- `Edit Contents`
- source application window
- confirmation dialog when present

**How To Use It**
1. Open `Window > Layers`.
2. Select the Smart Object layer.
3. Choose `Layer > Smart Objects > Edit Contents`.
4. Confirm with `OK` if Photoshop prompts you.
5. Edit the source file in the application that opens.
6. Save the source file.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the Smart Object layer and confirm any prompt before editing the source.
- `Menus`
  Use `Layer > Smart Objects > Edit Contents`.
- `Keyboard`
  Standard save shortcuts in the source application complete the update workflow.

**User Interface Behavior**
- Photoshop opens the source in the application best matched to the file format, such as Photoshop for PSD or Illustrator for AI.
- Saving the source updates the Smart Object in the document and any other linked instances that share it.

**Expected Result**
- The Smart Object's displayed content updates after the edited source is saved, while Smart Object flexibility remains intact.
