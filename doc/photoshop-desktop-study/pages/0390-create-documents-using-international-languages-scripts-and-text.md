# Create Documents Using International Languages, Scripts, and Text

- Entry Type: International text setup workflow
- Category: Text and typography > International text and languages
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/international-text-languages/create-documents-using-international-languages-scripts-and-text.html

**Function Summary**
International document setup prepares type layers for the language, script, font coverage, direction, and shaping behavior required by multilingual text.

**What This Function Does**
- Uses system input methods and Photoshop type tools to enter supported scripts.
- Applies fonts that include the needed glyphs.
- Sets language, direction, and paragraph behavior for correct shaping and reading order.

**Where You Find It**
- `Type Tool`, shortcut `T`.
- `Window > Character` and `Window > Paragraph`.
- Operating system keyboard/input method settings.

**Primary Interface Elements**
- Language menu.
- Font family menu.
- Paragraph direction controls.
- Composer and alignment settings.
- Text insertion cursor and paragraph text box.

**How To Use It**
1. Enable the needed keyboard or input method in the operating system.
2. Create or open a Photoshop document.
3. Select a font that supports the target script.
4. Enter text with the `Type Tool`.
5. Set language, direction, alignment, and paragraph behavior as needed.
6. Inspect shaping, punctuation, line breaks, and missing glyphs.

**Expected Result**
- The document contains editable international text that renders with proper script support.
