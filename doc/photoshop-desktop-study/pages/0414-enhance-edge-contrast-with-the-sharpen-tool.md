# Enhance Edge Contrast with the Sharpen Tool

- Entry Type: Local sharpen tool workflow
- Category: Effects and filters > Blur and sharpen
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/blur-sharpen/enhance-edge-contrast-sharpen-tool.html

**Function Summary**
The `Sharpen Tool` paints localized edge contrast onto image areas using brush-based sharpening strength.

**Where You Find It**
- Toolbar: `Sharpen Tool`, grouped with Blur and Smudge.

**Primary Interface Elements**
- Brush size, hardness, and strength.
- Protect Detail option where available.
- Brush cursor on canvas.

**How To Use It**
1. Select the `Sharpen Tool`.
2. Set a low strength for controlled sharpening.
3. Paint over edges or details that need more clarity.
4. Avoid repeated strokes over noise, skin, skies, or smooth gradients.

**Expected Result**
- Painted areas appear crisper with stronger edge contrast.
