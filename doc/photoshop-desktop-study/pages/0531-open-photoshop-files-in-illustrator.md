# Open Photoshop Files in Illustrator

- Entry Type: Illustrator handoff workflow
- Category: Share and collaborate > App integrations
- Source: https://helpx.adobe.com/photoshop/desktop/share-collaborate/app-integrations/open-photoshop-files-illustrator.html

**Function Summary**
Opening Photoshop files in Illustrator moves PSD artwork into Illustrator for vector layout, design, or illustration workflows.

**Where You Find It**
- Illustrator `File > Open` or Photoshop/Adobe integration handoff controls where available.

**Primary Interface Elements**
- PSD import/open options.
- Layer conversion or flattening choices.
- Illustrator layers and linked/embedded artwork.

**How To Use It**
1. Save the Photoshop document.
2. Open the file from Illustrator or use the available handoff command.
3. Choose import options for layers, text, or flattened appearance.
4. Review the file in Illustrator for layer and appearance changes.

**Expected Result**
- Photoshop artwork opens in Illustrator with available layer or appearance preservation based on import options.
