# Reposition Start and End Points

- Entry Type: Dynamic text control workflow
- Category: Text and typography > Work with dynamic text
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/work-with-dynamic-text/reposition-start-and-end-points.html

**Function Summary**
Start and end point controls define where dynamic/path-based text begins and ends within its editable structure.

**What This Function Does**
- Changes the visible range or placement of text along a path or dynamic text control line.
- Repositions the start marker, end marker, or text span without rewriting the characters.
- Helps fit text cleanly around curves, shapes, or controlled text regions.

**Where You Find It**
- On-canvas start and end controls for supported dynamic text.
- `Path Selection Tool` or related text control tool.
- Active type layer or dynamic text object.

**Primary Interface Elements**
- Start marker.
- End marker.
- Text baseline or path.
- Direction/flip indicators where applicable.

**How To Use It**
1. Select the dynamic text object or path-based type layer.
2. Locate the start and end controls on canvas.
3. Drag the start marker to change where the text begins.
4. Drag the end marker to change where the text stops or wraps.
5. Adjust formatting or path shape if the text becomes cramped.

**Expected Result**
- Dynamic text begins and ends at the adjusted positions while remaining editable.
