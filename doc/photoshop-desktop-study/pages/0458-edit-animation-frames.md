# Edit Animation Frames

- Entry Type: Frame editing workflow
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/edit-animation-frames.html

**Function Summary**
Editing animation frames changes the layer state, timing, order, or contents displayed by selected frames.

**Where You Find It**
- Timeline panel in Frame Animation mode.
- `Layers` panel for frame-specific layer visibility and properties.

**Primary Interface Elements**
- Frame thumbnails.
- Layer visibility eyes.
- Delay menus.
- Duplicate, trash, and tween controls.

**How To Use It**
1. Select a frame in the Timeline panel.
2. Adjust layer visibility, position, opacity, or effects for that frame.
3. Change delay time if needed.
4. Duplicate, delete, or reorder frames to refine motion.
5. Preview playback.

**Expected Result**
- The selected frame displays the updated layer state and timing.
