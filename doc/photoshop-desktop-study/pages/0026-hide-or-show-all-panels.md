# Hide or Show All Panels

- Entry Type: Workspace visibility control
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/hide-show-panels.html

**Function Summary**
This visibility shortcut temporarily hides interface panels so the canvas has more space. It can hide the entire panel interface or hide most panels while keeping the Tools panel and Options bar visible.

**What This Function Does**
- Toggles panel visibility across the Photoshop workspace
- Maximizes canvas space for editing and review
- Provides a faster alternative to manually closing panels
- Supports two visibility modes: full hide or partial hide

**Where You Use It**
- During editing when you want a larger unobstructed canvas
- While reviewing composition, retouching detail, or presenting an image

**Primary Interface Elements**
- panels
- Tools panel
- Options bar
- workspace edge hover area
- `Auto-Show Hidden Panels` preference

**How To Use It**
1. Press `Tab` to hide all panels, including the Tools panel and the Options bar.
2. Press `Tab` again to restore all hidden interface panels.
3. Press `Shift + Tab` to hide panels while leaving the Tools panel and Options bar visible.
4. Press `Shift + Tab` again to restore the hidden panels.
5. If `Auto-Show Hidden Panels` is enabled, move the pointer to the edge of the application window or monitor to reveal panels temporarily.

**Mouse, Keyboard, and Menu Input**
- `Keyboard`
  Use `Tab` for a full interface toggle and `Shift + Tab` for a partial panel toggle.
- `Mouse`
  With `Auto-Show Hidden Panels` enabled, hover near the screen edge to reveal hidden panels temporarily.
- `Menus`
  The hover behavior depends on the `Preferences > Workspace` setting for `Auto-Show Hidden Panels`.

**User Interface Behavior**
- `Tab` affects the full panel set, including the main tools and options controls.
- `Shift + Tab` keeps core tool-selection controls available while hiding most support panels.
- Hover reveal only works when the relevant workspace preference is enabled.
- Hidden panels return without losing their arrangement.

**Expected Result**
- The image canvas becomes less cluttered.
- You can inspect artwork with fewer distractions and restore the full interface instantly when needed.

**Practical Use Cases**
- Check composition and edges without panel interference.
- Paint or retouch in a less crowded workspace.
- Keep only the essential tools visible during focused editing.
