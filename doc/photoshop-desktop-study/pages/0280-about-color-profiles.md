# About Color Profiles

- Entry Type: Color management reference
- Category: Adjust color > Color profiles
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/color-profiles/about-color-profiles.html

**Function Summary**
Color profiles describe how devices and documents reproduce color so Photoshop can interpret color values consistently.

**What This Entry Is For**
- explains monitor, input, output, and document profiles
- clarifies why raw RGB or CMYK numbers need profile context
- helps prevent accidental color shifts between devices

**Primary Profile Types**
- monitor profiles
- input device profiles
- output device profiles
- document profiles

**Expected Result**
- Users understand why profile accuracy affects screen preview, print preparation, and cross-application color consistency.
