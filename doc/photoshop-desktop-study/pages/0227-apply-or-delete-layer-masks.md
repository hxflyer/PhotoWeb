# Apply or Delete Layer Masks

- Entry Type: Mask finalization workflow
- Category: Create masks > Layer masks
- Source: https://helpx.adobe.com/photoshop/desktop/create-masks/layer-masks/apply-or-delete-layer-masks.html

**Function Summary**
This workflow either bakes a layer mask's result into the layer or removes the mask without applying it.

**What This Function Does**
- permanently commits a mask effect
- removes a mask while keeping the original layer visible if deleted without applying
- helps move from non-destructive editing to final pixel output when needed

**Where You Find It**
- `Layers` panel controls

**Primary Interface Elements**
- mask thumbnail
- mask icon
- delete icon

**How To Use It**
1. Select the layer with the mask.
2. Apply the mask if you want the hidden areas permanently integrated into the layer result.
3. Delete the mask if you want to remove the mask object instead.

**User Interface Behavior**
- Smart Object restrictions may limit fully permanent mask application behavior in some cases.

**Expected Result**
- The mask is either committed to the layer result or removed from the layer structure.
