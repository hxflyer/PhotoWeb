# Snap to Image Edges Using Magnetic Lasso Tool

- Entry Type: Magnetic Lasso workflow
- Category: Make selections > Freehand selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/freehand-selections/snap-to-image-edges-using-magnetic-lasso-tool.html

**Function Summary**
The `Magnetic Lasso Tool` follows visible contrast edges automatically while you guide it around an object.

**What This Function Does**
- snaps selection borders to nearby edges
- places fastening points automatically
- helps with complex outlines on well-separated subjects

**Where You Find It**
- `Magnetic Lasso Tool` in the toolbar

**Primary Interface Elements**
- `Width`
- `Contrast`
- `Frequency`
- `Use Tablet Pressure`
- fastening points
- selection mode buttons

**How To Use It**
1. Select the `Magnetic Lasso Tool`.
2. Set `Width`, `Contrast`, and `Frequency` in the options bar.
3. Click once to place the starting point.
4. Drag along the object's edge while the tool snaps to visible boundaries.
5. Click manually to add fastening points where needed.
6. Close the selection at the start point or force-close it with a shortcut.

**Mouse, Keyboard, and Menu Input**
- Mouse: drag along the edge and click to add manual fastening points.
- Keyboard: press `Delete` to remove recent segments.
- Keyboard: press `Caps Lock` to display the width-based pointer.
- Keyboard: `]` increases width and `[` decreases width.
- Keyboard: double-click or use `Ctrl`-click on Windows or `Command`-click on macOS to close when not returning exactly to the start.

**User Interface Behavior**
- The selection border seeks nearby contrast transitions inside the specified width zone instead of following raw pointer movement exactly.

**Expected Result**
- A selection hugs a visible edge more accurately than a fully manual lasso on suitable images.
