# Supported File Formats in Photoshop

- Entry Type: File format support reference
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/supported-file-formats-photoshop.html

**Function Summary**
Supported file formats define what Photoshop can open, import, save, export, or place.

**Where You Find It**
- `File > Open`, `File > Save As`, `File > Save a Copy`, `File > Export`, and `File > Place Embedded/Linked`.

**Primary Interface Elements**
- Format menus.
- Import/open dialogs.
- Save and export option dialogs.

**Expected Result**
- Users can verify whether a required format is supported for editing, import, or output.
