# Align Layers

- Entry Type: Layer alignment workflow
- Category: Create and manage layers > Create layer compositions
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/create-layer-compositions/align-content-of-layers-and-groups.html

**Function Summary**
This workflow aligns two or more selected layers or groups along a shared edge or center axis. It is used for cleaner layouts, grid work, and consistent UI or graphic placement.

**What This Function Does**
- Aligns selected layers by top, bottom, left, right, or center positions
- Works with both layers and groups
- Speeds up precise layout construction

**Where You Find It**
- Layers panel
- `Layer > Align`

**Primary Interface Elements**
- selected layers or groups
- `Top Edges`
- `Vertical Centers`
- `Bottom Edges`
- `Left Edges`
- `Horizontal Centers`
- `Right Edges`

**How To Use It**
1. Select two or more layers or groups in the Layers panel.
2. Choose `Layer > Align`.
3. Select the alignment option you want.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Multi-select the layers and choose the alignment command from the menu.
- `Menus`
  Use `Layer > Align`.
- `Keyboard`
  No dedicated default hotkey is described for these alignment commands.

**User Interface Behavior**
- Photoshop uses the current selection bounds or group bounds as the alignment reference.
- The content moves to match the chosen edge or center rule.

**Expected Result**
- Selected layers line up cleanly according to the chosen alignment rule.
