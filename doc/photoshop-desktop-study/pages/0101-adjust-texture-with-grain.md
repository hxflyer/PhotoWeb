# Adjust Texture with Grain

- Entry Type: Texture adjustment workflow
- Category: Create and manage layers > Color adjustment and fill layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/color-adjustment-fill-layers/enhance-texture-with-grain.html

**Function Summary**
This adjustment layer adds film-like grain texture non-destructively. It is used to create texture, reduce an overly clean digital look, or unify composite elements visually.

**What This Function Does**
- Adds visible grain texture
- Controls grain strength, particle size, and randomness
- Keeps the effect editable through a dedicated adjustment layer

**Where You Find It**
- Layers panel
- `Create new fill or adjustment layer`
- `Grain`
- `Properties` panel

**Primary Interface Elements**
- `Amount`
- `Size`
- `Roughness`
- `Properties`
- layer name field

**How To Use It**
1. Open the image you want to texture.
2. In the Layers panel, create a new `Grain` adjustment layer.
3. In the `Properties` panel, adjust `Amount`, `Size`, and `Roughness`.
4. Name the layer if needed and confirm with `Enter` or `Return`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Create the adjustment and drag the grain sliders in the Properties panel.
- `Menus`
  The workflow starts from the adjustment-layer creation control in the Layers panel.
- `Keyboard`
  Use `Enter` or `Return` to confirm the layer name.

**User Interface Behavior**
- `Amount` changes how visible the grain is.
- `Size` changes the particle scale.
- `Roughness` changes how smooth or irregular the grain pattern feels.

**Expected Result**
- The image gains editable film-like surface texture without destructive pixel rewriting.
