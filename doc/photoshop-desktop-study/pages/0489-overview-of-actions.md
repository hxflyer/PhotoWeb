# Overview of Actions

- Entry Type: Actions reference
- Category: Automate tasks
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/overview-actions.html

**Function Summary**
Actions are recorded sequences of Photoshop commands that can be played back on one file or used in automated processing.

**Where You Find It**
- `Window > Actions`.

**Primary Interface Elements**
- Action sets.
- Individual actions.
- Recorded command steps.
- Checkboxes for enabling steps.
- Dialog toggles for pausing commands.
- Record, Stop, Play, New, and Delete buttons.

**Expected Result**
- Users understand the Actions panel structure before recording, editing, or playing actions.
