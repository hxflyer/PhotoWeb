# Set Image Size and Resolution

- Entry Type: Image Size dialog workflow
- Category: Crop, resize, and transform > Resize and adjust resolution
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/resize-adjust-resolution/set-image-size-and-resolution.html

**Function Summary**
This workflow uses the `Image Size` dialog to change pixel dimensions, print dimensions, resolution, and resampling behavior.

**What This Function Does**
- Changes width and height
- Changes print resolution
- Controls whether Photoshop resamples the image
- Shows a live preview of the result

**Where You Find It**
- `Image > Image Size`

**Primary Interface Elements**
- `Image Size`
- `Dimensions`
- `Fit To`
- `Width`
- `Height`
- `Resolution`
- `Resample`
- layer-style scaling settings icon

**How To Use It**
1. Choose `Image > Image Size`.
2. Adjust `Width`, `Height`, and `Resolution`.
3. Decide whether `Resample` should be enabled.
4. Use `Fit To` for preset sizing when appropriate.
5. Use the settings icon if you want layer styles to scale proportionally.
6. Review the preview before confirming.

**Expected Result**
- The image's print size, pixel size, or both change according to the selected settings.
