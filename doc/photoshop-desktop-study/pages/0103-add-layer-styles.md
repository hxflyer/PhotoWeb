# Add Layer Styles

- Entry Type: Layer-style application workflow
- Category: Create and manage layers > Apply layer effects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/apply-layer-effects/add-layer-styles.html

**Function Summary**
This workflow applies non-destructive layer styles such as shadows, glows, strokes, and overlays. Layer styles remain attached to the layer and update with its content instead of being painted directly into pixels.

**What This Function Does**
- Adds style effects from the Layers panel
- Adds style effects from the `Layer` menu
- Applies preset styles from the `Styles` panel
- Keeps effects editable through the Layer Style system

**Where You Find It**
- `Window > Layers`
- `Add a layer style` icon
- `Layer > Layer Style`
- `Window > Styles`

**Primary Interface Elements**
- `Add a layer style`
- `Layer Style` dialog
- `Styles` panel
- layer style arrow expander in the Layers panel

**How To Add a Style from the Layers Panel**
1. Open `Window > Layers`.
2. Select the layer you want to style.
3. Click `Add a layer style`.
4. Choose the effect you want.
5. Adjust the settings in the `Layer Style` dialog.
6. Confirm with `OK`.
7. Expand the style listing under the layer if you want to view the attached effects.

**How To Add a Style from the Layer Menu**
1. Select the target layer.
2. Choose `Layer > Layer Style`.
3. Pick the effect.
4. Adjust the settings and confirm with `OK`.

**How To Add a Preset Style**
1. Select the target layer.
2. Open `Window > Styles`.
3. Click a preset style thumbnail.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the layer, choose the effect, adjust the dialog controls, and apply the preset or custom style.
- `Menus`
  Use `Layer > Layer Style` or `Window > Styles`.
- `Keyboard`
  No dedicated default hotkey is described for adding layer styles in this entry.

**User Interface Behavior**
- Styles stay linked to the layer content and move with it.
- Multiple effects can be attached to one layer and expanded in the Layers panel.
- Preset styles are instant combinations of one or more effect settings.

**Expected Result**
- The selected layer gains editable visual effects without destructive painting or raster changes.
