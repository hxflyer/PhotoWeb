# Edit a Gradient

- Entry Type: Gradient editing workflow
- Category: Adjust color > Color effects and techniques
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/color-effects-techniques/edit-a-gradient.html

**Function Summary**
Gradient editing adjusts color stops, transition points, direction, length, and saved presets after a gradient is created.

**Primary Interface Elements**
- on-canvas gradient line
- color stops
- midpoint controls
- `Properties` panel
- `Save Preset`

**How To Use It**
1. Create or select a gradient.
2. Drag the on-canvas gradient line to change direction or length.
3. Select color stops to change colors.
4. Drag stops to adjust transition positions.
5. Click the line to add stops or drag stops away to remove them.
6. Use `Save Preset` to store the result.

**Expected Result**
- The gradient updates interactively and can be saved for future reuse.
