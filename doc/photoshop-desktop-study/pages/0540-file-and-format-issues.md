# File and Format Issues

- Entry Type: File troubleshooting reference
- Category: Troubleshoot
- Source: https://helpx.adobe.com/photoshop/desktop/troubleshoot/file-format-issues.html

**Function Summary**
File and format troubleshooting addresses files that fail to open, save, export, preview, or preserve expected Photoshop features.

**Where You Find It**
- Open/save/export dialogs.
- Error messages.
- File Handling preferences.
- Supported file format references.

**Primary Interface Elements**
- Format menus.
- Compatibility warnings.
- Error dialogs.
- Save options and color profile prompts.

**Expected Result**
- Users can identify whether the problem comes from unsupported format features, corruption, permissions, storage, or save/export settings.
