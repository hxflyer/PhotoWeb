# Manage Layer Visibility in Animation Frames

- Entry Type: Frame visibility workflow
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/manage-layer-visibility-animation-frames.html

**Function Summary**
Layer visibility per frame controls which layers appear in each frame of a frame animation.

**Where You Find It**
- Timeline panel and `Layers` panel in Frame Animation mode.

**Primary Interface Elements**
- Frame thumbnail selection.
- Layer visibility eye icons.
- New layer visibility options where available.

**How To Use It**
1. Select a frame.
2. In the `Layers` panel, show or hide layers for that frame.
3. Repeat for other frames.
4. Use unify options if one layer should be visible or hidden consistently.

**Expected Result**
- Each frame displays the intended combination of layers.
