# Adjustment Presets Overview

- Entry Type: Preset system reference
- Category: Create and manage layers > Color adjustment and fill layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/color-adjustment-fill-layers/adjustment-presets-overview.html

**Function Summary**
Adjustment Presets apply prebuilt combinations of adjustment settings so users can preview and apply looks quickly. They act as starting points for color correction, repair, monochrome treatment, and stylized effects.

**What This Feature Does**
- Shows ready-made adjustment recipes
- Lets users preview effects before applying them
- Creates an effect group in the layer stack for later refinement
- Supports different preset display modes

**Where You Find It**
- `Adjustments` panel
- `Presets` section
- `Properties` panel
- Layers panel effect group created after application

**Primary Interface Elements**
- `Presets`
- `Adjustments`
- `Properties`
- preset categories such as `Portraits`, `Landscape`, `Photo Repair`, `Creative`, `Black & White`, and `Cinematic`
- `List View`
- `Grid View`

**How To Use It**
1. Open the `Adjustments` panel.
2. Browse the preset categories.
3. Hover over a preset thumbnail to preview its effect on the canvas.
4. Click the preset you want to apply.
5. Refine the result afterward in the `Properties` panel.

**Display Options**
- `List View`
  Shows presets in a text-oriented list
- `Grid View`
  Shows presets as visual thumbnails

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Hover to preview, click to apply, and toggle between list and grid views with the panel control.
- `Menus`
  This feature lives primarily inside the Adjustments panel rather than a top-level menu.
- `Keyboard`
  No dedicated default hotkey is described for preset browsing.

**User Interface Behavior**
- Hovering previews the effect name and look on the canvas.
- Applying a preset creates a managed effect group in the Layers panel.
- Presets are intended as starting points and can be edited further after application.

**Expected Result**
- Users can reach polished looks faster while still keeping adjustments editable.
