# Crop Tool Options

- Entry Type: Crop settings reference
- Category: Crop, resize, and transform > Crop and straighten
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/crop-straighten/crop-tool-options.html

**Function Summary**
This reference explains the main settings that control how the `Crop Tool` frames, constrains, previews, and commits crop operations.

**What This Entry Is For**
- explains aspect-ratio presets
- explains width-height-resolution targeting
- explains destructive versus non-destructive crop behavior
- explains preview and shield settings

**Where You Find It**
- options bar while the `Crop Tool` is active

**Primary Interface Elements**
- `Ratio`
- `W x H x Resolution`
- preset ratios such as `1:1`, `4:5`, `2:3`, and `16:9`
- `Delete Cropped Pixels`
- `Auto Center Preview`
- `Show Cropped Area`
- `Enable Crop Shield`

**How To Use It**
1. Activate the `Crop Tool`.
2. Choose a ratio preset or enter a width-height-resolution target.
3. Decide whether `Delete Cropped Pixels` should be on or off.
4. Adjust preview controls such as crop shield or centered preview if they help your composition.
5. Resize and place the crop box, then commit.

**User Interface Behavior**
- Ratio settings constrain how the crop box behaves when dragged.
- `Delete Cropped Pixels` determines whether cropped image data is discarded or preserved off-canvas.

**Expected Result**
- Crop operations follow a predictable framing rule and preview style instead of default freeform behavior.
