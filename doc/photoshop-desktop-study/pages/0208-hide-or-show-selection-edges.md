# Hide or Show Selection Edges

- Entry Type: Selection-visibility command
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/hide-or-show-selection-edges.html

**Function Summary**
This command toggles the visibility of selection outlines without clearing the underlying selection.

**What This Function Does**
- hides marching-ants outlines for a cleaner preview
- reveals them again when continued editing is needed
- supports full extras toggling or selection-edge-only toggling

**Where You Find It**
- `View > Extras`
- `View > Show > Selection Edges`

**How To Use It**
1. Use `View > Extras` if you want to hide or show several overlay categories together.
2. Use `View > Show > Selection Edges` if you only want to toggle the selection outline.

**Mouse, Keyboard, and Menu Input**
- Menu path: `View > Extras`
- Menu path: `View > Show > Selection Edges`

**User Interface Behavior**
- Hiding the edge display does not cancel the selection; the selection remains active until changed or deselected.

**Expected Result**
- The selection border becomes visible or hidden while selection state remains intact.
