# Define a Feathered Edge for a Selection Tool

- Entry Type: Feather workflow
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/define-feathered-edges.html

**Function Summary**
`Feather` softens a selection by creating a blurred transition zone around its edge.

**What This Function Does**
- blurs the border of a selection
- softens fills, copies, deletes, and compositing operations
- can be applied before or after creating a selection

**Where You Find It**
- feather field in supported tool options bars
- `Select > Modify > Feather`

**Primary Interface Elements**
- `Feather`
- `Feather Radius`

**How To Use It**
1. Enter a feather value before selecting if you already know the needed softness.
2. Or make a selection first, then use `Select > Modify > Feather`.
3. Enter the feather radius and confirm.

**Mouse, Keyboard, and Menu Input**
- Menu path: `Select > Modify > Feather`

**User Interface Behavior**
- The feather effect usually becomes obvious after the selected area is moved, copied, filled, or otherwise applied.

**Expected Result**
- The selection transitions more gradually into surrounding pixels instead of ending abruptly.
