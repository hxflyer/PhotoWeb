# Flip or Move Text Along a Path

- Entry Type: Path text positioning workflow
- Category: Text and typography > Text on paths and shapes
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/text-on-paths-shapes/flip-move-text-along-path.html

**Function Summary**
Path text handles reposition text along its path and flip it to the opposite side of the path.

**What This Function Does**
- Moves the start point, end point, and text position along a path.
- Flips text across the path so it sits on the other side.
- Preserves editable text and path-based layout behavior.

**Where You Find It**
- `Path Selection Tool` and `Direct Selection Tool`.
- On-canvas path text handles.

**Primary Interface Elements**
- Start bracket.
- End bracket.
- Center or text position marker.
- Path outline and text baseline.

**How To Use It**
1. Select the type layer that uses path text.
2. Choose the `Path Selection Tool`.
3. Drag the start or end bracket to change the visible text range.
4. Drag the text position marker along the path to move the text.
5. Drag across the path to flip the text to the opposite side.

**Expected Result**
- Path text moves or flips while staying attached to the vector path.
