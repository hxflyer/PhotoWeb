# Overview of Animating Layer Properties

- Entry Type: Layer animation reference
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/overview-animating-layer-properties.html

**Function Summary**
Animating layer properties changes layer attributes over time with timeline keyframes.

**What This Function Describes**
- Common animatable properties include position, opacity, style, transform, and mask-related values depending on layer type.
- Photoshop interpolates values between keyframes in Video Timeline mode.
- Frame Animation mode uses frame-specific layer states instead of continuous keyframe interpolation.

**Where You Find It**
- `Window > Timeline`, Video Timeline mode.

**Primary Interface Elements**
- Layer property tracks.
- Stopwatch icons.
- Keyframe diamonds.
- Playhead and time ruler.

**Expected Result**
- Users understand which layer properties can drive motion or visual changes over time.
