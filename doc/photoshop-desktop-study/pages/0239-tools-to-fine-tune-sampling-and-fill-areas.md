# Tools to Fine-Tune Sampling and Fill Areas

- Entry Type: Content-Aware Fill tool reference
- Category: Repair and retouch > Remove objects and fill space
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/remove-objects-fill-space/tools-to-fine-tune-sampling-and-fill-areas.html

**Function Summary**
These tools control which pixels Photoshop can sample and which area receives the fill inside the `Content-Aware Fill` workspace.

**What This Function Does**
- adds or removes areas from the sampling overlay
- refines the original fill selection
- supports preview navigation while working

**Where You Find It**
- `Content-Aware Fill` workspace toolbar and options bar

**Primary Interface Elements**
- `Sampling Brush Tool`
- `Add to overlay area`
- `Subtract from overlay area`
- `Lasso`
- `Polygonal Lasso`
- `Expand`
- `Contract`
- `Reset`
- `Hand`
- `Zoom`
- preview zoom slider

**How To Use It**
1. Use the `Sampling Brush Tool` to include or exclude source texture areas.
2. Use lasso tools to refine the filled selection area.
3. Use `Expand`, `Contract`, or `Reset` for selection corrections.
4. Pan with the `Hand` tool and inspect detail with `Zoom`.

**Mouse, Keyboard, and Menu Input**
- Keyboard: `Alt` on Windows or `Option` on macOS temporarily switches Sampling Brush add/subtract behavior.
- Keyboard: `E` cycles selection modes.
- Keyboard: hold `Spacebar` to temporarily use the `Hand` tool.

**Expected Result**
- The fill uses better source pixels and targets a more accurate selected area.
