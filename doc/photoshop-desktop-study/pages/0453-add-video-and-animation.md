# Add Video and Animation

- Entry Type: Topic hub
- Category: Photoshop Desktop Help
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation.html

**Function Summary**
Video and animation features let Photoshop create frame animations, timeline animations, video layers, keyframes, and animated exports.

**Where You Find It**
- `Window > Timeline`.
- `File > Import > Video Frames to Layers`.
- `File > Export > Save for Web (Legacy)` or video render/export commands where available.

**Primary Interface Elements**
- Timeline panel.
- Frame Animation mode.
- Video Timeline mode.
- Frame thumbnails, layer visibility, keyframes, playback controls, delay time, and looping options.

**Expected Result**
- Users can build simple GIF-style animations or timeline-based motion from layers and video content.
