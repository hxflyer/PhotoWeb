# Overview of Unified Text Engine

- Entry Type: Text engine reference
- Category: Text and typography > International text and languages
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/international-text-languages/overview-unified-text-engine.html

**Function Summary**
The Unified Text Engine provides one modern text engine for handling a broad range of scripts and language behaviors in Photoshop.

**What This Function Describes**
- A single text engine supports Latin, East Asian, Middle Eastern, South Asian, and other supported scripts.
- Text shaping, glyph substitution, direction, and spacing depend on the chosen language, script, font, and paragraph settings.
- Documents from older versions may need review if text engine behavior changes layout metrics.

**Where You Find It**
- Type preferences and text behavior in current Photoshop versions.
- `Character` and `Paragraph` panels for language and paragraph controls.
- Type layers that contain multilingual text.

**Primary Interface Elements**
- Language settings.
- Paragraph direction controls.
- Composer and shaping behavior.
- Font coverage and glyph substitution.

**Expected Result**
- Users understand that Photoshop uses a unified engine to render supported scripts consistently in modern documents.
