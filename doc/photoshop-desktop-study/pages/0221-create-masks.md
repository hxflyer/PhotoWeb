# Create Masks

- Entry Type: Topic hub
- Category: Photoshop Desktop Help
- Source: https://helpx.adobe.com/photoshop/desktop/create-masks.html

**Function Summary**
This hub groups Photoshop masking features used to reveal, hide, blend, or isolate image content non-destructively.

**What This Entry Is For**
- introduces masking as an alternative to destructive erasing
- groups layer-mask and blend-image workflows
- acts as the main entry point for visibility control features

**Primary Function Areas**
- layer masks
- object-based mask creation
- mask linking and toggling
- image blending and auto-blending

**Expected Result**
- Users can navigate from this hub to the correct masking workflow based on whether they need local hiding, object isolation, or multi-image blending.
