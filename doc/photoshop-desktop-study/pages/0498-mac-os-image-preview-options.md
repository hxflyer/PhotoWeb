# Mac OS Image Preview Options

- Entry Type: macOS save preview reference
- Category: Save and export > Save files
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/save-files/mac-os-image-preview-options.html

**Function Summary**
macOS image preview options control preview data saved with files so the operating system and other apps can display thumbnails or previews.

**Where You Find It**
- macOS File Handling preferences and save options where available.

**Primary Interface Elements**
- Image preview setting.
- File Handling preferences.
- Save dialog preview-related options.

**Expected Result**
- Saved files include preview information suitable for macOS browsing and compatible applications.
