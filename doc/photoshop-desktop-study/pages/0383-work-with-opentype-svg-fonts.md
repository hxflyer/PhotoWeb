# Work with OpenType SVG Fonts

- Entry Type: Color font workflow
- Category: Text and typography > Characters and glyphs
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/characters-glyphs/work-with-opentype-svg-fonts.html

**Function Summary**
OpenType SVG fonts contain multicolor or graphic glyph artwork that can display as editable type in Photoshop.

**What This Function Does**
- Uses font glyphs that include color, gradients, or embedded SVG-style artwork.
- Supports decorative lettering, emoji-like characters, and display typography.
- Keeps the text layer editable while the font controls the glyph appearance.

**Where You Find It**
- Font family menu when an OpenType SVG font is installed or activated.
- `Window > Glyphs` for browsing available color glyphs.
- `Type Tool` text editing workflow.

**Primary Interface Elements**
- OpenType SVG font in the font list.
- Color glyph previews in the `Glyphs` panel.
- Type layer with editable text.

**How To Use It**
1. Select a type layer or create new text.
2. Choose an OpenType SVG font.
3. Open `Window > Glyphs` if you need to browse non-keyboard glyphs.
4. Double-click a glyph or type the corresponding character.
5. Scale or transform the type layer as needed.

**Expected Result**
- Text displays the color or graphic glyphs supplied by the OpenType SVG font.
