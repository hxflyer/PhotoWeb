# Add Layer Masks

- Entry Type: Layer mask workflow
- Category: Create masks > Layer masks
- Source: https://helpx.adobe.com/photoshop/desktop/create-masks/layer-masks/add-layer-masks.html

**Function Summary**
This workflow adds layer masks that reveal or hide all or part of a layer without deleting source pixels.

**What This Function Does**
- creates reveal-all or hide-all masks
- converts active selections into masks
- builds masks from existing transparency
- copies or duplicates masks between layers

**Where You Find It**
- `Layers` panel mask icon
- `Layer > Layer Mask`

**Primary Interface Elements**
- layer mask icon
- mask thumbnail
- `Reveal Selection`
- `Hide Selection`
- `From Transparency`

**How To Use It**
1. Select the layer or group you want to mask.
2. Add a full reveal or full hide mask depending on whether a selection is active and whether you use the modifier-click variant.
3. Use `Reveal Selection` or `Hide Selection` if you want the current selection to define the mask.
4. Copy or duplicate mask thumbnails between layers when you need to reuse a mask.

**Mouse, Keyboard, and Menu Input**
- Mouse: click the mask icon in the `Layers` panel.
- Mouse: `Alt`-click on Windows or `Option`-click on macOS to add a hide-all mask.
- Menu path: `Layer > Layer Mask`

**User Interface Behavior**
- The mask appears as a linked thumbnail beside the layer and can be edited independently from the layer pixels.

**Expected Result**
- Layer visibility is controlled non-destructively by the new mask.
