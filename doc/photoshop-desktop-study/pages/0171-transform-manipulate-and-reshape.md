# Transform, Manipulate, and Reshape

- Entry Type: Topic hub
- Category: Crop, resize, and transform
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/transform-manipulate-reshape.html

**Function Summary**
This hub groups Photoshop features for rotating, scaling, skewing, distorting, warping, and otherwise transforming layers, selections, and objects.

**What This Entry Is For**
- routes users to the correct transform command
- separates whole-image rotation from object-level transforms
- collects reshape and perspective-related tools in one place

**Primary Function Areas**
- transform command overview
- rotate and flip
- free transform and repeated transforms
- reference-point control
- object rotation and perspective changes

**Where You Access These Tools**
- `Edit > Transform`
- `Edit > Free Transform`
- contextual task bar for supported object features

**Expected Result**
- Users can move from this hub to the exact transform workflow needed for layout, correction, or creative distortion.
