# Work with Backdrops

- Entry Type: Liquify backdrop workflow
- Category: Effects and filters > Distortion
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/distortion/work-with-backdrops.html

**Function Summary**
Liquify backdrops display other layers behind or over the active Liquify preview to guide distortion alignment.

**Where You Find It**
- `Filter > Liquify`, Backdrop options.

**Primary Interface Elements**
- Show Backdrop checkbox.
- Layer source menu.
- Mode and opacity controls.
- Liquify preview.

**How To Use It**
1. Open Liquify on the target layer.
2. Enable `Show Backdrop`.
3. Choose which layer or composite to display as a reference.
4. Adjust opacity and mode.
5. Distort the active layer while comparing it to the backdrop.

**Expected Result**
- Other artwork remains visible as a reference while editing the active Liquify layer.
