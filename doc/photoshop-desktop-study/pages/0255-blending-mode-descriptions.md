# Blending Mode Descriptions

- Entry Type: Blending mode reference
- Category: Repair and retouch > Adjust light and tone
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/adjust-light-tone/blending-mode-descriptions.html

**Function Summary**
Blending modes define how one set of pixels mathematically interacts with another when painting, filling, editing, or compositing layers.

**What This Entry Is For**
- explains the purpose of blend mode groups
- helps users choose darker, lighter, contrast, comparison, or color-component effects
- clarifies tool and color-mode availability limits

**Primary Mode Groups**
- Normal
- Darken
- Lighten
- Contrast
- Comparative
- HSL

**Where You Find It**
- layer blend mode menu
- options bar for supported painting and editing tools
- fill and stroke workflows

**Expected Result**
- Users can predict whether a mode will replace, darken, lighten, contrast, compare, or transfer hue/saturation/luminosity information.
