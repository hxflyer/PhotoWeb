# Sharpen Controls with Smart Sharpen

- Entry Type: Smart Sharpen workflow
- Category: Effects and filters > Sharpening
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/sharpening/sharpen-controls-smart-sharpen.html

**Function Summary**
Smart Sharpen applies controlled edge sharpening with options for blur removal, noise reduction, and shadow/highlight protection.

**Where You Find It**
- `Filter > Sharpen > Smart Sharpen`.

**Primary Interface Elements**
- Preview window.
- Amount, Radius, Reduce Noise.
- Remove blur type menu.
- Shadow and Highlight fade controls.

**How To Use It**
1. Convert the layer to a Smart Object if you need editability.
2. Choose `Filter > Sharpen > Smart Sharpen`.
3. Set Amount for sharpening strength and Radius for edge width.
4. Use Reduce Noise to avoid gritty texture.
5. Adjust Shadow/Highlight controls if halos appear in tonal extremes.

**Expected Result**
- Image edges become clearer with more control than basic sharpen commands.
