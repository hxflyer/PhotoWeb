# Apply Filters from the Filter Gallery

- Entry Type: Filter Gallery workflow
- Category: Effects and filters > Get started with filters
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/get-started-filters/apply-filters-from-filter-gallery.html

**Function Summary**
The Filter Gallery previews and stacks gallery filters such as Artistic, Brush Strokes, Sketch, Stylize, Texture, and Distort effects.

**Where You Find It**
- `Filter > Filter Gallery`.

**Primary Interface Elements**
- Large preview pane.
- Filter category folders.
- Filter settings panel.
- Effect stack list with add/delete controls.

**How To Use It**
1. Select the target layer or Smart Object.
2. Choose `Filter > Filter Gallery`.
3. Select a filter category and effect.
4. Adjust effect settings while watching the preview.
5. Add more effects to the stack if needed.
6. Click `OK`.

**Expected Result**
- Photoshop applies one or more gallery filters to the target layer or Smart Object.
