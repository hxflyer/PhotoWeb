# View Linked Smart Object Properties

- Entry Type: Linked Smart Object inspection workflow
- Category: Create and manage layers > Smart objects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/smart-objects/view-linked-smart-object-properties.html

**Function Summary**
This workflow opens the Properties panel for a linked Smart Object so you can inspect its source path, size, and position, and perform link-related actions.

**What This Function Does**
- Shows the external source file path
- Shows linked-object size and document position
- Provides direct access to source editing and embedding actions

**Where You Find It**
- `Window > Layers`
- `Window > Properties`

**Primary Interface Elements**
- linked Smart Object layer
- `Properties` panel
- source file path
- dimensions
- `X` and `Y` position values

**How To Use It**
1. Open `Window > Layers`.
2. Select the linked Smart Object layer.
3. Open `Window > Properties`.
4. Review the source path, dimensions, and position information.
5. Use the available Properties actions if you want to edit the source file or embed the linked object.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the linked layer and inspect or trigger actions from the Properties panel.
- `Menus`
  Use `Window > Layers` and `Window > Properties`.
- `Keyboard`
  No dedicated default hotkey is described for this inspection workflow.

**User Interface Behavior**
- The Properties panel reflects the currently selected linked Smart Object.
- Photoshop can open the appropriate source application when you choose to edit the linked content.

**Expected Result**
- You can confirm where the linked file comes from and manage its relationship to the current document more confidently.
