# Change the Pixel Dimensions of Images

- Entry Type: Pixel-dimension workflow
- Category: Crop, resize, and transform > Resize and adjust resolution
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/resize-adjust-resolution/change-pixel-dimensions-of-an-image.html

**Function Summary**
This workflow changes the actual pixel count of an image for web, app, or screen delivery.

**What This Function Does**
- resizes the image in pixels rather than only changing print metadata
- affects file size and visible detail
- uses resampling to add or remove pixels

**Where You Find It**
- `Image > Image Size`

**Primary Interface Elements**
- `Pixel Dimensions`
- `Width`
- `Height`
- unit selectors
- `Resample`
- resampling method menu

**How To Use It**
1. Open `Image > Image Size`.
2. Make sure `Resample` is enabled.
3. Change `Width` and `Height` in pixel-based units.
4. Choose a suitable resampling method if you want finer control.
5. Confirm with `OK`.

**Mouse, Keyboard, and Menu Input**
- Menu path: `Image > Image Size`
- Mouse: click the unit menus and enter new numeric values.

**User Interface Behavior**
- Photoshop recalculates the pixel count immediately in the dialog preview.
- File size and image detail potential change as a result.

**Expected Result**
- The image is optimized to a new pixel-based size for digital delivery or performance needs.
