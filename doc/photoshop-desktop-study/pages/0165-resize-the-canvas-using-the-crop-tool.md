# Resize the Canvas Using the Crop Tool

- Entry Type: Crop-based canvas expansion workflow
- Category: Crop, resize, and transform > Crop and straighten
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/crop-straighten/resize-canvas-using-the-crop-tool.html

**Function Summary**
This workflow uses the `Crop Tool` to enlarge the canvas area without trimming existing image content.

**What This Function Does**
- extends the document boundaries
- keeps current pixels intact when set up non-destructively
- provides a quick visual way to add working space

**Where You Find It**
- `Crop Tool` in the toolbar

**Primary Interface Elements**
- `Crop Tool`
- crop boundary handles
- `Delete Cropped Pixels`
- options bar

**How To Use It**
1. Select the `Crop Tool`.
2. Turn off `Delete Cropped Pixels` if you want a non-destructive result.
3. Drag the crop handles outward beyond the current image edges.
4. Press `Enter` or `Return` to apply the new canvas bounds.

**Mouse, Keyboard, and Menu Input**
- Mouse: drag crop handles outward.
- Keyboard: `Enter` on Windows or `Return` on macOS to commit.

**User Interface Behavior**
- The crop boundary can extend outside the original image area.
- Existing image pixels remain where they are; Photoshop adds surrounding canvas area instead of trimming.

**Expected Result**
- The document gains extra empty canvas space around the original image.
