# Create Adjustment Layers

- Entry Type: Adjustment-layer creation workflow
- Category: Create and manage layers > Color adjustment and fill layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/color-adjustment-fill-layers/create-adjustment-layers.html

**Function Summary**
This feature creates new adjustment layers for non-destructive color and tonal editing. It is the standard starting point for reversible image correction in Photoshop.

**What This Function Does**
- Adds a new adjustment layer above the current stack
- Opens editable controls for the chosen correction type
- Preserves original image pixels by storing the correction separately

**Where You Find It**
- Layers panel
- `Create new fill or adjustment layer`
- `Properties` panel

**Primary Interface Elements**
- `Create new fill or adjustment layer`
- adjustment type list
- `Properties` panel
- layer name field

**How To Use It**
1. Open the image you want to correct.
2. In the Layers panel, click `Create new fill or adjustment layer`.
3. Choose the adjustment type you want.
4. Modify the available settings in the `Properties` panel.
5. Name the layer if needed and confirm with `Enter` or `Return`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Create the adjustment from the Layers panel and tune it in Properties.
- `Menus`
  The article emphasizes the Layers panel entry point for creation.
- `Keyboard`
  Use `Enter` or `Return` to finalize layer naming.

**User Interface Behavior**
- The new adjustment appears as its own layer in the stack.
- The Properties panel changes based on the adjustment type selected.
- The adjustment remains editable at any time until it is merged, rasterized, or removed.

**Expected Result**
- A new non-destructive correction layer is added and ready for fine tuning.
