# Shape, Path, and Pixel Mode Options

- Entry Type: Drawing mode reference
- Category: Draw shapes and paths > Draw lines and curves
- Source: https://helpx.adobe.com/photoshop/desktop/draw-shapes-paths/draw-lines-curves/shape-path-and-pixel-mode-options.html

**Function Summary**
Drawing mode determines whether a tool creates editable vector shapes, paths, or raster pixels.

**Primary Modes**
- `Shape`: creates an editable shape layer.
- `Path`: creates a vector path without direct fill pixels.
- `Pixels`: paints raster pixels directly onto the active layer.

**Primary Interface Elements**
- path operations
- fill
- stroke
- alignment
- caps
- corners
- weight
- `Properties` panel

**Expected Result**
- Users can choose the right drawing mode before creating geometry.
