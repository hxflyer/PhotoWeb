# Delete Workspaces

- Entry Type: Workspace management
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/delete-workspaces.html

**Function Summary**
This function removes a saved workspace from Photoshop's workspace list. It is used to clean up old or unused custom layouts so the workspace menu stays shorter and easier to manage.

**What This Function Does**
- Deletes a workspace entry from the available workspace list
- Removes outdated or unnecessary custom presets
- Helps keep the workspace picker focused on the layouts you actually use

**Where You Find It**
- `Choose a workspace` dropdown in the application bar
- Restore option for built-in presets: `Edit > Preferences > Workspace > Restore Default Workspaces` on Windows
- Restore option for built-in presets: `Photoshop > Settings > Workspace > Restore Default Workspaces` on macOS

**Primary Interface Elements**
- `Choose a workspace` dropdown
- `Delete Workspace` command
- workspace selection dialog
- workspace name dropdown
- `Delete` button
- workspace preferences screen

**How To Use It**
1. Open the `Choose a workspace` menu in the application bar.
2. Select `Delete Workspace`.
3. In the dialog, choose the workspace you want to remove.
4. Select `Delete` to confirm.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Use the workspace picker and dialog controls to choose and remove the target workspace.
- `Menus`
  The article centers this action around the workspace dropdown rather than a separate menu command.
- `Keyboard`
  No dedicated default hotkey is described for deleting workspaces.

**User Interface Behavior**
- Custom workspaces are removed from the workspace list after deletion.
- Deleted custom workspaces are not restored automatically; they need to be created again if you want them back.
- Built-in workspace presets may warn before deletion and can be restored later through workspace preferences.
- Deleting a workspace affects layout presets only, not open documents or image data.

**Expected Result**
- Unused workspace presets no longer appear in the selector.
- The list of available workspaces becomes cleaner and easier to scan.
- Only active or useful layouts remain available for quick switching.

**Practical Use Cases**
- Remove experimental layouts you no longer need.
- Clean up duplicate workspace presets after interface testing.
- Simplify Photoshop for a training or shared-machine environment.
