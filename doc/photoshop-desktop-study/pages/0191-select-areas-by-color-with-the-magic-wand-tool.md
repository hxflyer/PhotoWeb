# Select Areas by Color with the Magic Wand Tool

- Entry Type: Magic Wand workflow
- Category: Make selections > Automatic and color-based selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/automatic-color-based-selections/select-areas-by-color-with-the-magic-wand-tool.html

**Function Summary**
The `Magic Wand Tool` selects pixels with similar color values based on the color you click and the tolerance settings you choose.

**What This Function Does**
- selects by sampled color similarity
- can limit selection to touching regions or include matching colors elsewhere
- works well on flat backgrounds or simple color separations

**Where You Find It**
- `Magic Wand Tool` in the toolbar

**Primary Interface Elements**
- `Tolerance`
- `Sample Size`
- `Anti-alias`
- `Contiguous`
- `Sample All Layers`
- `Select and Mask`

**How To Use It**
1. Select the `Magic Wand Tool`.
2. Set `Tolerance`, `Contiguous`, and other options in the options bar.
3. Click the color area you want to select.
4. Add or subtract more clicks as needed.
5. Refine in `Select and Mask` if necessary.

**Mouse, Keyboard, and Menu Input**
- Mouse: click inside the target color area to sample and select.

**User Interface Behavior**
- Lower tolerance narrows the selected color range.
- `Contiguous` limits the selection to connected pixels; turning it off can select matching colors throughout the image.

**Expected Result**
- Areas with similar sampled color become selected without drawing a manual border.
