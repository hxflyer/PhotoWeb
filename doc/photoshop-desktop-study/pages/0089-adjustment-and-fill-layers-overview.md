# Adjustment and Fill Layers Overview

- Entry Type: Core concept reference
- Category: Create and manage layers > Color adjustment and fill layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/color-adjustment-fill-layers/adjustment-and-fill-layers-overview.html

**Function Summary**
Adjustment layers and fill layers are Photoshop's main non-destructive controls for color, tone, and graphic fills. They store instructions separately from pixel content so the original image data is preserved.

**What This Entry Explains**
- what adjustment layers are
- what fill layers are
- how they differ from direct pixel edits
- which common adjustment and fill layer types are available

**What Adjustment Layers Do**
- Apply color and tonal changes without rewriting original pixels
- Affect layers below them in the stack
- Keep edits adjustable later

**What Fill Layers Do**
- Add a solid color, gradient, or pattern as its own layer
- Act as design or compositing elements rather than as corrections to underlying pixels

**Common Adjustment Layer Types**
- `Levels` and `Curves`
  Adjust tonal range and color balance
- `Hue/Saturation`
  Changes color intensity and overall color behavior
- `Black & White`
  Converts color imagery to monochrome with controlled channel influence
- `Brightness/Contrast`
  Applies simpler lightness adjustments
- `Color Balance`
  Changes color mixture in shadows, midtones, and highlights
- `Selective Color`
  Fine-tunes specific color ranges

**Common Fill Layer Types**
- `Solid Color`
  Fills the layer with one chosen color
- `Gradient`
  Creates a color transition across the layer
- `Pattern`
  Fills the layer with a repeating pattern

**Expected Result**
- Users understand why adjustment and fill layers are safer and more flexible than direct destructive edits.

**Practical Use Cases**
- Add a reversible contrast correction above a photo.
- Build a gradient-based background without painting pixels permanently.
- Stack several color corrections while keeping the original image intact.
