# Straighten Tilted Photos

- Entry Type: Straighten workflow
- Category: Crop, resize, and transform > Crop and straighten
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/crop-straighten/straighten-tilted-photos.html

**Function Summary**
This workflow uses the `Straighten` control inside the crop workflow to level a crooked horizon or edge.

**What This Function Does**
- rotates the image based on a user-drawn guide line
- updates the crop frame automatically after the correction
- speeds up horizon and architecture leveling

**Where You Find It**
- `Crop Tool` options bar

**Primary Interface Elements**
- `Crop Tool`
- `Straighten`
- crop boundary
- `Done`
- contextual task bar

**How To Use It**
1. Select the `Crop Tool`.
2. Click `Straighten` in the options bar.
3. Drag a guide line along a horizon or other edge that should be level.
4. Adjust the crop boundary if needed.
5. Commit with `Done`, `Enter`, or `Return`.

**Mouse, Keyboard, and Menu Input**
- Mouse: draw the straighten line directly on the image.
- Keyboard: `Enter` on Windows or `Return` on macOS to apply.

**User Interface Behavior**
- Photoshop rotates the image to match the angle of the drawn guide.
- The crop frame updates to account for the rotated image bounds.

**Expected Result**
- The image is leveled and ready for final trimming.
