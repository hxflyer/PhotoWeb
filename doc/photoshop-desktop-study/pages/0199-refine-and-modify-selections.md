# Refine and Modify Selections

- Entry Type: Topic hub
- Category: Make selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections.html

**Function Summary**
This hub groups commands for moving, refining, softening, expanding, contracting, and otherwise modifying an existing selection.

**What This Entry Is For**
- covers post-selection cleanup instead of initial selection creation
- groups border edits, inversion, feathering, edge cleanup, and mask refinement
- serves as the entry point for turning rough selections into production-ready ones

**Primary Function Areas**
- moving selections and borders
- `Select and Mask`
- feathering and anti-aliasing
- expand, contract, border, inverse
- fringe and matte cleanup

**Expected Result**
- Users can find the right selection-adjustment tool after the initial subject area has already been selected.
