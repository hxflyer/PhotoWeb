# Use Firefly Boards with Photoshop

- Entry Type: Connected workflow integration
- Category: Generative AI in Photoshop
- Source: https://helpx.adobe.com/photoshop/desktop/generative-ai/open-firefly-boards.html

**Function Summary**
This workflow sends images between Photoshop and Firefly Boards so users can explore ideas in a board-based ideation environment and then return to Photoshop for more exact editing.

**What This Function Does**
- Sends Photoshop documents to Firefly Boards
- Opens Board content back into Photoshop as editable documents
- Connects ideation and precise editing in one flow

**Where You Find It**
- `Share > Send to Firefly Boards`
- cloud-document entry points from the Photoshop home screen, Share panel, and Export menu
- `Open in Photoshop` inside Firefly Boards

**Primary Interface Elements**
- `Share`
- `Send to Firefly Boards`
- board chooser
- `Open in Boards`
- `Open in Photoshop`
- Share panel
- Export menu

**How To Send a File from Photoshop to Boards**
1. Open a supported file such as a `PSDC` document.
2. Choose `Share > Send to Firefly Boards`.
3. Select an existing board or create a new one.
4. Choose `Open in Boards` to continue working there.

**How To Open a Board Image in Photoshop**
1. In Firefly Boards, select the image you want to continue editing.
2. Right-click the image.
3. Choose `Open in Photoshop`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Use the Share flow in Photoshop and the context menu inside Firefly Boards.
- `Menus`
  Start from `Share`, and for some cloud-document workflows use the home screen or Export entry points.
- `Keyboard`
  No dedicated default hotkey is described for the Board handoff workflow.

**User Interface Behavior**
- Photoshop hands the document off to a board-based ideation space without forcing you to rebuild the asset manually.
- Returning from Boards opens the image as a new Photoshop document for further editing.
- The workflow is designed for moving between concept exploration and production refinement.

**Expected Result**
- Creative exploration can happen in Boards while precise correction and finishing remain in Photoshop.

**Practical Use Cases**
- Collect several visual ideas on a board before choosing one to finish.
- Send a cloud document into an ideation flow and then reopen the strongest direction in Photoshop.
- Move concept assets between collaborative exploration and detailed retouching.
