# Set Foreground and Background Colors

- Entry Type: Color swatch workflow
- Category: Adjust color > Choose colors
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/choose-colors/set-foreground-and-background-colors.html

**Function Summary**
Foreground and background swatches define the active colors used by painting, fill, erasing, and many editing tools.

**Where You Find It**
- toolbar color swatches
- `Eyedropper Tool`

**Primary Interface Elements**
- foreground swatch
- background swatch
- switch colors icon
- default colors icon
- `Eyedropper Tool`
- `Sample Size`
- `Sample`

**How To Use It**
1. Click the foreground or background swatch to open the Color Picker.
2. Choose the new color.
3. Use the switch icon to swap foreground and background colors.
4. Use the default icon to restore black foreground and white background.
5. Use the `Eyedropper Tool` to sample colors from the canvas.

**Expected Result**
- Photoshop uses the chosen foreground and background colors for future tool actions.
