# Batch and Droplet Processing Options

- Entry Type: Automation options reference
- Category: Automate tasks > Batch processing
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/process-batch-files/batch-droplet-processing-options.html

**Function Summary**
Batch and droplet options control input source, destination, file naming, action playback, save behavior, and error handling.

**Where You Find It**
- `File > Automate > Batch`.
- `File > Automate > Create Droplet`.

**Primary Interface Elements**
- Source and Destination menus.
- Override action Open commands.
- Override action Save As commands.
- File naming tokens.
- Errors: Stop for Errors or Log Errors to File.

**Expected Result**
- Users can configure reliable unattended processing and avoid overwriting or mis-saving batch outputs.
