# Pattern Preview Best Practices

- Entry Type: Pattern Preview reference
- Category: Apply painting techniques > Create and fill with patterns
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/create-fill-patterns/pattern-preview-best-practices.html

**Function Summary**
`Pattern Preview` shows how the current document will repeat as a tiled pattern.

**Where You Find It**
- `View > Pattern Preview`

**How To Use It**
1. Enable `Pattern Preview`.
2. Edit the source tile while watching repeated edges.
3. Correct seams, spacing, and continuity before defining the pattern.

**Expected Result**
- The repeated pattern can be previewed and corrected before it becomes a preset.
