# Remove Objects and Fill the Area with Content-Aware Fill

- Entry Type: Content-Aware Fill workflow
- Category: Repair and retouch > Remove objects and fill space
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/remove-objects-fill-space/remove-objects-with-content-aware-fill.html

**Function Summary**
`Content-Aware Fill` removes selected content and fills the empty area using pixels sampled from the surrounding image.

**What This Function Does**
- removes unwanted objects from a selected area
- analyzes surrounding pixels for replacement texture
- provides a workspace for controlling sampling and previewing results

**Where You Find It**
- selection context menu
- `Edit > Content-Aware Fill` in common Photoshop workflows

**Primary Interface Elements**
- active selection
- `Content-Aware Fill` workspace
- sampling overlay
- preview panel
- `OK`

**How To Use It**
1. Select the object with a suitable selection tool.
2. Right-click inside the selection and choose `Content-Aware Fill`.
3. Adjust the sampling area in the workspace.
4. Review the preview.
5. Click `OK` to apply the fill.

**Expected Result**
- The selected object is removed and replaced with pixels that resemble the surrounding area.
