# Artboard Properties and Behaviors

- Entry Type: Artboard reference
- Category: Create and manage layers > Layout and design tools
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/layout-design-tools/artboard-properties.html

**Function Summary**
This entry summarizes the editable properties and behavioral rules that control how artboards are sized, positioned, displayed, and managed in Photoshop.

**What This Entry Covers**
- artboard dimensions and position
- preset and background options
- behavior controls for naming, moving, locking, and filtering

**Main Property Areas**
- `W` and `H`
- `X` and `Y`
- `Set Artboard to Preset`
- artboard background color

**Main Behavior Areas**
- artboard naming
- artboard movement and locking
- filtering layers by type
- grid and guide interaction
- auto-nesting behavior options

**Expected Result**
- Users can understand which artboard controls live in the Properties panel and which behaviors affect workspace organization.
