**Function Summary**
This hub groups tools that remove objects, backgrounds, and distractions, then replace the empty area with generated, sampled, or content-aware pixels.

**What This Entry Is For**
- organizes object-removal workflows
- connects background removal, Generate Background, Harmonize, Content-Aware Fill, and Remove tool features
- helps users choose between quick automatic cleanup and editable workspace-based cleanup

**Primary Function Areas**
- `Remove Background`
- `Generate Background`
- `Harmonize`
- `Content-Aware Fill`
- `Remove Tool`
- contextual task bar removal

**Expected Result**
