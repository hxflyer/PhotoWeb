# Move Reference Point for Transformations

- Entry Type: Transform origin workflow
- Category: Crop, resize, and transform > Transform, manipulate, and reshape
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/transform-manipulate-reshape/move-reference-point-for-transformations.html

**Function Summary**
This workflow changes the transform origin point used for rotation, scaling, and similar operations.

**What This Function Does**
- changes the pivot location for transforms
- allows more controlled off-center rotations and scaling
- works with standard Transform or Free Transform

**Where You Find It**
- `Edit > Transform`
- `Edit > Free Transform`
- options bar reference-point controls

**Primary Interface Elements**
- reference point display
- reference point location icon
- transform bounding box

**How To Use It**
1. Select the target layer, selection, or object.
2. Start `Edit > Transform` or `Edit > Free Transform`.
3. Show the reference point from the options bar if it is hidden.
4. Drag the reference point to the desired pivot location.
5. Perform the transform and commit it.

**Mouse, Keyboard, and Menu Input**
- Mouse: drag the reference point inside or outside the transform box.
- Keyboard: `Enter` or `Return` to commit.

**User Interface Behavior**
- The object continues to show its transform box while the pivot point becomes the center for rotation or scale calculations.

**Expected Result**
- Transforms behave relative to the new pivot instead of the default center point.
