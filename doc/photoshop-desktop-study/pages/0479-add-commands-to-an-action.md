# Add Commands to an Action

- Entry Type: Action editing workflow
- Category: Automate tasks > Actions
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/create-record-actions/add-commands-action.html

**Function Summary**
Adding commands inserts new recorded steps into an existing action.

**Where You Find It**
- Actions panel command list and Record button.

**Primary Interface Elements**
- Selected insertion point.
- Record and Stop buttons.
- Action command list.

**How To Use It**
1. Expand the action.
2. Select the command after which new steps should be added.
3. Click `Record`.
4. Perform the new commands.
5. Click `Stop`.
6. Test playback.

**Expected Result**
- New commands are added to the action sequence.
