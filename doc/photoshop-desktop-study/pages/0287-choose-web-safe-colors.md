# Choose Web-Safe Colors

- Entry Type: Web color selection reference
- Category: Adjust color > Choose colors
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/choose-colors/choose-web-safe-colors.html

**Function Summary**
Web-safe color controls help choose colors that stay within the limited classic browser-safe color set.

**Where You Find It**
- Adobe Color Picker web-safe controls

**How To Use It**
1. Open the Adobe Color Picker.
2. Enable or choose the web-safe color option when required.
3. Pick the closest available web-safe value.

**Expected Result**
- Photoshop uses a color value from the web-safe palette or a close web-safe equivalent.
