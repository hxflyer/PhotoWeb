# Convert Background and Regular Layers

- Entry Type: Layer type conversion
- Category: Create and manage layers > Get started with layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/get-started-layers/convert-background-and-regular-layers.html

**Function Summary**
This function changes whether a layer behaves as a locked background layer or as a regular editable layer. It is used when you need either the restrictions of a background or the flexibility of a normal layer.

**What This Function Does**
- Converts the background into a standard layer
- Converts a regular layer back into the background role
- Changes how the layer can be moved, stacked, and edited

**Where You Find It**
- Layers panel
- `Layer > New > Layer From Background`
- `Layer > New > Background From Layer`

**Primary Interface Elements**
- `Background` layer
- regular layer
- Layers panel
- `Layer From Background`
- `Background From Layer`

**How To Convert the Background to a Regular Layer**
1. Select the `Background` layer in the Layers panel.
2. Open `Layer > New > Layer From Background`.
3. Enter a layer name if desired and confirm with `OK`.

**How To Convert a Regular Layer to the Background**
1. Select the layer you want to convert.
2. Open `Layer > New > Background From Layer`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the target layer in the Layers panel and confirm the conversion dialog when converting from a background.
- `Menus`
  Use the relevant command from the `Layer > New` submenu.
- `Keyboard`
  No dedicated default hotkey is described for this conversion.

**User Interface Behavior**
- Converting from background to regular layer removes the special background status and makes the layer behave like other standard layers.
- Converting a regular layer to the background restores that special role in the document.

**Expected Result**
- The layer's behavior changes to match the editing flexibility or structural role you need.

**Practical Use Cases**
- Unlock a background so it can move or support transparency-related edits.
- Reassign a flat base layer as the document background.
