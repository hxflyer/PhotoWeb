# Create Star Shapes

- Entry Type: Polygon/Star shape workflow
- Category: Draw shapes and paths > Create shapes
- Source: https://helpx.adobe.com/photoshop/desktop/draw-shapes-paths/create-shapes/create-star-shapes.html

**Function Summary**
Star shape options create pointed polygon shapes using the Polygon tool settings.

**Primary Interface Elements**
- `Polygon Tool`
- sides or points
- star option
- indent or radius settings

**How To Use It**
1. Select the `Polygon Tool`.
2. Enable star-related options.
3. Set point count and indentation/radius.
4. Drag on the canvas.

**Expected Result**
- Photoshop creates an editable star shape.
