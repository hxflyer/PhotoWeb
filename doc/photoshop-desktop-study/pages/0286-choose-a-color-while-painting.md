# Choose a Color While Painting

- Entry Type: In-context color sampling workflow
- Category: Adjust color > Choose colors
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/choose-colors/choose-color-while-painting.html

**Function Summary**
This workflow samples or changes colors while a painting tool is active so users can keep painting without leaving the canvas.

**Primary Interface Elements**
- painting tool
- HUD color picker or color sampling shortcut
- foreground color
- Eyedropper behavior

**How To Use It**
1. Start with a painting tool such as the Brush.
2. Use the tool's color-sampling shortcut or picker.
3. Select a new color from the canvas or picker.
4. Continue painting with the updated foreground color.

**Expected Result**
- The active painting color changes without interrupting the painting workflow.
