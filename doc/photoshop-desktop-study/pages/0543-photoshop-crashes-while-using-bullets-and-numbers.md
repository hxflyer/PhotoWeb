- Entry Type: Crash troubleshooting workflow
- Category: Troubleshoot
- Source: https://helpx.adobe.com/photoshop/desktop/troubleshoot/photoshop-crashes-using-bullets-numbers.html

**Function Summary**
This troubleshooting workflow focuses on crashes triggered by bulleted or numbered text list features.

**Where You Find It**
- Text tool workflows involving paragraph lists.
- Crash reports and Adobe known issue references.

**Primary Interface Elements**
- Type layer.
- Bulleted/numbered list controls.
- Crash report dialog.
- Updates and known issue notes.

**How To Use It**
1. Save a copy of the document before editing list-heavy text.
2. Update Photoshop to the latest stable version available to you.
3. Test the list workflow in a new document.
4. Replace problematic fonts or simplify copied text formatting if needed.
5. Report repeatable crashes with the document and crash details.

**Expected Result**
- The crash condition is avoided, isolated, or escalated with useful diagnostic details.
