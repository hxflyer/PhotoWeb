# Apply a Hue or Saturation Adjustment

- Entry Type: Hue/Saturation adjustment workflow
- Category: Adjust color > Color corrections
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/color-corrections/apply-hue-saturation-adjustment.html

**Function Summary**
`Hue/Saturation` changes color hue, saturation strength, and lightness globally or by selected color ranges.

**Where You Find It**
- `Image > Adjustments > Hue/Saturation`
- `Adjustments` panel as an adjustment layer

**Primary Interface Elements**
- `Hue`
- `Saturation`
- `Lightness`
- color range menu
- targeted adjustment control
- `Colorize`

**Expected Result**
- Image colors shift, intensify, desaturate, or lighten/darken according to the selected controls.
