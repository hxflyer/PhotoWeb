# Resizing Parameters in Photoshop

- Entry Type: Image Size controls reference
- Category: Crop, resize, and transform > Resize and adjust resolution
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/resize-adjust-resolution/resizing-parameters-in-photoshop.html

**Function Summary**
This reference explains the key controls inside the `Image Size` dialog and how each one affects output.

**What This Entry Is For**
- clarifies the purpose of each resizing field
- helps users avoid distortion or unnecessary resampling
- connects resolution ranges to common use cases

**Where You Find It**
- `Image > Image Size`

**Primary Interface Elements**
- `Width`
- `Height`
- unit selectors
- aspect-ratio link
- `Resolution`
- `Resample`
- resampling method menu

**How To Use This Information**
1. Keep aspect-ratio linking on for normal proportional resizing.
2. Set `Resolution` based on print or screen needs.
3. Enable `Resample` only when you want the pixel count to change.
4. Choose units carefully so you are editing pixels, inches, or centimeters intentionally.

**User Interface Behavior**
- Linked dimensions update together when proportions are constrained.
- Resolution values have different meaning depending on whether `Resample` is enabled.

**Expected Result**
- You can interpret each `Image Size` control correctly before changing the document.
