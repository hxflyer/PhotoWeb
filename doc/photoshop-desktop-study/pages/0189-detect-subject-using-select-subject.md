# Detect Subject Using Select Subject

- Entry Type: Automatic subject-selection workflow
- Category: Make selections > Automatic and color-based selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/automatic-color-based-selections/detect-subject-using-select-subject.html

**Function Summary**
`Select Subject` automatically detects the main subject or subjects in an image and creates a selection from that analysis.

**What This Function Does**
- finds prominent subjects quickly
- replaces slower manual tracing for many images
- can be refined afterward with add, subtract, or `Select and Mask`

**Where You Find It**
- options bar for supported selection tools
- `Select and Mask` related workflows

**Primary Interface Elements**
- `Select Subject`
- `Add to Selection`
- `Subtract from Selection`
- `Select and Mask`

**How To Use It**
1. Activate a compatible selection tool such as `Object Selection`.
2. Click `Select Subject`.
3. Inspect the generated selection.
4. Refine with add or subtract modes if needed.
5. Open `Select and Mask` for edge cleanup if required.

**Mouse, Keyboard, and Menu Input**
- Mouse: click `Select Subject` in the options bar.

**User Interface Behavior**
- Photoshop analyzes the image and places marching-ants boundaries around the detected subject areas.

**Expected Result**
- The image subject becomes selected with minimal manual input.
