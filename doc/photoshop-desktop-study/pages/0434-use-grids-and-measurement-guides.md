# Use Grids and Measurement Guides

- Entry Type: Topic hub
- Category: Photoshop Desktop Help
- Source: https://helpx.adobe.com/photoshop/desktop/grids-guides-measurement.html

**Function Summary**
Grids, guides, Smart Guides, rulers, and measurement tools help align artwork, measure distances, set scale, and record measurement data.

**Where You Find It**
- `View` menu, `Window > Info`, rulers, guides, grids, and measurement-related commands.

**Primary Interface Elements**
- Rulers, guides, grid overlay, Smart Guides, measurement scale markers, Count/Measure tools, and Measurement Log.

**Expected Result**
- Users can position elements accurately and record real-world or pixel-based measurements.
