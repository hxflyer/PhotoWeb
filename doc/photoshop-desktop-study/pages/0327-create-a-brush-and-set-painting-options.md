# Create a Brush and Set Painting Options

- Entry Type: Brush configuration workflow
- Category: Apply painting techniques > Brushes and presets
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/brushes-presets/create-brush-set-painting-options.html

**Function Summary**
This workflow builds a custom brush by adjusting tip shape and painting behavior.

**Primary Interface Elements**
- brush tip shape
- size
- hardness
- spacing
- shape dynamics
- scattering
- texture
- smoothing

**How To Use It**
1. Open `Brush Settings`.
2. Choose a starting brush tip.
3. Adjust brush shape and dynamics.
4. Test the stroke.
5. Save as a preset if the brush should be reused.

**Expected Result**
- A custom brush behavior is ready for painting.
