# Resize Text

- Entry Type: Text sizing workflow
- Category: Text and typography > Get started with text
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/get-started-text/resize-text.html

**Function Summary**
Text resizing changes the visible size of a type layer by editing its font size, transforming the layer, or resizing its paragraph text box.

**What This Function Does**
- Changes character size through font-size controls.
- Scales the full type layer with transform handles.
- Resizes paragraph text bounds so text reflows inside a new box width or height.

**Where You Find It**
- Options bar and `Properties` panel font-size field.
- `Edit > Free Transform`, shortcut `Cmd+T` on macOS or `Ctrl+T` on Windows.
- Paragraph text bounding-box handles.

**Primary Interface Elements**
- Font size field.
- Transform bounding box and corner handles.
- Paragraph text box side and corner handles.
- Commit check mark for type or transform changes.

**How To Use It**
1. Select the type layer.
2. To resize characters, change the font-size value.
3. To scale the whole text object, use Free Transform and drag a corner handle.
4. To reflow paragraph text, use the `Type Tool` and drag the text-box handles.
5. Commit the change.

**Expected Result**
- Text appears at the new size while remaining editable unless the layer is rasterized.
