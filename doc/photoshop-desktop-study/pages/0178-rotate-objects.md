# Rotate Objects

- Entry Type: Rotate Object workflow
- Category: Crop, resize, and transform > Transform, manipulate, and reshape
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/transform-manipulate-reshape/rotate-objects.html

**Function Summary**
`Rotate Object` turns a flat layer into an interactively rotatable object that can be viewed from different angles.

**What This Function Does**
- simulates 3D-style object rotation from a 2D source layer
- provides on-canvas rotation controls
- supports later re-entry for refinement

**Where You Find It**
- `Edit > Rotate Object`
- contextual task bar when supported

**Primary Interface Elements**
- `Rotate Object`
- `Rotate`
- `Tilt`
- on-canvas rotation controls
- `Properties` panel
- `Done`

**How To Use It**
1. Open an image and select a suitable pixel or Smart Object layer.
2. Start `Rotate Object` from the menu or contextual task bar.
3. Drag directly on the canvas or use `Rotate` and `Tilt` controls.
4. Fine-tune values in the `Properties` panel if needed.
5. Commit with `Done`.

**Mouse, Keyboard, and Menu Input**
- Mouse: drag the on-canvas rotation controls.
- Menu path: `Edit > Rotate Object`

**User Interface Behavior**
- Photoshop may show a lower-resolution preview while you are adjusting the object, then render the full result after confirmation.

**Expected Result**
- The selected object appears rotated in depth or angle without replacing the source layer manually.
