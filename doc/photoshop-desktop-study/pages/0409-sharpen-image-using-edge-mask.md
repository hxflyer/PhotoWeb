# Sharpen Image Using Edge Mask

- Entry Type: Edge-protected sharpening workflow
- Category: Effects and filters > Sharpening
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/sharpening/sharpen-image-using-edge-mask.html

**Function Summary**
Edge mask sharpening restricts sharpening to detected edges so smooth areas such as skin or skies avoid extra noise.

**What This Function Does**
- Builds a mask from image edges.
- Applies sharpening through the mask.
- Reduces sharpening artifacts in flat tonal regions.

**Where You Find It**
- Selection, channels, masks, and sharpening filters.

**Primary Interface Elements**
- Edge mask or alpha channel.
- Sharpen filter dialog.
- Mask thumbnail or channel preview.

**Expected Result**
- Important edges sharpen while smooth areas remain cleaner.
