# Exclude Commands from an Action

- Entry Type: Action command toggle workflow
- Category: Automate tasks > Actions
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/create-record-actions/exclude-commands-action.html

**Function Summary**
Excluding commands temporarily disables recorded steps without deleting them from the action.

**Where You Find It**
- Actions panel command checkbox column.

**Primary Interface Elements**
- Command enable checkbox.
- Expanded action command list.
- Play button.

**How To Use It**
1. Expand the action in the Actions panel.
2. Clear the checkbox beside commands you want to skip.
3. Play the action.
4. Re-enable commands later by checking them again.

**Expected Result**
- Disabled commands are skipped during action playback.
