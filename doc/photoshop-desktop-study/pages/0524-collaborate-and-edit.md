# Collaborate and Edit

- Entry Type: Collaboration workflow
- Category: Share and collaborate
- Source: https://helpx.adobe.com/photoshop/desktop/share-collaborate/collaborate-edit.html

**Function Summary**
Collaborate and Edit shares editable Photoshop cloud work with other people so they can access or contribute according to permissions.

**Where You Find It**
- Share controls in Photoshop.
- Cloud documents and project areas.

**Primary Interface Elements**
- Invite or share dialog.
- Collaborator list.
- Permission controls.
- Comments, version, and sync status where available.

**How To Use It**
1. Save the document to a cloud-supported location.
2. Open the Share or Invite controls.
3. Add collaborators or generate a share link.
4. Set permissions.
5. Confirm sync status before collaborators begin work.

**Expected Result**
- Collaborators can access the shared Photoshop file according to the selected permissions.
