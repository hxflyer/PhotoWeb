# Neural Filter Categories and Output Options

- Entry Type: Neural Filters options reference
- Category: Effects and filters > Neural Filters
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/neural-filters/neural-filter-categories-output-options.html

**Function Summary**
Neural Filter categories group AI effects by purpose, while output options control whether the result edits the current layer or creates a safer editable result.

**What This Function Describes**
- Some filters are ready, some require download, and some may be beta or cloud-processed.
- Output choices can include current layer, duplicate layer, new layer, Smart Filter, or masked layer depending on the filter.
- Safer output options make it easier to compare, mask, or remove the effect.

**Where You Find It**
- `Filter > Neural Filters`, filter list and output dropdown.

**Primary Interface Elements**
- Filter category list.
- Status/download indicators.
- Output dropdown.
- Preview toggle.

**Expected Result**
- Users can choose both the correct Neural Filter and the safest output structure for later editing.
