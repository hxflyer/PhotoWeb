# Save as Photoshop PDF

- Entry Type: PDF save workflow
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/save-as-photoshop-pdf.html

**Function Summary**
Saving as Photoshop PDF exports a document to PDF while optionally preserving Photoshop editing capabilities.

**Where You Find It**
- `File > Save As` or `File > Save a Copy`, format `Photoshop PDF`.

**Primary Interface Elements**
- Save Adobe PDF dialog.
- Preset menu.
- Compression, output, security, and compatibility settings.
- Preserve Photoshop Editing Capabilities option.

**How To Use It**
1. Choose `File > Save As` or `Save a Copy`.
2. Select `Photoshop PDF`.
3. Choose an Adobe PDF preset.
4. Decide whether to preserve Photoshop editing capabilities.
5. Set compression, color, and security options.
6. Save the PDF.

**Expected Result**
- Photoshop creates a PDF for sharing, print, review, or editable Photoshop PDF workflows.
