# Technical Requirements And Installation

- Entry Type: Topic hub
- Category: Get started
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/technical-requirements-installation.html

**Function Summary**
This topic groups the setup-related reference pages needed before Photoshop desktop can be used reliably. It covers hardware requirements, GPU configuration, Windows codec support, and application language availability.

**What This Hub Is For**
- checking whether a system can run Photoshop
- validating GPU-related setup requirements
- resolving codec dependencies for certain file formats
- checking supported language availability before deployment

**Included Subtopics**
- Adobe Photoshop on desktop technical requirements
- Use the graphics processor
- Graphics processor (GPU) card usage
- Windows HEIF and HEVC codecs
- Photoshop language availability

**How To Use This Entry**
- Open this topic before installing Photoshop on a new device.
- Use it as the setup and compatibility branch for hardware, drivers, codecs, and localization.
- Move into the GPU-specific entries if Photoshop is running slowly or GPU features are unavailable.

**Expected Result**
- You can identify the correct prerequisite or compatibility page for your setup question.
