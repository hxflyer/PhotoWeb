# File Size

- Entry Type: Storage-size reference
- Category: Crop, resize, and transform > Resize and adjust resolution
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/resize-adjust-resolution/file-size.html

**Function Summary**
This reference describes what controls a Photoshop document's digital file size.

**What This Entry Is For**
- Explains why some images take more disk space than others
- Connects pixel dimensions, layers, and format choices to storage size
- Helps users balance image quality with portability and performance

**Primary Factors**
- pixel dimensions
- number of layers
- color depth
- channels
- file format and compression method

**How To Use This Information**
1. Reduce pixel dimensions if the image is larger than the target output requires.
2. Flatten or simplify layers when you no longer need edit flexibility.
3. Choose an export format with compression appropriate for web, print, or archive use.

**Mouse, Keyboard, and Menu Input**
- Typical related commands: `Image > Image Size`, `Layer > Flatten Image`, and export/save workflows.

**User Interface Behavior**
- Larger documents usually consume more memory, save more slowly, and take longer to render or print.
- Format choice can change saved file size significantly without changing visual dimensions.

**Expected Result**
- You can predict and control storage size more intentionally before saving or exporting.
