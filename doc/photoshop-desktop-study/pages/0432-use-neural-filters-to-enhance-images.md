# Use Neural Filters to Enhance Images

- Entry Type: Neural Filters workflow
- Category: Effects and filters > Neural Filters
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/neural-filters/use-neural-filters-enhance-images.html

**Function Summary**
Using Neural Filters applies selected AI effects to an image with previewed controls and a chosen output method.

**Where You Find It**
- `Filter > Neural Filters`.

**Primary Interface Elements**
- Neural Filters panel.
- Filter enable switches.
- Sliders and options for the active filter.
- Output menu and preview toggle.

**How To Use It**
1. Select the layer to enhance.
2. Choose `Filter > Neural Filters`.
3. Download the filter if Photoshop prompts for it.
4. Turn on the filter and adjust its controls.
5. Choose an output method.
6. Click `OK`.

**Expected Result**
- Photoshop applies the AI-enhanced result using the selected output type.
