# Place an Image into a Frame

- Entry Type: Frame content placement workflow
- Category: Create and manage layers > Layout and design tools
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/layout-design-tools/place-image-frame.html

**Function Summary**
This workflow inserts an image into a frame so the frame becomes a clipping container for that content. The placed image becomes a Smart Object and automatically fits the frame.

**What This Function Does**
- Places an image into a frame
- Supports embedded and linked placement
- Automatically masks and resizes the image to the frame boundary

**Where You Find It**
- `File > Place Embedded`
- `File > Place Linked`
- frame layer in the document

**Primary Interface Elements**
- frame layer
- `Place Embedded`
- `Place Linked`
- Smart Object content inside frame

**How To Use It**
1. Create or select a frame.
2. Choose `File > Place Embedded` or `File > Place Linked`.
3. Select the image file.
4. Place it into the frame.

**User Interface Behavior**
- The image becomes a Smart Object inside the frame.
- The frame clips the image automatically.
- Linked placement keeps the PSD lighter, while embedded placement stores the image inside the document.

**Expected Result**
- The frame displays the placed image as masked content.
