# Overview of Distortion Tools

- Entry Type: Liquify tool reference
- Category: Effects and filters > Distortion
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/distortion/overview-distortion-tools.html

**Function Summary**
Liquify distortion tools reshape pixels with brush-based tools for pushing, smoothing, twirling, puckering, bloating, shifting, and reconstructing.

**Where You Find It**
- `Filter > Liquify`, tool column inside the Liquify workspace.

**Primary Interface Elements**
- Forward Warp, Reconstruct, Smooth, Twirl, Pucker, Bloat, Push Left, Freeze Mask, and Thaw Mask tools.
- Brush Size, Density, Pressure, and Rate controls.
- Face-Aware Liquify controls where a face is detected.

**Expected Result**
- Users can choose the correct Liquify tool for localized reshaping.
