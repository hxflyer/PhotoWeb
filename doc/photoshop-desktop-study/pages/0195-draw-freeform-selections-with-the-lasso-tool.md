# Draw Freeform Selections with the Lasso Tool

- Entry Type: Lasso workflow
- Category: Make selections > Freehand selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/freehand-selections/draw-freeform-segments-of-a-selection-border.html

**Function Summary**
The `Lasso Tool` creates a selection by drawing a freehand border around the target area.

**What This Function Does**
- traces irregular shapes manually
- supports feathering and anti-aliasing
- can mix freehand and straight segments

**Where You Find It**
- `Lasso Tool` in the toolbar

**Primary Interface Elements**
- `Lasso Tool`
- `Feather`
- `Anti-alias`
- `Add to Selection`
- `Subtract from Selection`
- `Intersect with Selection`

**How To Use It**
1. Select the `Lasso Tool`.
2. Set `Feather` and `Anti-alias` if needed.
3. Choose the proper selection mode.
4. Drag freely around the target area.
5. Release to close the selection, or use modifier-based straight segments when necessary.

**Mouse, Keyboard, and Menu Input**
- Mouse: drag a freehand border around the target.
- Keyboard: hold `Alt` on Windows or `Option` on macOS while clicking to create straight-edged segments.

**User Interface Behavior**
- Photoshop turns the hand-drawn border into a live selection when you release and close the path.

**Expected Result**
- An irregular, hand-traced area becomes selected.
