# Tools and Resources

- Entry Type: Troubleshooting resource reference
- Category: Troubleshoot
- Source: https://helpx.adobe.com/photoshop/desktop/troubleshoot/tools-resources.html

**Function Summary**
Troubleshooting tools and resources collect system checks, preference resets, logs, GPU checks, and Adobe help resources used to diagnose Photoshop issues.

**Where You Find It**
- `Help` menu, Photoshop preferences, system information, Adobe Help pages, and Creative Cloud diagnostics where available.

**Primary Interface Elements**
- System Info.
- Preferences reset controls.
- GPU compatibility report.
- Crash/error messages.
- Adobe support links.

**Expected Result**
- Users know which diagnostic resources to use before applying a fix.
