# Erase Parts of an Image with the Eraser Tool

- Entry Type: Eraser Tool workflow
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/erase-parts-of-an-image-with-the-eraser-tool.html

**Function Summary**
The `Eraser Tool` removes pixels or restores pixels to a previous history state depending on the selected mode.

**Primary Interface Elements**
- `Brush`
- `Pencil`
- `Block`
- `Size`
- `Hardness`
- `Opacity`
- `Flow`
- `Erase to History`

**How To Use It**
1. Select the `Eraser Tool`.
2. Choose `Brush`, `Pencil`, or `Block` mode.
3. Set size, hardness, opacity, and flow when available.
4. Drag over the area to erase.
5. Enable `Erase to History` to restore from a history state instead.

**Expected Result**
- Pixels are erased to transparency, background color, or history state depending on layer type and tool settings.
