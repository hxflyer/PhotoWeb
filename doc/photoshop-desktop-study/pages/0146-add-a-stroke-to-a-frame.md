# Add a Stroke to a Frame

- Entry Type: Frame styling workflow
- Category: Create and manage layers > Layout and design tools
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/layout-design-tools/add-stroke-to-frame.html

**Function Summary**
This workflow adds a border to a frame using stroke settings in the layer-style system.

**What This Function Does**
- Adds a solid color, gradient, or pattern stroke to a frame
- Controls stroke size and position
- Enhances frame visibility and styling

**Where You Find It**
- Layers panel
- `fx`
- `Layer Style` dialog
- `Properties` panel

**Primary Interface Elements**
- frame layer
- `Stroke`
- `Fill Type`
- `Color Picker`
- `Size`
- `Position`
- `Inside`
- `Center`
- `Outside`

**How To Use It**
1. Select the frame layer.
2. Click `fx` in the Layers panel.
3. Open the `Stroke` settings.
4. Choose `Solid Color`, `Gradient`, or `Pattern`.
5. Set color, size, and position.
6. Confirm with `OK`.
