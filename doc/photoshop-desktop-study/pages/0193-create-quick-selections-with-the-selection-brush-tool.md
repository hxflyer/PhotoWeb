# Create Quick Selections with the Selection Brush Tool

- Entry Type: Selection Brush workflow
- Category: Make selections > Freehand selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/freehand-selections/create-quick-selections-with-selection-brush-tool.html

**Function Summary**
The `Selection Brush Tool` creates selections by painting an overlay directly onto the image.

**What This Function Does**
- paints selected areas like a mask
- supports add and subtract brush modes
- gives precise manual control with visual overlay feedback

**Where You Find It**
- `Selection Brush Tool` in the toolbar

**Primary Interface Elements**
- `Selection Brush Tool`
- overlay color
- `Opacity`
- `Size`
- `Hardness`
- `Add Brush`
- `Subtract Brush`

**How To Use It**
1. Select the `Selection Brush Tool`.
2. Paint over the area you want selected.
3. Adjust overlay color if the current preview is hard to see.
4. Tune `Opacity`, `Size`, and `Hardness`.
5. Switch between `Add Brush` and `Subtract Brush` to refine the selection.

**Mouse, Keyboard, and Menu Input**
- Mouse: paint the selection directly on the canvas.

**User Interface Behavior**
- Photoshop shows a tinted overlay while you paint, making it easier to see exactly what is being selected.

**Expected Result**
- A manually painted region becomes selected with brush-level control.
