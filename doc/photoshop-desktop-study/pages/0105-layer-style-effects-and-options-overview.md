# Layer Style Effects and Options Overview

- Entry Type: Layer-style reference
- Category: Create and manage layers > Apply layer effects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/apply-layer-effects/layer-style-effects-and-options-overview.html

**Function Summary**
This entry explains the Photoshop Layer Style system and the effects available in the `Layer Style` dialog. It serves as the reference map for how style-based shadows, glows, overlays, bevels, and blending settings are structured.

**What This Entry Covers**
- how to open the `Layer Style` dialog
- what the major effect families are
- how style checkboxes, blending options, and per-effect settings work

**Where You Find It**
- by double-clicking a layer outside its name and thumbnail
- by selecting the layer style icon in the Layers panel

**Primary Interface Elements**
- `Layer Style`
- `Styles`
- `Blending Options`
- `Bevel & Emboss`
- `Stroke`
- `Inner Shadow`
- `Inner Glow`
- `Satin`
- `Color Overlay`
- `Gradient Overlay`
- `Pattern Overlay`
- `Outer Glow`
- `Drop Shadow`

**Key Behavior**
- Each effect has a checkbox that turns it on with its current settings.
- `Blending Options` controls how the layer and the effects combine with the rest of the document.
- Multiple effects can be combined into one custom style.

**Common Option Families**
- angle and lighting controls
- opacity and blend mode
- contour and anti-alias
- size, spread, choke, and range
- highlight and shadow settings
- pattern, gradient, and texture options

**How To Use This Entry**
1. Open the `Layer Style` dialog from a styled or target layer.
2. Use this page as the option map for deciding which effect type and parameter family you need.
3. Enable the effect with its checkbox and then adjust its settings.

**Expected Result**
- Users understand where style effects live, how they are enabled, and which option groups control their appearance.
