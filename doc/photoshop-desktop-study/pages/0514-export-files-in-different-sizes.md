# Export Files in Different Sizes

- Entry Type: Multi-size export workflow
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/export-files-different-sizes.html

**Function Summary**
Multi-size export creates copies of the same asset at multiple scales for web, app, and screen-density workflows.

**Where You Find It**
- `File > Export > Export As`.
- Scale controls in the Export As dialog.

**Primary Interface Elements**
- Scale list.
- Add scale control.
- Suffix fields.
- Image size preview.

**How To Use It**
1. Open `Export As`.
2. Add one or more scale sizes.
3. Set each scale percentage and filename suffix.
4. Choose format and quality.
5. Export all sizes.

**Expected Result**
- Photoshop creates multiple exported files at the requested scale factors.
