# Save Your Work

- Entry Type: Core save workflow
- Category: Save and export > Save files
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/save-files/save-your-work.html

**Function Summary**
Saving your work writes the current document to disk or cloud storage so edits are preserved.

**Where You Find It**
- `File > Save`, `File > Save As`, `File > Save a Copy`.
- Shortcuts: `Cmd+S` on macOS or `Ctrl+S` on Windows.

**Primary Interface Elements**
- Save dialog.
- Format menu.
- Cloud document or local computer destination.
- Save options for layers, profiles, alpha channels, and compatibility where applicable.

**How To Use It**
1. Choose `File > Save` for an existing document.
2. Use `Save As` for a new editable master or new location.
3. Use `Save a Copy` when exporting a copy in a format that may not preserve all Photoshop features.
4. Choose the destination, filename, and format.
5. Confirm save options.

**Expected Result**
- Photoshop writes the document or copy to the selected destination.
