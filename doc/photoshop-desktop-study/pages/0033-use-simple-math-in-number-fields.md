# Use Simple Math in Number Fields

- Entry Type: Numeric input behavior
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/use-simple-math.html

**Function Summary**
Photoshop can evaluate simple math expressions directly inside numeric input fields. Instead of calculating values separately, you can type arithmetic into fields such as width, height, size, or transform values and let Photoshop resolve the result.

**What This Function Does**
- Performs arithmetic inside compatible numeric fields
- Replaces manual calculator use for common adjustments
- Supports quick proportional edits such as doubling, halving, adding offsets, or applying percentages

**Where You Use It**
- Width and height fields
- canvas and image size dialogs
- transform-related numeric controls
- font size and other measurement input fields

**Primary Interface Elements**
- numeric input field
- measurement value box
- width field
- height field
- `Enter` key confirmation

**How To Use It**
1. Click into a numeric field that accepts typed values.
2. Type a math expression instead of a final number.
3. Press `Enter` to make Photoshop calculate and apply the result.

**Supported Input Patterns**
- `30+45`
  Adds values
- `300/2`
  Divides a value
- `12*4`
  Multiplies a value
- `100/4`
  Replaces a field value with the computed result
- `+50` after an existing value, or `50+` before `200`
  Adjusts an existing number upward
- `3 cm * 50%`
  Applies percentage-based scaling while keeping units

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Click inside the numeric field you want to change.
- `Keyboard`
  Type the expression directly, then press `Enter` to apply it.
- `Menus`
  No separate menu command is needed; this is built into compatible number-entry fields.

**User Interface Behavior**
- Photoshop evaluates the expression and replaces it with the final resolved value.
- Units can remain part of the expression in many compatible fields.
- This is most useful in places where Photoshop already expects a valid numeric measurement.
- If the field does not support arithmetic input, Photoshop may reject the value instead of calculating it.

**Expected Result**
- Number changes become faster and more accurate.
- Proportional edits can be made without leaving the dialog or panel.

**Practical Use Cases**
- Halve an image dimension by typing `/2`.
- Increase a brush or type size by adding a fixed amount.
- Scale a measurement by percentage when adjusting layout or canvas values.
