# Copy and Paste Layer Styles

- Entry Type: Style transfer workflow
- Category: Create and manage layers > Apply layer effects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/apply-layer-effects/copy-and-paste-layer-styles.html

**Function Summary**
This workflow transfers a layer style from one layer to another. It is used to keep repeated design elements visually consistent without rebuilding the effect settings manually.

**What This Function Does**
- Copies the full style set from a source layer
- Pastes it onto one or more destination layers
- Supports repeated style reuse inside the current document

**Where You Find It**
- Layers panel context menu
- `Copy Layer Style`
- `Paste Layer Style`

**Primary Interface Elements**
- source layer
- destination layer
- `Copy Layer Style`
- `Paste Layer Style`

**How To Use It**
1. In the Layers panel, right-click the layer whose style you want to copy.
2. Choose `Copy Layer Style`.
3. Right-click the destination layer.
4. Choose `Paste Layer Style`.
5. Hold `Ctrl` on Windows or `Command` on macOS while selecting if you want to target multiple destination layers.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Open the context menu on the source and destination layers and use the copy and paste commands.
- `Keyboard`
  Use `Ctrl` or `Command` when building a multi-layer destination selection.
- `Menus`
  This workflow is centered on the Layers panel context menu.

**User Interface Behavior**
- The pasted style reproduces the source layer's effect configuration on the target layer.
- Repeated use is faster than rebuilding the same shadow, glow, or overlay setup manually each time.

**Expected Result**
- Destination layers gain the same visual styling as the source layer.
