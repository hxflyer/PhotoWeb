# Characters and Glyphs

- Entry Type: Topic hub
- Category: Text and typography
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/characters-glyphs.html

**Function Summary**
Character and glyph features control individual characters, alternate glyph forms, emoji, color glyph fonts, and font-specific character insertion.

**What This Area Controls**
- Browsing and inserting glyphs from a font.
- Choosing alternate glyph forms on canvas.
- Using OpenType SVG color fonts and emoji glyphs.
- Protecting characters from unsupported glyph substitution during font changes.

**Where You Find It**
- `Window > Glyphs`.
- `Window > Character`.
- On-canvas alternate glyph popup.
- Font and OpenType controls in the options bar and text panels.

**Primary Interface Elements**
- Glyphs panel grid.
- Font family and style selectors.
- Recently used glyphs.
- Alternate glyph popup.
- Character highlight and text insertion cursor.

**Expected Result**
- Users can insert exact characters and decorative alternates instead of relying only on keyboard input.
