# Replace Object Colors by Applying a Hue or Saturation Adjustment

- Entry Type: Object color replacement workflow
- Category: Adjust color > Selective color adjustments
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/selective-color-adjustments/replace-object-colors-by-applying-a-hue-or-saturation-adjustment.html

**Function Summary**
This workflow uses a selection mask and `Hue/Saturation` adjustment layer to change an object's color non-destructively.

**How To Use It**
1. Select the object with `Quick Selection` or another selection tool.
2. Add a `Hue/Saturation` adjustment layer.
3. Use the selection as the adjustment mask.
4. Adjust `Hue`, `Saturation`, and `Lightness`.
5. Enable `Colorize` if the original color affects the new result too strongly.
6. Paint the mask to refine the affected area.

**Expected Result**
- The selected object changes color while the original pixels remain editable underneath the adjustment layer.
