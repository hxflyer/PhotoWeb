# Photoshop File Formats Overview

- Entry Type: File format reference
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/photoshop-file-formats-overview.html

**Function Summary**
Photoshop file formats determine feature preservation, compression, compatibility, color support, transparency, animation, and delivery use.

**What This Function Describes**
- PSD and PSB preserve Photoshop editing features.
- JPEG is compact and flattened but lossy.
- PNG supports transparency and lossless web output.
- TIFF can support high-quality print and archival workflows.
- GIF supports limited-color animation and transparency.
- PDF supports document exchange and print workflows.

**Expected Result**
- Users can choose a format based on editability, quality, size, transparency, animation, and destination.
