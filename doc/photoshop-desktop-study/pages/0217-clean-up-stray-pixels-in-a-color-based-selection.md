# Clean Up Stray Pixels in a Color-Based Selection

- Entry Type: Smooth command workflow
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/clean-up-stray-pixels-in-color-based-selection.html

**Function Summary**
The `Smooth` command removes noisy protrusions and patchy stray pixels from a rough selection edge.

**What This Function Does**
- reduces jagged corners
- cleans isolated edge noise
- makes color-based selections less patchy

**Where You Find It**
- `Select > Modify > Smooth`

**Primary Interface Elements**
- `Smooth`
- `Sample Radius`

**How To Use It**
1. Make the initial selection.
2. Choose `Select > Modify > Smooth`.
3. Enter the sample radius and confirm.

**User Interface Behavior**
- Photoshop evaluates nearby selected and unselected pixels and regularizes the edge shape based on the radius value.

**Expected Result**
- The selection outline becomes cleaner and less noisy.
