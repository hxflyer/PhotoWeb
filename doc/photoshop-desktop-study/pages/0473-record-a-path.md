# Record a Path

- Entry Type: Action path recording workflow
- Category: Automate tasks > Actions
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/create-record-actions/record-path.html

**Function Summary**
Recording a path stores path creation or selection steps inside an action when those steps are supported by Photoshop action recording.

**Where You Find It**
- `Window > Actions`.
- Path tools and `Paths` panel during action recording.

**Primary Interface Elements**
- Actions panel recording state.
- Path tool controls.
- Paths panel.
- Recorded path-related commands.

**How To Use It**
1. Start recording an action.
2. Create or modify the path using supported path commands.
3. Watch the Actions panel for recorded path steps.
4. Stop recording.
5. Test the action on a copy of a document.

**Expected Result**
- Supported path operations are stored as commands inside the action.
