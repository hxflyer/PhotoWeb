# Remove Objects from Within the Contextual Task Bar

- Entry Type: Contextual removal workflow
- Category: Repair and retouch > Remove objects and fill space
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/remove-objects-fill-space/remove-objects-from-contextual-task-bar.html

**Function Summary**
The contextual task bar can remove a selected object directly after the user defines the unwanted area.

**What This Function Does**
- uses an active selection as the removal target
- exposes a fast `Remove` command near the canvas
- avoids opening the full Content-Aware Fill workspace for simple cleanup

**Where You Find It**
- contextual task bar after making a selection

**Primary Interface Elements**
- `Lasso Tool`
- active selection
- `Remove`
- contextual task bar

**How To Use It**
1. Open the image.
2. Select the unwanted area with the `Lasso Tool` or another selection tool.
3. Click `Remove` in the contextual task bar.

**Expected Result**
- Photoshop removes the selected object and fills the area automatically.
