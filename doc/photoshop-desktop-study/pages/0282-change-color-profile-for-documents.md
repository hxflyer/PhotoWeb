# Change Color Profile for Documents

- Entry Type: Color profile assignment/conversion workflow
- Category: Adjust color > Color profiles
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/color-profiles/change-color-profile-for-documents.html

**Function Summary**
Photoshop can either assign a new profile interpretation or convert document colors into another profile.

**Where You Find It**
- `Edit > Assign Profile`
- `Edit > Convert to Profile`

**Primary Interface Elements**
- `Don't Color Manage This Document`
- `Working RGB`
- `Profile`
- `Destination Space`
- `Engine`
- `Intent`
- `Use Black Point Compensation`
- `Use Dither`
- `Preview`

**How To Use It**
1. Use `Assign Profile` when the file's numbers need a different interpretation without changing values.
2. Use `Convert to Profile` when preparing colors for a different output profile.
3. Choose conversion options and preview the result.
4. Save in a format that preserves embedded profiles.

**User Interface Behavior**
- Assigning changes appearance by changing interpretation.
- Converting changes color values to preserve appearance in the destination profile.

**Expected Result**
- The document uses the intended profile behavior for display, print, or delivery.
