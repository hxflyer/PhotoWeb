# Filter the Layers Panel by Smart Objects

- Entry Type: Layer filtering workflow
- Category: Create and manage layers > Smart objects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/smart-objects/filter-the-layers-panel-by-smart-objects.html

**Function Summary**
This filtering workflow narrows the Layers panel to Smart Objects of specific types so users can identify embedded, linked, missing, or out-of-date objects more quickly.

**What This Function Does**
- Filters the Layers panel to Smart Object entries
- Separates linked-library, linked-local, missing, outdated, and embedded Smart Objects
- Reduces panel clutter in complex Smart Object-heavy documents

**Where You Find It**
- Layers panel filter controls
- Smart Object filtering dropdown
- Smart Object filter icons

**Primary Interface Elements**
- `Smart Object` filter type
- linked in library filter
- linked locally filter
- out-of-date linked filter
- missing linked filter
- embedded Smart Object filter
- `Turn layer filtering on or off`

**How To Use It**
1. In the Layers panel, open the filtering dropdown.
2. Choose `Smart Object` as the filter type.
3. Click the Smart Object filter icon that matches the object state you want to isolate.
4. Use the layer-filtering toggle to turn the filter off again when finished.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Choose the Smart Object filter type and click the relevant state icon.
- `Menus`
  The workflow is controlled from the Layers panel filter bar rather than a top-level application menu.
- `Keyboard`
  No dedicated default hotkey is described for this panel filter workflow.

**User Interface Behavior**
- Filtering changes which layers are shown in the panel, not the actual layer contents or visibility in the document.
- Different Smart Object states are represented by distinct icons for faster identification.

**Expected Result**
- Smart Object-heavy documents become easier to audit and manage.
