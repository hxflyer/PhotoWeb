# Apply or Cancel Content-Aware Fill Changes

- Entry Type: Content-Aware Fill commit controls
- Category: Repair and retouch > Remove objects and fill space
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/remove-objects-fill-space/apply-or-cancel-fill-changes.html

**Function Summary**
These controls determine whether a `Content-Aware Fill` result is applied, reset, or discarded.

**What This Function Does**
- applies a fill while keeping the workspace open
- commits a fill and exits the workspace
- cancels the operation
- resets settings for another attempt

**Primary Interface Elements**
- `Apply`
- `OK`
- `Cancel`
- `Reset`
- `Lasso`
- `Polygonal Lasso`

**How To Use It**
1. Make a selection and open `Content-Aware Fill`.
2. Adjust the fill and preview the result.
3. Use `Apply` to commit the current fill while continuing in the workspace.
4. Use `OK` to commit and close.
5. Use `Cancel` to discard, or `Reset` to restore fill settings.

**Mouse, Keyboard, and Menu Input**
- Keyboard: `Enter` on Windows or `Return` on macOS commits with `OK`.
- Keyboard: `Esc` cancels.

**Expected Result**
- The fill is applied, reset, or discarded according to the selected control.
