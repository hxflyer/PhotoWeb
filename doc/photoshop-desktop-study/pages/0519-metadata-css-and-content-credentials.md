# Metadata, CSS, and Content Credentials

- Entry Type: Topic hub
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/metadata-css-content-credentials.html

**Function Summary**
Metadata, CSS, and Content Credentials features attach descriptive, technical, design, copyright, and provenance information to Photoshop work.

**Where You Find It**
- `File > File Info`.
- Export dialogs.
- Content Credentials panel or preferences where available.
- Layer CSS copy/export features where available.

**Primary Interface Elements**
- File Info metadata fields.
- Export metadata options.
- Content Credentials preview/export controls.
- CSS copy or generate controls.

**Expected Result**
- Users can preserve useful document information, design handoff data, and provenance records during output.
