# Convert Embedded Smart Objects to Linked

- Entry Type: Embed-to-link conversion workflow
- Category: Create and manage layers > Smart objects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/smart-objects/convert-embedded-smart-objects-to-linked.html

**Function Summary**
This workflow converts an embedded Smart Object into a linked one so its source is saved as a separate file and the document can become lighter or more reusable across projects.

**What This Function Does**
- Moves embedded Smart Object source data into an external file
- Preserves existing transformations, filters, and effects while converting
- Creates a linked Smart Object from previously embedded content

**Where You Find It**
- Layers panel context menu
- `Convert to Linked`

**Primary Interface Elements**
- embedded Smart Object layer
- `Convert to Linked`
- save location chooser
- filename field
- link icon after conversion

**How To Use It**
1. Select the embedded Smart Object layer.
2. Right-click the layer and choose `Convert to Linked`.
3. Choose where to save the external source file.
4. Enter the filename and extension.
5. Save the file.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the layer, use the context menu, and choose the save path for the external source file.
- `Menus`
  The article centers this conversion on the layer context menu.
- `Keyboard`
  No dedicated default hotkey is described for this conversion path.

**User Interface Behavior**
- After conversion, the layer shows a link indicator in the Layers panel.
- Existing Smart Object transformations and effects are preserved while the storage model changes from embedded to linked.

**Expected Result**
- The Smart Object becomes externally referenced instead of stored fully inside the PSD.
