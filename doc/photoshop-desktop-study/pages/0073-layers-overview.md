# Layers Overview

- Entry Type: Core concept reference
- Category: Create and manage layers > Get started with layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/get-started-layers/layers-overview.html

**Function Summary**
Layers are the primary structure Photoshop uses to keep image content separated into editable components. They allow users to modify one part of a composition without directly changing other parts.

**What This Entry Explains**
- what a layer is
- why layers make non-destructive editing easier
- how different layer types support different workflows

**What Layers Do**
- Separate visual elements into individual editable units
- Allow repositioning, hiding, reordering, and editing without affecting every part of the image
- Make complex composites, designs, and retouching workflows manageable

**Basic Mental Model**
- Layers behave like stacked transparent sheets.
- Each sheet contains part of the final image.
- Changing one layer does not automatically rewrite the content on the others.

**Primary Layer Types Mentioned**
- `Adjustment Layer`
  Applies editable tonal or color corrections without directly rewriting pixel content
- `Fill Layer`
  Adds a solid color, gradient, or pattern fill as its own editable layer
- `Smart Object`
  Preserves source content and supports more flexible transformations and filters
- `Video Layer`
  Holds timeline-based video content inside a Photoshop document

**User Interface Areas Involved**
- Layers panel
- document canvas
- layer stack order
- visibility controls

**Expected Result**
- Users understand that Photoshop editing is built around independent stacked elements rather than one permanently flattened image.

**Practical Use Cases**
- Retouch one subject without altering the background.
- Combine text, photos, and adjustment layers in one composition.
- Test multiple creative directions while keeping source content safer.
