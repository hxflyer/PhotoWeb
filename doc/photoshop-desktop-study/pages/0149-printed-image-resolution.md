# Printed Image Resolution

- Entry Type: Print resolution concept reference
- Category: Crop, resize, and transform > Resize and adjust resolution
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/resize-adjust-resolution/printed-image-resolution.html

**Function Summary**
This entry explains how pixel dimensions and resolution interact in print workflows, especially through the concept of pixels per inch (`ppi`).

**What This Entry Explains**
- what printed image resolution means
- how image dimensions differ from resolution
- when resampling becomes necessary

**Key Concepts**
- `Resolution`
  The number of pixels assigned per printed inch
- `Image dimensions`
  The total pixel width and height of the file
- `Resampling`
  The process that changes the actual number of pixels

**User Interface Behavior**
- If image data is not resampled, changing resolution changes print size instead of changing the actual pixel count.
- Increasing `ppi` reduces print size when total image data stays constant.

**Expected Result**
- Users understand why print quality depends on both the amount of pixel data and how densely it is packed on paper.
