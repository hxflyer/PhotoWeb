# Delete or Cut Selected Pixels

- Entry Type: Selection removal command
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/delete-or-cut-selected-pixels.html

**Function Summary**
This command removes selected pixels either permanently from the current layer or by placing them on the clipboard.

**What This Function Does**
- clears selected pixels
- cuts selected pixels for later paste
- behaves differently on background layers versus standard layers

**Where You Find It**
- `Edit > Clear`
- `Edit > Cut`

**Primary Interface Elements**
- `Clear`
- `Cut`
- active selection

**How To Use It**
1. Make a selection.
2. Choose `Edit > Clear` to remove it without placing it on the clipboard, or press the delete key for the same effect.
3. Choose `Edit > Cut` if you want to remove it and keep it available for paste.

**Mouse, Keyboard, and Menu Input**
- Menu path: `Edit > Clear`
- Menu path: `Edit > Cut`
- Keyboard: `Backspace` on Windows or `Delete` on macOS for clear behavior.

**User Interface Behavior**
- Clearing pixels on a background layer fills the removed area with the background color.
- Clearing pixels on a normal layer leaves transparency.

**Expected Result**
- The selected pixels disappear from the source location, with or without clipboard storage depending on the chosen command.
