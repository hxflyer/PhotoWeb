# Save and Export

- Entry Type: Topic hub
- Category: Photoshop Desktop Help
- Source: https://helpx.adobe.com/photoshop/desktop/save-export.html

**Function Summary**
Save and export features write Photoshop work to PSD/PSB, cloud documents, PDFs, web assets, image formats, video, and metadata-aware outputs.

**Where You Find It**
- `File > Save`, `File > Save As`, `File > Save a Copy`, `File > Export`, and cloud document commands.

**Primary Interface Elements**
- Save dialogs, format menus, Export As dialog, Quick Export, Save for Web, Render Video, and Content Credentials options.

**Expected Result**
- Users can choose the correct output method for editable masters, sharing, printing, web delivery, video, or provenance metadata.
