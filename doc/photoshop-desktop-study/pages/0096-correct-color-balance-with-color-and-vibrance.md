# Correct Color Balance with Color and Vibrance

- Entry Type: Color-correction adjustment workflow
- Category: Create and manage layers > Color adjustment and fill layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/color-adjustment-fill-layers/correct-color-balance-with-color-and-vibrance.html

**Function Summary**
This adjustment workflow corrects color balance and color intensity by using `Temperature`, `Tint`, `Vibrance`, and `Saturation`. It is designed for natural-looking global color correction with editable layer-based controls.

**What This Function Does**
- Warms or cools image color with `Temperature`
- Pushes color toward green or magenta with `Tint`
- Increases muted color intensity with `Vibrance`
- Changes overall color strength with `Saturation`

**Where You Find It**
- Layers panel
- `Create new fill or adjustment layer`
- `Color and vibrance`
- `Properties` panel

**Primary Interface Elements**
- `Color and vibrance`
- `Temperature`
- `Tint`
- `Vibrance`
- `Saturation`
- layer name field

**How To Use It**
1. Open the image you want to correct.
2. In the Layers panel, choose `Create new fill or adjustment layer > Color and vibrance`.
3. In the `Properties` panel, adjust `Temperature`, `Tint`, `Vibrance`, and `Saturation`.
4. Name the layer if needed and confirm with `Enter` or `Return`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Create the adjustment and move the sliders in the Properties panel.
- `Menus`
  This adjustment is launched from the layer-creation control in the Layers panel.
- `Keyboard`
  Use `Enter` or `Return` to confirm layer naming.

**User Interface Behavior**
- The correction stays editable because it lives on an adjustment layer.
- `Vibrance` is better suited to boosting muted colors without overwhelming already-saturated regions.
- `Temperature` and `Tint` together act as the main white-balance correction controls.

**Expected Result**
- The image moves toward a more balanced and natural color appearance while remaining reversible.
