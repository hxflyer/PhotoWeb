# Measure and Scale

- Entry Type: Measurement workflow
- Category: Grids, guides, and measurement
- Source: https://helpx.adobe.com/photoshop/desktop/grids-guides-measurement/measure-scale.html

**Function Summary**
Measurement tools calculate distance, angle, area, count, and scale-based values from image content.

**Where You Find It**
- `Ruler Tool`, grouped with the Eyedropper tools.
- `Window > Info`.
- `Window > Measurement Log`.
- `Image > Analysis` commands where available.

**Primary Interface Elements**
- Ruler measurement line.
- Info panel values.
- Measurement scale settings.
- Measurement Log rows and columns.

**How To Use It**
1. Select a measurement tool such as the `Ruler Tool`.
2. Drag between two points to measure distance and angle.
3. Set a measurement scale if pixel values should represent real-world units.
4. Record measurements to the Measurement Log when needed.

**Expected Result**
- Photoshop reports pixel or scale-based measurement values.
