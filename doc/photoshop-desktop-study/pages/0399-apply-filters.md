# Apply Filters

- Entry Type: Filter application workflow
- Category: Effects and filters > Get started with filters
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/get-started-filters/apply-filters.html

**Function Summary**
Applying a filter runs a selected image effect on the active layer, selection, or Smart Object.

**Where You Find It**
- `Filter` menu.
- Last-used filter shortcut: `Cmd+F` on macOS or `Ctrl+F` on Windows.

**Primary Interface Elements**
- Active layer or active selection.
- Filter dialog with preview and settings.
- Smart Filter entry when the layer is a Smart Object.

**How To Use It**
1. Select the layer or make a selection to limit the effect.
2. Convert the layer to a Smart Object if you want editable filter settings.
3. Choose a filter from the `Filter` menu.
4. Adjust preview and settings.
5. Confirm the dialog.

**Expected Result**
- The selected filter changes the target image area or appears as an editable Smart Filter.
