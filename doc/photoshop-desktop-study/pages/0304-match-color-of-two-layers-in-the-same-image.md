# Match Color of Two Layers in the Same Image

- Entry Type: Layer-to-layer Match Color workflow
- Category: Adjust color > Selective color adjustments
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/selective-color-adjustments/match-color-of-two-layers-in-the-same-image.html

**Function Summary**
This workflow makes one layer's color balance resemble another layer in the same document.

**Where You Find It**
- `Image > Adjustments > Match Color`

**How To Use It**
1. Select the target layer.
2. Open `Image > Adjustments > Match Color`.
3. Set `Source` to the current document.
4. Choose the source layer from the `Layer` menu.
5. Adjust selection, neutralize, luminance, color intensity, and fade options.
6. Click `OK`.

**Expected Result**
- The selected target layer better matches the chosen source layer's color appearance.
