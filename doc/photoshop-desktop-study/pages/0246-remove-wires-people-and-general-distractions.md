# Remove Wires, People, and General Distractions

- Entry Type: Remove Tool automatic detection workflow
- Category: Repair and retouch > Remove objects and fill space
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/remove-objects-fill-space/remove-wires-people-distractions.html

**Function Summary**
`Find Distractions` lets the `Remove Tool` automatically detect and remove categories such as wires, people, and general distractions.

**What This Function Does**
- detects common distraction categories
- removes wires and cables automatically
- detects people for review and removal
- supports general distraction detection and manual refinement

**Where You Find It**
- `Remove Tool > Find Distractions` in the options bar

**Primary Interface Elements**
- `Remove Tool`
- `Find Distractions`
- `Wires and Cables`
- `People`
- `General Distractions`
- `Sample All Layers`
- checkmark commit button

**How To Use It**
1. Open the image and select the `Remove Tool`.
2. Choose `Find Distractions`.
3. Select `Wires and Cables`, `People`, or `General Distractions`.
4. Review detected regions when Photoshop provides an overlay.
5. Commit with the checkmark.

**Expected Result**
- Photoshop removes detected distractions using automatic replacement pixels.
