# Troubleshoot Scratch Disk Full Errors in Photoshop

- Entry Type: Scratch disk error workflow
- Category: Troubleshoot
- Source: https://helpx.adobe.com/photoshop/desktop/troubleshoot/troubleshoot-scratch-disk-full-errors-photoshop.html

**Function Summary**
Scratch disk full troubleshooting frees or reassigns temporary storage when Photoshop cannot complete operations due to insufficient scratch space.

**Where You Find It**
- Scratch disk error message.
- Scratch Disk preferences.
- Operating system storage tools.

**Primary Interface Elements**
- Scratch disk drive list.
- Free space indicator.
- Startup scratch disk selector where supported.
- System storage cleanup tools.

**How To Use It**
1. Close Photoshop if possible.
2. Free space on the scratch disk by removing unneeded files.
3. Choose a different scratch disk with more free space.
4. Restart Photoshop.
5. Reduce document size, history states, or open-file load if the problem continues.

**Expected Result**
- Photoshop has enough temporary disk space to open, save, or edit documents.
