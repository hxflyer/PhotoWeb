# Video and Animation Export Formats

- Entry Type: Animation export reference
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/video-animation-export-formats.html

**Function Summary**
Video and animation export formats determine whether timeline work is saved as a video file, image sequence, or GIF-style animation.

**Where You Find It**
- `File > Export > Render Video`.
- `File > Export > Save for Web (Legacy)` for GIF-style frame animation.

**Primary Interface Elements**
- Render Video dialog.
- Format and preset menus.
- Size, frame rate, range, and destination controls.
- GIF format and looping options in Save for Web.

**Expected Result**
- Users choose the correct output format for video playback, image sequences, or animated web graphics.
