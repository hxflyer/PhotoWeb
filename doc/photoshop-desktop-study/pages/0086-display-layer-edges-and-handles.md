# Display Layer Edges and Handles

- Entry Type: Layer visual aid control
- Category: Create and manage layers > Transform and manipulate layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/transform-manipulate-layers/display-layer-edges-and-handles.html

**Function Summary**
This display workflow shows visual boundaries and transform controls for the active layer, making alignment, resizing, and rotation easier during layout work.

**What This Function Does**
- Shows layer boundary edges
- Shows transform handles for direct resizing and rotation
- Improves visual feedback during movement and positioning

**Where You Find It**
- `View > Show > Layer Edges`
- `Move` tool
- `Show Transform Controls` in the Options bar

**Primary Interface Elements**
- layer edges
- `Move` tool
- `Show Transform Controls`
- transform handles

**How To Show Layer Edges**
1. Choose `View > Show > Layer Edges`.

**How To Show Transform Handles**
1. Select the `Move` tool.
2. Enable `Show Transform Controls` in the Options bar.
3. Drag the displayed handles to resize or rotate the layer content.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Enable the controls and drag the handles directly on the canvas.
- `Menus`
  Use `View > Show > Layer Edges`.
- `Keyboard`
  No dedicated default hotkey is described in this entry.

**User Interface Behavior**
- Layer edges display the content boundary for the active layer.
- Transform handles appear only when the relevant Move tool option is enabled.
- The handles provide immediate on-canvas manipulation feedback.

**Expected Result**
- The active layer becomes easier to position and transform precisely.
