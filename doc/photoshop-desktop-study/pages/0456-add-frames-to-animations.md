# Add Frames to Animations

- Entry Type: Frame creation workflow
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/add-frames-to-animations.html

**Function Summary**
Adding frames expands a frame animation by inserting additional states into the Timeline panel.

**Where You Find It**
- `Window > Timeline`, Frame Animation mode.
- New frame button in the Timeline panel.

**Primary Interface Elements**
- New frame icon.
- Selected frame thumbnail.
- Frame order in the Timeline panel.

**How To Use It**
1. Open the Timeline panel in Frame Animation mode.
2. Select the frame after which the new frame should be added.
3. Click the new frame button.
4. Change layer visibility, position, or effects for the new frame.

**Expected Result**
- A new frame appears in the animation sequence.
