# Convert an Image to Bitmap Mode

- Entry Type: Bitmap conversion workflow
- Category: Adjust color > Color modes
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/color-modes/convert-image-to-bitmap-mode.html

**Function Summary**
Bitmap mode converts an image into a one-bit black-and-white pixel structure.

**Where You Find It**
- `Image > Mode > Bitmap`

**How To Use It**
1. Convert to Grayscale first if required.
2. Choose `Image > Mode > Bitmap`.
3. Set output resolution and conversion method.
4. Confirm the conversion.

**Expected Result**
- The image becomes a black-and-white bitmap using the selected method.
