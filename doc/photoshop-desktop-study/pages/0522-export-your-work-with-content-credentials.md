# Export Your Work with Content Credentials

- Entry Type: Content Credentials export workflow
- Category: Save and export > Metadata, CSS, and Content Credentials
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/metadata-css-content-credentials/export-work-content-credentials.html

**Function Summary**
Exporting with Content Credentials writes or publishes selected provenance information with the output file.

**Where You Find It**
- Export workflow with Content Credentials enabled.
- Content Credentials export or publish controls where available.

**Primary Interface Elements**
- Export dialog.
- Content Credentials toggle/status.
- Attach or publish options.
- File format compatibility indicators.

**How To Use It**
1. Prepare the document and enable Content Credentials.
2. Preview the credentials.
3. Choose an export method and compatible format.
4. Confirm credential attachment or publishing options.
5. Export the file.

**Expected Result**
- The exported file carries or references the selected Content Credentials data.
