# Select Animation Frames

- Entry Type: Frame selection workflow
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/select-animation-frames.html

**Function Summary**
Selecting animation frames determines which frame or group of frames receives edits, delays, deletion, duplication, or tweening.

**Where You Find It**
- Timeline panel in Frame Animation mode.

**Primary Interface Elements**
- Frame thumbnails.
- Selected frame highlight.
- `Shift`-click and `Cmd/Ctrl`-click multi-selection behavior.

**How To Use It**
1. Click a frame thumbnail to select one frame.
2. `Shift`-click to select a continuous range.
3. `Cmd/Ctrl`-click to add or remove individual frames from the selection.
4. Apply delay, duplicate, delete, or tween commands to the selected frames.

**Expected Result**
- The chosen frames are highlighted and ready for frame-level edits.
