# Retouch Images with the Clone Stamp Tool

- Entry Type: Clone Stamp workflow
- Category: Repair and retouch > Heal and clone
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/heal-clone/retouch-images-with-the-clone-stamp-tool.html

**Function Summary**
The `Clone Stamp Tool` copies pixels from one source point and paints them onto another area.

**What This Function Does**
- duplicates texture or detail exactly from a sampled source
- repairs image areas manually
- supports opacity, flow, blend mode, alignment, and layer sampling controls

**Where You Find It**
- `Clone Stamp Tool` in the toolbar

**Primary Interface Elements**
- `Mode`
- `Opacity`
- `Flow`
- `Aligned`
- `Sample`
- `Current Layer`
- `Current & Below`
- `All Layers`

**How To Use It**
1. Select the `Clone Stamp Tool`.
2. Set blend mode, opacity, flow, alignment, and sample source.
3. `Alt`-click on Windows or `Option`-click on macOS to define the source point.
4. Paint over the target area.

**Mouse, Keyboard, and Menu Input**
- Mouse: paint over the repair area.
- Keyboard: `Alt`-click on Windows or `Option`-click on macOS to sample.

**User Interface Behavior**
- With `Aligned` on, the sample point tracks relative to the brush movement.
- With `Aligned` off, each stroke restarts from the original sample point.

**Expected Result**
- Source pixels are copied into the painted target area with the selected brush and blending behavior.
