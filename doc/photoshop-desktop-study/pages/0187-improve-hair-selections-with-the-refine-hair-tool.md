# Improve Hair Selections with the Refine Hair Tool

- Entry Type: Hair-refinement workflow
- Category: Make selections > Automatic and color-based selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/automatic-color-based-selections/make-improved-hair-selections.html

**Function Summary**
`Refine Hair` improves difficult hair edges inside the `Select and Mask` workspace.

**What This Function Does**
- analyzes fine hair strands and soft transitions
- improves masks around flyaway hair
- combines with `Select and Mask` refinement settings

**Where You Find It**
- `Select and Mask` workspace

**Primary Interface Elements**
- `Refine Hair`
- `Refine Mode`
- `Color Aware`
- `Object Aware`
- `Edge Detection`
- `Smooth`
- `Feather`
- `Contrast`
- `Shift Edge`
- `Output Settings`

**How To Use It**
1. Start with a subject selection.
2. Open `Select and Mask`.
3. Click `Refine Hair`.
4. Adjust edge and global refinement controls in the `Properties` panel.
5. Choose an output mode and confirm.

**Mouse, Keyboard, and Menu Input**
- Mouse: click `Refine Hair`, then adjust the sliders in `Properties`.

**User Interface Behavior**
- Photoshop updates the preview mask to recover wispy edge detail and reduce harsh cutouts around hair.

**Expected Result**
- Hair edges look more natural and better integrated with the background or new composite.
