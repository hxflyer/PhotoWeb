# Convert a Color Photo to Grayscale Mode

- Entry Type: Grayscale conversion workflow
- Category: Adjust color > Color modes
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/color-modes/convert-color-photo-to-grayscale-mode.html

**Function Summary**
Grayscale mode removes color channels and keeps image luminance as gray values.

**Where You Find It**
- `Image > Mode > Grayscale`

**How To Use It**
1. Open the color image.
2. Choose `Image > Mode > Grayscale`.
3. Confirm the prompt if Photoshop asks to discard color information.

**Expected Result**
- The image becomes a grayscale document with color data removed.
