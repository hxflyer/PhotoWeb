# Create Projects and Add Files

- Entry Type: Project organization workflow
- Category: Share and collaborate
- Source: https://helpx.adobe.com/photoshop/desktop/share-collaborate/create-projects-add-files.html

**Function Summary**
Projects organize related Photoshop files and assets in a shared workspace for creative work.

**Where You Find It**
- Photoshop Home screen or project area where available.
- Adobe cloud project controls.

**Primary Interface Elements**
- Project list.
- New Project control.
- Add files button.
- File thumbnails and project members.

**How To Use It**
1. Open the project area.
2. Create a new project and name it clearly.
3. Add Photoshop documents or related assets.
4. Organize files for the team or workflow.
5. Share the project if collaboration is needed.

**Expected Result**
- Files are grouped into a project workspace.
