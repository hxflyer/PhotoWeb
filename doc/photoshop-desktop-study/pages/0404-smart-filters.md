# Smart Filters

- Entry Type: Non-destructive filter workflow
- Category: Effects and filters > Get started with filters
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/get-started-filters/smart-filters.html

**Function Summary**
Smart Filters apply filters to Smart Objects so their settings, blending, visibility, and mask can be changed later.

**Where You Find It**
- `Filter > Convert for Smart Filters`.
- `Layer > Smart Objects > Convert to Smart Object`.
- Smart Filter list under the Smart Object layer in the `Layers` panel.

**Primary Interface Elements**
- Smart Object layer icon.
- Smart Filters stack under the layer.
- Smart Filter mask thumbnail.
- Filter visibility eye, blending options icon, and editable filter name.

**How To Use It**
1. Select a layer and convert it to a Smart Object.
2. Apply a filter from the `Filter` menu.
3. Double-click the filter name to edit settings later.
4. Paint on the Smart Filter mask to hide or reveal the effect.
5. Use the filter eye or blending options for visibility and mixing.

**Expected Result**
- The filter remains editable and non-destructive.
