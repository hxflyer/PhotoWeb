# Set Up Paragraph Formatting

- Entry Type: Paragraph formatting workflow
- Category: Text and typography > Get started with text
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/get-started-text/set-up-paragraph-formatting.html

**Function Summary**
Paragraph formatting controls the layout behavior of lines and paragraphs inside paragraph text.

**What This Function Does**
- Sets alignment, justification, indentation, paragraph spacing, and hyphenation.
- Controls how text reflows inside a paragraph text box.
- Applies formatting to selected paragraphs or the active insertion paragraph.

**Where You Find It**
- `Window > Paragraph`.
- `Properties` panel with a type layer selected.
- Options bar for basic alignment controls.

**Primary Interface Elements**
- Alignment and justification buttons.
- Left, right, and first-line indent fields.
- Space before and space after fields.
- Hyphenation and composer options where available.

**How To Use It**
1. Select the paragraph text layer.
2. Highlight paragraphs or place the insertion cursor inside a paragraph.
3. Open `Window > Paragraph` or use the `Properties` panel.
4. Set alignment, indents, spacing, and hyphenation.
5. Resize the text box if the new formatting causes reflow.

**Expected Result**
- Paragraph text reflows and aligns according to the selected paragraph settings.
