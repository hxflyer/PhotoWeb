# Performance and Stability Issues

- Entry Type: Troubleshooting workflow
- Category: Troubleshoot
- Source: https://helpx.adobe.com/photoshop/desktop/troubleshoot/performance-stability-issues.html

**Function Summary**
Performance and stability troubleshooting addresses slow tools, lag, crashes, freezes, memory pressure, GPU problems, plug-ins, and preference corruption.

**Where You Find It**
- `Preferences > Performance`.
- Scratch Disk preferences.
- Plug-in folders.
- Adobe troubleshooting resources.

**Primary Interface Elements**
- Memory Usage slider.
- Graphics Processor settings.
- History States and Cache settings.
- Scratch disk free space.

**How To Use It**
1. Update Photoshop, GPU drivers, and the operating system.
2. Check scratch disk free space and RAM usage.
3. Test with GPU disabled or advanced GPU settings adjusted.
4. Disable third-party plug-ins for isolation.
5. Reset preferences only after safer checks fail.

**Expected Result**
- Common performance and crash causes are isolated and reduced.
