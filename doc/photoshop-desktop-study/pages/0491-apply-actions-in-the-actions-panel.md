# Apply Actions in the Actions Panel

- Entry Type: Action playback workflow
- Category: Automate tasks
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/apply-actions-actions-panel.html

**Function Summary**
Applying an action plays its recorded commands on the active document.

**Where You Find It**
- `Window > Actions`, Play button.

**Primary Interface Elements**
- Selected action.
- Play button.
- Dialog toggles and command enable checkboxes.
- Playback progress and any action dialogs.

**How To Use It**
1. Open the document to process.
2. Open the Actions panel.
3. Select the action to run.
4. Check that required commands and dialog toggles are set correctly.
5. Click Play.
6. Review the result and undo if the action is not correct.

**Expected Result**
- Photoshop runs the selected action on the active document.
