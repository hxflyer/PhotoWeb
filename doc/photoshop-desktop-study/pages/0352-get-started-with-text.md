# Get Started with Text

- Entry Type: Topic hub
- Category: Text and typography
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/get-started-text.html

**Function Summary**
The basic text workflow covers creating, editing, formatting, moving, resizing, coloring, rotating, copying, and pasting editable type layers.

**What This Area Controls**
- Point text for short labels and headings.
- Paragraph text boxes for bounded multi-line copy.
- Character formatting such as font, style, size, color, tracking, leading, and anti-aliasing.
- Paragraph formatting such as alignment, indentation, spacing, hyphenation, and justification.
- On-canvas transforms for moving, resizing, rotating, and committing type.

**Where You Find It**
- `Type Tool` in the toolbar, shortcut `T`.
- `Type` menu for type-specific commands.
- Options bar when a type tool is active.
- `Properties`, `Character`, `Paragraph`, and `Layers` panels.

**Primary Interface Elements**
- Text insertion cursor.
- Text bounding box and resize handles.
- Commit check mark and cancel icon in the options bar.
- Font family, style, size, alignment, color swatch, warp, and anti-aliasing controls.

**Expected Result**
- Users can perform everyday text operations without converting the text to pixels, keeping the type layer editable for later design changes.
