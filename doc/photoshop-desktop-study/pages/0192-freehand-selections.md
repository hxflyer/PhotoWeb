# Freehand Selections

- Entry Type: Topic hub
- Category: Make selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/freehand-selections.html

**Function Summary**
This hub groups manual selection methods used when automatic tools are not precise enough.

**What This Entry Is For**
- organizes hand-drawn and point-based selection tools
- groups lasso, polygonal lasso, magnetic lasso, selection brush, and color-range workflows
- serves as the entry point for user-guided boundary creation

**Primary Function Areas**
- brush-painted selections
- freeform lasso drawing
- polygonal edge tracing
- magnetic edge snapping
- color-range targeting

**Expected Result**
- Users can choose the manual selection tool that best matches the shape and complexity of the target edge.
