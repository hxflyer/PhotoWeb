# Scale Layer Effects

- Entry Type: Layer-style scaling control
- Category: Create and manage layers > Apply layer effects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/apply-layer-effects/scale-layer-effects.html

**Function Summary**
This control scales the size of a layer's attached style effects so shadows, glows, bevels, and similar effects stay visually proportional after resizing the layer.

**What This Function Does**
- Scales all effects in one layer style proportionally
- Keeps shadows, glows, and bevels balanced after transformations
- Helps avoid oversized or undersized effects after layer resizing

**Where You Find It**
- Layers panel
- `Layer Style` dialog
- style `Scale` control

**Primary Interface Elements**
- target layer
- layer style icon
- `Layer Style`
- `Scale`
- `OK`

**How To Use It**
1. Select the layer with effects you want to resize.
2. Open the `Layer Style` dialog by double-clicking the style indicator beside the layer.
3. Adjust the `Scale` slider to the percentage you want.
4. Confirm with `OK`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Open the dialog and drag the `Scale` slider.
- `Menus`
  The article centers this action on the Layer Style dialog rather than a separate top-level command.
- `Keyboard`
  No dedicated default hotkey is described for effect scaling.

**User Interface Behavior**
- The effect scale changes while the layer content itself remains the same size unless separately transformed.
- This is especially useful when the visual effects no longer match the resized layer proportions.

**Expected Result**
- Attached effects appear more balanced relative to the resized layer.

**Practical Use Cases**
- Resize styled text and keep its shadow looking correct.
- Scale a button or badge layer without ending up with oversized glows.
