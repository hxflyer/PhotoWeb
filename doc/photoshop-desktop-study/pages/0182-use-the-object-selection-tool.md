# Use the Object Selection Tool

- Entry Type: Object Selection workflow
- Category: Make selections > Get started with selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/get-started-selections/select-objects-with-object-selection-tool.html

**Function Summary**
The `Object Selection Tool` finds and selects distinct objects by hover detection or by analyzing a user-drawn rectangle or lasso region.

**What This Function Does**
- detects separate objects automatically
- supports rectangle or lasso targeting
- supports add, subtract, and intersect selection modes

**Where You Find It**
- `Object Selection Tool` in the toolbar

**Primary Interface Elements**
- `Object Selection Tool`
- `Object Finder`
- `Rectangle`
- `Lasso`
- `New Selection`
- `Add to Selection`
- `Subtract from Selection`
- `Intersect with Selection`
- `Sample All Layers`
- `Hard Edge`

**How To Use It**
1. Select the `Object Selection Tool`.
2. Let Photoshop highlight an object on hover, or turn off automatic finding and draw a `Rectangle` or `Lasso` around the target.
3. Choose a selection mode such as add or subtract if you are refining an existing selection.
4. Enable `Sample All Layers` if the visible stack should be considered.
5. Refine further with `Select and Mask` if needed.

**Mouse, Keyboard, and Menu Input**
- Mouse: hover to preview an object, or drag a rectangle or lasso around the target.

**User Interface Behavior**
- Hover highlighting previews what Photoshop thinks the object boundary is before you commit.
- Selection mode buttons determine whether the next action replaces or modifies the current selection.

**Expected Result**
- A distinct object or region becomes selected without hand-tracing the full edge.
