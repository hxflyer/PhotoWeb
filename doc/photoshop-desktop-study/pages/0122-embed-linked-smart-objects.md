# Embed Linked Smart Objects

- Entry Type: Link-to-embed conversion workflow
- Category: Create and manage layers > Smart objects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/smart-objects/embed-linked-smart-objects.html

**Function Summary**
This workflow converts linked Smart Objects into embedded ones so the Photoshop document becomes more self-contained and no longer depends on the external file path.

**What This Function Does**
- Embeds one linked Smart Object into the document
- Embeds all linked Smart Objects at once when needed
- Removes reliance on external source files for the embedded object

**Where You Find It**
- Layers panel context menu
- `Embed Linked`
- `Layer > Smart Objects > Embed All Linked`

**Primary Interface Elements**
- linked Smart Object layer
- `Embed Linked`
- `Embed All Linked`
- link icon in the Layers panel

**How To Embed One Linked Smart Object**
1. Open `Window > Layers`.
2. Select the linked Smart Object layer.
3. Right-click it and choose `Embed Linked`.

**How To Embed All Linked Smart Objects**
1. Choose `Layer > Smart Objects > Embed All Linked`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Right-click the linked object to embed it directly from the Layers panel.
- `Menus`
  Use `Layer > Smart Objects > Embed All Linked` for a document-wide conversion.
- `Keyboard`
  No dedicated default hotkey is described for embedding linked Smart Objects.

**User Interface Behavior**
- After embedding, the link icon disappears because the object is now stored inside the Photoshop document.
- The document becomes more portable because it no longer depends on the original external file path for that object.

**Expected Result**
- The selected linked Smart Object becomes embedded and travels with the PSD.
