# Create Frame-Based Animations

- Entry Type: Frame animation reference
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/create-frame-based-animations.html

**Function Summary**
Frame-based animation plays a sequence of frames, with each frame displaying a chosen combination of layer visibility, position, and effects.

**Where You Find It**
- `Window > Timeline > Create Frame Animation`.

**Primary Interface Elements**
- Frame thumbnails.
- Layer visibility and layer properties.
- Delay time and looping controls.
- Tweening command.

**Expected Result**
- Users understand frame animation as a layer-state sequence suited for GIFs, simple web graphics, and short looping motion.
