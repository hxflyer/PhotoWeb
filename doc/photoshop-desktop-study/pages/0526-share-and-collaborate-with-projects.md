# Share and Collaborate with Projects

- Entry Type: Project sharing workflow
- Category: Share and collaborate
- Source: https://helpx.adobe.com/photoshop/desktop/share-collaborate/share-collaborate-projects.html

**Function Summary**
Project sharing gives collaborators access to a grouped set of Photoshop files and related assets.

**Where You Find It**
- Project workspace sharing controls.

**Primary Interface Elements**
- Invite button.
- Collaborator list.
- Permissions.
- Project file list and sync status.

**How To Use It**
1. Open the project.
2. Choose the share or invite control.
3. Add collaborators.
4. Set access permissions.
5. Monitor project sync and file updates.

**Expected Result**
- Collaborators can work with the shared project files according to permissions.
