# Adjust a Selection Manually

- Entry Type: Add/subtract selection workflow
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/adjust-a-selection-manually.html

**Function Summary**
This workflow manually expands or reduces an existing selection with any compatible selection tool.

**What This Function Does**
- adds new areas to an existing selection
- subtracts unwanted areas from it
- preserves consistent edge behavior when feather and anti-alias settings match

**Where You Find It**
- selection tool options bar

**Primary Interface Elements**
- `Add to Selection`
- `Subtract from Selection`
- `Feather`
- `Anti-alias`

**How To Use It**
1. Make an initial selection.
2. Match `Feather` and `Anti-alias` settings to the original selection behavior.
3. Use `Add to Selection` or hold `Shift` while selecting to include more area.
4. Use `Subtract from Selection` or hold `Alt` on Windows or `Option` on macOS while selecting to remove area.

**Mouse, Keyboard, and Menu Input**
- Mouse: drag or click with the active selection tool to modify the selection.
- Keyboard: hold `Shift` to add.
- Keyboard: hold `Alt` on Windows or `Option` on macOS to subtract.

**User Interface Behavior**
- The pointer shows a plus or minus indicator when modifier keys are active.

**Expected Result**
- The selection boundary grows or shrinks exactly where the follow-up selection action is applied.
