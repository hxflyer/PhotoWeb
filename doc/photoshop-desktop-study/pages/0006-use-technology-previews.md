# Use Technology Previews

- Entry Type: Preferences feature
- Category: What's new
- Source: https://helpx.adobe.com/photoshop/desktop/whats-new/enable-and-use-technology-previews.html

**Function Summary**
Technology Previews is a settings area that lets you turn selected experimental Photoshop features on or off before they become regular production features. It is intended for optional testing, not guaranteed-stable daily use.

**What This Function Does**
- exposes in-development Photoshop features
- allows per-feature enable or disable control
- separates experimental behavior from standard release defaults
- supports early testing and feedback workflows

**Where You Find It**
- Windows: `Edit > Settings > Technology Previews`
- macOS: `Photoshop > Settings > Technology Previews`

**How To Use It**
1. Open the `Technology Previews` settings screen.
2. Review the available preview options.
3. Turn individual preview features on or off depending on your testing needs.
4. Select `OK`.
5. Restart Photoshop if required so the change takes effect.

**User Interface Behavior**
- Technology previews are managed from preferences, not from a normal tool panel.
- Availability depends on the current Photoshop build.
- Preview options may disappear, change name, or move into the regular product over time.

**Expected Result**
- Enabled preview items become available in Photoshop after the required restart.
- Disabled preview items stay outside the active workspace.

**Practical Usage Notes**
- Use this section for testing, not for mission-critical production by default.
- If a preview causes instability, turn it off first before broader troubleshooting.
