# Draw an Arrow

- Entry Type: Line Tool arrowhead workflow
- Category: Draw shapes and paths > Draw lines and curves
- Source: https://helpx.adobe.com/photoshop/desktop/draw-shapes-paths/draw-lines-curves/draw-an-arrow.html

**Function Summary**
Arrowhead options add a start or end arrowhead to a line.

**Primary Interface Elements**
- `Line Tool`
- `Start`
- `End`
- `Width`
- `Length`
- `Concavity`
- contextual task bar

**How To Use It**
1. Select the `Line Tool`.
2. Enable `Start` or `End` arrowhead.
3. Set width, length, and concavity.
4. Draw the line.

**Expected Result**
- The line is created with an arrowhead at the selected end.
