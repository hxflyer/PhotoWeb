# Supported International Scripts

- Entry Type: Script support reference
- Category: Text and typography > International text and languages
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/international-text-languages/supported-international-scripts.html

**Function Summary**
Supported international scripts identify the writing systems Photoshop can shape, display, and edit through the Unified Text Engine.

**What This Function Describes**
- Script support depends on Photoshop version, operating system input support, and font glyph coverage.
- Correct rendering may require script-appropriate fonts and language settings.
- Unsupported or missing glyphs can appear as fallback characters, boxes, or incorrect substitutions.

**Where You Find It**
- Adobe support reference for supported scripts.
- `Character` panel language and font controls.
- Photoshop type layers containing multilingual text.

**Primary Interface Elements**
- Language and script settings.
- Font selection and missing glyph indicators.
- Text rendering on canvas.

**Expected Result**
- Users can verify whether Photoshop and the chosen font can support the script required by the document.
