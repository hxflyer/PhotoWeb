# Overview of Guides, Grids, and Smart Guides

- Entry Type: Alignment overlay reference
- Category: Grids, guides, and measurement
- Source: https://helpx.adobe.com/photoshop/desktop/grids-guides-measurement/overview-guides-grids-smart-guides.html

**Function Summary**
Guides, grids, and Smart Guides are non-printing overlays that support spacing, alignment, and layout decisions.

**What This Function Describes**
- Guides are user-placed horizontal or vertical reference lines.
- Grids are repeating document overlays.
- Smart Guides appear temporarily when moving or transforming objects near alignment points.

**Where You Find It**
- `View > Guides`.
- `View > Show > Grid`.
- `View > Show > Smart Guides`.
- `View > Snap` and `View > Snap To`.

**Expected Result**
- Users know which alignment overlay to use for fixed layout guides, repeated spacing, or live snapping feedback.
