# Clean Up Image Layers

- Entry Type: Layer maintenance workflow
- Category: Create and manage layers > Transform and manipulate layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/transform-manipulate-layers/clean-up-layers.html

**Function Summary**
Layer Cleanup automatically removes empty layers or renames layers based on detected content. It is used to simplify and standardize layer stacks in busy documents.

**What This Function Does**
- Deletes empty pixel and text layers
- Renames layers automatically based on content
- Applies cleanup to the full document or only to selected layers

**Where You Find It**
- Layers panel
- `Layer Cleanup` icon and dialog

**Primary Interface Elements**
- `Layer Cleanup`
- `Delete empty layers`
- `Rename layers`
- `Entire document`
- `Selected layers`
- `Apply`

**How To Delete Empty Layers**
1. Open the image you want to clean.
2. In the Layers panel, click the `Layer Cleanup` icon.
3. Choose `Delete empty layers`.
4. Choose the cleanup scope: `Entire document` or `Selected layers`.
5. Select `Apply`.

**How To Rename Layers Automatically**
1. Open the image.
2. In the Layers panel, click the `Layer Cleanup` icon.
3. Choose `Rename layers`.
4. Choose the cleanup scope.
5. Select `Apply`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Open the cleanup dialog from the Layers panel and choose the cleanup mode and scope.
- `Menus`
  The article centers this feature on the Layers panel icon rather than a top-level menu path.
- `Keyboard`
  No dedicated default hotkey is described for Layer Cleanup.

**User Interface Behavior**
- Empty-layer cleanup removes blank pixel and text layers only.
- Automatic renaming applies descriptive names based on layer content when supported.
- If no layers are deleted during cleanup, the document may not contain any removable empty layers.

**Expected Result**
- The layer stack becomes shorter, clearer, and easier to understand.

**Practical Use Cases**
- Remove accidental blank layers after long retouching sessions.
- Standardize layer names before handing a file to another designer.
- Reduce clutter in layered concept files.
