# Adjust Preferences

- Entry Type: Application settings management
- Category: Get started > Settings and preferences
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/settings-and-preferences/adjust-preferences.html

**Function Summary**
This feature opens Photoshop's central Preferences dialog, where app-wide behavior is configured. It is the main entry point for changing interface, workspace, performance, file-handling, and other core application settings.

**What This Function Does**
- Opens the full Preferences dialog
- Lets users change settings across multiple categories
- Saves customized application behavior for future sessions

**Where You Find It**
- `Edit > Preferences` on Windows
- `Photoshop > Settings` on macOS
- Shortcut: `Ctrl + K` on Windows, `Command + K` on macOS

**Primary Interface Elements**
- Preferences dialog
- category list
- `General`
- `Interface`
- `Workspace`
- performance and file-handling sections
- `OK`

**How To Use It**
1. Open Preferences from the application menu or press `Ctrl + K` / `Command + K`.
2. Select the category you want to edit, such as `General`, `Interface`, or `Workspace`.
3. Change the settings in that category.
4. Confirm with `OK`.
5. Close Photoshop normally if you want to ensure your updated settings are written out safely.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Click through the category list and change the options you need.
- `Keyboard`
  Press `Ctrl + K` on Windows or `Command + K` on macOS to open Preferences quickly.
- `Menus`
  Use the platform-specific Preferences path from the top application menu.

**User Interface Behavior**
- Preferences are stored when Photoshop closes normally.
- Preference data may be lost if Photoshop crashes or is force-quit before settings are saved correctly.
- Settings can migrate from an earlier version to a newer Photoshop installation depending on the upgrade path.

**Expected Result**
- Photoshop behavior changes according to the options set in the chosen preference category.

**Practical Use Cases**
- Adjust interface visibility and scaling.
- Change performance or scratch-disk behavior.
- Modify workspace, file handling, and tool behavior from one central control area.
