# Create Embedded Smart Objects

- Entry Type: Embedded Smart Object creation workflow
- Category: Create and manage layers > Smart objects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/smart-objects/create-embedded-smart-objects.html

**Function Summary**
This workflow places or converts content into an embedded Smart Object so the source stays inside the Photoshop document while remaining more flexible than a normal raster layer.

**What This Function Does**
- Places files as embedded Smart Objects
- Opens files directly as Smart Objects
- Converts one or more existing layers into a Smart Object
- Preserves source content for later editing

**Where You Find It**
- `File > Place Embedded`
- `File > Open As Smart Object`
- `Layer > Smart Objects > Convert to Smart Object`
- `Edit > Paste` with `Smart Object` in the paste dialog

**Primary Interface Elements**
- Smart Object thumbnail
- `Place Embedded`
- `Open As Smart Object`
- `Convert to Smart Object`
- `Smart Object` paste option

**How To Create an Embedded Smart Object**
1. Use one of the supported entry paths:
   `File > Place Embedded`
   `File > Open As Smart Object`
   `Layer > Smart Objects > Convert to Smart Object`
   `Edit > Paste` and choose `Smart Object` when pasting supported vector artwork
2. Once created, double-click the Smart Object thumbnail in the Layers panel to edit its contents.
3. Save the Smart Object contents to update every instance in the document.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Place files, convert selected layers, and double-click the Smart Object thumbnail to edit.
- `Menus`
  Use the `File`, `Layer`, or `Edit` paths appropriate to the source content.
- `Keyboard`
  No dedicated default hotkey is described for embedded Smart Object creation.

**User Interface Behavior**
- Embedded Smart Object data lives inside the Photoshop file rather than in a separate linked source.
- Photoshop updates all document instances of that embedded object when its contents are edited and saved.

**Expected Result**
- Source content becomes protected and more flexible while remaining self-contained inside the document.
