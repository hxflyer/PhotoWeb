# About Fonts

- Entry Type: Font reference
- Category: Text and typography > Select and manage fonts
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/select-manage-fonts/about-fonts.html

**Function Summary**
Fonts define the visual design, metrics, glyph coverage, and available typographic features used by Photoshop type layers.

**What This Function Describes**
- Font families, styles, weights, widths, and glyph sets.
- Installed system fonts and activated Adobe Fonts.
- Differences between standard fonts, OpenType fonts, variable fonts, and color fonts.
- Why missing glyphs or missing fonts can change a design.

**Where You Find It**
- Font family menu in the options bar, `Properties`, and `Character` panel.
- Operating system font installation tools.
- Creative Cloud font activation where available.

**Primary Interface Elements**
- Font preview list.
- Font family and style controls.
- Glyph coverage in the `Glyphs` panel.
- Missing font and missing glyph indicators.

**Expected Result**
- Users understand that a font is more than a visual style: it also controls spacing, available characters, OpenType features, and layout behavior.
