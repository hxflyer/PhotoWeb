# Apply Transformations

- Entry Type: Transform workflow reference
- Category: Crop, resize, and transform > Transform, manipulate, and reshape
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/transform-manipulate-reshape/apply-transformations.html

**Function Summary**
This reference covers applying transform modes to active content and repeating the last transform with `Again`.

**What This Function Does**
- applies standard transform operations to the active target
- repeats a previously used transform on another target
- supports layers, selections, shapes, and similar transformable content

**Where You Find It**
- `Edit > Transform`
- `Edit > Transform > Again`
- `Edit > Free Transform`

**Primary Interface Elements**
- transform handles
- `Scale`
- `Rotate`
- `Skew`
- `Distort`
- `Perspective`
- `Warp`
- `Again`
- `Done`

**How To Use It**
1. Select the active layer or selection.
2. Open `Edit > Transform` and choose a transform mode.
3. Adjust the content on-canvas and commit the result.
4. To repeat that same transform on another object, select the new target and use `Edit > Transform > Again`.

**Mouse, Keyboard, and Menu Input**
- Menu path: `Edit > Transform`
- Menu path: `Edit > Transform > Again`
- Keyboard: `Enter` or `Return` to commit an interactive transform.

**User Interface Behavior**
- Only the active layer or selected content is transformed.
- `Again` is available after a transform has already been performed in the session.

**Expected Result**
- The selected target is transformed once or receives a repeated version of the previous transform.
