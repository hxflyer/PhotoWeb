# Batch Process Files

- Entry Type: Batch command workflow
- Category: Automate tasks > Batch processing
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/process-batch-files/batch-process-files.html

**Function Summary**
Batch processing runs a selected action across a folder, open files, imported files, or another source set.

**Where You Find It**
- `File > Automate > Batch`.

**Primary Interface Elements**
- Play section for action set and action.
- Source selector.
- Destination selector.
- File naming controls.
- Override Open/Save commands and error options.

**How To Use It**
1. Create and test the action first.
2. Choose `File > Automate > Batch`.
3. Select the action set and action.
4. Choose the source files.
5. Choose destination and naming options.
6. Run on a small test folder before processing the full batch.

**Expected Result**
- Photoshop applies the selected action to each file in the batch source.
