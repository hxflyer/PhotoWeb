# Save Skin Tones Settings as a Preset

- Entry Type: Color Range preset workflow
- Category: Make selections > Freehand selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/freehand-selections/save-skin-tones-settings-as-a-preset.html

**Function Summary**
This workflow stores `Color Range` skin-tone settings so the same selection behavior can be reused later.

**What This Function Does**
- saves tuned skin-tone selection settings
- supports later reload in other documents
- reduces repeated setup work for portrait workflows

**Where You Find It**
- `Select > Color Range`

**Primary Interface Elements**
- `Skin Tones`
- `Detect Faces`
- `Fuzziness`
- `Selection Preview`
- `Save`
- `Load`

**How To Use It**
1. Open `Select > Color Range`.
2. Choose `Skin Tones`.
3. Enable `Detect Faces` if needed and tune `Fuzziness`.
4. Preview the result.
5. Click `Save` to store the preset.
6. Later, return to the same dialog and click `Load` to reuse it.

**Mouse, Keyboard, and Menu Input**
- Menu path: `Select > Color Range`
- Mouse: click `Save` and `Load` inside the dialog.

**User Interface Behavior**
- Photoshop stores the current dialog settings as a reusable preset file for later selection sessions.

**Expected Result**
- Skin-tone targeting can be recalled consistently without rebuilding the settings from scratch.
