# Create a New Pattern

- Entry Type: Pattern creation workflow
- Category: Apply painting techniques > Create and fill with patterns
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/create-fill-patterns/create-new-pattern.html

**Function Summary**
This workflow saves the current document or selected area as a reusable pattern preset.

**Where You Find It**
- `Edit > Define Pattern`

**How To Use It**
1. Create or select the pattern source.
2. Use `Pattern Preview` if checking repeat behavior.
3. Choose `Edit > Define Pattern`.
4. Name and save the pattern.

**Expected Result**
- The artwork is stored as a pattern preset for future fills.
