# Stroke a Selection or Layer with Color

- Entry Type: Stroke command workflow
- Category: Apply painting techniques > Fill objects, selections, and layers
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/fill-objects-selections-layers/stroke-selection-layer-color.html

**Function Summary**
`Stroke` adds a colored outline to a selection or layer.

**Where You Find It**
- `Edit > Stroke`

**Primary Interface Elements**
- stroke width
- `Inside`
- `Center`
- `Outside`
- blend `Mode`
- `Opacity`
- `Preserve Transparency`

**How To Use It**
1. Choose a foreground color.
2. Select the area or layer to outline.
3. Choose `Edit > Stroke`.
4. Set width, location, blending, opacity, and transparency behavior.
5. Click `OK`.

**Expected Result**
- A colored border is applied to the selected boundary or layer pixels.
