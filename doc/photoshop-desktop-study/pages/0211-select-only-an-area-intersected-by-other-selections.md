# Select Only an Area Intersected by Other Selections

- Entry Type: Intersect-selection workflow
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/select-area-intersected-by-other-selections.html

**Function Summary**
This selection mode keeps only the overlapping area shared by the current selection and a new selection action.

**What This Function Does**
- reduces a selection to its overlap area
- supports precise targeting where two boundaries should meet
- works with any compatible selection tool

**Where You Find It**
- selection tool options bar

**Primary Interface Elements**
- `Intersect with Selection`

**How To Use It**
1. Start with an existing selection.
2. Activate any selection tool.
3. Choose `Intersect with Selection`, or hold the intersection modifier while making the second selection.
4. Drag or click over the area you want to keep.

**Mouse, Keyboard, and Menu Input**
- Keyboard: hold `Alt + Shift` on Windows or `Option + Shift` on macOS while making the new selection.

**User Interface Behavior**
- Photoshop discards all non-overlapping areas and preserves only the intersection of the old and new selection.

**Expected Result**
- The active selection shrinks to the shared area between the two selection boundaries.
