# Add Artboards to the Current Document

- Entry Type: Artboard expansion workflow
- Category: Create and manage layers > Layout and design tools
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/layout-design-tools/add-artboards-current-document.html

**Function Summary**
This workflow adds more artboards to an existing Photoshop document, either by drawing new ones or by using directional add controls beside an existing artboard.

**What This Function Does**
- Draws new artboards of custom or preset size
- Adds adjacent artboards from the current artboard layout
- Supports duplication of an existing artboard and its contents

**Where You Find It**
- `Artboard` tool grouped with `Move`
- size dropdown in the Options bar
- directional add icons around selected artboards

**Primary Interface Elements**
- `Artboard` tool
- `Move` tool group
- `Size` dropdown
- directional add icons
- Layers panel artboard naming

**How To Draw a New Artboard**
1. Select the `Artboard` tool from the Move-tool group.
2. Choose a preset size if needed.
3. Drag on the canvas to create the artboard.
4. Rename it in the Layers panel if desired.

**How To Add Another Artboard from an Existing One**
1. Select the `Artboard` tool.
2. Click an existing artboard.
3. Use one of the directional add icons to create a new artboard beside it.
4. Hold `Option` on macOS or `Alt` on Windows to duplicate the artboard with its contents.
