# Draw Lines and Line Segments

- Entry Type: Line Tool workflow
- Category: Draw shapes and paths > Draw lines and curves
- Source: https://helpx.adobe.com/photoshop/desktop/draw-shapes-paths/draw-lines-curves/draw-lines-line-segments.html

**Function Summary**
The `Line Tool` creates straight line segments as shapes, paths, or pixels.

**Primary Interface Elements**
- `Line Tool`
- weight
- fill
- stroke
- shape/path/pixels mode

**How To Use It**
1. Select the `Line Tool`.
2. Set mode, weight, fill, and stroke.
3. Drag to create the line.
4. Hold `Shift` to constrain the angle when needed.

**Expected Result**
- A straight line segment is created with the selected styling.
