# Use the Actions Panel

- Entry Type: Actions panel workflow
- Category: Automate tasks
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/use-actions-panel.html

**Function Summary**
The Actions panel records, organizes, edits, plays, imports, exports, and deletes Photoshop actions.

**Where You Find It**
- `Window > Actions`.

**Primary Interface Elements**
- Action set folders.
- Action rows and command steps.
- Panel buttons for Stop, Record, Play, New Set, New Action, and Delete.
- Panel menu for load, save, insert, duplicate, and playback options.

**How To Use It**
1. Open `Window > Actions`.
2. Expand action sets to inspect actions and commands.
3. Select an action and click Play to run it.
4. Use New Action and Record to create a new workflow.
5. Use panel menu commands to save, load, or edit actions.

**Expected Result**
- Users can operate the central panel used for Photoshop action automation.
