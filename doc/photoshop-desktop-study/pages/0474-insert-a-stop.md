# Insert a Stop

- Entry Type: Action stop workflow
- Category: Automate tasks > Actions
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/create-record-actions/insert-a-stop.html

**Function Summary**
An action stop pauses playback and displays a message so the user can read instructions or perform a manual step.

**Where You Find It**
- Actions panel menu: `Insert Stop`.

**Primary Interface Elements**
- Stop message dialog.
- Allow Continue option.
- Action command list.

**How To Use It**
1. Select the action step where the pause should occur.
2. Choose `Insert Stop` from the Actions panel menu.
3. Type the instruction message.
4. Enable `Allow Continue` if playback can resume after the message.
5. Test the action.

**Expected Result**
- Action playback pauses at the inserted stop and shows the message.
