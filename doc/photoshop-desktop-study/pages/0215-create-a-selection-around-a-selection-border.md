# Create a Selection Around a Selection Border

- Entry Type: Border selection workflow
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/create-selection-around-selection-border.html

**Function Summary**
The `Border` command turns an existing selection into a ring-shaped selection centered on its edge.

**What This Function Does**
- selects a band of pixels around an existing edge
- helps isolate halos, outlines, or cleanup zones
- expands both inward and outward from the original edge

**Where You Find It**
- `Select > Modify > Border`

**Primary Interface Elements**
- `Border`
- border width field

**How To Use It**
1. Make a selection.
2. Choose `Select > Modify > Border`.
3. Enter the border width in pixels.
4. Confirm the command.

**Expected Result**
- The original solid selection becomes a border-only selection centered around the previous boundary.
