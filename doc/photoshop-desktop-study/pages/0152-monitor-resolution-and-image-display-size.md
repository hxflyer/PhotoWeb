# Monitor Resolution and Image Display Size

- Entry Type: Display behavior reference
- Category: Crop, resize, and transform > Resize and adjust resolution
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/resize-adjust-resolution/monitor-resolution-and-image-display-size.html

**Function Summary**
This reference explains why the same image can appear larger or smaller on different monitors even when zoom is set to `100%`.

**What This Entry Is For**
- Explains how monitor pixel density affects onscreen size
- Clarifies the difference between image pixels and physical screen size
- Helps users judge preview size more realistically

**Where You See This Effect**
- document window at `100%` zoom
- `View > Print Size`
- any workspace where image magnification matters

**Primary Interface Elements**
- document tab
- zoom percentage
- monitor resolution
- image pixel dimensions

**How To Use This Information**
1. Treat `100%` zoom as a pixel-for-pixel preview, not a guaranteed physical-size preview.
2. Compare image pixel dimensions with your monitor density when evaluating screen presentation.
3. Use `View > Print Size` when you need an approximate print preview instead of a pixel preview.

**Mouse, Keyboard, and Menu Input**
- Menu path: `View > Print Size`
- Mouse: use zoom controls in the document window when evaluating different display sizes.

**User Interface Behavior**
- Larger monitors with lower density can make a `100%` view appear physically larger.
- Smaller, denser displays can make the same image appear physically smaller while preserving the same pixel count.

**Expected Result**
- You can judge whether a size difference is caused by zoom level, image dimensions, or monitor density.
