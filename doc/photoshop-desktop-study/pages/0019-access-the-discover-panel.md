# Access the Discover Panel

- Entry Type: Interface feature
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/access-discover-panel.html

**Function Summary**
The Discover panel is Photoshop's in-app help and workflow-assistance surface. It combines search, guided help, contextual recommendations, tutorials, and quick actions in one panel so you can find tools or complete common tasks without leaving the application.

**What This Function Does**
- Opens a searchable help and command surface inside Photoshop
- Shows contextual recommendations based on the active document or current task
- Provides direct access to tutorials, help articles, and quick actions
- Helps users find tools, menu commands, and next-step operations faster

**Where You Find It**
- `Help > Photoshop Help`
- The `Search for tools, help, and more` field or icon in the upper-right area of the workspace
- Keyboard shortcut: `Ctrl + F` on Windows, `Cmd + F` on macOS

**Primary Interface Elements**
- Discover panel
- search field
- quick actions
- contextual recommendations
- help articles
- tutorial cards

**How To Use It**
1. Open Photoshop and make sure the app has an active internet connection.
2. Open the panel with `Help > Photoshop Help`, the top-right search control, or `Ctrl + F` / `Cmd + F`.
3. Type a tool name, command name, or workflow goal into the search field.
4. Review the returned results for tutorials, feature explanations, and direct actions.
5. Select a result to open guidance, jump to a help topic, or launch an available quick action.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Use the top-right search entry point or click a recommended card inside the panel.
- `Keyboard`
  Press `Ctrl + F` on Windows or `Cmd + F` on macOS to open the panel directly.
- `Menus`
  Open it from `Help > Photoshop Help`.

**User Interface Behavior**
- The panel appears as an in-app overlay or side panel rather than a full separate workspace.
- Results change depending on what you search for and what Photoshop detects in the current editing context.
- Quick actions may offer one-click operations when Photoshop can identify a likely next step.
- If the app is offline, Discover content can be limited or unavailable.

**Expected Result**
- You can search Photoshop help, find related tools, and launch task guidance from one place.
- Newer users can discover features faster without scanning multiple panels and menus.
- Repetitive workflows can start more quickly when a matching quick action is available.

**Practical Use Cases**
- Find where a tool or panel is located when you do not remember the menu path.
- Search for a workflow such as removing a background or replacing the sky.
- Use quick actions for common edits without manually opening several panels first.
