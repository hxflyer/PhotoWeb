# Duplicate Layers

- Entry Type: Layer reuse workflow
- Category: Create and manage layers > Get started with layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/get-started-layers/duplicate-layers.html

**Function Summary**
This function creates a copy of an existing layer or layer group. It is used when you want to reuse content, preserve an editable original, or branch a design element without rebuilding it.

**What This Function Does**
- Copies a layer or group
- Preserves the original while creating a reusable duplicate
- Allows the duplicate to be named and optionally sent to another document

**Where You Find It**
- Layers panel
- `Layer > Duplicate Layer`

**Primary Interface Elements**
- selected layer or group
- `Duplicate Layer`
- name field
- destination document selector
- `OK`

**How To Use It**
1. Select the layer or layer group in the Layers panel.
2. Open `Layer > Duplicate Layer`.
3. Enter a name for the duplicate if needed.
4. Choose the destination document if you want the copy placed elsewhere.
5. Confirm with `OK`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the source layer or group and complete the duplicate dialog.
- `Menus`
  Use `Layer > Duplicate Layer`.
- `Keyboard`
  No dedicated default hotkey is described for this dialog-driven action.

**User Interface Behavior**
- Photoshop keeps the source unchanged and creates a separate editable copy.
- The duplicate can remain in the current document or be placed into another destination document.

**Expected Result**
- A second editable version of the chosen layer or group becomes available for reuse or variation work.

**Practical Use Cases**
- Duplicate a text layer before trying a different treatment.
- Create alternate versions of a photo element without losing the original.
- Copy a grouped asset into another open document.
