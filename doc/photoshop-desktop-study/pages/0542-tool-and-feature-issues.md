# Tool and Feature Issues

- Entry Type: Tool troubleshooting reference
- Category: Troubleshoot
- Source: https://helpx.adobe.com/photoshop/desktop/troubleshoot/tool-feature-issues.html

**Function Summary**
Tool and feature troubleshooting addresses tools that do not respond, missing options, disabled commands, incorrect cursors, or unexpected behavior.

**Where You Find It**
- Tool options bar.
- `Window > Tools`.
- Preferences and workspace reset commands.
- Tool preset and keyboard shortcut settings.

**Primary Interface Elements**
- Active tool icon.
- Options bar.
- Tool presets.
- Layer/selection state.
- Mode, bit depth, and document type indicators.

**Expected Result**
- Users can check common causes such as wrong layer type, active selection, locked layer, unsupported color mode, hidden UI, or corrupted preferences.
