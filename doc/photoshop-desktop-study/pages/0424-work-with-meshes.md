# Work with Meshes

- Entry Type: Liquify mesh workflow
- Category: Effects and filters > Distortion
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/distortion/work-with-meshes.html

**Function Summary**
Liquify meshes store distortion data so a Liquify deformation can be saved, loaded, reused, or applied consistently.

**Where You Find It**
- `Filter > Liquify`, mesh controls in the Liquify workspace.

**Primary Interface Elements**
- Load Mesh and Save Mesh controls.
- Mesh overlay where available.
- Liquify preview.

**How To Use It**
1. Open Liquify and create a distortion.
2. Save the mesh if the deformation should be reused.
3. Load a saved mesh on a compatible image or layer.
4. Adjust or confirm the result.

**Expected Result**
- Liquify distortion settings can be reused or transferred through mesh files.
