# Manage Scale Markers

- Entry Type: Scale marker workflow
- Category: Grids, guides, and measurement
- Source: https://helpx.adobe.com/photoshop/desktop/grids-guides-measurement/manage-scale-markers.html

**Function Summary**
Scale markers place visible reference bars in an image to show the active measurement scale.

**Where You Find It**
- `Image > Analysis > Place Scale Marker` where available.

**Primary Interface Elements**
- Scale marker dialog.
- Length, font, color, and location settings.
- Marker layer or overlay.

**How To Use It**
1. Set a measurement scale.
2. Choose the scale marker command.
3. Set marker length, label, color, and position.
4. Confirm and place the marker.

**Expected Result**
- A visible scale marker appears in the document for reference or output.
