# Reset Preferences

- Entry Type: Troubleshooting and factory-reset workflow
- Category: Get started > Settings and preferences
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/settings-and-preferences/reset-preferences.html

**Function Summary**
This workflow resets Photoshop preference files to their default state. It is commonly used when Photoshop behaves unexpectedly, performance becomes unstable, or corrupted settings are suspected.

**What This Function Does**
- Deletes or replaces the current preference set with default files
- Clears corrupted or problematic stored settings
- Helps troubleshoot interface, performance, and behavior issues

**Where You Find It**
- Launch-time reset shortcut
- `Edit > Preferences > General` on Windows
- `Photoshop > Settings > General` on macOS
- Photoshop settings folder in the operating system

**Primary Reset Methods**
- launch-time keyboard reset
- `Reset Preferences On Quit`
- manual settings-folder reset

**How To Reset with the Launch Shortcut**
1. Close Photoshop completely.
2. Hold `Alt + Ctrl + Shift` on Windows or `Option + Command + Shift` on macOS while launching Photoshop.
3. Confirm the prompt to delete the Photoshop settings file.
4. Let Photoshop open with default preferences.

**How To Reset from Preferences**
1. Open Photoshop.
2. Go to `Edit > Preferences > General` on Windows or `Photoshop > Settings > General` on macOS.
3. Select `Reset Preferences On Quit`.
4. Confirm with `OK`.
5. Close Photoshop and relaunch it.

**How To Reset Manually**
1. Close Photoshop completely.
2. Navigate to the Photoshop settings folder for your operating system.
3. Back up the folder first if you may need your custom settings later.
4. Delete or rename the original settings folder.
5. Launch Photoshop so fresh default preference files are created automatically.

**Mouse, Keyboard, and Menu Input**
- `Keyboard`
  Use the launch-time modifier shortcut for the fastest full reset.
- `Menus`
  Use the `General` preferences page when you want Photoshop to reset on the next quit.
- `Mouse`
  Manual reset requires file management in Finder or File Explorer.

**User Interface Behavior**
- Resetting preferences removes customized settings such as interface choices, tool behavior, and other stored configuration.
- Photoshop rebuilds fresh default preference files after the reset.
- Backing up the settings folder first is recommended if your custom setup might need to be restored later.

**Expected Result**
- Photoshop returns to a cleaner default state that can resolve issues caused by damaged or conflicting preferences.

**Practical Use Cases**
- Fix unstable behavior after a crash.
- Troubleshoot unusual tool, panel, or performance problems.
- Start fresh after a long period of accumulated customizations.
