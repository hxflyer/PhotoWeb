# Fill Text with Image

- Entry Type: Clipping mask text effect
- Category: Text and typography > Type layers and creation
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/type-layers-creation/fill-text-with-image.html

**Function Summary**
Filling text with an image uses a type layer as a clipping shape so the image appears only inside the letterforms.

**What This Function Does**
- Uses the opacity shape of editable text to mask an image layer.
- Keeps the text layer editable unless it is rasterized or converted.
- Lets users reposition or transform the image independently from the text.

**Where You Find It**
- `Layer > Create Clipping Mask`.
- Keyboard: `Option+Cmd+G` on macOS or `Alt+Ctrl+G` on Windows.
- Layers panel context menu: `Create Clipping Mask`.

**Primary Interface Elements**
- Type layer below the image layer.
- Image layer clipped to the text layer.
- Down-arrow clipping mask indicator in the `Layers` panel.

**How To Use It**
1. Create or select a type layer.
2. Place an image layer directly above the type layer.
3. Select the image layer.
4. Choose `Layer > Create Clipping Mask`.
5. Use the `Move Tool` or Free Transform to position the image inside the text.

**Expected Result**
- The image is visible only through the text shapes.
