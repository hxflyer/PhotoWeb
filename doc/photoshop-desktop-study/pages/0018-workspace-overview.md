# Workspace Overview

- Entry Type: Interface overview
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/workspace-overview.html

**Function Summary**
The Photoshop workspace is the main editing environment that combines the application bar, document window, tools, panels, and contextual controls. This page describes the core interface structure rather than a single edit command.

**Primary Workspace Elements**
- application bar
- panels
- tools panel
- document window
- options bar
- notifications
- Contextual Task Bar

**What Each Area Does**
- `Application bar`
  Provides workspace switching, menus, and top-level application controls
- `Panels`
  Show editing, organization, and adjustment controls
- `Tools panel`
  Provides the main image-editing and creation tools
- `Document window`
  Displays the active file and editing canvas
- `Options bar`
  Shows settings for the currently selected tool
- `Contextual Task Bar`
  Presents likely next actions based on the current selection or task

**How To Use This Workspace**
- Use the tools panel to choose the active editing tool.
- Use the options bar to adjust that tool's settings.
- Use panels to manage layers, properties, history, and other support controls.
- Use the Contextual Task Bar when Photoshop surfaces a direct next-step action for the current task.

**Windows-Specific Note**
- On Windows, `Win + Z` opens Snap Layouts for arranging Photoshop windows in screen layouts.

**Expected Result**
- You should understand how the main Photoshop workspace is divided and which area to use for tools, settings, files, and next-step actions.
