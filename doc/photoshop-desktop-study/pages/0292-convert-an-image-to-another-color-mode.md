# Convert an Image to Another Color Mode

- Entry Type: Color mode conversion workflow
- Category: Adjust color > Color modes
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/color-modes/convert-image-to-another-color-mode.html

**Function Summary**
Color mode conversion changes how Photoshop stores and interprets image color channels.

**Where You Find It**
- `Image > Mode`

**How To Use It**
1. Save a copy if the conversion may discard data.
2. Choose `Image > Mode`.
3. Select the target mode.
4. Respond to any flattening, profile, or data-loss prompts.

**Expected Result**
- The image changes to the selected color mode for the next editing or output workflow.
