# Save Custom Workspaces

- Entry Type: Workspace management
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/save-custom-workspaces.html

**Function Summary**
This function saves the current Photoshop interface layout as a reusable workspace preset. A saved workspace can include panel positions, keyboard shortcuts, and menu customizations, making it easier to switch between different editing setups.

**What This Function Does**
- Captures the current arrangement of panels and tools
- Stores optional shortcut and menu customizations with the layout
- Creates a named workspace that can be recalled later
- Helps build task-specific setups for editing, painting, retouching, or design work

**Where You Find It**
- `Window > Workspace > New Workspace`
- Saved workspaces later appear in the `Choose a workspace` menu in the application bar

**Primary Interface Elements**
- floating panels
- docked panels
- `Window` menu
- `Workspace` submenu
- `New Workspace` dialog
- `Capture` options
- `Save` button
- `Choose a workspace` dropdown

**How To Use It**
1. Arrange the Photoshop interface the way you want it.
2. Move, dock, or group panels until the layout matches your preferred workflow.
3. Open `Window > Workspace > New Workspace`.
4. Enter a name for the new workspace.
5. In the `Capture` section, choose whether to store `Panel Locations`, `Keyboard Shortcuts`, and `Menus`.
6. Select `Save`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Drag panel tabs to dock, group, stack, or float them before saving the layout.
- `Menus`
  Use `Window > Workspace > New Workspace` to open the save dialog.
- `Keyboard`
  No default direct hotkey is described for saving a workspace. The saved workspace can optionally include your existing shortcut set.

**User Interface Behavior**
- Photoshop stores the workspace name in the workspace picker after you save it.
- If `Panel Locations` is enabled, the current panel arrangement becomes part of the workspace preset.
- If `Keyboard Shortcuts` or `Menus` are enabled, those custom interface behaviors are tied to the workspace.
- The saved workspace becomes selectable like the built-in presets.

**Expected Result**
- You can return to the same interface layout without rebuilding it manually.
- Different jobs can use different saved workspace presets.
- Personalized shortcuts and menu visibility can travel with that workspace if captured.

**Practical Use Cases**
- Save one workspace for photo retouching and another for compositing.
- Keep a compact panel setup for smaller displays.
- Create a training workspace with only the panels and menu items you want to expose.
