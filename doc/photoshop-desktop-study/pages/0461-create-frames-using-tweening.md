# Create Frames Using Tweening

- Entry Type: Tweening workflow
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/create-frames-using-tweening.html

**Function Summary**
Tweening automatically generates intermediate frames between two animation states.

**Where You Find It**
- Timeline panel in Frame Animation mode, Tween button or panel menu.

**Primary Interface Elements**
- Tween dialog.
- Frames to Add field.
- Parameters such as Position, Opacity, and Effects.

**How To Use It**
1. Create a start frame and an end frame with different layer states.
2. Select a frame and open Tween.
3. Choose how many frames to add.
4. Select which properties Photoshop should interpolate.
5. Preview playback.

**Expected Result**
- Photoshop inserts transitional frames that smooth movement, opacity changes, or effects.
