# Export Artboards as Files

- Entry Type: Artboard file export workflow
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/export-artboards-as-files.html

**Function Summary**
Exporting artboards as files saves each artboard as a separate image file.

**Where You Find It**
- `File > Export > Artboards to Files`.

**Primary Interface Elements**
- Artboard selection.
- Destination folder.
- File name prefix.
- File type and quality settings.

**How To Use It**
1. Name and organize artboards.
2. Choose `File > Export > Artboards to Files`.
3. Select destination and format.
4. Choose export options.
5. Run the export.

**Expected Result**
- Photoshop creates separate files for the selected artboards.
