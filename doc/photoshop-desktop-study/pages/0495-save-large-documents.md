# Save Large Documents

- Entry Type: Large document save workflow
- Category: Save and export > Save files
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/save-files/save-large-documents.html

**Function Summary**
Large document saving uses formats such as PSB when a file exceeds standard PSD size limits.

**Where You Find It**
- `File > Save As` or `File > Save a Copy`, format menu.

**Primary Interface Elements**
- Large Document Format (`PSB`) option.
- Save As dialog.
- Maximize Compatibility option.

**How To Use It**
1. Choose `File > Save As` or `Save a Copy`.
2. Select `Large Document Format` if the file is too large for PSD.
3. Keep layers and compatibility options as needed.
4. Save the file.

**Expected Result**
- Large layered documents are preserved in a format designed for oversized Photoshop files.
