# Home Screen Overview

- Entry Type: Interface overview
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/homescreen-overview.html

**Function Summary**
The Photoshop Home screen is the start surface shown when the app opens without an active document. It is used to create files, open existing work, access recent items, and reach learning or cloud-file entry points.

**What This Screen Does**
- shows recent Photoshop documents
- provides `New file` and `Open` entry points
- exposes learning resources and feature highlights
- gives access to file groups such as shared cloud files and Lightroom assets

**Primary Interface Elements**
- `Home`
- `New file`
- `Open`
- `Learn`
- `Your files`
- `Shared with you`
- `Lightroom photos`
- `Deleted`
- `Recents`

**How To Use It**
1. Launch Photoshop without opening a document.
2. Use `New file` to create a document from templates or a blank setup.
3. Use `Open` to load an existing file.
4. Browse `Recents` to reopen recent work quickly.
5. Open `Learn` if you want guided tutorials.
6. Open `Your files` to access cloud, shared, Lightroom, or deleted items.

**User Interface Behavior**
- While editing a document, selecting `Home` in the options bar returns you to this screen.
- Pressing `Esc` exits the Home screen and returns to the active editing view when applicable.
- The recent-file count can be changed in `Edit > Preferences > File Handling`.

**Expected Result**
- You can start new work, reopen existing files, or navigate Photoshop learning and cloud-file entry points from a single launch surface.
