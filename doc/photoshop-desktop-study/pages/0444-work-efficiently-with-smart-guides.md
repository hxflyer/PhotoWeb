# Work Efficiently with Smart Guides

- Entry Type: Smart Guides workflow
- Category: Grids, guides, and measurement
- Source: https://helpx.adobe.com/photoshop/desktop/grids-guides-measurement/work-efficiently-smart-guides.html

**Function Summary**
Smart Guides show temporary alignment and spacing hints while moving, transforming, or drawing objects.

**Where You Find It**
- `View > Show > Smart Guides`.
- `View > Snap` and `View > Snap To`.

**Primary Interface Elements**
- Temporary magenta guide lines.
- Distance and alignment hints.
- Snap feedback while dragging.

**How To Use It**
1. Enable `View > Show > Smart Guides`.
2. Move or transform a layer, shape, selection, or type object.
3. Watch for Smart Guide lines indicating alignment to edges, centers, or spacing.
4. Release the mouse when the object snaps or aligns.

**Expected Result**
- Objects align quickly to nearby layers, document edges, centers, or spacing relationships.
