# Add Emoji Glyphs

- Entry Type: Emoji insertion workflow
- Category: Text and typography > Characters and glyphs
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/characters-glyphs/add-emoji-glyphs.html

**Function Summary**
Emoji glyph insertion adds emoji characters to editable Photoshop type layers through an emoji-capable font or system input.

**What This Function Does**
- Inserts emoji as text characters rather than pasted pixels.
- Uses the appearance supplied by the active emoji or color font.
- Allows emoji to be resized, moved, transformed, and styled as part of a type layer.

**Where You Find It**
- Operating system emoji picker.
- `Window > Glyphs` with an emoji-capable font selected.
- Active type layer or `Type Tool` insertion point.

**Primary Interface Elements**
- Text insertion cursor.
- Emoji picker or Glyphs panel.
- Emoji/color font selection.
- Editable type layer.

**How To Use It**
1. Create or select a type layer.
2. Place the text insertion cursor where the emoji should appear.
3. Use the system emoji picker or choose an emoji glyph from the `Glyphs` panel.
4. Adjust size, alignment, and spacing like other text.

**Expected Result**
- The selected emoji appears as editable text within the type layer.
