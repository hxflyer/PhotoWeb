# Change Transparency Preferences

- Entry Type: Display preference
- Category: Create and manage layers > Get started with layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/get-started-layers/change-transparency-preferences.html

**Function Summary**
This preference changes how Photoshop displays transparent areas in the workspace. It affects only the checkerboard preview pattern, not the actual transparency data in the image.

**What This Function Does**
- Changes the checkerboard size used to represent transparency
- Changes the checkerboard colors used behind transparent pixels
- Helps users distinguish transparency from image content more easily

**Where You Find It**
- `Edit > Preferences > Transparency & Gamut` on Windows
- `Photoshop > Preferences > Transparency & Gamut` on macOS

**Primary Interface Elements**
- `Transparency & Gamut`
- `Transparency Settings`
- `Grid Size`
- `Grid Colors`
- `OK`

**How To Use It**
1. Open the `Transparency & Gamut` preferences page.
2. In `Transparency Settings`, choose the `Grid Size`.
3. Choose the `Grid Colors`.
4. Confirm with `OK`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Open the preference screen and choose the grid size and colors from the available controls.
- `Menus`
  Use the platform-specific Preferences path for transparency display settings.
- `Keyboard`
  No dedicated default hotkey is described for this preference page.

**User Interface Behavior**
- The checkerboard preview updates how transparent areas look inside Photoshop.
- Changing these settings does not alter exported transparency or pixel content.
- Higher-contrast grid colors can make transparent regions easier to identify in complex documents.

**Expected Result**
- Transparent areas become easier to read in the workspace based on the user's visual preference.
