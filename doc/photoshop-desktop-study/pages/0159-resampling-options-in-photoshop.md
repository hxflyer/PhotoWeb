# Resampling Options in Photoshop

- Entry Type: Resampling method reference
- Category: Crop, resize, and transform > Resize and adjust resolution
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/resize-adjust-resolution/resampling-options.html

**Function Summary**
This reference explains the interpolation methods Photoshop can use when `Resample` is enabled.

**What This Entry Is For**
- helps users choose the right method for enlargement or reduction
- improves sharpness and artifact control during resizing
- clarifies when to use automatic versus manual method selection

**Where You Find It**
- `Image > Image Size > Resample` menu

**Primary Interface Elements**
- `Automatic`
- `Preserve Details 2.0`
- `Preserve Details`
- `Bicubic Smoother`
- `Bicubic Sharper`
- `Bicubic`
- `Nearest Neighbor`
- `Bilinear`

**How To Use It**
1. Open `Image > Image Size`.
2. Enable `Resample`.
3. Open the method menu next to `Resample`.
4. Choose a method based on whether you are enlarging, reducing, or preserving hard edges.

**User Interface Behavior**
- `Automatic` lets Photoshop choose based on the resize direction.
- detail-preserving methods usually work better for enlargement.
- sharpening-oriented methods are often better for reduction.

**Expected Result**
- Resized images retain more appropriate edge behavior, smoothness, or detail depending on the selected algorithm.
