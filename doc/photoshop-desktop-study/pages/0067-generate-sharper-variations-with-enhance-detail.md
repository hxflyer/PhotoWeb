# Generate Sharper Variations with Enhance Detail

- Entry Type: Generative variation refinement
- Category: Create, open, and import images > Create images
- Source: https://helpx.adobe.com/photoshop/desktop/create-open-import-images/create-images/generate-sharper-variations-with-enhance-detail.html

**Function Summary**
Enhance Detail creates a sharpened duplicate of an existing generative variation so you can compare a clearer, more detailed version against the original. It is a refinement step, not a standalone generation command.

**What This Function Does**
- Sharpens an existing generative variation
- Produces a refined duplicate rather than replacing the original
- Helps generated content blend more cleanly into the surrounding image

**Where You Find It**
- `Window > Properties`
- generative variation controls after a `Generative Fill` result
- `Enhance Detail` icon

**Primary Interface Elements**
- `Properties` panel
- variation thumbnails
- `Enhance Detail` icon
- original variation
- enhanced variation copy

**How To Use It**
1. Generate content first with `Generative Fill`.
2. Open the `Properties` panel with `Window > Properties` if it is not already visible.
3. Select the generated variation you want to refine.
4. Click the `Enhance Detail` icon.
5. Compare the enhanced duplicate with the original variation and keep the version you prefer.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the variation, click the refinement icon, and compare the results visually.
- `Menus`
  Use `Window > Properties` to reveal the variation controls.
- `Keyboard`
  No dedicated default hotkey is described for `Enhance Detail`.

**User Interface Behavior**
- Photoshop creates a duplicate refined version rather than destroying the original variation.
- Only generated variations larger than `1024` pixels are described as eligible for this refinement.
- The user can compare both versions and keep whichever blends better.

**Expected Result**
- A sharper alternate version becomes available for selection, making fine-detail integration easier in some images.

**Practical Use Cases**
- Improve clarity in a generated object before final compositing.
- Compare a softer original generation against a more detailed version.
- Refine a nearly correct generative fill result without regenerating from scratch.
