# Get Started

- Entry Type: Topic hub
- Category: Photoshop Desktop Help
- Source: https://helpx.adobe.com/photoshop/desktop/get-started.html

**Function Summary**
This entry is the starting hub for first-use Photoshop desktop topics. It groups installation, workspace setup, interface basics, toolbar and panel setup, preferences, and other early onboarding references.

**What This Hub Is For**
- onboarding to Photoshop desktop
- locating setup and installation guidance
- learning the main workspace structure
- finding introductory preference and interface topics

**Included Subtopics**
- Technical requirements and installation
- Learn the basics
- Set up toolbars and panels
- Settings and preferences

**How To Use This Entry**
- Start here if you are new to Photoshop desktop or setting up a new machine.
- Open the installation branch for hardware and setup requirements.
- Open the basics branch for workspace, home screen, and panel behavior.
- Use the preference branch when you need to customize Photoshop behavior rather than edit an image.

**Interface Context**
- This is a documentation hub, not a Photoshop tool or command.
- It organizes beginner and setup documentation into logical entry points.

**Expected Result**
- You can quickly decide which onboarding or setup topic to open next.
