# Generate Images Guided by a Reference Image

- Entry Type: Prompt-and-reference image generation feature
- Category: Create, open, and import images > Create images
- Source: https://helpx.adobe.com/photoshop/desktop/create-open-import-images/create-images/generate-images-using-reference-image.html

**Function Summary**
This feature creates original images from a text prompt while using a reference image to influence style or composition. It is used when prompt text alone is not enough to control the look or structure of the generated image.

**What This Function Does**
- Generates a new image from a prompt
- Uses a reference image to influence either visual style or scene composition
- Combines prompt guidance with stronger visual direction

**Where You Find It**
- new document workflow
- `Generate Image`
- Contextual Task Bar
- `Generate Image` dialog

**Primary Interface Elements**
- `Generate Image`
- `Reference image`
- `Style`
- `Composition`
- `Choose Image`
- `Gallery`
- prompt entry box
- `Generate`

**How To Use It**
1. Start in a new document.
2. Select `Generate Image` from the Contextual Task Bar.
3. In the dialog, choose whether the reference image should guide `Style` or `Composition`.
4. Select `Choose Image` to upload a reference image, or pick a sample from the `Gallery`.
5. Enter the text prompt describing the image you want to generate.
6. Select `Generate`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Upload the reference image, choose style or composition guidance, enter the prompt, and generate.
- `Menus`
  This workflow is launched through the contextual generation controls in a new document.
- `Keyboard`
  No dedicated default hotkey is described for this dialog workflow.

**User Interface Behavior**
- `Style` guidance affects visual character and rendering feel.
- `Composition` guidance affects layout and scene arrangement more directly.
- The result is built from the combination of prompt text and reference-image influence rather than from either one alone.

**Expected Result**
- Generated images stay closer to the intended look or arrangement, especially when the prompt needs stronger visual anchoring.

**Practical Use Cases**
- Match a concept image's composition while changing the subject matter through prompts.
- Reuse a visual style direction across multiple generated concepts.
- Start ideation with stronger creative consistency than prompt-only generation.
