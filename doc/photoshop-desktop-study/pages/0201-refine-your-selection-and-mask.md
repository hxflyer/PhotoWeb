# Refine Your Selection and Mask

- Entry Type: Select and Mask reference
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/refine-your-selection-and-mask.html

**Function Summary**
`Select and Mask` provides the main workspace for previewing, refining, and outputting difficult selections.

**What This Function Does**
- previews the selection against different backgrounds
- refines hair, fur, edges, and soft transitions
- outputs the result as a selection, mask, or new layer-based result

**Where You Find It**
- `Select and Mask` from supported selection workflows

**Primary Interface Elements**
- `View Mode`
- `Onion Skin`
- `Marching Ants`
- `Overlay`
- `On Black`
- `On White`
- `Black and White`
- `On Layers`
- `Show Edge`
- `Show Original`
- `High Quality Preview`
- `Transparency`
- `Opacity`
- `Color Aware`
- `Object Aware`

**How To Use It**
1. Enter `Select and Mask` from an existing selection.
2. Choose a `View Mode` that makes edge problems easy to see.
3. Switch between `Color Aware` and `Object Aware` based on the subject.
4. Use refinement tools and edge controls while watching the live preview.
5. Output the result to the destination format you need.

**Mouse, Keyboard, and Menu Input**
- Keyboard: press `F` to cycle view modes.
- Keyboard: press `X` to temporarily hide view overlays.

**User Interface Behavior**
- Different view modes reveal different edge problems, so the same selection can look cleaner or dirtier depending on the preview.

**Expected Result**
- A rough selection becomes a cleaner mask or selection ready for compositing or local edits.
