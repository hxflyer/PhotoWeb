# Create Text Selection Borders

- Entry Type: Text-to-selection workflow
- Category: Text and typography > Type layers and creation
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/type-layers-creation/create-text-selection-borders.html

**Function Summary**
Text selection borders create an active selection shaped like the visible letters of a type layer.

**What This Function Does**
- Loads the opacity shape of text as a selection.
- Lets users fill, mask, stroke, copy, or edit areas based on letterforms.
- Keeps the original type layer intact unless later edits change or rasterize it.

**Where You Find It**
- `Cmd`-click a type layer thumbnail on macOS or `Ctrl`-click on Windows.
- Selection-related commands after the text outline is active.

**Primary Interface Elements**
- Marching ants around the letter shapes.
- Type layer thumbnail in the `Layers` panel.
- Active selection shown on the canvas.

**How To Use It**
1. Locate the type layer in the `Layers` panel.
2. `Cmd/Ctrl`-click the layer thumbnail to load the text shape as a selection.
3. Use the selection for a mask, fill, stroke, copy, or adjustment.
4. Deselect with `Cmd+D` or `Ctrl+D` when finished.

**Expected Result**
- An active selection matches the text letterforms.
