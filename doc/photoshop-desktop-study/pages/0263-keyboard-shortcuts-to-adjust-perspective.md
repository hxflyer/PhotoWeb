# Keyboard Shortcuts to Adjust Perspective

- Entry Type: Perspective Warp shortcut reference
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/keyboard-shortcuts-to-adjust-perspective.html

**Function Summary**
This reference lists keyboard controls for switching modes and manipulating `Perspective Warp` planes.

**Primary Shortcuts**
- `L`: layout mode
- `W`: warp mode
- `Enter` or `Return`: commit
- `H`: temporarily hide extras or access view-related behavior depending on context
- `Shift`: straighten or toggle straightening behavior on selected edges

**Expected Result**
- Users can adjust perspective faster without relying only on toolbar controls.
