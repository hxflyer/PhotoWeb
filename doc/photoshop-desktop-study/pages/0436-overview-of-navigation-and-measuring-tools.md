# Overview of Navigation and Measuring Tools

- Entry Type: Tool reference
- Category: Grids, guides, and measurement
- Source: https://helpx.adobe.com/photoshop/desktop/grids-guides-measurement/overview-navigation-measuring-tools.html

**Function Summary**
Navigation and measuring tools control how users inspect, move around, zoom, sample, and measure within a document.

**Where You Find It**
- Toolbar tools such as `Hand Tool`, `Zoom Tool`, `Eyedropper Tool`, `Ruler Tool`, and `Count Tool`.
- `Window > Navigator`, `Window > Info`, and `Window > Measurement Log`.

**Primary Interface Elements**
- Zoom percentage, hand panning cursor, ruler line, angle/distance readouts, info values, and count labels.

**Expected Result**
- Users can inspect artwork at the right scale and capture measurement values from the canvas.
