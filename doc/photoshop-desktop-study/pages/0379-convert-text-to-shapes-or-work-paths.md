# Convert Text to Shapes or Work Paths

- Entry Type: Text conversion workflow
- Category: Text and typography > Type layers and creation
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/type-layers-creation/convert-text-shapes-work-paths.html

**Function Summary**
Text conversion turns editable characters into vector shapes or a work path for direct outline editing.

**What This Function Does**
- `Convert to Shape` turns text into a shape layer made from letter outlines.
- `Create Work Path` creates a path from the text outlines while leaving the type layer available.
- Enables anchor-point editing, custom vector effects, and precise outline manipulation.

**Where You Find It**
- `Type > Convert to Shape`.
- `Type > Create Work Path`.
- Layer context menu for selected type layers where available.

**Primary Interface Elements**
- Shape layer in the `Layers` panel.
- Work Path in the `Paths` panel.
- Anchor points and direction handles.

**How To Use It**
1. Select the type layer.
2. Choose `Type > Convert to Shape` for editable vector outlines as a shape layer.
3. Choose `Type > Create Work Path` if you need a path outline while keeping the type layer.
4. Edit outlines with path and selection tools.

**Expected Result**
- Photoshop creates vector outlines from the text; shape conversion removes normal text editability, while work-path creation preserves the original type layer.
