# Copy and Paste Selections

- Entry Type: Selection transfer workflow
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/copy-and-paste-selections.html

**Function Summary**
This workflow copies or cuts selected image content and pastes it as new content within the same document or into another document.

**What This Function Does**
- copies selected pixels to the clipboard
- pastes selected content as a new layer
- supports direct drag-copy behavior with the `Move Tool`

**Where You Find It**
- `Edit > Copy`
- `Edit > Copy Merged`
- `Edit > Cut`
- `Edit > Paste`
- `Move Tool`

**Primary Interface Elements**
- `Copy`
- `Copy Merged`
- `Cut`
- `Paste`
- `Move Tool`
- `Free Transform`

**How To Use It**
1. Make a selection.
2. Use `Edit > Copy`, `Copy Merged`, or `Cut` depending on whether you want layer-only, merged, or removed content.
3. Open the destination document or target area.
4. Use `Edit > Paste` to create a new pasted layer, or use the `Move Tool` with modifier keys to drag-copy directly.
5. Transform the pasted content if the document resolutions differ.

**Mouse, Keyboard, and Menu Input**
- Menu path: `Edit > Copy`, `Edit > Copy Merged`, `Edit > Cut`, `Edit > Paste`
- Mouse: hold `Alt` on Windows or `Option` on macOS and drag with the `Move Tool` to duplicate while moving.

**User Interface Behavior**
- Pasted content appears on a new layer.
- Pixel dimensions are preserved when pasting across files, which can create apparent size mismatches if document resolution differs.

**Expected Result**
- Selected image content is duplicated or moved into a new editable layer context.
