# Modify Text Paths

- Entry Type: Path editing workflow
- Category: Text and typography > Text on paths and shapes
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/text-on-paths-shapes/modify-text-paths.html

**Function Summary**
Modifying a text path changes the vector path or shape that controls how attached text flows.

**What This Function Does**
- Edits anchor points, curves, and handles on the path used by text.
- Reflows path text or area text as the path changes.
- Keeps the text layer editable while changing the layout path.

**Where You Find It**
- `Direct Selection Tool` for anchor and handle edits.
- `Path Selection Tool` for selecting and moving the whole path.
- `Pen Tool` for adding or deleting anchor points.

**Primary Interface Elements**
- Anchor points.
- Direction handles.
- Path outline.
- Text baseline or area boundary.

**How To Use It**
1. Select the path text layer.
2. Choose the `Direct Selection Tool`.
3. Click the path to reveal anchor points and handles.
4. Drag anchor points or direction handles to reshape the path.
5. Review the text reflow and adjust start/end handles if needed.

**Expected Result**
- The text follows the edited path or reflows inside the edited shape.
