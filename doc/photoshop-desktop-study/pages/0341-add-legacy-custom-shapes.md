# Add Legacy Custom Shapes

- Entry Type: Custom shape library workflow
- Category: Draw shapes and paths > Create shapes
- Source: https://helpx.adobe.com/photoshop/desktop/draw-shapes-paths/create-shapes/add-legacy-custom-shapes.html

**Function Summary**
This workflow restores older Photoshop custom shape libraries to the shape preset list.

**Where You Find It**
- `Shapes` panel menu

**How To Use It**
1. Open the `Shapes` panel.
2. Open the panel menu.
3. Choose the legacy shapes option.
4. Browse the added shape groups.

**Expected Result**
- Legacy custom shapes become available for the `Custom Shape Tool`.
