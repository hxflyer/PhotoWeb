# Remove Imperfections with the Spot Healing Brush

- Entry Type: Spot Healing workflow
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/spot-healing-brush-tool.html

**Function Summary**
The `Spot Healing Brush Tool` removes small blemishes by automatically sampling nearby pixels.

**What This Function Does**
- repairs dust, blemishes, scratches, and small flaws
- blends replacement texture with surrounding tone
- works without manually defining a source point

**Where You Find It**
- `Spot Healing Brush Tool` in the toolbar

**Primary Interface Elements**
- brush size
- `Content-Aware`
- `Create Texture`
- `Proximity Match`
- `Sample All Layers`

**How To Use It**
1. Select the `Spot Healing Brush Tool`.
2. Set brush size slightly larger than the imperfection.
3. Choose the healing type and sampling options.
4. Click or drag over the flaw.

**Expected Result**
- The marked imperfection is replaced with blended surrounding detail.
