# Text and Typography

- Entry Type: Topic hub
- Category: Photoshop Desktop Help
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography.html

**Function Summary**
Text and typography features create editable type layers, control font appearance, place text on paths or inside shapes, insert glyphs, and support multilingual scripts.

**What This Area Controls**
- Text layer creation, editing, movement, resizing, rotation, and color.
- Font selection, font activation, missing-font replacement, font matching, and OpenType controls.
- Decorative type workflows such as warped type, text filled with imagery, text on paths, and text converted to vector shapes.
- Character-level controls for glyphs, emoji, alternates, and OpenType SVG color fonts.
- International text behavior through language, script, direction, shaping, and composer settings.

**Where You Find It**
- `Type Tool` in the toolbar.
- `Type` menu in the application menu bar.
- `Window > Character`, `Window > Paragraph`, `Window > Glyphs`, `Window > Properties`, and `Window > Layers`.
- Text-specific controls in the options bar and contextual on-canvas controls.

**Primary Interface Elements**
- Text insertion cursor and text bounding box.
- Type layer entries in the `Layers` panel.
- Font family, font style, size, anti-aliasing, alignment, color, warp, and commit/cancel controls.
- Character, Paragraph, Properties, and Glyphs panels.

**Expected Result**
- Users can choose the correct Photoshop type workflow for editable design text, layout text, font management, decorative lettering, or multilingual production.
