# Resolve Program Error While Saving Files

- Entry Type: Save error troubleshooting workflow
- Category: Troubleshoot
- Source: https://helpx.adobe.com/photoshop/desktop/troubleshoot/resolve-program-error-saving-files.html

**Function Summary**
Save error troubleshooting resolves program errors caused by file corruption, unsupported features, permissions, storage, GPU issues, or preferences.

**Where You Find It**
- Save error dialogs.
- File Handling preferences.
- Scratch Disk and Performance preferences.

**Primary Interface Elements**
- Error message.
- Save As, Save a Copy, and format menu.
- Layer and channel structure.
- Storage permission and free-space indicators.

**How To Use It**
1. Try `Save As` or `Save a Copy` to a new local location.
2. Check available disk and scratch disk space.
3. Simplify problematic layers, channels, or paths in a duplicate file.
4. Try another format such as PSD or PSB if size/features require it.
5. Update Photoshop or reset preferences if the problem persists.

**Expected Result**
- The document saves successfully or the failing file/format condition is isolated.
