# Hover Layer Bounds in the Move Tool

- Entry Type: Move tool preview feature
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/hover-layer-bounds-in-the-move-tool.html

**Function Summary**
`Hover Layer Bounds` previews the outline of a layer or group when you hover with the `Move Tool`.

**What This Function Does**
- reveals object boundaries before selection
- highlights the matching layer in the `Layers` panel
- improves picking accuracy in dense documents

**Where You Find It**
- `Move Tool` options
- `Guides, Grid & Slices` settings

**Primary Interface Elements**
- `Move Tool`
- `Auto-Select`
- `Layer`
- `Group`
- hover boundary outline
- layer highlight

**How To Use It**
1. Activate the `Move Tool`.
2. Enable hover-boundary behavior from the tool's additional options if needed.
3. Hover over content on the canvas to preview its bounds.
4. Use `Auto-Select` set to `Layer` or `Group` depending on the level you want.
5. Adjust appearance settings in `Guides, Grid & Slices` if the outline is hard to see.

**Mouse, Keyboard, and Menu Input**
- Mouse: hover over content to display bounds.
- Keyboard: use `[` and `]` to cycle overlapping hover candidates.

**User Interface Behavior**
- Photoshop outlines the hovered object and highlights the corresponding layer in the `Layers` panel before you click.

**Expected Result**
- Layer targeting becomes easier when multiple objects overlap or sit close together.
