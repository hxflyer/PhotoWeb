# Control the Movement of a Selection

- Entry Type: Selection movement controls reference
- Category: Make selections > Refine and modify selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/refine-modify-selections/control-the-movement-of-a-selection.html

**Function Summary**
This reference explains the keyboard-assisted ways to move a selection with angle constraints and pixel-precise nudging.

**What This Entry Is For**
- supports precise repositioning
- avoids accidental diagonal drift
- helps align selections accurately

**Where You Find It**
- selection movement with the `Move Tool`

**Primary Interface Elements**
- `Move Tool`
- active selection

**How To Use It**
1. Make a selection and activate the `Move Tool`.
2. Drag normally to reposition it.
3. Hold `Shift` while dragging to constrain movement to `45°` increments.
4. Use arrow keys for `1`-pixel nudges.
5. Use `Shift` plus arrow keys for `10`-pixel nudges.

**Mouse, Keyboard, and Menu Input**
- Mouse: drag to move.
- Keyboard: `Shift` constrains drag angle.
- Keyboard: arrow keys nudge by `1`.
- Keyboard: `Shift` plus arrow keys nudge by `10`.

**Expected Result**
- The selection moves in a controlled, predictable way suitable for detailed alignment work.
