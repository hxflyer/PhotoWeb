# View Full-Resolution Preview in the Preview Panel

- Entry Type: Content-Aware Fill preview navigation
- Category: Repair and retouch > Remove objects and fill space
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/remove-objects-fill-space/view-full-resolution-preview-in-the-preview-panel.html

**Function Summary**
The preview controls let users inspect a `Content-Aware Fill` result at higher magnification before committing it.

**What This Function Does**
- zooms the preview up to detailed inspection levels
- pans across the preview image
- helps catch repeating texture, edge artifacts, or poor fill choices

**Primary Interface Elements**
- `Preview` panel
- zoom slider
- zoom percentage box
- `Zoom Tool`
- `Hand Tool`

**How To Use It**
1. Open the `Content-Aware Fill` workspace from a selection.
2. Use the preview panel zoom slider or percentage field.
3. Use the `Zoom Tool` to inspect a specific area.
4. Use the `Hand Tool` to pan across the preview.

**Mouse, Keyboard, and Menu Input**
- Mouse: drag the preview zoom slider.
- Keyboard: press `H` while using another tool to temporarily switch to the `Hand Tool`.

**Expected Result**
- The fill result can be checked at full detail before it is applied to the document.
