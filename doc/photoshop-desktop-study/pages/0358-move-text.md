# Move Text

- Entry Type: Text positioning workflow
- Category: Text and typography > Get started with text
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/get-started-text/move-text.html

**Function Summary**
Moving text repositions an editable type layer on the canvas without changing the text content.

**What This Function Does**
- Moves point text or paragraph text as a layer.
- Lets users align type with other artwork, guides, or layout elements.
- Preserves the type layer and its formatting.

**Where You Find It**
- `Move Tool`, shortcut `V`.
- Active `Type Tool` outside the text-editing insertion area.
- `Layers` panel for selecting the target type layer.

**Primary Interface Elements**
- Move Tool cursor.
- Layer bounding box when transform controls are visible.
- Smart Guides, alignment guides, and canvas coordinates.

**How To Use It**
1. Select the type layer.
2. Press `V` for the `Move Tool`.
3. Drag the text with the mouse to a new location.
4. Use arrow keys for small nudges, or `Shift` plus arrow keys for larger nudges.
5. Use Smart Guides or alignment controls to position text precisely.

**Expected Result**
- The type layer moves to the new canvas position.
