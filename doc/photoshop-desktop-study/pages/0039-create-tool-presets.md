# Create Tool Presets

- Entry Type: Tool configuration
- Category: Get started > Set up toolbars and panels
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/set-up-toolbars-panels/create-tool-preset.html

**Function Summary**
Tool presets save a tool together with chosen settings so the same behavior can be recalled quickly later. This is useful when you repeat the same brush, selection, or adjustment setup across multiple projects.

**What This Function Does**
- Stores the current settings of a tool as a reusable preset
- Speeds up repeated workflows by removing repeated setup steps
- Lets you filter, sort, and display presets in different ways

**Where You Find It**
- `Window > Tool Presets`
- Tool Presets dropdown near the selected tool in the upper-left area of the Options bar

**Primary Interface Elements**
- `Tool Presets` panel
- `Create New Tool Preset`
- Options bar
- panel menu
- `Show All Tool Presets`
- `Sort by Tool`
- `Show Current Tool Presets`
- `Text Only`
- `Small List`
- `Large List`

**How To Use It**
1. Select the tool you want to save.
2. Set its options in the Options bar.
3. Open `Window > Tool Presets`, or use the preset dropdown beside the active tool.
4. Click `Create New Tool Preset`.
5. Enter a descriptive preset name and confirm with `OK`.
6. To reuse it later, select the preset from the Tool Presets panel or dropdown.

**Preset Management Options**
- `Show All Tool Presets`
  Displays every loaded preset
- `Sort by Tool`
  Groups presets by tool type
- `Show Current Tool Presets`
  Filters the list to presets for the active tool only
- `Text Only`, `Small List`, `Large List`
  Changes how the presets are displayed visually

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Set tool options, create a preset, and click a saved preset when you want to apply it.
- `Menus`
  Use `Window > Tool Presets` to open the preset panel.
- `Keyboard`
  No dedicated default hotkey is described for creating tool presets.

**User Interface Behavior**
- The preset stores the active tool together with its current options.
- Selecting a preset reconfigures the active tool to match the stored setup.
- Tool Presets can be shown as a full panel or accessed through the compact dropdown in the Options bar.

**Expected Result**
- Frequently used tool configurations become one-click selections instead of multi-step setups.

**Practical Use Cases**
- Save brush variants for sketching, masking, and cleanup work.
- Store favorite selection tool settings for repeated asset extraction.
- Maintain consistent tool behavior across multiple files or projects.
