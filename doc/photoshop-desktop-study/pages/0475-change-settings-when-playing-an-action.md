# Change Settings When Playing an Action

- Entry Type: Action dialog toggle workflow
- Category: Automate tasks > Actions
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/create-record-actions/change-settings-playing-action.html

**Function Summary**
Dialog toggles let an action pause at selected commands so users can change settings during playback.

**Where You Find It**
- Actions panel dialog toggle column beside recorded commands.

**Primary Interface Elements**
- Dialog toggle icon.
- Recorded command list.
- Playback dialogs for enabled commands.

**How To Use It**
1. Open the Actions panel.
2. Expand the action.
3. Click the dialog toggle beside commands that need manual settings.
4. Play the action.
5. Adjust settings when Photoshop pauses at that command.

**Expected Result**
- The action runs automatically except at commands where user input is enabled.
