# Choose a Spot Color

- Entry Type: Spot color selection workflow
- Category: Adjust color > Choose colors
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/choose-colors/choose-a-spot-color.html

**Function Summary**
Spot color selection chooses a named ink color from a color library for print workflows.

**Primary Interface Elements**
- `Color Libraries`
- spot color book
- named color swatches

**How To Use It**
1. Open the Color Picker.
2. Open `Color Libraries`.
3. Choose the appropriate spot color library.
4. Select the named spot color.

**Expected Result**
- The selected spot color can be used in print-oriented artwork or channels.
