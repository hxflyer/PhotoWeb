# Remove Unwanted Objects and Distractions

- Entry Type: Remove Tool workflow
- Category: Repair and retouch > Remove objects and fill space
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/remove-objects-fill-space/remove-unwanted-objects-and-distractions.html

**Function Summary**
The `Remove Tool` removes brushed or circled distractions and fills the result automatically.

**What This Function Does**
- removes unwanted objects from photos
- works through brush-like marking on the canvas
- automatically creates replacement pixels

**Where You Find It**
- `Remove Tool` in the toolbar

**Primary Interface Elements**
- `Remove Tool`
- brush size
- options bar

**How To Use It**
1. Open the image.
2. Select the `Remove Tool`.
3. Set the brush slightly larger than the distraction.
4. Draw a loop or brush over the unwanted area.

**Expected Result**
- The marked object is removed and the surrounding area is filled automatically.
