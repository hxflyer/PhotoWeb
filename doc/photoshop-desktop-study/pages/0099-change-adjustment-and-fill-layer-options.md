# Change Adjustment and Fill Layer Options

- Entry Type: Adjustment and preset management workflow
- Category: Create and manage layers > Color adjustment and fill layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/color-adjustment-fill-layers/change-adjustment-and-fill-layer-options.html

**Function Summary**
This workflow focuses on editing existing adjustment and fill layers and on working with the updated adjustment preset interface. It explains how preset management and adjustment access have been streamlined.

**What This Function Does**
- Lets users refine existing adjustment and fill layer settings
- Keeps those edits non-destructive
- Introduces the updated preset organization and navigation layout

**Where You Find It**
- existing adjustment or fill layers
- `Adjustments Presets` panel
- updated flyout menu

**Primary Interface Elements**
- selected adjustment or fill layer
- `Adjustments Presets`
- `Presets`
- `Single adjustments`
- enhanced preset icons
- drag-and-drop preset groups
- updated flyout menu

**How To Use It**
1. Select the adjustment or fill layer you want to change.
2. Edit its settings or mask to refine where the effect appears.
3. Use the updated `Adjustments Presets` interface if you want to browse, organize, or reapply presets more efficiently.

**Preset Interface Changes Covered**
- `Tabbed navigation`
  Separates `Presets` from `Single adjustments`
- `Enhanced preset icons`
  Makes visual identification clearer
- `Organized preset groups`
  Supports drag-and-drop arrangement and grouping
- `Updated flyout access`
  Makes saved presets easier to reach

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the layer, adjust settings, drag presets into groups, and use the flyout menu.
- `Menus`
  This workflow is mainly handled through panels rather than a classic top-level menu path.
- `Keyboard`
  No dedicated default hotkey is described for the updated preset management behavior.

**User Interface Behavior**
- Adjustment and fill layers remain fully editable until they are merged or rasterized.
- Masks still define where the effect appears.
- The revised preset layout is intended to reduce friction when moving between one-off adjustments and reusable looks.

**Expected Result**
- Existing non-destructive correction layers become easier to refine and easier to manage through the newer preset interface.
