# Export Artboards as PDF

- Entry Type: Artboard PDF export workflow
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/export-artboards-as-pdf.html

**Function Summary**
Exporting artboards as PDF turns one or more Photoshop artboards into PDF pages or files for review or sharing.

**Where You Find It**
- `File > Export > Artboards to PDF`.

**Primary Interface Elements**
- Source artboard selection.
- Destination folder.
- PDF options.
- Multi-page or separate document behavior where available.

**How To Use It**
1. Organize the document artboards.
2. Choose `File > Export > Artboards to PDF`.
3. Select which artboards to export.
4. Choose destination and PDF options.
5. Run the export.

**Expected Result**
- Photoshop exports artboards into PDF output.
