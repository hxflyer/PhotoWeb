# Save and Apply Settings in Match Color

- Entry Type: Match Color preset workflow
- Category: Adjust color > Selective color adjustments
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/selective-color-adjustments/save-and-apply-settings-in-match-color.html

**Function Summary**
`Match Color` statistics can be saved and reloaded so similar color adjustments can be applied across files.

**Where You Find It**
- `Image > Adjustments > Match Color`

**Primary Interface Elements**
- `Image Statistics`
- `Save Statistics`
- `Load Statistics`

**How To Use It**
1. Open `Match Color`.
2. Use `Save Statistics` to save the current color settings.
3. Later, open `Match Color` again.
4. Use `Load Statistics` to apply the saved settings.

**Expected Result**
- A saved color-matching setup can be reused for consistency across images.
