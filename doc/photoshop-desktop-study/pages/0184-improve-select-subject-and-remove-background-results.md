# Improve Select Subject and Remove Background Results

- Entry Type: Processing-mode preference workflow
- Category: Make selections > Automatic and color-based selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/automatic-color-based-selections/improved-select-subject-and-remove-background-results.html

**Function Summary**
This preference controls whether `Select Subject` and `Remove Background` use device processing or cloud processing.

**What This Function Does**
- changes the processing source behind automatic selection features
- can improve detail handling, especially around hair or fine edges
- affects selection quality and performance tradeoffs

**Where You Find It**
- `Preferences > Image Processing`

**Primary Interface Elements**
- `Select Subject and Remove Background`
- `Device`
- `Cloud`
- `Select and Mask`

**How To Use It**
1. Open an image.
2. Open `Preferences > Image Processing`.
3. Choose `Device` or `Cloud` for `Select Subject and Remove Background`.
4. Run your selection workflow and refine it in `Select and Mask` if needed.

**Mouse, Keyboard, and Menu Input**
- Menu path on Windows: `Edit > Preferences > Image Processing`
- Menu path on macOS: `Photoshop > Settings > Image Processing`

**User Interface Behavior**
- `Device` processing is generally faster and local.
- `Cloud` processing may provide different interpretation quality depending on the subject and edge complexity.

**Expected Result**
- Automatic subject and background selections use the chosen processing mode for later selection commands.
