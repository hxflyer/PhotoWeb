# Import Preset Style Libraries

- Entry Type: Style-library import workflow
- Category: Create and manage layers > Apply layer effects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/apply-layer-effects/import-preset-style-libraries.html

**Function Summary**
This workflow loads preset style libraries into Photoshop so additional reusable layer styles become available in the Styles panel.

**What This Function Does**
- Imports `.asl` style library files
- Expands the available set of reusable preset styles
- Supports faster application of common text, button, and decorative effects

**Where You Find It**
- `Window > Styles`
- Styles panel menu
- `Import Styles`

**Primary Interface Elements**
- `Styles`
- panel menu
- `Import Styles`
- `.asl` style library file
- `Open`

**How To Use It**
1. Open the `Styles` panel with `Window > Styles`.
2. Open the Styles panel menu.
3. Choose `Import Styles`.
4. Browse to the `.asl` style library you want to load.
5. Select the file and choose `Open`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Open the Styles panel menu, browse to the library file, and import it.
- `Menus`
  Use `Window > Styles` and then the panel menu's `Import Styles` command.
- `Keyboard`
  No dedicated default hotkey is described for style-library import.

**User Interface Behavior**
- Imported styles become available in the Styles panel for later application.
- Style libraries extend reusable non-destructive effects rather than painting fixed pixels into layers.

**Expected Result**
- Additional preset looks become available for quick application to layers.
