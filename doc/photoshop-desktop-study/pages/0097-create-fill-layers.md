# Create Fill Layers

- Entry Type: Fill-layer creation workflow
- Category: Create and manage layers > Color adjustment and fill layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/color-adjustment-fill-layers/create-fill-layers.html

**Function Summary**
This workflow creates editable fill layers for solid colors, gradients, or patterns. Fill layers add design content non-destructively and stay editable after they are created.

**What This Function Does**
- Adds a `Solid Color` fill layer
- Adds a `Gradient` fill layer
- Adds a `Pattern` fill layer
- Keeps fill content separate from the original pixels

**Where You Find It**
- `Layer > New Fill Layer`

**Primary Interface Elements**
- `New Fill Layer`
- `Solid color`
- `Gradient`
- `Pattern`
- layer name field

**How To Use It**
1. Open the image or document where you want the fill.
2. Choose `Layer > New Fill Layer`.
3. Select `Solid color`, `Gradient`, or `Pattern`.
4. Name the layer and confirm with `Enter` or `Return`.
5. Adjust the fill-specific settings after creation if Photoshop opens the related control dialog.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Choose the fill type from the menu and confirm the layer creation settings.
- `Menus`
  Use `Layer > New Fill Layer`.
- `Keyboard`
  Use `Enter` or `Return` for naming confirmation.

**User Interface Behavior**
- The fill appears as its own separate layer in the stack.
- Because the fill is layer-based, it can be hidden, masked, blended, or reordered like other layers.

**Expected Result**
- A new editable fill element is added without directly painting or permanently altering the original image pixels.
