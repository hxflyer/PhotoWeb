# Explore Beyond the Canvas with Generative Expand

- Entry Type: AI-assisted canvas extension feature
- Category: Create, open, and import images > Create images
- Source: https://helpx.adobe.com/photoshop/desktop/create-open-import-images/create-images/explore-beyond-the-canvas-with-generative-expand.html

**Function Summary**
Generative Expand extends an image beyond its current canvas boundaries by generating matching content into newly added space. It is used when you need more background, a wider composition, or extra room for layout changes.

**What This Function Does**
- Enlarges the canvas area
- Fills newly exposed regions with AI-generated content
- Keeps expansion tied to the existing image context
- Returns multiple result variations for review

**Where You Find It**
- `Crop` tool workflow
- Contextual Task Bar
- `Generative Expand`
- `Properties` panel for returned variations

**Primary Interface Elements**
- `Crop` tool
- crop handles
- `Generative Expand`
- model picker
- prompt field
- `Generate`
- `Properties` panel

**How To Use It**
1. Select the `Crop` tool from the Tools panel.
2. Drag the crop handles outward to enlarge the canvas beyond the current image edges.
3. Select `Generative Expand` from the Contextual Task Bar.
4. Choose the AI model from the model picker when available.
5. Leave the prompt blank if you want Photoshop to extend the image automatically based on surrounding content.
6. Select `Generate`.
7. Review the returned variations in the `Properties` panel and choose the one that fits best.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Drag the crop boundary outward, then trigger generation from the task bar and choose a variation.
- `Menus`
  This workflow is driven mainly from the Crop tool and contextual generative controls.
- `Keyboard`
  Any Crop tool shortcut depends on the user's Photoshop setup, but the generative step itself is primarily mouse-driven.

**User Interface Behavior**
- Photoshop treats the newly added canvas area as the target region for generated content.
- A blank prompt relies on contextual image continuation rather than explicit instruction.
- Returned results appear as selectable variations rather than one irreversible fill.

**Expected Result**
- The image extends beyond its original boundaries with generated content that attempts to match the existing scene.

**Practical Use Cases**
- Widen an image for a banner or social layout.
- Add headroom or side space without manual cloning.
- Reframe a composition after the original crop feels too tight.
