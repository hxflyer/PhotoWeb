- Entry Type: Tool family reference
- Category: Draw shapes and paths > Create shapes
- Source: https://helpx.adobe.com/photoshop/desktop/draw-shapes-paths/create-shapes/drawing-tools-overview.html

**Function Summary**
This reference lists the main tools for drawing shapes, paths, and vector segments.

**Primary Tools**
- rectangle and ellipse tools
- polygon and star tools
- line tool
- custom shape tool
- Pen tools
- path selection tools

**Expected Result**
- Users can pick the correct drawing tool before creating vector or pixel-based artwork.
