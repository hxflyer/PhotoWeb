# Rearrange Document Windows

- Entry Type: Window management
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/rearrange-document-windows.html

**Function Summary**
This function controls how multiple open documents are displayed inside Photoshop. It lets you reorder tabs, float documents into separate windows, redock them into tab groups, and build viewing layouts that make multi-file work easier.

**What This Function Does**
- Reorders document tabs
- Separates documents into floating windows
- Redocks floating windows into tabbed groups
- Helps compare or edit several files at the same time

**Where You Find It**
- Document tab area at the top of open files
- `Window > Arrange`

**Primary Interface Elements**
- document tab
- tab group
- floating document window
- `Window` menu
- `Arrange` submenu
- `Float in Window`
- `Float All in Windows`

**How To Use It**
1. Open more than one document in Photoshop.
2. Drag a document tab left or right to change its position in the tab row.
3. Drag a tab away from the tab group to undock it into a floating window.
4. Drag a floating document back toward the tab bar of another document group to redock it.
5. Drag one document window onto another to build stacked or tiled viewing arrangements.
6. Drag a selection over another document tab when you need to switch documents while keeping the drag operation active.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Drag tabs and document windows to reorder, float, redock, or tile them.
- `Menus`
  Use `Window > Arrange > Float in Window` for the current document, or `Window > Arrange > Float All in Windows` to separate all open documents.
- `Keyboard`
  No dedicated default hotkey is described for this workspace action.

**User Interface Behavior**
- Dragging a tab out of a group creates a separate floating document window.
- Dragging a floating window back into a tab area merges it into that tab group again.
- Floating windows can be placed side by side for comparison work.
- Rearranging document windows changes viewing layout only and does not alter file content.

**Expected Result**
- Multiple documents become easier to compare, transfer content between, or manage in parallel.
- You can choose between a compact tabbed workflow and a more spread-out floating-window workflow.

**Practical Use Cases**
- Compare two retouched versions of the same image side by side.
- Keep a reference image visible while editing another file.
- Move assets between several open Photoshop documents more efficiently.
