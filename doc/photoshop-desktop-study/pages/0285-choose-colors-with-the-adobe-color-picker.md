# Choose Colors with the Adobe Color Picker

- Entry Type: Color Picker workflow
- Category: Adjust color > Choose colors
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/choose-colors/choose-colors-with-adobe-color-picker.html

**Function Summary**
The Adobe Color Picker selects exact colors using visual fields, sliders, numeric values, and color-model inputs.

**Primary Interface Elements**
- color field
- hue slider
- new/current color preview
- HSB values
- RGB values
- Lab values
- CMYK values
- hexadecimal field

**How To Use It**
1. Open the Color Picker from a foreground/background swatch or compatible color control.
2. Drag in the color field or hue slider.
3. Enter numeric values for precise color.
4. Confirm the selected color.

**Expected Result**
- The selected color becomes active for the chosen swatch or color control.
