# Mask All Objects in a Layer

- Entry Type: Auto-mask workflow
- Category: Make selections > Automatic and color-based selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/automatic-color-based-selections/mask-all-objects-in-a-layer.html

**Function Summary**
`Mask All Objects` detects multiple objects in a layer and creates separate masks for them automatically.

**What This Function Does**
- finds multiple objects in one layer
- creates independent masks for each detected object
- speeds up multi-object isolation workflows

**Where You Find It**
- layer context menu in the `Layers` panel

**Primary Interface Elements**
- `Layers` panel
- layer context menu
- `Mask All Objects`

**How To Use It**
1. Select a layer in the `Layers` panel.
2. Right-click on Windows or `Control`-click on macOS.
3. Choose `Mask All Objects`.

**User Interface Behavior**
- Photoshop analyzes the selected layer and generates separate masks for detected objects instead of asking for manual tracing first.

**Expected Result**
- Object masks are created automatically for the chosen layer.
