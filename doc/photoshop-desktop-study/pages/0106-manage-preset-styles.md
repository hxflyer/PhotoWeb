# Manage Preset Styles

- Entry Type: Preset style management workflow
- Category: Create and manage layers > Apply layer effects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/apply-layer-effects/manage-preset-styles.html

**Function Summary**
This workflow creates, renames, deletes, and exports preset styles. It is used when a layer effect configuration needs to be saved as a reusable library item.

**What This Function Does**
- Saves a configured layer effect as a preset style
- Renames and deletes existing preset styles
- Exports selected styles as a reusable library

**Where You Find It**
- Layers panel
- `Add a layer style`
- `Layer Style` dialog
- `Window > Styles`
- style panel more-options menu

**Primary Interface Elements**
- `New Style`
- `Styles`
- `Delete style`
- `Export Selected Styles`
- `Save`

**How To Create a Preset Style**
1. Select a layer in the Layers panel.
2. Add a default layer effect from the layer style menu.
3. Adjust the look in the `Layer Style` dialog.
4. Click `New Style`.
5. Enter a name and confirm.

**How To Rename a Preset Style**
1. Open `Window > Styles`.
2. Select the preset style.
3. Double-click its name and enter the new one.

**How To Delete a Preset Style**
1. Open `Window > Styles`.
2. Select the style to remove.
3. Choose `Delete style`.
4. Confirm the deletion.

**How To Save a Style Library**
1. Open `Window > Styles`.
2. Select the preset style you want to export.
3. Choose `Export Selected Styles` from the more-options menu.
4. Name the style library and save it.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Build the style, save it as a preset, then manage it from the Styles panel.
- `Menus`
  Use `Window > Styles` for preset management.
- `Keyboard`
  Use normal text-entry confirmation when naming or renaming styles.

**User Interface Behavior**
- Styles saved through `New Style` become reusable entries in the Styles panel.
- Exported style libraries can be reused later as `.asl`-based style collections through Photoshop's import workflows.

**Expected Result**
- Reusable layer-style looks become part of a manageable preset library.
