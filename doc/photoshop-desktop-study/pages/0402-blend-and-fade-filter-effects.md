# Blend and Fade Filter Effects

- Entry Type: Filter blending workflow
- Category: Effects and filters > Get started with filters
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/get-started-filters/blend-fade-filter-effects.html

**Function Summary**
Blend and fade controls reduce or mix a filter result with the original image after the filter is applied.

**Where You Find It**
- `Edit > Fade` immediately after applying a destructive filter.
- Smart Filter blending options under a Smart Filter entry in the `Layers` panel.

**Primary Interface Elements**
- Fade dialog with `Opacity` and `Mode`.
- Smart Filter blending options icon.
- Blend Mode and Opacity controls.

**How To Use It**
1. Apply a filter.
2. Immediately choose `Edit > Fade` if the filter was destructive.
3. Or double-click the Smart Filter blending options icon if it is a Smart Filter.
4. Adjust opacity and blend mode.
5. Confirm the dialog.

**Expected Result**
- The filter effect is mixed more subtly or blended with the image using the selected mode.
