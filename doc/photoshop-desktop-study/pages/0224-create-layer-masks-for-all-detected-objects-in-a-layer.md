# Create Layer Masks for All Detected Objects in a Layer

- Entry Type: Automatic multi-mask workflow
- Category: Create masks > Layer masks
- Source: https://helpx.adobe.com/photoshop/desktop/create-masks/layer-masks/create-layer-masks-for-all-detected-objects-in-a-layer.html

**Function Summary**
`Mask All Objects` analyzes one layer and creates separate object masks for everything it detects.

**What This Function Does**
- detects multiple objects automatically
- creates organized mask groups for each object
- enables object-specific adjustments without manual tracing

**Where You Find It**
- `Layer > Mask All Objects`

**Primary Interface Elements**
- `Mask All Objects`
- mask groups
- layer mask thumbnails
- `Hue/Saturation`
- `Show Transform Controls`

**How To Use It**
1. Select a single layer.
2. Choose `Layer > Mask All Objects`.
3. Open the created mask group for the object you want.
4. Apply adjustments or filters to the masked object as needed.
5. Inspect or highlight masks on-canvas when necessary.

**User Interface Behavior**
- Photoshop creates grouped masks rather than a single combined mask, which makes per-object editing easier.

**Expected Result**
- Each detected object in the layer becomes separately targetable through its own mask structure.
