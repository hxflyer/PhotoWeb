# Set Sample Sources for Cloning and Healing

- Entry Type: Clone Source workflow
- Category: Repair and retouch > Heal and clone
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/heal-clone/set-sample-sources-for-cloning-and-healing.html

**Function Summary**
This workflow defines and switches between multiple pixel sources for the `Clone Stamp` and `Healing Brush` tools.

**What This Function Does**
- stores up to five sample sources
- samples from the current document, another open document, or animation frames
- lets users switch sources without resampling every time

**Where You Find It**
- `Window > Clone Source`
- `Clone Stamp Tool`
- `Healing Brush Tool`

**How To Use It**
1. Select `Clone Stamp` or `Healing Brush`.
2. `Alt`-click on Windows or `Option`-click on macOS to set a source.
3. Open `Window > Clone Source`.
4. Select another source slot and sample again if needed.
5. Paint using the chosen source slot.

**Expected Result**
- The repair tool paints from the selected stored source.
