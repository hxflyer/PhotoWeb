# Create Artboard Documents

- Entry Type: Artboard document creation workflow
- Category: Create and manage layers > Layout and design tools
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/layout-design-tools/create-artboard-documents.html

**Function Summary**
This workflow creates new artboard-based documents or converts existing layered content into artboards.

**What This Function Does**
- Starts a new document with artboards enabled
- Converts selected existing layers or groups into artboards
- Supports preset artboard sizes for device and layout workflows

**Where You Find It**
- `File > New`
- `Artboards` checkbox in the New Document dialog
- `File > Open`
- layer context menu options such as `Artboard from Layers` and `Artboard from Group`

**Primary Interface Elements**
- `New Document`
- `Artboards`
- preset sizes such as `Mobile`
- `Artboard from Layers`
- `Artboard from Group`

**How To Create a New Artboard Document**
1. Choose `File > New`.
2. Enable the `Artboards` checkbox.
3. Choose a preset size if needed.
4. Name the document.
5. Select `Create`.

**How To Convert Existing Content into Artboards**
1. Open the Photoshop file.
2. Select the layers you want to turn into an artboard.
3. Right-click and choose `Artboard from Layers`.
4. Use `Artboard from Group` when starting from a grouped layer set.

**Expected Result**
- One or more artboards become available as design containers inside the document.
