# Switch Workspaces

- Entry Type: Workspace management
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/switch-workspaces.html

**Function Summary**
This function changes the active Photoshop workspace so the interface matches the type of task you are doing. Switching workspaces updates which panels are visible, how they are arranged, and which saved layout preset Photoshop applies.

**What This Function Does**
- Loads a different workspace preset
- Changes visible panels and panel arrangement
- Lets you move between built-in and custom interface layouts quickly
- Reduces setup time when changing from one kind of work to another

**Where You Find It**
- `Choose a workspace` dropdown in the application bar
- `Window > Workspace`

**Primary Interface Elements**
- `Choose a workspace` dropdown
- built-in workspace presets
- custom workspace names
- `Window` menu
- `Workspace` submenu

**How To Use It**
1. Open the `Choose a workspace` menu in the application bar.
2. Select a built-in preset such as `Essentials` or another saved workspace.
3. Wait for Photoshop to rearrange the visible panels and restore the selected layout.
4. If preferred, use `Window > Workspace` and choose the workspace from the menu list instead.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Click the workspace dropdown and select the layout you want.
- `Menus`
  Use `Window > Workspace` to switch without using the application-bar control.
- `Keyboard`
  No default dedicated hotkey is listed for switching workspaces.

**User Interface Behavior**
- Photoshop loads the selected layout immediately after you choose it.
- The workspace can change which panels are docked, grouped, or hidden.
- Custom workspace names appear alongside built-in presets in the same selection flow.
- The change affects the interface layout, not the image content in the active document.

**Expected Result**
- The workspace layout matches the current editing task more closely.
- Relevant panels become easier to reach.
- Time spent manually opening and rearranging panels is reduced.

**Related Assets**
- Workspace picker example: [Choose a workspace dropdown](../images/0021-switch-workspaces/01-photoshop-interface-displaying-the-choos.jpg)

**Practical Use Cases**
- Move from a general layout to a photography-focused setup.
- Load a custom painting or design workspace before starting specialized work.
- Return to `Essentials` when the interface becomes cluttered or unfamiliar.
