# Modify Fill and Stroke for Shapes

- Entry Type: Shape appearance workflow
- Category: Draw shapes and paths > Create shapes
- Source: https://helpx.adobe.com/photoshop/desktop/draw-shapes-paths/create-shapes/modify-fill-stroke-for-shapes.html

**Function Summary**
Shape fill and stroke controls change the interior color and outline of editable shape layers.

**Primary Interface Elements**
- fill swatch
- stroke swatch
- stroke width
- stroke style
- alignment
- caps and corners
- `Properties` panel

**How To Use It**
1. Select the shape layer.
2. Adjust fill color or set it to none.
3. Adjust stroke color, width, and style.
4. Use the `Properties` panel for additional options.

**Expected Result**
- The selected shape updates with the chosen fill and outline styling.
