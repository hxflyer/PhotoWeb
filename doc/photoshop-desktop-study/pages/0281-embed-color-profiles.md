# Embed Color Profiles

- Entry Type: Save option workflow
- Category: Adjust color > Color profiles
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/color-profiles/embed-color-profiles.html

**Function Summary**
Embedding a color profile stores the document's color interpretation inside the saved file.

**Where You Find It**
- `File > Save As`
- document status information area

**Primary Interface Elements**
- `Embed Color Profile`
- `Document Profile`
- supported file formats such as PSD, TIFF, JPEG, PDF, and PSB

**How To Use It**
1. Choose `File > Save As`.
2. Select a format that supports profiles.
3. Enable `Embed Color Profile`.
4. Save the file.
5. Use the document information area to view the current profile when needed.

**Expected Result**
- Other devices and applications can interpret the file's color more consistently.
