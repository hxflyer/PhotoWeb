# Set a Global Lighting Angle for All Layers

- Entry Type: Shared layer-style lighting control
- Category: Create and manage layers > Apply layer effects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/apply-layer-effects/set-a-global-lighting-angle-for-all-layers.html

**Function Summary**
Global Light defines one shared light direction for layer effects that depend on lighting, such as shadows and bevels. It helps keep multiple styled layers visually consistent.

**What This Function Does**
- Sets one shared lighting angle for supported effects
- Controls light altitude in addition to direction
- Synchronizes directional effects across multiple layers when `Use Global Light` is enabled

**Where You Find It**
- `Layer > Layer Style > Global Light`
- inside the `Layer Style` dialog for effects such as `Drop Shadow`, `Inner Shadow`, and `Bevel & Emboss`

**Primary Interface Elements**
- `Global Light`
- `Angle`
- `Altitude`
- `Use Global Light`
- `Drop Shadow`
- `Inner Shadow`
- `Bevel & Emboss`

**How To Set the Global Lighting Angle**
1. Open `Layer > Layer Style > Global Light`.
2. Set the `Angle` by entering a value or dragging the angle control.
3. Adjust `Altitude` to control the height of the light source.
4. Confirm with `OK`.

**How To Use It from a Layer Effect**
1. Open the `Layer Style` dialog by double-clicking a layer.
2. Choose a directional effect such as `Drop Shadow`, `Inner Shadow`, or `Bevel & Emboss`.
3. Enable `Use Global Light`.
4. Adjust `Angle` and `Altitude`.
5. Confirm with `OK`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Drag the angle control and adjust altitude directly in the dialog.
- `Menus`
  Use `Layer > Layer Style > Global Light` for the global control path.
- `Keyboard`
  No dedicated default hotkey is described for this dialog command.

**User Interface Behavior**
- Changing Global Light updates supported effects across layers that use the shared lighting option.
- It helps prevent mismatched shadows and highlights across layered designs.

**Expected Result**
- Directional layer effects appear to share one coherent light source across the document.
