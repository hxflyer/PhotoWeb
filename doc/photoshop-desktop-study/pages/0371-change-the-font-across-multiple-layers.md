# Change the Font Across Multiple Layers

- Entry Type: Multi-layer font workflow
- Category: Text and typography > Select and manage fonts
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/select-manage-fonts/change-font-across-multiple-layers.html

**Function Summary**
Multi-layer font changes apply the same font family or style to several selected type layers at once.

**What This Function Does**
- Updates font styling across multiple type layers in a single operation.
- Helps standardize typography in mockups, layouts, and imported designs.
- Leaves each type layer editable.

**Where You Find It**
- `Layers` panel for selecting multiple type layers.
- Options bar, `Properties`, or `Character` panel font controls.

**Primary Interface Elements**
- Multiple selected type layers in the `Layers` panel.
- Font family and style menus.
- Character and Properties panels.

**How To Use It**
1. In the `Layers` panel, `Shift`-click or `Cmd/Ctrl`-click multiple type layers.
2. Choose a new font family or style from the options bar, `Properties`, or `Character` panel.
3. Review each layer for reflow, overset text, or spacing changes.

**Expected Result**
- All selected type layers use the chosen font settings.
