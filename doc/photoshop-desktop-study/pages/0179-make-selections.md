# Make Selections

- Entry Type: Topic hub
- Category: Photoshop Desktop Help
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections.html

**Function Summary**
This hub groups Photoshop tools and commands for selecting pixels, isolating subjects, refining edges, and creating masks.

**What This Entry Is For**
- organizes the selection toolset
- separates automatic, freehand, and refinement-based workflows
- acts as the main entry point for masking and targeted editing operations

**Primary Function Areas**
- object and subject detection
- color-based selections
- freehand selections
- selection modification
- masking workflows

**Expected Result**
- Users can navigate to the correct selection method based on whether they need automatic detection, manual tracing, or later refinement.
