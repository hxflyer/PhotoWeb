# Automate Tasks

- Entry Type: Topic hub
- Category: Photoshop Desktop Help
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks.html

**Function Summary**
Automation features repeat Photoshop operations through actions, batches, droplets, presets, and file-processing commands.

**Where You Find It**
- `Window > Actions`.
- `File > Automate`.
- `File > Scripts`.

**Primary Interface Elements**
- Actions panel, action sets, recorded commands, Batch dialog, Droplet dialog, Image Processor, and automation presets.

**Expected Result**
- Users can turn repeated editing or export steps into reusable automated workflows.
