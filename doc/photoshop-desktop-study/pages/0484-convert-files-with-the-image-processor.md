# Convert Files with the Image Processor

- Entry Type: Image Processor workflow
- Category: Automate tasks > Batch processing
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/process-batch-files/convert-files-image-processor.html

**Function Summary**
The Image Processor converts multiple files to selected output formats and optional dimensions.

**Where You Find It**
- `File > Scripts > Image Processor`.

**Primary Interface Elements**
- Source folder/open images selection.
- Save location.
- JPEG/PSD/TIFF output checkboxes.
- Resize to Fit.
- Run Action and copyright/metadata options where available.

**How To Use It**
1. Choose `File > Scripts > Image Processor`.
2. Select open images or a source folder.
3. Choose a destination folder.
4. Select output formats and quality settings.
5. Enable Resize to Fit or Run Action if needed.
6. Click `Run`.

**Expected Result**
- Photoshop processes the source images into the chosen file formats.
