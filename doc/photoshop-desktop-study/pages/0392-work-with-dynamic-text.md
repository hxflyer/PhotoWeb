# Work with Dynamic Text

- Entry Type: Topic hub
- Category: Text and typography
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/work-with-dynamic-text.html

**Function Summary**
Dynamic text workflows focus on text objects that can be adjusted with live formatting, resizing, and start/end controls instead of being treated as flat pixels.

**What This Area Controls**
- Dynamic text object behavior.
- Formatting and resizing controls.
- Start and end point repositioning.
- On-canvas handles that affect text flow or placement.

**Where You Find It**
- Active dynamic text object or supported type layer.
- `Type Tool`, `Move Tool`, and relevant on-canvas handles.
- `Properties`, `Character`, and `Paragraph` panels.

**Primary Interface Elements**
- Editable text object.
- Bounding box or path/endpoint handles.
- Start and end markers.
- Formatting controls in text panels.

**Expected Result**
- Users can identify and adjust text that uses live dynamic controls for layout and placement.
