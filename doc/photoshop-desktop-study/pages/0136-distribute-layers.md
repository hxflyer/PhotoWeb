# Distribute Layers

- Entry Type: Layer spacing workflow
- Category: Create and manage layers > Create layer compositions
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/create-layer-compositions/distribute-layers.html

**Function Summary**
This workflow spaces selected layers evenly based on edges, centers, or overall horizontal and vertical gaps.

**What This Function Does**
- Evenly spaces layers along horizontal or vertical axes
- Supports distribution by edges, centers, and spacing intervals
- Helps build cleaner grids and balanced layouts

**Where You Find It**
- Layers panel
- `Layer > Distribute`

**Primary Interface Elements**
- selected layers
- `Top Edges`
- `Vertical Centers`
- `Bottom Edges`
- `Left Edges`
- `Horizontal Centers`
- `Right Edges`
- `Horizontally`
- `Vertically`

**How To Use It**
1. Select the layers you want to distribute.
2. Choose `Layer > Distribute`.
3. Select the distribution rule you want.

**User Interface Behavior**
- Photoshop repositions layers so the chosen edges, centers, or spacing intervals become even.

**Expected Result**
- Multiple objects become evenly spaced with less manual measurement.
