# Select a Frame and Its Content

- Entry Type: Frame selection workflow
- Category: Create and manage layers > Layout and design tools
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/layout-design-tools/select-frame-content.html

**Function Summary**
This workflow switches between selecting the frame container and selecting the content inside it, allowing separate control over the mask and the image.

**What This Function Does**
- Selects frame and content together
- Selects only the placed image
- Selects only the frame border
- Supports frame/content targeting from the Layers panel

**Where You Find It**
- canvas selection behavior
- Layers panel thumbnails

**Primary Interface Elements**
- inset image area
- frame border
- frame thumbnail
- content thumbnail

**How To Use It**
1. Click the inset image on the canvas to select the frame and image together.
2. Double-click the image to select only the content.
3. Click the frame border to select only the frame.
4. Use the Layers panel thumbnails if you prefer panel-based targeting.
