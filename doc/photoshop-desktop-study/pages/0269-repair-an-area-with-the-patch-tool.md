# Repair an Area with the Patch Tool

- Entry Type: Patch Tool workflow
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/repair-a-selected-area-with-the-patch-tool.html

**Function Summary**
The `Patch Tool` repairs a selected area by dragging it to or from a cleaner source area.

**Primary Interface Elements**
- `Patch Tool`
- `Source`
- `Destination`
- `Normal`
- `Content-Aware`
- `Transparent`
- `Use Pattern`
- `Diffusion`
- `Structure`
- `Color`
- `Sample All Layers`

**How To Use It**
1. Select the `Patch Tool`.
2. Draw a selection around the area to repair.
3. Choose `Source` or `Destination`.
4. Adjust patch settings.
5. Drag the selected area to the source or destination position.
6. Release to apply.

**Expected Result**
- The selected area is repaired using sampled pixels or pattern/content-aware blending.
