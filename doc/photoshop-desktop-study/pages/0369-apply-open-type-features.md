# Apply OpenType Features

- Entry Type: OpenType formatting workflow
- Category: Text and typography > Select and manage fonts
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/select-manage-fonts/apply-open-type-features.html

**Function Summary**
Applying OpenType features enables font-specific typographic substitutions and behaviors on selected text.

**What This Function Does**
- Turns on features such as ligatures, swashes, alternates, fractions, ordinals, and stylistic sets.
- Applies only features supported by the active font.
- Changes glyph appearance while keeping text editable.

**Where You Find It**
- `Window > Character`.
- Character panel OpenType section or panel menu.
- On-canvas alternate glyph popup for supported characters.

**Primary Interface Elements**
- Character selection highlight.
- OpenType feature icons and menu options.
- Glyph alternate popup.
- Glyphs panel for manual glyph insertion.

**How To Use It**
1. Select a type layer or highlight the characters to affect.
2. Choose an OpenType font.
3. Open `Window > Character`.
4. Enable the needed OpenType feature.
5. If alternates appear on canvas, click the desired alternate glyph.

**Expected Result**
- Photoshop substitutes supported glyph forms while preserving editable text.
