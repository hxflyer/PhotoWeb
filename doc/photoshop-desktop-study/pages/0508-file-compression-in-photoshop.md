# File Compression in Photoshop

- Entry Type: Compression reference
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/file-compression-photoshop.html

**Function Summary**
File compression reduces file size using lossless or lossy methods depending on the format and settings.

**What This Function Describes**
- Lossless compression reduces size without visible data loss.
- Lossy compression discards image information to create smaller files.
- Higher compression can reduce quality, create artifacts, or slow saving/opening.
- Format choice determines available compression controls.

**Where You Find It**
- Save, Save As, Export As, Save for Web, TIFF, JPEG, PNG, and PDF option dialogs.

**Expected Result**
- Users can balance file size, image quality, compatibility, and performance.
