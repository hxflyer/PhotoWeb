# Delete Preset Brushes

- Entry Type: Brush preset management workflow
- Category: Apply painting techniques > Brushes and presets
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/brushes-presets/delete-preset-brushes.html

**Function Summary**
Deleting a preset removes it from the active brush preset library.

**How To Use It**
1. Open the `Brushes` panel.
2. Select the brush preset.
3. Use the delete command or panel trash control.
4. Confirm the deletion if prompted.

**Expected Result**
- The selected brush preset is removed from the visible preset list.
