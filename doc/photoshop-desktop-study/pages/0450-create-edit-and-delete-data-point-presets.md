# Create, Edit, and Delete Data Point Presets

- Entry Type: Measurement preset workflow
- Category: Grids, guides, and measurement
- Source: https://helpx.adobe.com/photoshop/desktop/grids-guides-measurement/create-edit-delete-data-point-presets.html

**Function Summary**
Data point presets save reusable sets of Measurement Log fields for repeated measurement tasks.

**Where You Find It**
- `Image > Analysis > Select Data Points`.
- Measurement Log panel options.

**Primary Interface Elements**
- Data point checklist.
- Preset name field.
- Save, delete, and manage preset controls.

**How To Use It**
1. Open the data point selection dialog.
2. Choose the measurement fields to include.
3. Save the selection as a preset.
4. Edit or delete presets when measurement requirements change.

**Expected Result**
- The Measurement Log can quickly switch between saved data field sets.
