# Export the Contents of an Embedded Smart Object

- Entry Type: Smart Object export workflow
- Category: Create and manage layers > Smart objects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/smart-objects/export-the-contents-of-an-embedded-smart-object.html

**Function Summary**
This workflow saves the source content of an embedded Smart Object out to a separate file so it can be reused or edited outside the current Photoshop document.

**What This Function Does**
- Extracts embedded Smart Object source data into a standalone file
- Preserves the original source format when possible
- Makes embedded content easier to reuse in external workflows

**Where You Find It**
- `Window > Layers`
- `Layer > Smart Objects > Export Contents`

**Primary Interface Elements**
- selected Smart Object
- `Export Contents`
- save location chooser
- filename field

**How To Use It**
1. Open `Window > Layers`.
2. Select the embedded Smart Object layer.
3. Choose `Layer > Smart Objects > Export Contents`.
4. Choose a save location and filename.
5. Save the exported file.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Select the Smart Object and choose the export location from the save dialog.
- `Menus`
  Use `Layer > Smart Objects > Export Contents`.
- `Keyboard`
  No dedicated default hotkey is described for this export path.

**User Interface Behavior**
- Photoshop exports the Smart Object in its original source format when possible.
- Multi-layer Smart Object content may be exported as `PSB` when appropriate.

**Expected Result**
- The embedded Smart Object becomes available as a separate file outside the PSD.
