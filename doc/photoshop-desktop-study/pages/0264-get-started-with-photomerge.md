# Get Started with Photomerge

- Entry Type: Topic hub
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/get-started-with-photomerge.html

**Function Summary**
This hub introduces `Photomerge`, Photoshop's workflow for combining multiple source images into panoramas or related merged compositions.

**Primary Function Areas**
- panoramic image creation
- 360-degree panorama workflows
- source-image alignment
- merge layout options

**Expected Result**
- Users can choose a Photomerge workflow based on whether they need a standard panorama or a 360-degree composition.
