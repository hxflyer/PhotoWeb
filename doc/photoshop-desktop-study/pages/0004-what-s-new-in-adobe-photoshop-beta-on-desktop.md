# What’s New In Adobe Photoshop (Beta) On Desktop

- Entry Type: Beta release summary page
- Category: What's new
- Source: https://helpx.adobe.com/photoshop/desktop/whats-new/whats-new-in-photoshop-beta-on-desktop.html

**Function Summary**
This page summarizes newly available features in the Photoshop beta desktop app. It is intended for users who want to test early-access capabilities before they ship to the standard Photoshop release.

**What This Entry Covers**
- newly added beta-only features
- version-grouped beta highlights
- experimental AI model updates
- early creative workflow additions

**Current Beta Highlights Mentioned Here**
- Dynamic Text for circle, arch, and bow layouts
- Firefly Fill & Expand (beta) model improvements for generative workflows

**How To Use This Entry**
- Open this page when you want to see what has recently entered the beta channel.
- Use it to compare beta-only functionality against the stable desktop release.
- Treat each listed feature as early-access behavior that may change before final release.

**User Interface Context**
- This is a beta feature summary, not an in-app tool panel.
- It points you toward features that are available for testing inside the separate Photoshop beta application.

**Expected Result**
- You can identify which new capabilities are currently limited to Photoshop beta.
- You can decide whether a feature is mature enough for testing or should wait for the stable release.
