# Colorize a Grayscale Image or Create a Monotone Effect

- Entry Type: Hue/Saturation colorize workflow
- Category: Adjust color > Color corrections
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/color-corrections/colorize-grayscale-image-create-monotone-effect.html

**Function Summary**
`Colorize` applies a single hue-based tint to grayscale or desaturated imagery.

**Where You Find It**
- `Hue/Saturation` adjustment

**How To Use It**
1. Add or open a `Hue/Saturation` adjustment.
2. Enable `Colorize`.
3. Adjust `Hue`, `Saturation`, and `Lightness`.

**Expected Result**
- The image takes on a controlled monochrome or tinted color effect.
