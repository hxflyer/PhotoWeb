# Create Panoramic Images with Photomerge

- Entry Type: Photomerge panorama workflow
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/create-panoramic-images-with-photomerge.html

**Function Summary**
`Photomerge` aligns overlapping photos and combines them into a panorama.

**Where You Find It**
- `File > Automate > Photomerge`

**Primary Interface Elements**
- layout options
- source file list
- `Blend Images Together`
- `Vignette Removal`
- `Geometric Distortion Correction`
- `Content Aware Fill Transparent Areas`

**How To Use It**
1. Choose `File > Automate > Photomerge`.
2. Add the source images.
3. Choose a layout mode.
4. Enable blending and correction options as needed.
5. Run the merge and inspect the layered result.

**Expected Result**
- Photoshop creates a panorama from multiple overlapping source images.
