# Auto-Blend Layers Command Overview

- Entry Type: Auto-Blend reference
- Category: Create masks > Blend images
- Source: https://helpx.adobe.com/photoshop/desktop/create-masks/blend-images/auto-blend-layers-command-overview.html

**Function Summary**
`Auto-Blend Layers` analyzes multiple aligned layers and adds masks to combine them into a smoother composite.

**What This Function Does**
- blends multiple exposures or focus states
- adds masks automatically
- supports focus stacking and certain layer-combination tasks

**Where You Find It**
- `Edit > Auto-Blend Layers`

**Primary Interface Elements**
- `Auto-Blend Layers`
- blend method options
- `Stack Images`
- `Panorama`

**How To Use It**
1. Prepare multiple aligned layers in an RGB or Grayscale document.
2. Select the layers to combine.
3. Run `Edit > Auto-Blend Layers`.
4. Choose the blend method that matches the task.

**User Interface Behavior**
- Photoshop creates masks on individual layers to hide weaker regions and preserve the best parts of each source.
- The command is not available for unsupported layer types such as Smart Objects and some special layer categories.

**Expected Result**
- The layer stack merges into a more seamless composite with less manual masking work.
