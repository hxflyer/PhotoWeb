# Edit Text

- Entry Type: Text editing workflow
- Category: Text and typography > Get started with text
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/get-started-text/edit-text.html

**Function Summary**
Text editing changes the wording or formatting of an existing editable type layer.

**What This Function Does**
- Lets users insert, delete, replace, or select characters inside a type layer.
- Applies character or paragraph formatting to selected text ranges.
- Keeps the layer as editable text after the edit is committed.

**Where You Find It**
- `Type Tool`, shortcut `T`.
- `Layers` panel type layer.
- Options bar, `Properties`, `Character`, and `Paragraph` panels.

**Primary Interface Elements**
- Text insertion cursor.
- Highlighted character selection.
- Type-layer bounding box.
- Commit check mark and cancel icon.

**How To Use It**
1. Select the type layer in the `Layers` panel.
2. Press `T` to activate the `Type Tool`.
3. Click inside the text and edit with the keyboard.
4. Drag across characters to select a range, then change formatting if needed.
5. Commit the edit with the check mark, `Esc`, or by selecting another tool.

**Expected Result**
- The selected type layer updates while remaining editable.
