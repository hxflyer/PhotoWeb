# Change Text Color

- Entry Type: Character color workflow
- Category: Text and typography > Get started with text
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/get-started-text/change-text-color.html

**Function Summary**
Text color controls set the fill color of selected characters or the entire type layer.

**What This Function Does**
- Changes all text on a selected type layer when no characters are highlighted.
- Changes only highlighted characters when a text range is selected.
- Uses the standard Photoshop color picker, swatches, and numeric color values.

**Where You Find It**
- Options bar color swatch when the `Type Tool` is active.
- `Properties` panel and `Character` panel color controls.
- `Window > Swatches` for saved colors.

**Primary Interface Elements**
- Text color swatch.
- Color Picker dialog.
- Swatches panel.
- Highlighted text selection.

**How To Use It**
1. Select a type layer or highlight specific characters with the `Type Tool`.
2. Click the text color swatch in the options bar, `Properties`, or `Character` panel.
3. Choose a color with the picker, swatches, or numeric color fields.
4. Confirm the color and commit the text edit.

**Expected Result**
- The selected text displays the new color while remaining editable.
