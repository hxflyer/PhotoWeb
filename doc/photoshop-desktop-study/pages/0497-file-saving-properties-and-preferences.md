# File Saving Properties and Preferences

- Entry Type: File handling preference reference
- Category: Save and export > Save files
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/save-files/file-saving-properties-preferences.html

**Function Summary**
File saving preferences define compatibility, previews, extension behavior, recovery, cloud defaults, and related save options.

**Where You Find It**
- macOS: `Photoshop > Settings > File Handling`.
- Windows: `Edit > Preferences > File Handling`.

**Primary Interface Elements**
- Image previews.
- File extension behavior.
- Maximize PSD/PSB compatibility.
- Automatically Save Recovery Information.
- Cloud document save location preferences.

**Expected Result**
- Saved files match the user's compatibility, preview, recovery, and workflow preferences.
