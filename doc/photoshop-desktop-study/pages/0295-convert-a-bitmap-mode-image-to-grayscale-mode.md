# Convert a Bitmap Mode Image to Grayscale Mode

- Entry Type: Bitmap-to-grayscale conversion workflow
- Category: Adjust color > Color modes
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/color-modes/convert-bitmap-mode-image-to-grayscale-mode.html

**Function Summary**
This conversion changes a one-bit bitmap image into grayscale so more editing options become available.

**Where You Find It**
- `Image > Mode > Grayscale`

**How To Use It**
1. Open the bitmap image.
2. Choose `Image > Mode > Grayscale`.
3. Set the size ratio if prompted.
4. Confirm the conversion.

**Expected Result**
- The file becomes a grayscale image that supports tonal editing beyond black and white pixels.
