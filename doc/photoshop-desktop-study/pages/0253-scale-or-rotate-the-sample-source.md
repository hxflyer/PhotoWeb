# Scale or Rotate the Sample Source

- Entry Type: Clone Source transform workflow
- Category: Repair and retouch > Heal and clone
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/heal-clone/scale-or-rotate-the-sample-source.html

**Function Summary**
These `Clone Source` controls resize or rotate sampled pixels before they are painted.

**What This Function Does**
- scales the cloned or healed source
- rotates the source to match target perspective or texture direction
- preserves or breaks width-height proportions

**Where You Find It**
- `Window > Clone Source`

**Primary Interface Elements**
- source slots
- `W`
- `H`
- maintain aspect ratio
- `Angle`
- reset transform icon

**How To Use It**
1. Select `Clone Stamp` or `Healing Brush`.
2. Set a sample source.
3. Open `Clone Source`.
4. Choose the source slot.
5. Adjust `W`, `H`, or `Angle`.
6. Paint with the transformed source.

**Expected Result**
- The sampled texture or detail is painted at the chosen scale and rotation.
