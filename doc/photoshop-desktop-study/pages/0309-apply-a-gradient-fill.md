# Apply a Gradient Fill

- Entry Type: Gradient Tool workflow
- Category: Adjust color > Color effects and techniques
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/color-effects-techniques/apply-gradient-fill.html

**Function Summary**
The `Gradient Tool` applies a smooth color transition across a layer or selected area.

**Where You Find It**
- `Gradient Tool` in the toolbar

**Primary Interface Elements**
- gradient preset bar
- `Linear`
- `Radial`
- `Angle`
- `Reflected`
- `Diamond`
- `Dither`
- `Reverse`
- `Properties` panel
- contextual task bar

**How To Use It**
1. Select the area to receive the gradient, or leave no selection to affect the whole layer.
2. Select the `Gradient Tool`.
3. Choose a preset and gradient type.
4. Enable `Dither` or `Reverse` if needed.
5. Drag across the canvas to define direction and length.
6. Edit the result in the `Properties` panel or contextual task bar.

**Expected Result**
- Photoshop creates a gradient transition using the chosen preset, type, and drag direction.
