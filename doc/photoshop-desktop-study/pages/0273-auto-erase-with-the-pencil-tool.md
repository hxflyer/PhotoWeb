# Auto Erase with the Pencil Tool

- Entry Type: Pencil color-replacement mode
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/auto-erase-with-the-pencil-tool.html

**Function Summary**
`Auto Erase` makes the `Pencil Tool` switch between foreground and background colors based on the color where the stroke begins.

**Where You Find It**
- `Pencil Tool` options bar

**Primary Interface Elements**
- foreground color
- background color
- `Auto Erase`

**How To Use It**
1. Set foreground and background colors.
2. Select the `Pencil Tool`.
3. Enable `Auto Erase`.
4. Drag across the image.

**User Interface Behavior**
- Starting on the foreground color paints with the background color.
- Starting elsewhere paints with the foreground color.

**Expected Result**
- The Pencil Tool quickly swaps between two color states while painting.
