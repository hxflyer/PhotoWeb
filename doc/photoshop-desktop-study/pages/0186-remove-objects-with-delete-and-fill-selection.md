# Remove Objects with Delete and Fill Selection

- Entry Type: Selection removal workflow
- Category: Make selections > Automatic and color-based selections
- Source: https://helpx.adobe.com/photoshop/desktop/make-selections/automatic-color-based-selections/remove-objects-with-delete-and-fill-selection.html

**Function Summary**
`Delete and Fill Selection` removes selected content and automatically fills the gap using surrounding image information.

**What This Function Does**
- deletes the selected object
- generates replacement content in the cleared area
- shortens common cleanup workflows

**Where You Find It**
- contextual menu on an active selection

**Primary Interface Elements**
- active selection
- context menu
- `Delete and Fill Selection`

**How To Use It**
1. Select the unwanted object with any suitable selection tool.
2. Right-click on Windows or `Control`-click on macOS inside the selected area.
3. Choose `Delete and Fill Selection`.

**Mouse, Keyboard, and Menu Input**
- Mouse: right-click the selection on Windows.
- Mouse: `Control`-click the selection on macOS.

**User Interface Behavior**
- Photoshop removes the selected object and fills the cleared area automatically in one command.

**Expected Result**
- The selected object disappears and the area is patched with context-aware replacement pixels.
