# Soften Hard Edges or Reduce Detail with the Blur Tool

- Entry Type: Local blur tool workflow
- Category: Effects and filters > Blur and sharpen
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/blur-sharpen/soften-hard-edges-reduce-detail-blur-tool.html

**Function Summary**
The `Blur Tool` paints localized softness onto image areas using brush-based blur strength.

**Where You Find It**
- Toolbar: `Blur Tool`, grouped with Sharpen and Smudge.

**Primary Interface Elements**
- Brush size, hardness, and strength.
- Sample All Layers option where available.
- Brush cursor on canvas.

**How To Use It**
1. Select the `Blur Tool`.
2. Set brush size and strength in the options bar.
3. Paint over hard edges or areas with too much detail.
4. Build the blur gradually with multiple strokes.

**Expected Result**
- Painted areas become softer and less detailed.
