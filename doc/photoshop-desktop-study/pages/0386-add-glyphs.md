# Add Glyphs

- Entry Type: Glyph insertion workflow
- Category: Text and typography > Characters and glyphs
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/characters-glyphs/add-glyphs.html

**Function Summary**
The `Glyphs` panel inserts characters, symbols, alternates, and special forms that may not be available from the keyboard.

**What This Function Does**
- Displays the glyph set included in the selected font.
- Inserts symbols, accented characters, alternates, ornaments, and special typographic forms.
- Helps users choose exact characters for professional typography.

**Where You Find It**
- `Window > Glyphs`.
- Active type layer with the text insertion cursor placed.

**Primary Interface Elements**
- Glyph grid.
- Font family and style menus.
- Recently used glyphs.
- Glyph category or display filters where available.
- Double-click insertion behavior.

**How To Use It**
1. Select or create a type layer.
2. Place the text insertion cursor where the glyph should be inserted.
3. Open `Window > Glyphs`.
4. Choose the font and glyph category if needed.
5. Double-click the glyph to insert it.

**Expected Result**
- The selected glyph is inserted into the editable type layer.
