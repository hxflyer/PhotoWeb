# List Of Technology Preview Features

- Entry Type: Feature catalog
- Category: What's new
- Source: https://helpx.adobe.com/photoshop/desktop/whats-new/list-of-technology-preview-features.html

**Function Summary**
This page lists individual Photoshop features that are currently or historically exposed through the Technology Previews system. It acts as a reference catalog for experimental items rather than a single operational feature.

**What This Entry Covers**
- preview features available through Technology Previews
- feature names and activation locations
- examples of preview-only capabilities before full release

**Preview Features Mentioned Here**
- Preserve Details 2.0 Upscale
- Content-Aware Tracing Tool
- HDR-related color-management and preview improvements
- opening JPEG files without a Background layer
- Modern User Interface preview items

**How To Use This Entry**
- Use it after opening the Technology Previews settings so you know what each preview item is intended to expose.
- Treat it as a lookup page for experimental features rather than as a workflow tutorial.
- Open the related detailed feature pages when you want actual usage steps for a specific preview tool.

**User Interface Context**
- Some listed preview items appear in preferences.
- Some become available in tools, dialogs, or menu entries only after they are enabled.
- Availability depends on Photoshop version and platform support.

**Expected Result**
- You can identify which experimental features are available for testing and where they surface in Photoshop after activation.
