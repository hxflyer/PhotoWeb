# Retouch a Large Area with the Healing Brush Tool

- Entry Type: Healing Brush workflow
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/healing-brush-tool.html

**Function Summary**
The `Healing Brush Tool` uses a manually selected source and blends it with the target area's tone, lighting, and texture.

**Where You Find It**
- `Healing Brush Tool` in the toolbar

**Primary Interface Elements**
- `Mode`
- `Source`
- `Sampled`
- `Pattern`
- `Aligned`
- `Sample`
- `Diffusion`

**How To Use It**
1. Select the `Healing Brush Tool`.
2. Set brush size, hardness, and spacing.
3. Choose sampled or pattern source.
4. `Alt`-click on Windows or `Option`-click on macOS to set a sample point.
5. Paint over the area to repair.

**Expected Result**
- The sampled detail is blended into the target area instead of copied exactly.
