# Adjust Formatting and Resize Text

- Entry Type: Dynamic text formatting workflow
- Category: Text and typography > Work with dynamic text
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/work-with-dynamic-text/adjust-formatting-resize-text.html

**Function Summary**
Dynamic text formatting and resizing change the appearance, size, or layout bounds of a live text object.

**What This Function Does**
- Adjusts font family, style, size, color, spacing, and paragraph attributes.
- Resizes the text object or text area through handles or transform controls.
- Reflows text according to the new formatting and bounds.

**Where You Find It**
- `Type Tool` and on-canvas text handles.
- `Properties`, `Character`, and `Paragraph` panels.
- Free Transform for object-level scaling.

**Primary Interface Elements**
- Text bounding box.
- Resize handles.
- Font and paragraph controls.
- Commit and cancel controls.

**How To Use It**
1. Select the dynamic text object or type layer.
2. Use text panels to change font, size, spacing, color, and paragraph behavior.
3. Drag the text bounding handles to resize the live text area.
4. Use Free Transform only when you want object-level scaling.
5. Review wrapping, clipping, and alignment.

**Expected Result**
- The text updates visually and remains editable through its live controls.
