# Overview of OpenType Fonts

- Entry Type: OpenType reference
- Category: Text and typography > Select and manage fonts
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/select-manage-fonts/overview-open-type-fonts.html

**Function Summary**
OpenType fonts can contain advanced typographic features such as ligatures, alternates, swashes, fractions, ordinals, stylistic sets, and expanded glyph coverage.

**What This Function Describes**
- OpenType features are built into the font, not generated by Photoshop independently.
- Available controls depend on the selected font and selected characters.
- OpenType features improve professional typography, logo lettering, editorial text, and multilingual typesetting.

**Where You Find It**
- `Window > Character`.
- Character panel menu and OpenType controls.
- On-canvas alternate glyph popup when alternates are available.
- `Window > Glyphs` for browsing the font's glyph set.

**Primary Interface Elements**
- Ligature, discretionary ligature, contextual alternate, swash, stylistic alternate, fraction, ordinal, and stylistic set controls.
- OpenType icons in the Character panel.
- Glyphs panel glyph grid.

**Expected Result**
- Users understand which advanced typography controls may appear for OpenType fonts and why availability changes by font.
