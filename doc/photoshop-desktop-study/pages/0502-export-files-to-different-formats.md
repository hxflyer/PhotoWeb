# Export Files to Different Formats

- Entry Type: Export format workflow
- Category: Save and export
- Source: https://helpx.adobe.com/photoshop/desktop/save-export/export-files-different-formats.html

**Function Summary**
Export commands create flattened or optimized copies in formats such as PNG, JPEG, GIF, SVG-related assets, and other delivery formats.

**Where You Find It**
- `File > Export > Export As`.
- `File > Export > Quick Export As`.
- `File > Export > Save for Web (Legacy)`.

**Primary Interface Elements**
- Export As dialog.
- Format, quality, transparency, size, scale, color space, and metadata controls.
- Export location preferences.

**Expected Result**
- Users can generate delivery copies while preserving the original Photoshop document.
