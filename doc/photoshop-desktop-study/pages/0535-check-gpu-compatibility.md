# Check GPU Compatibility

- Entry Type: GPU diagnostic workflow
- Category: Troubleshoot
- Source: https://helpx.adobe.com/photoshop/desktop/troubleshoot/check-gpu-compatibility.html

**Function Summary**
GPU compatibility checks whether the graphics processor and driver meet Photoshop requirements for GPU-accelerated features.

**Where You Find It**
- `Help > GPU Compatibility`.
- `Preferences > Performance`.

**Primary Interface Elements**
- GPU compatibility report.
- Use Graphics Processor checkbox.
- Advanced GPU settings.
- Detected graphics card and driver information.

**How To Use It**
1. Run the GPU compatibility check.
2. Review detected GPU, driver status, and unsupported items.
3. Update the graphics driver or operating system if needed.
4. Toggle GPU features only for troubleshooting.

**Expected Result**
- Users know whether GPU issues may be causing display, performance, or feature problems.
