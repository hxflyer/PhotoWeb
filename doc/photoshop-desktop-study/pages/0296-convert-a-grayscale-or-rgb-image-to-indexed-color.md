# Convert a Grayscale or RGB Image to Indexed Color

- Entry Type: Indexed Color conversion workflow
- Category: Adjust color > Color modes
- Source: https://helpx.adobe.com/photoshop/desktop/adjust-color/color-modes/convert-grayscale-rgb-image-to-indexed-color.html

**Function Summary**
Indexed Color reduces an image to a limited color table, often for lightweight web or palette-based graphics.

**Where You Find It**
- `Image > Mode > Indexed Color`

**How To Use It**
1. Start with an RGB or Grayscale image.
2. Choose `Image > Mode > Indexed Color`.
3. Choose palette, color count, transparency, and dithering options.
4. Confirm the conversion.

**Expected Result**
- Photoshop converts the image into a palette-based indexed document.
