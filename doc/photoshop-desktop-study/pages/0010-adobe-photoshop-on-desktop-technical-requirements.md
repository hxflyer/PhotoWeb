# Adobe Photoshop On Desktop Technical Requirements

- Entry Type: System requirements reference
- Category: Get started > Technical requirements and installation
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/technical-requirements-installation/adobe-photoshop-on-desktop-technical-requirements.html

**Function Summary**
This page defines the minimum and recommended system specifications required to install and run Photoshop desktop. It is a compatibility reference, not an editing feature or workflow tool.

**What This Entry Is For**
- checking install compatibility before deployment
- comparing minimum vs recommended hardware
- confirming version-specific support requirements
- validating platform readiness for Photoshop updates

**What You Find Here**
- requirements for current 27.x releases and later
- older-version requirement references such as 26.x
- operating-system and platform requirement groupings
- hardware baseline guidance for supported desktop environments

**How To Use This Entry**
- Check this page before installing Photoshop on a new workstation.
- Use the recommended values, not just the minimum values, when planning production hardware.
- Recheck the version-specific section when upgrading Photoshop across major release families.

**User Interface Context**
- This is a technical reference page, not an in-app preference panel.
- It supports installation planning and compatibility checks outside the editing workflow.

**Expected Result**
- You should know whether a target system is suitable for installing and running Photoshop.
- You should know whether the machine only meets the minimum baseline or is strong enough for better performance.
