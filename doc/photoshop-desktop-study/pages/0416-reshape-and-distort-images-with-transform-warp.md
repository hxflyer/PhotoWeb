# Reshape and Distort Images with Transform Warp

- Entry Type: Transform Warp workflow
- Category: Effects and filters > Distortion
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/distortion/reshape-distort-images-transform-warp.html

**Function Summary**
Transform Warp bends and reshapes a layer with a warp grid, preset warp shapes, and draggable control points.

**Where You Find It**
- `Edit > Transform > Warp`.
- Free Transform options bar warp button.

**Primary Interface Elements**
- Warp grid.
- Corner points, edge handles, and Bezier-style controls.
- Preset warp shape menu.
- Grid split controls where available.

**How To Use It**
1. Select the layer or Smart Object.
2. Choose `Edit > Transform > Warp`.
3. Drag grid points, handles, or mesh areas to reshape the layer.
4. Use presets or split warp lines for more control.
5. Commit the transform.

**Expected Result**
- The layer bends or reshapes according to the warp grid.
