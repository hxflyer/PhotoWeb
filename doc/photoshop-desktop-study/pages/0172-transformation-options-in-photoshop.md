# Transformation Options in Photoshop

- Entry Type: Transform command reference
- Category: Crop, resize, and transform > Transform, manipulate, and reshape
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/transform-manipulate-reshape/transformation-options-in-adobe-photoshop.html

**Function Summary**
This reference lists the standard transform modes available from Photoshop's `Transform` submenu.

**What This Entry Is For**
- explains which transform modes Photoshop provides
- helps users choose between scale, rotate, skew, distort, perspective, and warp
- serves as a menu-level reference rather than a single action

**Where You Find It**
- `Edit > Transform`

**Primary Interface Elements**
- `Scale`
- `Rotate`
- `Skew`
- `Distort`
- `Perspective`
- `Warp`
- `Rotate 180°`
- `Rotate 90° Clockwise`
- `Rotate 90° Counter Clockwise`
- `Flip Horizontal`
- `Flip Vertical`

**How To Use It**
1. Select the active layer, mask, shape, or selected pixels you want to affect.
2. Open `Edit > Transform`.
3. Choose the transform type that matches the adjustment you need.
4. Adjust the object on-canvas if the chosen mode uses handles or a mesh.
5. Commit the result.

**User Interface Behavior**
- Some commands apply immediately, while others enter an interactive transform state with handles or control points.

**Expected Result**
- The selected content changes geometry according to the chosen transform mode.
