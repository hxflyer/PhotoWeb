# Add Text

- Entry Type: Type Tool workflow
- Category: Text and typography > Get started with text
- Source: https://helpx.adobe.com/photoshop/desktop/text-typography/get-started-text/add-text.html

**Function Summary**
The `Type Tool` adds editable text to a Photoshop document by creating a new type layer.

**What This Function Does**
- Creates point text when you click once on the canvas.
- Creates paragraph text when you drag a rectangular text box.
- Preserves characters as editable type until the layer is rasterized, converted, or merged into pixels.

**Where You Find It**
- Toolbar: `Type Tool`, shortcut `T`.
- Tool variants: `Horizontal Type Tool`, `Vertical Type Tool`, and type mask variants.
- Options bar and `Properties` panel for font and formatting controls.

**Primary Interface Elements**
- Canvas insertion cursor for point text.
- Paragraph text bounding box with corner and side handles.
- Options bar controls for font family, style, size, alignment, color, anti-aliasing, warp, commit, and cancel.

**How To Use It**
1. Press `T` or select the `Type Tool`.
2. Click the canvas for point text, or drag with the mouse to define a paragraph text box.
3. Type the content.
4. Adjust font, size, alignment, and color from the options bar or `Properties` panel.
5. Press `Esc`, click the check mark, or select another tool to commit the text.

**Expected Result**
- Photoshop creates a new editable type layer in the `Layers` panel.
