# Overwrite a Single Command

- Entry Type: Action command replacement workflow
- Category: Automate tasks > Actions
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/create-record-actions/overwrite-single-command.html

**Function Summary**
Overwriting a command replaces one recorded action step without rebuilding the entire action.

**Where You Find It**
- Actions panel command list and record controls.

**Primary Interface Elements**
- Selected command step.
- Record and Stop buttons.
- Replacement recorded command.

**How To Use It**
1. Expand the action and select the command to replace.
2. Start recording at that position.
3. Perform the replacement command.
4. Stop recording.
5. Test the action to confirm the new step works.

**Expected Result**
- The selected action command is replaced by the newly recorded command.
