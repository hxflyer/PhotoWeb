# Specify a Delay Time in Frame Animations

- Entry Type: Frame timing workflow
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/specify-delay-time-frame-animations.html

**Function Summary**
Delay time controls how long each frame remains visible during playback.

**Where You Find It**
- Timeline panel frame delay menu under each frame thumbnail.

**Primary Interface Elements**
- Delay time dropdown.
- Custom delay value.
- Selected frame thumbnails.

**How To Use It**
1. Select one or more frames.
2. Click the delay value below a frame thumbnail.
3. Choose a preset delay or enter a custom time.
4. Preview playback and adjust pacing.

**Expected Result**
- Selected frames hold for the chosen duration during playback and export.
