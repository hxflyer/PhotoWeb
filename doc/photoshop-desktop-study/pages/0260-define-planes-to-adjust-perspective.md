# Define Planes to Adjust Perspective

- Entry Type: Perspective Warp layout workflow
- Category: Repair and retouch > Clean and restore images
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/clean-restore-images/define-planes-to-adjust-perspective.html

**Function Summary**
`Perspective Warp` begins by defining flat planes in the image that Photoshop can later warp or straighten.

**Where You Find It**
- `Edit > Perspective Warp`

**Primary Interface Elements**
- layout mode
- quad plane
- perspective edges

**How To Use It**
1. Select the image or layer.
2. Choose `Edit > Perspective Warp`.
3. Draw quads along architectural or object planes.
4. Align quad edges with real straight lines in the image.

**User Interface Behavior**
- Photoshop uses the drawn quads as the geometric structure for later perspective correction.

**Expected Result**
- The image contains defined planes ready for warp adjustment.
