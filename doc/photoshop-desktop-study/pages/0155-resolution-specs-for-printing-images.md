# Resolution Specs for Printing Images

- Entry Type: Print-sizing reference
- Category: Crop, resize, and transform > Resize and adjust resolution
- Source: https://helpx.adobe.com/photoshop/desktop/crop-resize-transform/resize-adjust-resolution/resolution-specs-for-printing-images.html

**Function Summary**
This reference outlines practical print-resolution expectations, especially the common `300 ppi` target.

**What This Entry Is For**
- Helps users prepare files for photo and document printing
- Explains when lower resolution is acceptable
- Shows how to preview approximate print size onscreen

**Typical Guidance**
- `300 ppi` for close-viewed prints
- lower effective resolution for large prints viewed from farther away
- monitor preview is only approximate

**Where You Find Related Commands**
- `Image > Image Size`
- `View > Print Size`

**How To Use This Information**
1. Decide how closely the printed piece will be viewed.
2. Set image size and resolution for that use case.
3. Use `View > Print Size` for an approximate onscreen check.

**Mouse, Keyboard, and Menu Input**
- Menu path: `View > Print Size`
- Menu path: `Image > Image Size`

**User Interface Behavior**
- `Print Size` depends on monitor setup, so it is a guide rather than a precise proofing system.

**Expected Result**
- You can match resolution to viewing distance and printing needs instead of using one value blindly.
