# Get New Variations of Generated Content

- Entry Type: Generative variation control
- Category: Generative AI in Photoshop
- Source: https://helpx.adobe.com/photoshop/desktop/generative-ai/get-new-variations-of-generated-content-with-generate-similar.html

**Function Summary**
`Generate Similar` creates fresh alternatives based on an existing generated result. It is used when one variation is close to what you want but needs related options rather than a completely different prompt or workflow.

**What This Function Does**
- Builds new results from an existing generated variation
- Keeps the user inside the current generative workflow
- Helps refine output by exploring nearby alternatives

**Where You Find It**
- `Variations` tab in the `Properties` panel
- ellipsis menu for a generated variation

**Primary Interface Elements**
- generated variation thumbnail
- `Variations` tab
- `Properties` panel
- ellipsis menu
- `Generate Similar`

**How To Use It**
1. Generate content with a supported workflow such as `Generative Fill`, `Generative Expand`, `Generate Background`, or `Generate Image`.
2. Open the `Variations` tab in the `Properties` panel.
3. Find the generated variation that is closest to the result you want.
4. Open its ellipsis menu.
5. Choose `Generate Similar`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Browse the variation thumbnails, open the per-variation menu, and select `Generate Similar`.
- `Menus`
  This function is controlled from the Properties panel rather than the main application menu.
- `Keyboard`
  No dedicated default hotkey is described for this variation action.

**User Interface Behavior**
- Photoshop updates the canvas to a comparable new result derived from the chosen variation.
- The feature works as a refinement step after generation rather than as a starting command.
- It is most useful when a result is directionally correct but not the final preferred version.

**Expected Result**
- Users receive additional variations that stay close to the currently selected generated content.

**Practical Use Cases**
- Refine the composition of a generated object without rewriting the prompt.
- Explore alternate lighting, texture, or shape outcomes from a promising result.
- Generate nearby options for quick visual comparison before committing.
