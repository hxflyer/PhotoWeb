# Rearrange Commands Within an Action

- Entry Type: Action order workflow
- Category: Automate tasks > Actions
- Source: https://helpx.adobe.com/photoshop/desktop/automate-tasks/create-record-actions/rearrange-commands-within-action.html

**Function Summary**
Rearranging commands changes the order in which an action plays its recorded steps.

**Where You Find It**
- Actions panel command list.

**Primary Interface Elements**
- Expanded action steps.
- Drag-and-drop insertion line.

**How To Use It**
1. Expand the action.
2. Drag a command step to a new location in the list.
3. Release when the insertion line appears in the desired position.
4. Play the action on a test file.

**Expected Result**
- The action runs commands in the new order.
