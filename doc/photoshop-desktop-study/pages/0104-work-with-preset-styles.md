# Work with Preset Styles

- Entry Type: Preset style workflow
- Category: Create and manage layers > Apply layer effects
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/apply-layer-effects/work-with-preset-styles.html

**Function Summary**
This workflow applies, copies, and organizes preset layer styles. It is used to speed up repeated effect work and keep style application consistent across layers.

**What This Function Does**
- Applies preset styles from the `Styles` panel
- Copies a style from one layer to another
- Changes how style presets are displayed
- Supports grouping and renaming styles

**Where You Find It**
- `Window > Styles`
- Layers panel
- style panel menu

**Primary Interface Elements**
- `Styles`
- preset style thumbnail
- `Text Only`
- `Small Thumbnail`
- `Large Thumbnail`
- `Small List`
- `Large List`
- style folder group

**How To Apply a Preset Style**
1. Select the layer you want to style.
2. Open `Window > Styles`.
3. Hover over a style to preview its name and effect.
4. Click the style to apply it.

**How To Copy a Style from Another Layer**
1. Select the source layer in the Layers panel.
2. `Alt`-drag on Windows or `Option`-drag on macOS the layer effect icon from the source layer to the destination layer.

**How To Change the Styles Panel Display**
1. Open the panel menu in `Styles`.
2. Choose `Text Only`, `Small Thumbnail`, `Large Thumbnail`, `Small List`, or `Large List`.
3. Create groups with the folder icon if needed.
4. Rename a style by double-clicking it and entering a new name.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Click preset thumbnails, drag effects between layers, and change the panel view from the menu.
- `Keyboard`
  Use `Alt` or `Option` during the drag-based style-copy workflow, and `Enter` or `Return` after renaming.
- `Menus`
  Use `Window > Styles` to reveal the preset style panel.

**User Interface Behavior**
- Hover previews help identify styles before they are committed.
- The display mode changes whether styles are shown as text, thumbnails, or mixed list items.
- Grouping and renaming help maintain large style libraries.

**Expected Result**
- Preset styles become faster to reuse and easier to organize.
