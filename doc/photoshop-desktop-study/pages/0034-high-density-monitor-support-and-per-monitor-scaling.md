# High-Density Monitor Support and Per-Monitor Scaling

- Entry Type: Display compatibility and interface scaling
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/high-density-monitor-support-and-per-monitor-scaling.html

**Function Summary**
This entry describes how Photoshop handles high-density displays and how interface scaling behaves across monitors with different resolutions. It is primarily relevant on Windows systems with mixed-display setups or very high pixel-density screens.

**What This Function Does**
- Keeps Photoshop usable on high-resolution displays
- Supports different scale factors across multiple monitors
- Provides Photoshop-specific UI scaling controls for some interface areas
- Helps text and dialogs remain readable across 4K, 5K, or mixed-resolution setups

**Where You Configure It**
- Windows display settings: `Start > Settings > System > Display`
- Photoshop interface settings: `Edit > Preferences > Interface > UI Scaling`

**Primary Interface Elements**
- Windows `Scale and Layout` controls
- Photoshop `UI Scaling` setting
- `Auto`
- `100%`
- `200%`
- high-density monitor
- per-monitor scaling

**How To Use It**
1. On Windows, open `Start > Settings > System > Display`.
2. Select each monitor and choose the scale factor you want under `Scale and Layout`.
3. In Photoshop, open `Edit > Preferences > Interface`.
4. Set `UI Scaling` to `Auto`, `100%`, or `200%` depending on your display needs.
5. Test Photoshop across the connected monitors and adjust either the Windows scale or Photoshop UI scaling if text and dialogs feel too small or too large.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Change display scaling in Windows settings and Photoshop interface preferences.
- `Menus`
  Use Photoshop's `Edit > Preferences > Interface` for app-level UI scaling.
- `Keyboard`
  No dedicated default hotkey is described for scaling controls.

**User Interface Behavior**
- High-density displays show sharper text and image detail, but the interface may need scaling to remain comfortable to use.
- On Windows, different monitors can use different scaling percentages.
- Photoshop's `UI Scaling` setting affects only certain interface areas and dialogs, such as `File Info` and `Camera Raw`, rather than every on-screen element equally.
- `Auto` follows the primary monitor's scale behavior between the supported Photoshop UI sizes.

**Expected Result**
- Photoshop remains readable and visually sharp across mixed monitor configurations.
- Users can balance interface size and screen real estate more effectively.

**Practical Use Cases**
- Use a 4K laptop display and a lower-resolution external monitor together.
- Increase scaling on an 8K or dense display where Photoshop text is too small.
- Reduce scaling when you want more workspace on a very large monitor.
