# Specify Looping in Frame Animations

- Entry Type: Animation loop workflow
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/specify-looping-frame-animations.html

**Function Summary**
Looping controls whether a frame animation plays once, a set number of times, or forever.

**Where You Find It**
- Timeline panel looping menu below the frame thumbnails.

**Primary Interface Elements**
- Looping dropdown.
- `Once`, `Forever`, or custom repeat count options.
- Playback preview controls.

**How To Use It**
1. Open the Timeline panel in Frame Animation mode.
2. Click the looping dropdown.
3. Choose `Once`, `Forever`, or another repeat count.
4. Preview the animation.

**Expected Result**
- The animation repeats according to the selected looping behavior.
