# Boost Workflows with the Contextual Task Bar

- Entry Type: Interface feature
- Category: Get started > Learn the basics
- Source: https://helpx.adobe.com/photoshop/desktop/get-started/learn-the-basics/boost-workflows-with-the-contextual-task-bar.html

**Function Summary**
The Contextual Task Bar is an on-canvas control strip that surfaces likely next actions for the current selection or editing step. Instead of making you search through menus and panels, Photoshop places a compact task-specific interface near the work area.

**What This Function Does**
- Shows context-aware actions directly on the canvas
- Updates available controls based on the selected object, layer, or editing state
- Reduces menu and panel switching during common workflows
- Speeds up tasks by presenting the next most useful commands in place

**Where You Find It**
- On the canvas while working in supported workflows
- Near the active selection, object, or task area

**Primary Interface Elements**
- Contextual Task Bar
- action buttons
- workflow-specific controls
- selection-aware options
- on-canvas command strip

**How To Use It**
1. Open a document and begin a supported workflow such as selecting an object or editing a layer.
2. Watch for the Contextual Task Bar to appear on the canvas.
3. Review the buttons Photoshop surfaces for the current task.
4. Click the action you want instead of opening separate panels or menus.
5. Continue editing and let the bar update as the task changes.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Click the on-canvas buttons that appear in the task bar.
- `Keyboard`
  The article does not list a dedicated default hotkey for opening or using this feature.
- `Menus`
  This feature is primarily used directly from the canvas rather than from a traditional menu path.

**User Interface Behavior**
- The bar changes dynamically according to what Photoshop detects in the current context.
- It appears close to the working area to keep attention on the canvas.
- Available controls differ depending on whether you are making selections, editing image content, or working with layers.
- The feature acts as a shortcut surface, not a replacement for the full menus and panels.

**Expected Result**
- Common actions become easier to reach.
- Fewer interface jumps are needed during repetitive workflows.
- Newer users can discover likely next steps more naturally while editing.

**Practical Use Cases**
- Start follow-up actions immediately after making a selection.
- Work through guided image-editing steps without opening multiple side panels.
- Keep focus on the canvas during fast iteration work.
