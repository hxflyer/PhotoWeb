# Select an AI Model for Generative Control

- Entry Type: Generative workflow option
- Category: Generative AI in Photoshop
- Source: https://helpx.adobe.com/photoshop/desktop/generative-ai/select-an-ai-model-for-generative-control.html

**Function Summary**
This feature lets users choose which AI model powers a generative fill or expand request. Model choice changes the style, interpretation, and behavior of generated results, giving more control over the output.

**What This Function Does**
- Exposes a model selector during supported generative workflows
- Separates Adobe models from partner models
- Lets users compare how different models respond to the same prompt and selection

**Where You Find It**
- Supported generative workflows such as `Generative Fill`
- Contextual Task Bar model selector

**Primary Interface Elements**
- selection tool, including `Selection Brush`
- `Generative Fill`
- model selector
- `Adobe` model group
- `Partner` model group
- prompt field
- `Generate`

**How To Use It**
1. Open an image with `File > Open`.
2. Make a selection for the area you want to add, remove, or modify.
3. Start `Generative Fill`.
4. In the Contextual Task Bar, open the model selector.
5. Review the available model groups and hover over models for extra details when available.
6. Choose the model you want.
7. Enter the prompt and select `Generate`.

**Mouse, Keyboard, and Menu Input**
- `Mouse`
  Make the selection, choose the model from the selector, and generate from the task bar.
- `Menus`
  Use `File > Open` to begin with a source image.
- `Keyboard`
  Tool shortcuts depend on the selection method you use; the model feature itself is mainly mouse-driven in the generative UI.

**User Interface Behavior**
- The model selector appears only in supported generative workflows.
- Adobe and partner models are grouped separately for clarity.
- Changing the model can change the character of the result even when the prompt and selection remain the same.

**Expected Result**
- Generated output can be tuned more intentionally by choosing the model best suited to the task.

**Practical Use Cases**
- Compare several model interpretations of the same generative fill.
- Use one model for realism and another for a more stylized or concept-driven result.
- Retry a difficult prompt with a different model before changing the selection.
