# Transform Image to Video

- Entry Type: Image-to-video integration workflow
- Category: Share and collaborate > App integrations
- Source: https://helpx.adobe.com/photoshop/desktop/share-collaborate/app-integrations/transform-image-to-video.html

**Function Summary**
Image-to-video integration uses a still image as the starting point for generating or creating video through supported Adobe workflows.

**Where You Find It**
- App integration or generative video entry point where available.

**Primary Interface Elements**
- Source image preview.
- Prompt or motion settings where available.
- Generate or continue-in-app command.
- Result preview and export options.

**How To Use It**
1. Select or prepare the source image.
2. Open the image-to-video integration.
3. Enter prompt or motion settings if provided.
4. Generate or continue the workflow in the connected app.
5. Export or refine the result.

**Expected Result**
- The still image is used as input for a video-generation or motion workflow.
