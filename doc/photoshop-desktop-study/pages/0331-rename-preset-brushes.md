# Rename Preset Brushes

- Entry Type: Brush preset management workflow
- Category: Apply painting techniques > Brushes and presets
- Source: https://helpx.adobe.com/photoshop/desktop/apply-painting-techniques/brushes-presets/rename-preset-brushes.html

**Function Summary**
Renaming a brush preset changes its display name in the brush library.

**How To Use It**
1. Open the `Brushes` panel.
2. Select the preset.
3. Choose the rename command from the panel menu or context menu.
4. Enter the new name.

**Expected Result**
- The selected brush preset appears under the new name.
