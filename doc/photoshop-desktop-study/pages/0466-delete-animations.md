# Delete Animations

- Entry Type: Animation removal workflow
- Category: Video and animation
- Source: https://helpx.adobe.com/photoshop/desktop/video-animation/delete-animations.html

**Function Summary**
Deleting animations removes frames, timeline tracks, or animation data from the document.

**Where You Find It**
- Timeline panel trash icon and panel menu.

**Primary Interface Elements**
- Selected frame thumbnails or timeline tracks.
- Delete/trash control.
- Timeline panel menu.

**How To Use It**
1. Select the frame, frames, or animation item to remove.
2. Click the trash icon or choose the delete command from the Timeline panel menu.
3. Confirm if Photoshop asks.
4. Save a copy first if the animation may be needed later.

**Expected Result**
- Selected animation data is removed from the Timeline panel.
