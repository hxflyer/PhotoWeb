# Video Layers Overview

- Entry Type: Layer type reference
- Category: Create and manage layers > Get started with layers
- Source: https://helpx.adobe.com/photoshop/desktop/create-manage-layers/get-started-layers/video-layers-overview.html

**Function Summary**
Video layers let Photoshop treat moving footage as editable layered content. They extend the layer model into timeline-based work so users can combine motion with still-image editing tools.

**What This Entry Explains**
- how video layers behave in the Layers panel
- how Photoshop edits moving frames with familiar layer concepts
- which kinds of video-related layers are available

**What Video Layers Do**
- Bring video content into the normal layer stack
- Allow frame-level or timeline-based edits
- Support masks, filters, painting, and adjustments on motion content
- Let still graphics, text, and motion elements be mixed in one composition

**Primary Video Layer Types**
- `Video Layer`
  Holds moving footage and supports timeline-based editing
- `Adjustment Layer`
  Applies non-destructive correction across frames
- `Blank Video Layer`
  Creates an empty timeline for frame-by-frame or hand-drawn animation work

**User Interface Areas Involved**
- Layers panel
- timeline or frame editing workflow
- adjustment and mask controls

**User Interface Behavior**
- Video layers appear in the Layers panel like standard layers.
- Frame content can be edited while the layer still behaves as part of the stacked layer structure.
- Adjustment layers and masks can affect video non-destructively across time.

**Expected Result**
- Users understand that Photoshop can manage moving image content through the same layer logic used for still compositions.

**Practical Use Cases**
- Add moving footage behind static text and graphics.
- Paint or correct selected frames in a short clip.
- Build simple frame-by-frame animation inside a layered Photoshop document.
