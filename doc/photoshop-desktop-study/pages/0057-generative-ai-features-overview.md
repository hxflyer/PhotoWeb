# Generative AI Features Overview

- Entry Type: Capability overview
- Category: Generative AI in Photoshop
- Source: https://helpx.adobe.com/photoshop/desktop/generative-ai/generative-ai-features-overview.html

**Function Summary**
This entry summarizes the main generative AI capabilities available in Photoshop. It is a capability map rather than a single feature command, helping users understand which generative tool fits a given creative task.

**Main Capabilities Covered**
- `Harmonize`
  Blends inserted objects into a background by adjusting lighting, color, shadow, and local detail
- `Generate Image`
  Creates original images from descriptive prompts
- `Generative Expand`
  Extends content beyond the current canvas boundary
- `Generate Similar`
  Creates related variations from an existing generated result
- `Generative Upscale`
  Improves output quality and detail in supported workflows

**Where You Encounter These Features**
- Contextual Task Bar
- generative properties and variation controls
- feature-specific tool flows within Photoshop's AI section

**How To Use This Entry**
1. Identify the result you need: creation, expansion, harmonization, or variation.
2. Open the feature-specific page for that task.
3. Use the corresponding generative workflow rather than treating this page as a direct control surface.

**Expected Result**
- Users can choose the correct generative feature faster and avoid using the wrong AI workflow for the job.
