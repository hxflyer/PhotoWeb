# Review and Refine General Distractions

- Entry Type: Remove Tool refinement workflow
- Category: Repair and retouch > Remove objects and fill space
- Source: https://helpx.adobe.com/photoshop/desktop/repair-retouch/remove-objects-fill-space/review-and-refine-general-distractions.html

**Function Summary**
This workflow reviews detected general distractions before the `Remove Tool` commits the cleanup.

**What This Function Does**
- shows detected distractions with a colored overlay
- allows users to add or remove targeted areas
- supports category filtering for distraction detection

**Where You Find It**
- `Remove Tool > Find Distractions > General Distractions`

**Primary Interface Elements**
- `Find Distractions`
- `General Distractions`
- `Find`
- overlay opacity
- category toggles
- checkmark commit button

**How To Use It**
1. Open the image and select the `Remove Tool`.
2. Choose `Find Distractions > General Distractions`.
3. Click `Find`.
4. Review highlighted areas and brush to refine the target.
5. Commit with the checkmark.

**Expected Result**
- Only the intended general distractions are removed after review.
