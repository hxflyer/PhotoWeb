# Get Precise Distortions with Split Warp

- Entry Type: Split Warp workflow
- Category: Effects and filters > Distortion
- Source: https://helpx.adobe.com/photoshop/desktop/effects-filters/distortion/get-precise-distortions-split-warp.html

**Function Summary**
Split Warp adds custom horizontal and vertical split lines to the warp grid for more localized distortion control.

**Where You Find It**
- `Edit > Transform > Warp` options bar split controls.

**Primary Interface Elements**
- Warp grid.
- Split line controls.
- Intersection points and handles.
- Commit/cancel controls.

**How To Use It**
1. Select a layer and enter Warp.
2. Add split lines from the options bar or by modifier-clicking where supported.
3. Drag points and handles around the split to isolate the distortion.
4. Commit when the shape matches the target.

**Expected Result**
- Only chosen regions of the layer warp strongly while nearby regions can remain more stable.
